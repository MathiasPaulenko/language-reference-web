## 概述

名词化将动词、形容词或从句转化为名词。

- **西班牙语**：使用冠词+不定式（el correr）、lo+形容词（lo bueno）和派生后缀（-ción, -miento, -dad）。名词性从句中常出现虚拟式。
- **英语**：使用动名词-ing、that从句和派生后缀（-tion, -ment, -ness, -ity）。常需冠词the。
- **汉语**：使用 的 作为万能名词化助词用于从句（他说的）、形容词（美丽的）和动词（吃的）。动词永不变化。

## 西班牙语

### 不定式+冠词

西班牙语通过冠词将不定式名词化：

- El correr es bueno para la salud.（跑步对健康有益。）
- El leer amplía la mente.（阅读开阔思维。）
- El cantar es su pasión.（唱歌是他的热情。）

**注意**：无冠词时，不定式在句中作动词而非名词。

### Lo + 形容词

西班牙语用 lo + 形容词来名词化形容词：

- Lo bueno es que...（好的方面是……）
- Lo difícil del asunto...（这件事的难点……）
- Lo importante es intentarlo.（重要的是尝试。）
- Me gusta lo español.（我喜欢西班牙的东西。）

**注意**：Lo 是中性的——无性无数。它作为抽象名词使用。

### 派生后缀

| 后缀 | 例子 |
|---|---|
| -ción | la decisión（决定），la educación（教育） |
| -miento | el pensamiento（思想），el movimiento（运动） |
| -dad / -tad / -itud | la libertad（自由），la igualdad（平等） |
| -ez / -eza | la belleza（美丽），la fortaleza（力量） |
| -ura | la dulzura（甜蜜），la bravura（勇敢） |
| -ía | la tristeza（悲伤），la mentira（谎言） |
| -or / -ora | el cantor（歌手），la cantora |
| -ero / -era | el panadero（面包师），la panadera |

### 带虚拟式的名词性从句

西班牙语在某些动词后的名词性从句中使用虚拟式：

- Espero que venga.（我希望他能来。）
- Es necesario que estudies.（你需要学习。）
- El hecho de que haya llegado...（他已经到达的事实……）

### 分词作形容词/名词

- El herido fue llevado al hospital.（伤者被送往医院。）
- Los ancianos merecen respeto.（老年人应受尊重。）
- Lo dicho, dicho está.（说出去的话，泼出去的水。）

### 副动词（西班牙语中不作名词）

西班牙语副动词（-ando / -iendo）**不**作名词：

- ✅ El correr es bueno.（不定式名词化）
- ❌ El corriendo es bueno.（副动词不是名词）

## 英语

### 动名词 -ing 作名词

英语使用动名词作为动词性名词：

- Running is good for your health.（跑步对你的健康有益。）
- Swimming is my favorite sport.（游泳是我最喜欢的运动。）
- I enjoy reading.（我喜欢阅读。）

### That 从句

带 that 的从句可作名词：

- I know that he is right.（我知道他是对的。）
- The fact that she left surprised us.（她离开的事实让我们惊讶。）
- That he won is incredible.（他赢了真是不可思议。）

### The + 形容词

英语用 the + 形容词指群体：

- The rich should help the poor.（富人应帮助穷人。）
- The good always prevails.（善终将胜利。）
- I prefer the blue one.（我更喜欢蓝色的。）

### 派生后缀

| 后缀 | 例子 |
|---|---|
| -tion / -sion / -ation | education, decision, creation |
| -ment | movement, government, development |
| -ness | happiness, kindness, darkness |
| -ity | reality, ability, diversity |
| -ance / -ence | importance, difference, existence |
| -al | arrival, approval, denial |
| -er / -or | runner, actor, teacher |
| -ist | artist, scientist, pianist |
| -ism | criticism, optimism, tourism |

### 带 to 的不定式

带 to 的不定式可作名词：

- To err is human.（犯错人皆难免。）
- I want to leave.（我想离开。）
- The decision to sell was difficult.（出售的决定很难。）

### -ing vs 不定式

| 动名词（-ing） | 不定式（to） |
|---|---|
| 动词性名词 | 动词补语 |
| I enjoy swimming. | I want to swim. |
| Swimming is fun. | To swim is fun.（正式） |

## 汉语

### 的 de（万能名词化助词）

的 是汉语中最通用的名词化工具。它几乎可以将任何成分转化为名词。

#### 从句 + 的

- 他说的**话**（他说的话）
- 我想买的**东西**（我想买的东西）
- 他来的**事**（他来的事）
- 你做的**很好**（你做的很好）

**模式**：主语 + 动词 + **的** + 通用名词（话, 事, 人, 东西）

#### 形容词 + 的

- 美丽的**风景**（美丽的风景）
- 漂亮的**衣服**（漂亮的衣服）
- 红的比绿的**好**（红的比绿的好）

**注意**：的 将形容词转化为定语修饰语或与通用名词连用时转化为抽象名词。

#### 动词 + 的 作名词

- 吃**的**（吃的——"被吃的东西"）
- 喝**的**（喝的——"被喝的东西"）
- 穿**的**（穿的——"被穿的东西"）
- 卖**的**（卖的——"被卖的东西"）

**用法**：你要吃**的**还是喝**的**？（你要吃的还是喝的？）

#### 所 suǒ（正式）

所 + 动词 + 的 = 正式名词化：

- 所谓**的**（所谓的）
- 所有**的**（所有的）
- 所见**的**（所见的）

### 动词作名词（不变形）

许多汉语动词同时可作名词，无需变化：

| 动词 | 名词 |
|---|---|
| 决定 | 决定 |
| 研究 | 研究 |
| 变化 | 变化 |
| 发展 | 发展 |
| 问题 | 问题 |

### 形容词作名词（不变形）

汉语形容词可直接作名词：

- 美（美/美丽）
- 好（好的东西）
- 大（大的东西）
- 新（新的东西）

### 语境名词化

汉语名词化无需形态变化——由语境决定：

- 学习很重要。（学习很重要。——动词作主语）
- 我喜欢学习。（我喜欢学习。——动词作宾语）

## 一览对比

| 策略 | 西班牙语 | 英语 | 汉语 |
|----------|---------|---------|---------|
| 不定式/动名词作名词 | El + 不定式 | -ing（动名词） | 不变动词+语境 |
| 形容词→名词 | lo + 形容词 | the + 形容词 | 不变形容词（上下文） |
| 名词性从句 | el hecho de que | that从句 | 主语+动词+的+通用名词 |
| 派生后缀 | -ción, -miento, -dad | -tion, -ment, -ness, -ity | 无（复合构词） |
| 动词变形 | 有时 | 有时（-ing, -ed） | 从不 |
| 冠词需求 | 是（常需） | 是（常需） | 否 |

## 语境中的例子

### 跑步是好的
- **ES**: El correr es bueno.（冠词+不定式）
- **EN**: Running is good.（动名词-ing）
- **ZH**: 跑步很好。（不变动词——跑步 = 跑步）

### 决定
- **ES**: la decisión（-sión）
- **EN**: the decision（-sion）
- **ZH**: 决定（动词/名词相同）

### 好的东西
- **ES**: lo bueno（lo+形容词）
- **EN**: the good（the+形容词）
- **ZH**: 好的（形容词+的 作抽象名词）

## 常见错误

1. **西班牙语母语者学英语**：混淆动名词与分词：I am interesting → I am interested

2. **英语母语者学西班牙语**：用副动词作名词：El corriendo → El correr

3. **英语母语者学汉语**：名词性从句中省略 的：他说话 → 他说**的**话

4. **汉语母语者学英语**：需要不定式处用动名词：I want going → I want to go
