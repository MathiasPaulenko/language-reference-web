## Resumen

El registro es la forma en que el lenguaje varía según el contexto social, incluyendo formalidad, medio (hablado/escrito) y relación social.

- **Español**: Formalidad gramaticalizada a través de *tú/usted* (distinción de forma verbal). Variación regional en uso de pronombres (*vos* en Río de la Plata, *vosotros* en España).
- **Inglés**: Sin distinción T-V gramatical. Formalidad expresada mediante elección de vocabulario, verbos modales, títulos e indirectez.
- **Chino**: Distinción de pronombres (*你/您*). Títulos honoríficos. Alternativas de vocabulario para contextos formales.

## Español

### Tú vs usted

El español tiene una distinción gramaticalizada de formalidad:

| Persona | Uso | Ejemplo |
|---|---|---|
| **tú** | Amigos, familia, pares | ¿Cómo estás, **tú**? |
| **usted** | Desconocidos, autoridad, mayores | ¿Cómo está **usted**? |
| **vos** | Regional (Rioplatense) | ¿Cómo andás, **vos**? |
| **vosotros** | España (plural informal) | ¿Cómo estáis, **vosotros**? |

### Verbos con distinción de formalidad

| Infinitivo | tú | usted |
|---|---|---|
| venir | vienes | viene |
| estar | estás | está |
| poder | puedes | puede |
| tener | tienes | tiene |

### Regionalismo

| Región | Uso | Notas |
|---|---|---|
| España | tú/vosotros/usted/ustedes | Vosotros = plural informal |
| Latinoamérica | tú/usted/ustedes | Ustedes = plural universal |
| Río de la Plata | vos | Reemplaza a tú |

### Escrito vs hablado

| Característica | Hablado | Escrito |
|---|---|---|
| Contracciones | Sí | No (formal) |
| Subjuntivo | Menos común | Más común |
| Vocabulario | Coloquial | Formal |

## Inglés

### Sin distinción T-V

El inglés no distingue formalidad gramaticalmente en el pronombre:

| Contexto | Uso | Ejemplo |
|---|---|---|
| Informal | you + contracciones | How **are you**? |
| Formal | you + títulos | How **are you**, Professor Smith? |
| Muy formal | Sir/Madam + would | **Would** you please... |

### Suavizadores modales

| Suavizador | Uso | Ejemplo |
|---|---|---|
| would | peticiones formales | **Would** you help me? |
| could | peticiones formales | **Could** you open the door? |
| might | sugerencia formal | You **might** want to... |
| please | cortesía | **Please** sit down. |

### Títulos

| Título | Uso | Ejemplo |
|---|---|---|
| Mr. / Ms. | General | **Mr.** Smith |
| Mrs. / Miss | Mujeres casada/soltera | **Mrs.** Johnson |
| Dr. | Doctor | **Dr.** Williams |
| Professor | Profesor universitario | **Professor** Lee |
| Sir / Madam | Formal | Yes, **Sir**. |

### Escrito vs hablado

| Característica | Hablado | Escrito (formal) |
|---|---|---|
| Contracciones | Sí (I'm, don't) | No (I am, do not) |
| Vocabulario | Coloquial | Formal |
| Oraciones | Cortas | Completas |

| Pronombre | Uso | Ejemplo |

## Errores comunes

1. **Hispanohablantes**: Usar tú inapropiadamente → Con desconocidos usar **usted**

2. **Angloparlantes**: No usar 您 en chino → Con mayores usar <ruby>您<rt>nín</rt></ruby><ruby>好<rt>hǎo</rt></ruby>

3. **Chinos**: Pensar que el tono basta para formalidad en inglés → Usar títulos: Professor Smith

4. **Todos**: Usar contracciones en escritura formal → do not (NO don't); cannot (NO can't)
