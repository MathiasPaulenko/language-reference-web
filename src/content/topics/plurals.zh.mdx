## 概述

复数标记表示不止一个。每种语言的处理方式不同。

- **西班牙语**：加-s或-es。冠词和形容词必须在性和数上一致。许多以辅音结尾的名词加-es。
- **英语**：加-s或-es。许多不规则复数。不可数名词不复数化。
- **汉语**：名词不变形。复数通过数词+量词、语境或们men表示。

## 西班牙语

### 规则复数

| 规则 | 单数 | 复数 |
|---|---|---|
| 元音加-s | libro | libros |
| 辅音加-es | mes | meses |
| -z变-ces | lápiz | lápices |
| -ión（加重音） | nación | naciones |
| 不变（某些） | el lunes | los lunes |

### 不规则复数

| 单数 | 复数 | 说明 |
|---|---|---|
| el padre | los padres | 冠词变化 |
| el rey | los reyes | 加-es |
| el pie | los pies | 内部变化 |
| la mujer | las mujeres | -es |
| el café | los cafés | 保留重音 |

### 一致

- **冠词**：el → los; la → las; un → unos; una → unas
- **形容词**：rojo → rojos; alta → altas; bueno → buenos
- **动词**：动词不因主语数而变化（与英语不同）

### 集合名词

- *la gente piensa* → 复数动词一致 (people think)
- *la familia es grande* → 单数动词一致

## 英语

### 规则复数

| 规则 | 单数 | 复数 |
|---|---|---|
| -s | book | books |
| -es (s, x, z, ch, sh) | box | boxes |
| -ies (辅音+y) | baby | babies |
| -ves (f, fe) | knife | knives |

### 不规则复数

| 单数 | 复数 |
|---|---|
| man | men |
| woman | women |
| child | children |
| foot | feet |
| tooth | teeth |
| mouse | mice |
| goose | geese |
| person | people |
| sheep | sheep (不变) |
| deer | deer (不变) |

### 不可数名词

| 类别 | 例子 |
|---|---|
| 液体 | water, milk, coffee |
| 材料 | wood, metal, cotton |
| 抽象 | advice, information, knowledge |
| 活动 | homework, research, work |

**注**：不复数化。用"some"、"much"、"a piece of"量化。

### 集合名词

- The team **is** playing well (强调群体)
- The team **are** arguing (强调个体)

## 汉语

### 形式不变

| 单数/复数 | 语境 |
|---|---|
| <ruby>书<rt>shū</rt></ruby> | 书或书们（取决于语境） |
| <ruby>人<rt>rén</rt></ruby> | 人或人们 |
| <ruby>桌<rt>zhuō</rt></ruby><ruby>子<rt>zi</rt></ruby> | 桌子或桌子们 |

### 们标记（用于人）

| 代词 | 复数 |
|---|---|
| <ruby>我<rt>wǒ</rt></ruby> → | <ruby>我<rt>wǒ</rt></ruby><ruby>们<rt>men</rt></ruby> (我们) |
| <ruby>你<rt>nǐ</rt></ruby> → | <ruby>你<rt>nǐ</rt></ruby><ruby>们<rt>men</rt></ruby> (你们) |
| <ruby>他<rt>tā</rt></ruby> → | <ruby>他<rt>tā</rt></ruby><ruby>们<rt>men</rt></ruby> (他们) |
| <ruby>她<rt>tā</rt></ruby> → | <ruby>她<rt>tā</rt></ruby><ruby>们<rt>men</rt></ruby> (她们) |

**注**：们只用于代词和某些人名词：<ruby>朋<rt>péng</rt></ruby><ruby>友<rt>yǒu</rt></ruby><ruby>们<rt>men</rt></ruby> ✓, <ruby>书<rt>shū</rt></ruby><ruby>们<rt>men</rt></ruby> ✗

### 量词+数词

| 数词 | 量词 | 名词 |
|---|---|---|
| <ruby>三<rt>sān</rt></ruby> | <ruby>本<rt>běn</rt></ruby> | <ruby>书<rt>shū</rt></ruby> (书) |
| <ruby>五<rt>wǔ</rt></ruby> | <ruby>个<rt>gè</rt></ruby> | <ruby>人<rt>rén</rt></ruby> (人) |
| <ruby>两<rt>liǎng</rt></ruby> | <ruby>张<rt>zhāng</rt></ruby> | <ruby>纸<rt>zhǐ</rt></ruby> (纸) |

**注**：名词不变；量词+数词表示复数。

### 些表示"一些"

| 表达 | 意义 |
|---|---|
| <ruby>这<rt>zhè</rt></ruby><ruby>些<rt>xiē</rt></ruby><ruby>书<rt>shū</rt></ruby> | 这些书（一些） |
| <ruby>那<rt>nà</rt></ruby><ruby>些<rt>xiē</rt></ruby><ruby>人<rt>rén</rt></ruby> | 那些人 |

## 快速比较

| 特征 | 西班牙语 | 英语 | 汉语 |
|---|---|---|---|
| 形式变化 | 有 (-s / -es) | 有 (-s / -es / 不规则) | 无 |
| 冠词一致 | 有 (el → los) | 无 (the) | 无 (无冠词) |
| 形容词一致 | 有 (rojo → rojos) | 无 | 无 |
| 人称标记 | -s / -es | -s / -es | 们 (有限) |
| 与数词搭配 | 有 (tres libros) | 有 (three books) | 无 (三本书) |
| 集合名词 | 复数动词 | 单数或复数 | 语境 |
| 不可数名词 | 一些 | 许多 | 很少限制 |

### 不规则
- **ES**: el hijo → los hijos; el pie → los pies
- **EN**: man → men; child → children; tooth → teeth

- **ES**: Los libros rojos son interesantes. (rojos 与 libros 一致)

## 常见错误

1. **西班牙语母语者**：在汉语名词上加-s → <ruby>书<rt>shū</rt></ruby><ruby>们<rt>men</rt></ruby> 错误

2. **英语母语者**：在汉语中强制复数一致 → 汉语形容词不变

3. **西班牙语母语者**：在西班牙语中按英语类比使用不规则复数 → "hiji" 而不是 "hijos"

4. **英语母语者**：将西班牙语不可数名词当作可数名词 → "el agua" 并非在所有语境中都像 "las aguas" 那样复数化
