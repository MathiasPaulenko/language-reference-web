## Visión general

La negación invierte el valor de verdad de una oración. Los idiomas difieren en cómo marcan la negación y qué palabras requieren soporte auxiliar.

- **Español**: Niega con *no* antes del verbo. La negación doble es obligatoria: *no + nada/nadie/ninguno/nunca*. Sin auxiliar *do*.
- **Inglés**: Requiere *do-support* para verbos principales: *do/does/did + not + infinitivo*. La negación doble es no estándar.
- **Chino**: Usa <ruby>不<rt>bù</rt></ruby> para negar intenciones/hábitos/estados y <ruby>没<rt>méi</rt></ruby> para acciones no completadas. Sin auxiliar adicional.

## Español

### Negación simple

Coloca *no* antes del verbo conjugado:

- *No como carne.* (No como carne.)
- *No es verdad.* (No es verdad.)
- *No estoy listo.* (No estoy listo.)
- *No hay pan.* (No hay pan.)

### Negación doble (obligatoria)

El español requiere negación doble: *no* + palabra negativa:

| Afirmativo | Negativo |
|---|---|
| algo | **no** ... **nada** |
| alguien | **no** ... **nadie** |
| alguno / alguna | **no** ... **ninguno / ninguna** |
| siempre | **no** ... **nunca / jamás** |
| también | **no** ... **tampoco** |
| o ... o | **no** ... **ni** ... **ni** |

**Ejemplos**:

- *No vi **nada**.* (No vi nada.)
- *No vino **nadie**.* (No vino nadie.)
- *No tengo **ningún** problema.* (No tengo ningún problema.)
- *No **nunca** voy allí.* (Nunca voy allí.)
- *Yo **tampoco**.* (Yo tampoco.)
- *No le gusta **ni** el café **ni** el té.* (No le gusta ni el café ni el té.)

**Nota**: Las palabras negativas pueden preceder al verbo, en cuyo caso *no* se omite:

- ***Nadie** vino.* (Nadie vino.)
- ***Nunca** lo haré.* (Nunca lo haré.)
- ***Nada** me gusta.* (Nada me gusta.)

### Negación imperativa

- *No toques eso.* (No toques eso.)
- *No te vayas todavía.* (No te vayas todavía.)

### Negación con artículo indefinido

- *No es **un** médico.* (No es un médico.)
- *No tengo **ningún** coche.* (No tengo ningún coche.)

### Negación con el subjuntivo

- *No creo que **venga**.* (No creo que venga. — subjuntivo)
- *No es posible que **sea** cierto.* (No es posible que sea cierto.)

## Inglés

### Do-support para verbos principales

El inglés requiere *do/does/did* para negar verbos principales:

- *I **do not** like coffee.* (No me gusta el café.)
- *She **does not** speak Spanish.* (Ella no habla español.)
- *They **did not** arrive on time.* (Ellos no llegaron a tiempo.)

**Nota**: Los modales y el verbo *to be* usan negación directa:

- *I **am not** tired.* (No estoy cansado.)
- *He **cannot** swim.* (Él no puede nadar.)
- *They **will not** come.* (Ellos no vendrán.)

### Contracciones comunes

| Completo | Contracción |
|---|---|
| do not | don't |
| does not | doesn't |
| did not | didn't |
| is not | isn't |
| are not | aren't |
| was not | wasn't |
| were not | weren't |
| have not | haven't |
| has not | hasn't |
| had not | hadn't |
| will not | won't |
| would not | wouldn't |
| cannot | can't |
| could not | couldn't |
| should not | shouldn't |
| must not | mustn't |
| need not | needn't |

### Negación con palabras de significado positivo

Después de una negación, el inglés estándar usa palabras de significado positivo:

- *I didn't see **anything**.* (No vi nada.)
- *She doesn't know **anyone**.* (Ella no conoce a nadie.)
- *We haven't been **anywhere**.* (No hemos estado en ningún lugar.)

**Nota**: El inglés coloquial a veces usa negación doble (*I didn't see nothing*), pero esto es no estándar.

### Negación de sustantivos

- *There is **no** bread.* (No hay pan.)
- *I have **no** money.* (No tengo dinero.)
- *She is **not** a doctor.* (Ella no es doctora.)

### Negación imperativa

- *Do **not** touch.* (No toques.)
- *Don't go.* (No vayas.)

## Chino

### 不 bù (negación general)

<ruby>不<rt>bù</rt></ruby> niega intenciones, hábitos, estados y acciones futuras:

- <ruby>我<rt>wǒ</rt></ruby><ruby>不<rt>bù</rt></ruby><ruby>吃<rt>chī</rt></ruby><ruby>肉<rt>ròu</rt></ruby>。(No como carne. — hábito)
- <ruby>他<rt>tā</rt></ruby><ruby>不<rt>bù</rt></ruby><ruby>是<rt>shì</rt></ruby><ruby>医<rt>yī</rt></ruby><ruby>生<rt>shēng</rt></ruby>。(No es médico.)
- <ruby>我<rt>wǒ</rt></ruby><ruby>不<rt>bù</rt></ruby><ruby>知<rt>zhī</rt></ruby><ruby>道<rt>dào</rt></ruby>。(No sé.)
- <ruby>明<rt>míng</rt></ruby><ruby>天<rt>tiān</rt></ruby><ruby>不<rt>bù</rt></ruby><ruby>下<rt>xià</rt></ruby><ruby>雨<rt>yǔ</rt></ruby>。(Mañana no lloverá.)

### 没 méi (negación de acción completada)

<ruby>没<rt>méi</rt></ruby> niega que una acción se haya completado:

- <ruby>我<rt>wǒ</rt></ruby><ruby>没<rt>méi</rt></ruby><ruby>吃<rt>chī</rt></ruby><ruby>饭<rt>fàn</rt></ruby>。(No comí. / No he comido.)
- <ruby>他<rt>tā</rt></ruby><ruby>没<rt>méi</rt></ruby><ruby>来<rt>lái</rt></ruby>。(No vino.)
- <ruby>没<rt>méi</rt></ruby><ruby>有<rt>yǒu</rt></ruby><ruby>钱<rt>qián</rt></ruby>。(No hay dinero.)

**Regla clave**:

- <ruby>不<rt>bù</rt></ruby> + verbo = no (hábito / intención / estado)
- <ruby>没<rt>méi</rt></ruby> + verbo = no (acción no completada)
- <ruby>没<rt>méi</rt></ruby><ruby>有<rt>yǒu</rt></ruby> = no hay / no tener (existencia)

### 没有 méiyǒu (no tener / no haber)

<ruby>没<rt>méi</rt></ruby><ruby>有<rt>yǒu</rt></ruby> niega la posesión o existencia:

- <ruby>我<rt>wǒ</rt></ruby><ruby>没<rt>méi</rt></ruby><ruby>有<rt>yǒu</rt></ruby><ruby>车<rt>chē</rt></ruby>。(No tengo coche.)
- <ruby>这<rt>zhè</rt></ruby><ruby>里<rt>lǐ</rt></ruby><ruby>没<rt>méi</rt></ruby><ruby>有<rt>yǒu</rt></ruby><ruby>人<rt>rén</rt></ruby>。(Aquí no hay nadie.)

### 别 bié (negación imperativa)

<ruby>别<rt>bié</rt></ruby> niega órdenes o consejos:

- <ruby>别<rt>bié</rt></ruby><ruby>碰<rt>pèng</rt></ruby>！(¡No toques!)
- <ruby>别<rt>bié</rt></ruby><ruby>担<rt>dān</rt></ruby><ruby>心<rt>xīn</rt></ruby>。(No te preocupes.)
- <ruby>别<rt>bié</rt></ruby><ruby>忘<rt>wàng</rt></ruby><ruby>了<rt>le</rt></ruby>。(No lo olvides.)

### 双重否定

El chino a veces usa la estructura 什么都不 (todo + 都 + no):

- <ruby>我<rt>wǒ</rt></ruby><ruby>什<rt>shén</rt></ruby><ruby>么<rt>me</rt></ruby><ruby>都<rt>dōu</rt></ruby><ruby>不<rt>bù</rt></ruby><ruby>知<rt>zhī</rt></ruby><ruby>道<rt>dào</rt></ruby>。(No sé nada. — literal: "todo no sé")
- <ruby>他<rt>tā</rt></ruby><ruby>什<rt>shén</rt></ruby><ruby>么<rt>me</rt></ruby><ruby>都<rt>dōu</rt></ruby><ruby>没<rt>méi</rt></ruby><ruby>做<rt>zuò</rt></ruby>。(No hizo nada.)

**Nota**: Esta estructura es gramatical pero diferente de la negación doble española o del inglés.

### 不 vs 没：resumen

| Contexto | 使用 |
|---|---|
| Hábitos / preferencias | <ruby>不<rt>bù</rt></ruby> |
| Estados / descripciones | <ruby>不<rt>bù</rt></ruby> |
| Intenciones / planes futuros | <ruby>不<rt>bù</rt></ruby> |
| Acciones no completadas | <ruby>没<rt>méi</rt></ruby> |
| Existencia / posesión | <ruby>没<rt>méi</rt></ruby><ruby>有<rt>yǒu</rt></ruby> |
| Órdenes negativas | <ruby>别<rt>bié</rt></ruby> |

## Errores comunes

1. **Hispanohablantes aprendiendo inglés**: Omitiendo do-support: *I no eat* → *I **do not** eat*

2. **Angloparlantes aprendiendo español**: Olvidando la negación doble: *No vi algo* → *No vi **nada***

3. **Angloparlantes aprendiendo chino**: Usando <ruby>不<rt>bù</rt></ruby> para todo: *<ruby>我<rt>wǒ</rt></ruby><ruby>不<rt>bù</rt></ruby><ruby>吃<rt>chī</rt></ruby><ruby>了<rt>le</rt></ruby><ruby>午<rt>wǔ</rt></ruby><ruby>饭<rt>fàn</rt></ruby>* → *<ruby>我<rt>wǒ</rt></ruby><ruby>没<rt>méi</rt></ruby><ruby>吃<rt>chī</rt></ruby><ruby>午<rt>wǔ</rt></ruby><ruby>饭<rt>fàn</rt></ruby>* (acción específica no completada)

4. **Sinohablantes aprendiendo inglés**: Negación doble no estándar: *I don't know nothing* → *I don't know **anything***
