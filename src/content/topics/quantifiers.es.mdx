## Resumen

Los cuantificadores expresan cantidad: todo, algo, nada, muchos, pocos. Interactúan con la negación de maneras complejas.

- **Español**: *Alguno* (alguno/ninguno) cambia a *ninguno* (ninguno) en contextos negativos. *Todo* concuerda en género y número. La concordancia negativa significa que múltiples negativos se refuerzan mutuamente.
- **Inglés**: La distinción some/any depende de la polaridad (afirmativa, negativa, interrogativa). *Few/a few* y *little/a little* tienen implicaciones opuestas. No hay concordancia negativa.
- **Chino**: *<ruby>都<rt>dū</rt></ruby>* es el adverbio cuantificador universal. *<ruby>什<rt>shí</rt></ruby><ruby>么<rt>mó</rt></ruby>...<ruby>都<rt>dū</rt></ruby>/<ruby>也<rt>yě</rt></ruby>* crea negación universal. Sin distinción contable/masivo. La negación usa *<ruby>不<rt>bù</rt></ruby>/<ruby>没<rt>méi</rt></ruby>* según el aspecto.

## Español

### Cuantificadores universales

| Cuantificador | Uso | Ejemplo |
|---|---|---|
| todo / toda | Todo el conjunto | **Todo** el mundo vino |
| todos / todas | Todo el conjunto (plural) | **Todos** los días |
| cada | Cada elemento | **Cada** persona |

### Cuantificadores existenciales

| Cuantificador | Uso | Ejemplo |
|---|---|---|
| algún / alguna | Algo (afirmativo) | Tengo **algún** problema |
| algunos / algunas | Algunos (afirmativo) | **Algunos** vinieron |
| algo | Algo incontable | Quiero **algo** de agua |
| alguien | Alguien | **Alguien** llamó |

### Cuantificadores negativos

| Cuantificador | Uso | Ejemplo |
|---|---|---|
| ningún / ninguna | Ninguno (con no) | No tengo **ningún** problema |
| nadie | Nadie | No vino **nadie** |
| nada | Nada | No hizo **nada** |

### Concordancia negativa

En español, múltiples negativos se refuerzan:

- No vino **nadie**.
- No hizo **nada**.
- No tengo **ningún** problema.
- **Nunca** dijo **nada** a **nadie**.

**Nota**: A diferencia del inglés, el español permite y de hecho prefiere múltiples negativos en la misma oración.

### Mucho / poco

| Cuantificador | Significado | Ejemplo |
|---|---|---|
| mucho / muchos | Mucho | Tengo **mucho** dinero |
| poco / pocos | Poco | Tengo **poco** dinero |
| bastante / bastantes | Suficiente | Tengo **bastante** tiempo |

## Inglés

### Some vs any

| Polaridad | Usar | Ejemplo |
|---|---|---|
| Afirmativa | some | I have **some** money. |
| Negativa | any | I don't have **any** money. |
| Interrogativa | any | Do you have **any** money? |
| Ofrecimiento | some | Would you like **some** tea? |

**Nota**: *Some* también puede usarse en preguntas cuando se espera una respuesta afirmativa.

### Few / a few

| Cuantificador | Significado | Implicación |
|---|---|---|
| **few** | Pocos | Negativa (casi ninguno) |
| **a few** | Algunos | Positiva (suficientes) |
| **little** | Poco | Negativa (casi nada) |
| **a little** | Un poco | Positiva (suficiente) |

**Ejemplos**:
- I have **few** friends. (= casi ninguno, me siento solo)
- I have **a few** friends. (= algunos, estoy bien)

### All / every / each

| Cuantificador | Uso | Ejemplo |
|---|---|---|
| **all** | Todo el conjunto | **All** students passed. |
| **every** | Cada elemento (singular) | **Every** student passed. |
| **each** | Cada elemento (individual) | **Each** student passed. |

### Neither / either / both

| Cuantificador | Significado | Ejemplo |
|---|---|---|
| **both** | Ambos | **Both** books are good. |
| **either** | Cualquiera de dos | **Either** book is fine. |
| **neither** | Ninguno de dos | **Neither** book is good. |

## Comparación rápida

| Característica | Español | Inglés | Chino |
|---|---|---|---|
| Universal | todo(s) / toda(s) | all / every / each | 都 / 所有 |
| Existencial | algún / algunos / algo | some / a few / several | 一些 / 几个 |
| Cualquiera | alguno / ninguno | any | 任何 / 什么 |
| Nada / ninguno | ninguno / nadie / nada | none / nobody / nothing | 没有 / 什么都不 |
| Ambos | ambos / los dos | both / either / neither | 两个都 / 两边 |
| Concordancia negativa | Sí | No | No (usar 都) |
| Contable/masivo | Sí (muchos/mucho) | Sí (many/much) | No |

## Errores comunes

1. **Hispanohablantes**: Usar "some" en negativos del inglés → I don't have **any** money (no some)

2. **Angloparlantes**: Usar "alguno" en negativos del español → No tengo **ninguno** (no alguno)

3. **Angloparlantes**: Olvidar <ruby>都<rt>dū</rt></ruby> en universales del chino → <ruby>大<rt>dà</rt></ruby><ruby>家<rt>jiā</rt></ruby>**<ruby>都<rt>dōu</rt></ruby>**<ruby>来<rt>lái</rt></ruby><ruby>了<rt>le</rt></ruby>（不能只说 大家来了）

4. **Chinos**: Traducir 什么 literalmente como "any" → <ruby>什<rt>shén</rt></ruby><ruby>么<rt>me</rt></ruby> + <ruby>都<rt>dū</rt></ruby> + Neg = "nada / ninguno"
