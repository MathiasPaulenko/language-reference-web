## Visión general

Los morfemas son las unidades más pequeñas de significado en una palabra. Los idiomas difieren drásticamente en cómo combinan los morfemas.

- **Español**: Morfología flexiva (género, número, persona, tiempo, modo) y derivacional. Prefijos y sufijos latinos/griegos productivos. Los morfemas son a menudo vinculantes.
- **Inglés**: Morfología flexiva mínima (plural -s, pasado -ed, progresivo -ing) pero derivación extensa (prefijos/sufijos latinos/griegos/germánicos). Palabra compuesta común.
- **Chino**: Sin inflexión. Todas las palabras son morfemas libres. La composición es la estrategia principal. Las partículas funcionales (<ruby>的<rt>de</rt></ruby>, <ruby>了<rt>le</rt></ruby>, <ruby>着<rt>zhe</rt></ruby>, <ruby>过<rt>guò</rt></ruby>) son morfemas separados, no afijos vinculantes.

## Español

### Morfología flexiva

El español **flexiona** los verbos y los sustantivos/adjetivos:

| Categoría | Ejemplo | Morfema |
|---|---|---|
| Plural | gato → gato**s** | -s |
| Género | gato / gat**a** | -o / -a |
| Persona (verbo) | habl**o**, habl**as**, habl**a** | -o, -as, -a |
| Tiempo (verbo) | habl**é**, habl**aba**, habl**aré** | -é, -aba, -aré |
| Modo (verbo) | habl**e** (subjuntivo) | -e |
| Participio | habl**ado** | -ado |
| Gerundio | habl**ando** | -ando |

### Derivación por prefijos

| Prefijo | Significado | Ejemplo |
|---|---|---|
| in-/im- | negación | imposible, incompleto |
| des- | negación / reversión | descontento, deshacer |
| re- | otra vez | rehacer, reescribir |
| anti- | contra | antiguo, antinatural |
| super- | sobre | supermercado, superhombre |
| sub- | bajo | subterráneo, submarino |
| pre- | antes | prehistoria, prever |
| post- | después | postguerra, posmoderno |
| auto- | uno mismo | automóvil, autobiografía |

### Derivación por sufijos

| Sufijo | Significado | Ejemplo |
|---|---|---|
| -oso | cualidad | peligroso, hermoso |
| -ico/-ica | relativo | histórico, político |
| -al | relativo | musical, cultural |
| -mente | adverbio | rápidamente, felizmente |
| -ción/-sión | sustantivo acción | educación, decisión |
| -dor/-dora | agente | trabajador, cantadora |
| -ble | capacidad | comible, legible |
| -ito/-ita | diminutivo | gatito, mesita |
| -ote/-ota | aumentativo | casona, mujerota |
| -ismo/-ista | doctrina / partidario | socialismo, pianista |

### Composición

- limpiaparabrisas (limpiar + parabrisas)
- sacacorchos (sacar + corchos)
- abrelatas (abrir + latas)
- paraguas (parar + aguas)
- rascacielos (rascar + cielos)
- ferrocarril (ferro + carril — del latín)
- parabrisas (parar + brisas)

### Morfemas libres vs vinculantes

| Morfema libre | Morfema vinculante |
|---|---|
| gato, casa, alto | -oso, -ción, in-, des- |
| caminar, beber, leer | -ar, -er, -ir (no significan nada solos) |

El español tiene ambos tipos, pero la derivación usa principalmente afijos vinculantes.

## Inglés

### Morfología flexiva mínima

El inglés tiene solo ~8 morfemas flexivos:

| Categoría | Ejemplo | Morfema |
|---|---|---|
| Plural | cat → cat**s** | -s / -es / -ies |
| Posesivo | cat → cat**'s** | -'s |
| 3ra persona sg | walk → walk**s** | -s |
| Pasado | walk → walk**ed** | -ed |
| Progresivo | walk → walk**ing** | -ing |
| Participio | walk → walk**ed** | -ed |
| Comparativo | tall → tall**er** | -er |
| Superlativo | tall → tall**est** | -est |

### Derivación por prefijos

| Prefijo | Significado | Ejemplo |
|---|---|---|
| un- | negación (germánico) | unhappy, unkind |
| in-/im-/ir-/il- | negación (latino) | impossible, illegal, irregular |
| dis- | negación / reversión | dislike, disconnect |
| re- | otra vez | rewrite, rebuild |
| anti- | contra | antiviral, antifreeze |
| super- | sobre | superhuman, supermarket |
| sub- | bajo | submarine, subzero |
| pre- | antes | prehistoric, preview |
| post- | después | postwar, postgraduate |
| auto- | uno mismo | automobile, automatic |
| non- | no | nonstop, nonviolent |
| over- / under- | exceso / insuficiencia | overeat, undercooked |
| mis- | mal | misunderstand, misplace |
| co- | juntos | cooperate, coauthor |
| ex- | anterior | ex-president, ex-wife |
| self- | uno mismo | self-aware, self-control |

### Derivación por sufijos

| Sufijo | Significado | Ejemplo |
|---|---|---|
| -ful | lleno de | beautiful, helpful |
| -less | sin | homeless, fearless |
| -ness | estado / cualidad | happiness, kindness |
| -ment | resultado / acción | government, movement |
| -tion/-sion | sustantivo acción | education, decision |
| -er/-or | agente | worker, actor |
| -able/-ible | capacidad | readable, legible |
| -ous | cualidad | dangerous, nervous |
| -ive | tendencia | active, creative |
| -ly | adverbio | quickly, happily |
| -en | hacerse | darken, strengthen |
| -ize/-ise | hacer | modernize, apologize |
| -hood | estado colectivo | childhood, neighborhood |
| -ship | estado / relación | friendship, leadership |
| -let | diminutivo | piglet, booklet |
| -y / -ie | diminutivo afectivo | doggy, daddy |

### Composición

- blackboard (black + board)
- toothbrush (tooth + brush)
- notebook (note + book)
- sunrise (sun + rise)
- handwriting (hand + writing)
- bedroom (bed + room)
- dishwasher (dish + washer)
- snowman (snow + man)
- raincoat (rain + coat)
- overlook (over + look — también verbo compuesto)

### Dobletes del inglés

Mismo concepto, diferentes orígenes:

| Nativo/Germánico | Latín/Francés |
|---|---|
| kingly | royal |
| time | temporal |
| holy | sacred |
| ask | inquire |
| freedom | liberty |

### Análisis morfológico

- **un-happi-ness**: un- (negación) + happy (raíz) + -ness (sustantivo)
- **re-write-able**: re- (otra vez) + write (raíz) + -able (capacidad)
- **co-oper-ate**: co- (juntos) + oper (trabajar — del latín) + -ate (verbo)

## Chino

### Sin inflexión

El chino **no** inflecta los sustantivos, verbos o adjetivos:

- <ruby>猫<rt>māo</rt></ruby> = gato / gatos (no cambia)
- <ruby>走<rt>zǒu</rt></ruby> = caminar / caminó / caminará (no cambia)
- <ruby>大<rt>dà</rt></ruby> = grande (no género, no número)

### Composición (estrategia principal)

| Palabra compuesta | Componentes | Significado |
|---|---|---|
| <ruby>电<rt>diàn</rt></ruby><ruby>脑<rt>nǎo</rt></ruby> | eléctrico + cerebro | computadora |
| <ruby>电<rt>diàn</rt></ruby><ruby>话<rt>huà</rt></ruby> | eléctrico + hablar | teléfono |
| <ruby>火<rt>huǒ</rt></ruby><ruby>车<rt>chē</rt></ruby> | fuego + vehículo | tren |
| <ruby>飞<rt>fēi</rt></ruby><ruby>机<rt>jī</rt></ruby> | volar + máquina | avión |
| <ruby>洗<rt>xǐ</rt></ruby><ruby>手<rt>shǒu</rt></ruby><ruby>间<rt>jiān</rt></ruby> | lavar + mano + habitación | baño |
| <ruby>超<rt>chāo</rt></ruby><ruby>市<rt>shì</rt></ruby> | super + mercado | supermercado |
| <ruby>地<rt>dì</rt></ruby><ruby>铁<rt>tiě</rt></ruby> | tierra + hierro | metro |
| <ruby>公<rt>gōng</rt></ruby><ruby>共<rt>gòng</rt></ruby><ruby>汽<rt>qì</rt></ruby><ruby>车<rt>chē</rt></ruby> | público + vapor + vehículo | autobús |
| <ruby>北<rt>běi</rt></ruby><ruby>京<rt>jīng</rt></ruby> | norte + capital | Beijing |

### Morfemas libres

Cada carácter chino es un morfema libre con significado:

- <ruby>人<rt>rén</rt></ruby> = persona
- <ruby>大<rt>dà</rt></ruby> = grande
- <ruby>水<rt>shuǐ</rt></ruby> = agua
- <ruby>走<rt>zǒu</rt></ruby> = caminar
- <ruby>来<rt>lái</rt></ruby> = venir
- <ruby>上<rt>shàng</rt></ruby> = arriba
- <ruby>下<rt>xià</rt></ruby> = abajo

### Partículas funcionales (no afijos)

| Partícula | Uso | Ejemplo |
|---|---|---|
| <ruby>的<rt>de</rt></ruby> | adjetivo / posesivo | <ruby>美<rt>měi</rt></ruby><ruby>丽<rt>lì</rt></ruby>**的**<ruby>花<rt>huā</rt></ruby> (flor hermosa) |
| <ruby>了<rt>le</rt></ruby> | cambio de estado / completado | <ruby>吃<rt>chī</rt></ruby>**了** (comió / terminó de comer) |
| <ruby>着<rt>zhe</rt></ruby> | progresivo continuo | <ruby>走<rt>zǒu</rt></ruby>**着** (está caminando) |
| <ruby>过<rt>guò</rt></ruby> | experiencia pasada | <ruby>去<rt>qù</rt></ruby>**过** (ha ido antes) |
| <ruby>地<rt>de</rt></ruby> | adverbio | <ruby>快<rt>kuài</rt></ruby>**地**<ruby>走<rt>zǒu</rt></ruby> (caminar rápidamente) |
| <ruby>得<rt>de</rt></ruby> | complemento de grado | <ruby>走<rt>zǒu</rt></ruby>**得**<ruby>很<rt>hěn</rt></ruby><ruby>快<rt>kuài</rt></ruby> (caminar muy rápido) |

**Importante**: Estas partículas son **palabras separadas**, no afijos vinculantes. No se fusionan con la raíz.

### Reduplicación

El chino usa la reduplicación para diversos efectos:

- <ruby>看<rt>kàn</rt></ruby><ruby>看<rt>kàn</rt></ruby> (eche un vistazo — suave)
- <ruby>人<rt>rén</rt></ruby><ruby>人<rt>rén</rt></ruby> (todas las personas — distributivo)
- <ruby>妈<rt>mā</rt></ruby><ruby>妈<rt>ma</rt></ruby> (mamá — afectivo)
- <ruby>爸<rt>bà</rt></ruby><ruby>爸<rt>ba</rt></ruby> (papá — afectivo)

### Morfemas afijos (muy limitados)

- <ruby>子<rt>zi</rt></ruby>: <ruby>桌<rt>zhuō</rt></ruby><ruby>子<rt>zi</rt></ruby> (mesa), <ruby>椅<rt>yǐ</rt></ruby><ruby>子<rt>zi</rt></ruby> (silla) — nominalizador leve
- <ruby>儿<rt>ér</rt></ruby>: <ruby>花<rt>huā</rt></ruby><ruby>儿<rt>ér</rt></ruby> (florcita — dialecto del norte)
- <ruby>们<rt>men</rt></ruby>: <ruby>我<rt>wǒ</rt></ruby><ruby>们<rt>men</rt></ruby> (nosotros), <ruby>朋<rt>péng</rt></ruby><ruby>友<rt>yǒu</rt></ruby><ruby>们<rt>men</rt></ruby> (amigos — plural humano)

### Comparación: morfema de plural

- 3 cat**s** (inglés: sufijo)
- 3 gato**s** (español: sufijo)
- <ruby>三<rt>sān</rt></ruby><ruby>只<rt>zhī</rt></ruby><ruby>猫<rt>māo</rt></ruby> (chino: <ruby>猫<rt>māo</rt></ruby> no cambia — el numeral + clasificador expresa cantidad)

## Comparación de un vistazo

| Estrategia | Español | Inglés | Chino |
|----------|---------|---------|---------|
| Inflexión | Extensiva (verbo, género, número) | Mínima (plural, pasado, -ing) | Ninguna |
| Derivación (prefijos) | Latino/griego productivo | Germánico/latino/griego | Ninguna (morfemas libres) |
| Derivación (sufijos) | Extensiva (-oso, -ción, -mente) | Extensiva (-ness, -ful, -ly, -tion) | Muy limitada (-子, -儿, -们) |
| Composición | Común (limpiaparabrisas) | Muy común (blackboard) | **Estrategia principal** (电脑) |
| Morfemas libres | Algunos | Algunos | **Todos** |
| Partículas funcionales | Clíticos vinculantes | Auxiliares separados | **Partículas separadas** (的, 了, 着, 过) |

## Errores comunes

1. **Hispanohablantes/angloparlantes aprendiendo chino**: Esperando inflexión verbal: *<ruby>他<rt>tā</rt></ruby><ruby>走<rt>zǒu</rt></ruby><ruby>了<rt>le</rt></ruby><ruby>过<rt>guò</rt></ruby><ruby>去<rt>qù</rt></ruby>* → <ruby>他<rt>tā</rt></ruby><ruby>走<rt>zǒu</rt></ruby><ruby>过<rt>guò</rt></ruby><ruby>去<rt>qù</rt></ruby><ruby>了<rt>le</rt></ruby> (la partícula <ruby>了<rt>le</rt></ruby> sigue al verbo, no se fusiona)

2. **Sinohablantes aprendiendo inglés**: Olvidando el plural: *three book* → *three book**s***

3. **Sinohablantes aprendiendo español**: Olvidando la concordancia de género: *un persona* → *una persona*

4. **Angloparlantes aprendiendo español**: Añadiendo -s a todo: *la problema* → *el problema* (masculino irregular)
