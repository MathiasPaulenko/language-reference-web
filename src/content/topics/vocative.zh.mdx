## 概述

呼格是直接称呼某人的形式或策略。

- **西班牙语**：没有特殊的呼格形态。人名和头衔直接使用。语调（升调、逗号停顿）标记呼格。Oye... 是常见的引起注意用语。
- **英语**：没有呼格形态。直接称呼使用人名、头衔或引起注意的词语（hey, excuse me）。语调将呼格与句子其余部分分开。
- **汉语**：没有呼格形态。使用助词（<ruby>喂<rt>wèi</rt></ruby>, <ruby>哎<rt>āi</rt></ruby>）和头衔。在公共场合，<ruby>不<rt>bù</rt></ruby><ruby>好<rt>hǎo</rt></ruby><ruby>意<rt>yì</rt></ruby><ruby>思<rt>sī</rt></ruby> 或 <ruby>麻<rt>má</rt></ruby><ruby>烦<rt>fán</rt></ruby><ruby>一<rt>yī</rt></ruby><ruby>下<rt>xià</rt></ruby> 缓和打扰。

## 西班牙语

### 用名字直接称呼

名字不变地使用：

- *María, ven aquí.*
- *Juan, escúchame.*
- *Pedro, ¿dónde estás?*

**语调**：名字用升调，短暂停顿。

### 作为呼格的头衔

- *Señor García, un momentito.*
- *Señora López, por favor.*
- *Doctor Martínez, ¿puedo hablar con usted?*
- *Don Juan, buenos días.*
- *Doña María, ¿cómo está?*
- *Profesora, tengo una pregunta.*

### 引起注意

| 表达 | 用途 | 语域 |
|---|---|---|
| Oiga | 注意（正式，对陌生人） | 正式 |
| Oye | 注意（非正式） | 非正式 |
| Mira | 看/听 | 非正式 |
| Venga | 来吧/嘿 | 非正式 |
| Eh / Ey | 嘿 | 非常非正式 |
| Ea | 嘿（西班牙） | 非正式 |
| Chico / Chica | 嘿，小子/姑娘 | 非正式 |
| Muchacho / Muchacha | 嘿，年轻人 | 非正式 |

### 礼貌引起注意

- *Disculpe, señor...*
- *Perdone, señora...*
- *Con permiso...*
- *Perdón...*

### 亲密/家人

- *Mamá, ¿dónde estás?*
- *Papá, ven.*
- *Hijo, escucha.*
- *Mi amor, te quiero.*
- *Cariño, ven aquí.*

### 书信

- *Querido Juan:*
- *Estimado señor García:*
- *Querida familia:*

## 英语

### 用名字直接称呼

- *María, come here.*
- *John, listen to me.*
- *Sarah, where are you?*

**注意**：书写中逗号标记呼格。语调在口语中区分呼格。

### 头衔

| 头衔 | 用途 |
|---|---|
| Mr. + 姓 | 男士正式 |
| Mrs. + 姓 | 已婚女士正式 |
| Ms. + 姓 | 女士正式（中性） |
| Miss + 姓 | 未婚女士正式（过时） |
| Dr. + 姓 | 博士 |
| Professor + 姓 | 大学教授 |
| sir | 男士（服务、正式） |
| ma'am | 女士（服务、正式） |
| officer | 警察 |
| waiter / waitress | 餐厅（逐渐减少） |
| nurse | 护士 |
| Your Honor | 法官 |

### 引起注意

| 表达 | 用途 | 语域 |
|---|---|---|
| Excuse me | 注意、道歉、通过 | 中性 |
| Hey | 注意 | 非正式 |
| Hi / Hello | 问候 | 中性 |
| Listen | 注意 | 非正式 |
| Look | 注意 | 非正式 |
| Pardon me | 道歉、注意 | 正式 |
| Sorry | 道歉 | 中性 |
| Yo | 注意（美国） | 非常非正式 |
| Psst | 悄悄引起注意 | 非正式 |

### 礼貌用语

- *Excuse me, sir...*
- *Pardon me, ma'am...*
- *I'm sorry to bother you...*
- *Could I have your attention, please?*

### 亲密/家人

- *Mom, where are you?*
- *Dad, come here.*
- *Honey, listen to me.*
- *Babe / Baby, I love you.*
- *Sweetie, come here.*
- *Guys, listen up.*

### 警告/紧急

- *Watch out!*
- *Look out!*
- *Heads up!*
- *Attention, everyone!*

### 书信

- *Dear John,*
- *Dear Mr. Smith,*
- *To whom it may concern,*
- *Dear Sir or Madam,*
- *Hello María,*（非正式邮件）

## 常见错误

1. **英语者学西班牙语**：对陌生人用 Hola 引起注意 → *Oiga*（正式）/ *Oye*（非正式）

2. **西班牙语者学英语**：在所有语境用 Oiga → *Excuse me*（正式）/ *Hey*（非正式）

3. **英语者学汉语**：所有注意场景都用 <ruby>你<rt>nǐ</rt></ruby><ruby>好<rt>hǎo</rt></ruby> → <ruby>不<rt>bù</rt></ruby><ruby>好<rt>hǎo</rt></ruby><ruby>意<rt>yì</rt></ruby><ruby>思<rt>sī</rt></ruby>（公共场合）/ <ruby>喂<rt>wèi</rt></ruby>（电话）

4. **汉语者学英语**：面对面用 <ruby>喂<rt>wèi</rt></ruby> → *Hey* / *Excuse me*
