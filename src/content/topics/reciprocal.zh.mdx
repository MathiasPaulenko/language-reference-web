## 概述

相互结构表达相互动作：他们拥抱，我们理解。

- **西班牙语**：用反身代词 *se*（及其变体 *nos, os*）与复数主语。相互和反身形式相同；语境消除歧义。
- **英语**：用专门的相互代词 *each other* 和 *one another*。它们在语法上与反身代词 (*themselves*) 不同。
- **汉语**：用前置副词 *互相* 或 *相互*。无专门相互代词。反身 (*自己*) 和相互 (*互相*) 完全区分。

## 西班牙语

### 相互代词

西班牙语用与反身相同的代词，但复数主语和相互意义：

| 人称 | 反身/相互 | 相互例句 |
|---|---|---|
| 第一人称复数 | nos | **Nos** ayudamos |
| 第二人称复数 | os | **Os** llamáis |
| 第三人称复数 | se | **Se** quieren |

### 反身/相互歧义

西班牙语中，同一形式可能是反身或相互：

| 句子 | 可能含义 |
|---|---|
| Se lavan. | 各自洗自己（反身） |
| Se lavan. | 互相洗（相互） |
| Se miran. | 各自看自己（反身） |
| Se miran. | 互相对视（相互） |

**注**：语境消除歧义，不是语法。

### 常见相互动词

| 动词 | 例句 |
|---|---|
| quererse | Se quieren mucho. |
| amarse | Se aman. |
| ayudarse | Nos ayudamos. |
| pegarse | Se pegan. |
| entenderse | Se entienden bien. |
| respetarse | Se respetan. |

### 相互表达

| 表达 | 含义 |
|---|---|
| mutuo / mutua | 相互 |
| común | 共同 |
| compartido | 共享 |
| reciproco | 相互 |

## 英语

### 相互代词

英语有专门的相互代词：

| 代词 | 用法 | 例句 |
|---|---|---|
| each other | 两人 | They love **each other**. |
| one another | 三人及以上 | They help **one another**. |

**注**：现代实践中，*each other* 和 *one another* 可互换。

### 反身/相互区分

| 反身 | 相互 |
|---|---|
| They washed **themselves**. | They washed **each other**. |
| They saw **themselves** in the mirror. | They saw **each other** at the park. |
| They hurt **themselves**. | They hurt **each other**. |

### 相互所有格

| 结构 | 例句 |
|---|---|
| each other's + 名词 | They held **each other's** hands. |
| one another's + 名词 | They respect **one another's** opinions. |

### 一起

| 表达 | 含义 | 例句 |
|---|---|---|
| together | 一起 | They work **together**. |
| mutually | 相互 | They **mutually** agreed. |
| with each other | 彼此 | They talk **with each other**. |

## 快速比较

| 特征 | 西班牙语 | 英语 | 汉语 |
|---|---|---|---|
| 相互形式 | se + 复数 | each other / one another | 互相 / 相互 |
| 与反身歧义 | 有 | 无 | 无 |
| 词类 | 代词 | 代词 | 副词 |
| 相互所有格 | 无专门 | each other's | 互相 + 的 |
| 需复数主语 | 是 | 是 | 是 |

## 常见错误

1. **西班牙语母语者**：英语中省略 each other → They help **each other**（不说 They help）

2. **英语母语者**：西班牙语中混淆反身和相互 → *Se lavan* 可两者；语境决定

3. **英语母语者**：汉语中互相与单数主语连用 → 需复数：<ruby>他<rt>tā</rt></ruby><ruby>们<rt>men</rt></ruby><ruby>互<rt>hù</rt></ruby><ruby>相<rt>xiāng</rt></ruby>...

4. **汉语母语者**：英语中 each other 与单数主语连用 → **They** love each other（不说 He）
