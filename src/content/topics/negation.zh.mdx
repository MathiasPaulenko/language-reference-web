## 概述

否定反转句子的真值。不同语言标记否定的方式和支持助动词的需求各不相同。

- **西班牙语**：用 no 放在动词前否定。双重否定是必需的：no + nada/nadie/ninguno/nunca。无 do 支持。
- **英语**：主动词否定需要 do 支持：do/does/did + not + 不定式。双重否定是非标准的。
- **汉语**：不 否定意图/习惯/状态，没 否定未完成的动作。无需额外助动词。

## 西班牙语

### 简单否定

将 no 放在变位动词前：

- No como carne.（我不吃肉。）
- No es verdad.（这不是真的。）
- No estoy listo.（我没准备好。）
- No hay pan.（没有面包。）

### 双重否定（必需）

西班牙语要求双重否定：no + 否定词：

| 肯定 | 否定 |
|---|---|
| algo | no ... nada |
| alguien | no ... nadie |
| alguno / alguna | no ... ninguno / ninguna |
| siempre | no ... nunca / jamás |
| también | no ... tampoco |
| o ... o | no ... ni ... ni |

**例子**：

- No vi nada.（我什么也没看见。）
- No vino nadie.（没人来。）
- No tengo ningún problema.（我没有任何问题。）
- No nunca voy allí.（我从不去那里。）
- Yo tampoco.（我也不。）
- No le gusta ni el café ni el té.（他既不喜欢咖啡也不喜欢茶。）

**注意**：否定词可置于动词前，此时省略 no：

- Nadie vino.（没人来。）
- Nunca lo haré.（我永远不会做。）
- Nada me gusta.（我什么都不喜欢。）

### 祈使否定

- No toques eso.（别碰那个。）
- No te vayas todavía.（先别走。）

### 带不定冠词的否定

- No es un médico.（他不是医生。）
- No tengo ningún coche.（我没有任何车。）

### 带虚拟式的否定

- No creo que venga.（我不认为他会来。——虚拟式）
- No es posible que sea cierto.（这不可能是真的。）

## 英语

### 主动词否定需要 do 支持

英语否定主动词需要 do/does/did：

- I do not like coffee.（我不喜欢咖啡。）
- She does not speak Spanish.（她不说西班牙语。）
- They did not arrive on time.（他们没有准时到达。）

**注意**：情态动词和 to be 直接否定：

- I am not tired.（我不累。）
- He cannot swim.（他不会游泳。）
- They will not come.（他们不会来。）

### 常见缩略形式

| 完整 | 缩略 |
|---|---|
| do not | don't |
| does not | doesn't |
| did not | didn't |
| is not | isn't |
| are not | aren't |
| was not | wasn't |
| were not | weren't |
| have not | haven't |
| has not | hasn't |
| had not | hadn't |
| will not | won't |
| would not | wouldn't |
| cannot | can't |
| could not | couldn't |
| should not | shouldn't |
| must not | mustn't |
| need not | needn't |

### 否定后用肯定意义词

标准英语在否定后用肯定意义词：

- I didn't see anything.（我什么也没看见。）
- She doesn't know anyone.（她不认识任何人。）
- We haven't been anywhere.（我们哪儿也没去过。）

**注意**：英语口语有时使用双重否定（I didn't see nothing），但非标准。

### 名词否定

- There is no bread.（没有面包。）
- I have no money.（我没有钱。）
- She is not a doctor.（她不是医生。）

### 祈使否定

- Do not touch.（别碰。）
- Don't go.（别走。）

## 汉语

### 不 bù（一般否定）

不 否定意图、习惯、状态和将来动作：

- 我不吃肉。（我不吃肉。——习惯）
- 他不是医生。（他不是医生。）
- 我不知道。（我不知道。）
- 明天不下雨。（明天不会下雨。）

### 没 méi（已完成动作否定）

没 否定动作已完成：

- 我没吃饭。（我没吃饭。/ 我没有吃饭。）
- 他没来。（他没来。）
- 没有钱。（没有钱。）

**关键规则**：

- 不 + 动词 = 不（习惯/意图/状态）
- 没 + 动词 = 没（动作未完成）
- 没有 = 没有/不存在

### 没有 méiyǒu（没有/不存在）

没有 否定拥有或存在：

- 我没有车。（我没有车。）
- 这里没有人。（这里没有人。）

### 别 bié（祈使否定）

别 否定命令或建议：

- 别碰！（别碰！）
- 别担心。（别担心。）
- 别忘了。（别忘了。）

### 双重否定

汉语有时使用 什么都不 结构：

- 我什么都不知道。（我什么都不知道。——字面："什么都不知道"）
- 他什么都没做。（他什么都没做。）

**注意**：这种结构是语法正确的，但与西班牙语或英语的双重否定不同。

### 不 vs 没：总结

| 语境 | 使用 |
|---|---|
| 习惯/偏好 | 不 |
| 状态/描述 | 不 |
| 意图/将来计划 | 不 |
| 未完成动作 | 没 |
| 存在/拥有 | 没有 |
| 否定命令 | 别 |

## 一览对比

| 结构 | 西班牙语 | 英语 | 汉语 |
|----------|---------|---------|---------|
| 简单动词否定 | no + verbo | do/does/did + not + verbo | 不 + verbo |
| 已完成动作否定 | no + pretérito | did + not + infinitivo | 没 + verbo |
| 名词否定 | no es / no hay | is not / there is no | 不是 / 没有 |
| 祈使否定 | no + imperativo | do + not + infinitivo | 别 + verbo |
| 双重否定 | 必需（no + nada） | 非标准（anything） | 什么...都 + 不 |
| 否定助动词 | 无 | do/does/did | 无 |

## 常见错误

1. **西班牙语母语者学英语**：省略 do 支持：I no eat → I do not eat

2. **英语母语者学西班牙语**：忘记双重否定：No vi algo → No vi nada

3. **英语母语者学汉语**：所有否定都用 不：我不吃了午饭 → 我没吃午饭（特定动作未完成）

4. **汉语母语者学英语**：非标准双重否定：I don't know nothing → I don't know anything
