## 概述

被动语态将焦点从施事者转移到受事者。

- **西班牙语**：被动语态相对罕见。说西班牙语者偏好主动语态或无人称 se 结构。使用时，过去分词在性和数上与主语一致。
- **英语**：被动语态极为普遍——用于正式写作、科技文本，以及施事未知或不重要时。
- **汉语**：用 <ruby>被<rt>bèi</rt></ruby> 标记被动。历史上带有负面或对抗性语气（某件坏事发生在主语身上），但现代用法正在扩展。

## 西班牙语

### 真被动（ser + 过去分词）

用 *ser* + 过去分词构成。分词在性和数上一致：

- *La carta **fue escrita** por María.*（这封信由玛丽亚写的。）
- *Las casas **fueron vendidas**.*（这些房子被卖了。）
- *El libro **será publicado** el mes que viene.*（这本书将于下个月出版。）

**重要**：这里的分词是形容词——必须一致。

### 状态被动（estar + 过去分词）

*Estar* + 分词描述**状态或结果**，而非动作：

- *La puerta **está cerrada**.*（门是关着的。——状态）
- *La ventana **está rota**.*（窗户是破的。——结果）

**区别**：

- *La puerta **fue cerrada** por Juan.*（动作被动——胡安关了门）
- *La puerta **está cerrada**.*（状态被动——门处于关闭状态）

### 无人称 se（偏好替代被动）

西班牙语通常避免被动。相反，使用 *se* + 第三人称动词：

- *Se **venden** casas.*（房屋出售。）
- *Se **habla** español.*（说西班牙语。）
- *Se **dice** que...*（据说……）

这是表达被动意义最自然的方式。

### 被动反身

某些动词使用反身形式表达被动意义：

- *El vino **se vende** bien.*（这酒卖得很好。）
- *La fruta **se come** fresca.*（水果新鲜吃。）

## 英语

### Be-passive（标准）

用 *be* + 过去分词构成。施事可选：

- *The letter **was written** (by John).*
- *The house **is being built**.*
- *The problem **has been solved**.*

### 被动中的时态

任何时态都可以变为被动：

| 时态 | 主动 | 被动 |
|---|---|---|
| 一般现在时 | They make cars here. | Cars **are made** here. |
| 现在进行时 | They are building a house. | A house **is being built**. |
| 一般过去时 | Shakespeare wrote Hamlet. | Hamlet **was written** by Shakespeare. |
| 现在完成时 | Someone has stolen my bike. | My bike **has been stolen**. |
| 将来时 | They will announce it tomorrow. | It **will be announced** tomorrow. |
| 情态动词 | You can see it from here. | It **can be seen** from here. |

### Get-passive（非正式）

*Get* 在非正式语境中替代 *be*，常带有对抗性：

- *I **got fired**.*（我被解雇了。）
- *He **got hurt**.*（他受伤了。）
- *They **got married**.*（他们结婚了。）

### by 引导施事

施事由 *by* 引入：

- *The book **was written by** J.K. Rowling.*
- *The window **was broken by** the wind.*

当施事未知或不重要时，省略：

- *My car **was stolen**.*（我不知道是谁干的。）
- *Spanish **is spoken** here.*（说话者无关紧要。）

### 为什么使用被动？

1. **施事未知**：*My bike was stolen.*
2. **施事明显/不重要**：*The bill was passed.*（议会通过——明显）
3. **焦点在受事**：*The window was broken.*（不想说谁打破了）
4. **正式/科技文体**：*It was observed that...*

## 汉语

### 被字句（<ruby>被<rt>bèi</rt></ruby> + 施事 + 动词）

<ruby>被<rt>bèi</rt></ruby> 标记被动。施事（若提及）置于 <ruby>被<rt>bèi</rt></ruby> 和动词之间：

- <ruby>他<rt>tā</rt></ruby><ruby>被<rt>bèi</rt></ruby><ruby>老<rt>lǎo</rt></ruby><ruby>师<rt>shī</rt></ruby><ruby>批<rt>pī</rt></ruby><ruby>评<rt>píng</rt></ruby><ruby>了<rt>le</rt></ruby>。（他被老师批评了。）
- <ruby>我<rt>wǒ</rt></ruby><ruby>的<rt>de</rt></ruby><ruby>车<rt>chē</rt></ruby><ruby>被<rt>bèi</rt></ruby><ruby>偷<rt>tōu</rt></ruby><ruby>了<rt>le</rt></ruby>。（我的车被偷了。）

**结构**：主语 + <ruby>被<rt>bèi</rt></ruby> +（施事）+ 动词 + <ruby>了<rt>le</rt></ruby>

### 历史对抗性语气

传统上，<ruby>被<rt>bèi</rt></ruby> 暗示发生了不好的事情：

- <ruby>被<rt>bèi</rt></ruby><ruby>打<rt>dǎ</rt></ruby>（被打）
- <ruby>被<rt>bèi</rt></ruby><ruby>骂<rt>mà</rt></ruby>（被骂）
- <ruby>被<rt>bèi</rt></ruby><ruby>骗<rt>piàn</rt></ruby>（被骗）

现代汉语正在扩展此用法，但 <ruby>被<rt>bèi</rt></ruby> 仍带有轻微的被动对抗意味。

### 中性替代形式

对于中性或积极的被动，汉语常使用主动语态或其他结构：

- <ruby>由<rt>yóu</rt></ruby>...<ruby>做<rt>zuò</rt></ruby>（由……做）——正式、中性
- <ruby>受<rt>shòu</rt></ruby>...<ruby>欢<rt>huān</rt></ruby><ruby>迎<rt>yíng</rt></ruby>（受……欢迎）——被……欢迎
- 直接省略施事：<ruby>这<rt>zhè</rt></ruby><ruby>本<rt>běn</rt></ruby><ruby>书<rt>shū</rt></ruby><ruby>卖<rt>mài</rt></ruby><ruby>完<rt>wán</rt></ruby><ruby>了<rt>le</rt></ruby>。（这本书卖完了。）

### 叫/让作为口语替代

<ruby>叫<rt>jiào</rt></ruby> 和 <ruby>让<rt>ràng</rt></ruby> 可在口语中替代 <ruby>被<rt>bèi</rt></ruby>：

- <ruby>我<rt>wǒ</rt></ruby><ruby>叫<rt>jiào</rt></ruby><ruby>人<rt>rén</rt></ruby><ruby>骗<rt>piàn</rt></ruby><ruby>了<rt>le</rt></ruby>。（我被人骗了。）
- <ruby>他<rt>tā</rt></ruby><ruby>让<rt>ràng</rt></ruby><ruby>狗<rt>gǒu</rt></ruby><ruby>咬<rt>yǎo</rt></ruby><ruby>了<rt>le</rt></ruby>。（他被狗咬了。）

### 给作为口语被动

<ruby>给<rt>gěi</rt></ruby> 在北方口语中：

- <ruby>我<rt>wǒ</rt></ruby><ruby>给<rt>gěi</rt></ruby><ruby>骂<rt>mà</rt></ruby><ruby>了<rt>le</rt></ruby>。（我被骂了。）

### 动词不变形

汉语动词在被动中不变形：

- <ruby>打<rt>dǎ</rt></ruby>（打）→ <ruby>被<rt>bèi</rt></ruby><ruby>打<rt>dǎ</rt></ruby>（被打）——同一动词

## 常见错误

1. **英语者学西班牙语**：过度使用被动：*La decisión fue tomada* → *Se tomó la decisión*（无人称 se 更自然）

2. **英语者学西班牙语**：忘记一致：*La carta fue escrito* → *La carta fue **escrita***

3. **英语者学汉语**：在所有情况下使用 <ruby>被<rt>bèi</rt></ruby>：*<ruby>这<rt>zhè</rt></ruby><ruby>本<rt>běn</rt></ruby><ruby>书<rt>shū</rt></ruby><ruby>被<rt>bèi</rt></ruby><ruby>写<rt>xiě</rt></ruby><ruby>了<rt>le</rt></ruby>* → *<ruby>这<rt>zhè</rt></ruby><ruby>本<rt>běn</rt></ruby><ruby>书<rt>shū</rt></ruby><ruby>是<rt>shì</rt></ruby><ruby>作<rt>zuò</rt></ruby><ruby>者<rt>zhě</rt></ruby><ruby>写<rt>xiě</rt></ruby><ruby>的<rt>de</rt></ruby>*（主动对中性陈述更自然）

4. **汉语者学英语**：省略 be：*The letter written by him* → *The letter **was** written by him*
