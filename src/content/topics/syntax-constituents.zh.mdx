## 概述

句法研究单词如何组合成更大的单位。

- **西班牙语**：基本 SVO 语序，但屈折变化丰富使词序相对自由。成分结构与英语平行。形容词通常后置。
- **英语**：SVO 语序严格。成分结构明确：NP, VP, PP, AdjP, AdvP。移位规则（疑问句形成、被动化）依赖层级结构。
- **中文**：基本 SVO 语序，但话题化广泛。成分通过助词（的, 地, 得）而非屈折变化来标记。前置关系从句。

| 补语化短语 | que vino | CP |
| 不定式短语 | leer el libro | SInf |

### 灵活语序

| 语序 | 例句 | 语境 |
|---|---|---|
| SVO (中性) | El estudiante lee un libro. | 基本语序 |
| VSO (疑问) | ¿Lee el estudiante un libro? | 疑问句 |
| OVS (有标记) | Un libro lo lee el estudiante. | 宾语强调 |
| VS (起始) | Llegó Juan. | 动词强调 |

### 形容词位置

| 位置 | 意义 | 例句 |
|---|---|---|
| 前置 | 主观/情感化 | **viejo** amigo (亲爱的) |
| 后置 | 客观/描述性 | amigo **viejo** (年长的) |

## 语境中的例子

### 基本句子
- **西班牙语**：El estudiante lee un libro.
- **英语**：The student reads a book.
- **中文**：<ruby>学<rt>xué</rt></ruby><ruby>生<rt>shēng</rt></ruby><ruby>读<rt>dú</rt></ruby><ruby>书<rt>shū</rt></ruby>。

### 话题化
- **西班牙语**：Un libro **lo** lee el estudiante.
- **英语**：A book the student reads.
- **中文**：<ruby>书<rt>shū</rt></ruby>，<ruby>学<rt>xué</rt></ruby><ruby>生<rt>shēng</rt></ruby><ruby>读<rt>dú</rt></ruby><ruby>了<rt>le</rt></ruby>。

### 关系从句
- **西班牙语**：el libro **que leí**
- **英语**：the book **that I read**
- **中文**：<ruby>我<rt>wǒ</rt></ruby><ruby>读<rt>dú</rt></ruby><ruby>的<rt>de</rt></ruby><ruby>书<rt>shū</rt></ruby>

## 常见错误

1. **所有使用者**：假设中文 = 英语 → 中文使用话题-述语结构

2. **英语使用者**：西班牙语前置形容词 → estudiante **alto**（不是 alto estudiante）

3. **中文使用者**：将英语助动词视为独立从句 → [VP has been reading]

4. **所有使用者**：认为无屈折 = 无结构 → 中文使用助词
