## 概述

介词是表示关系的小词——位置、时间、方向、原因等。

- **西班牙语**：介词比英语少，但与冠词结合（*al, del*）。两个棘手的介词（*por* vs *para*）都译为"for"，但用法不同。
- **英语**：许多介词有细微区别。短语动词（*give up, look after, turn on*）是重大挑战。介词悬空在疑问句中被允许。
- **汉语**：使用少量核心介词（<ruby>在<rt>zài</rt></ruby>、<ruby>从<rt>cóng</rt></ruby>、<ruby>到<rt>dào</rt></ruby>、<ruby>对<rt>duì</rt></ruby>）。后置词（<ruby>上<rt>shàng</rt></ruby>、<ruby>下<rt>xià</rt></ruby>、<ruby>里<rt>lǐ</rt></ruby>、<ruby>外<rt>wài</rt></ruby>）常跟在名词后表示位置。

## 西班牙语

### 核心介词

| 介词 | 主要用法 | 例句 |
|---|---|---|
| a | 方向 / 时间 | Voy **a** casa, **a** las tres |
| de | 所有 / 来源 | El libro **de** Juan, soy **de** España |
| en | 位置 / 时间 | Estoy **en** casa, **en** verano |
| por | 原因 / 方式 / 价格 | Lo hice **por** ti, paso **por** aquí |
| para | 目的 / 目的地 | Es **para** ti, salgo **para** Madrid |
| con | 伴随 / 工具 | Voy **con** María, corto **con** cuchillo |
| sin | 缺乏 | Voy **sin** dinero |
| sobre | 上面 / 主题 | El libro **sobre** la mesa, hablo **sobre** política |
| entre | 两者之间 | **Entre** tú y yo |
| desde | 起点 | **Desde** Madrid hasta Barcelona |
| hasta | 终点 | **Hasta** mañana |

### 与冠词的缩合

| 组合 | 形式 | 例句 |
|---|---|---|
| a + el | **al** | Voy **al** cine |
| de + el | **del** | Vengo **del** trabajo |

**注**：不与la缩合：a + la = a la；de + la = de la。

### por vs para

| por | para |
|---|---|
| 原因（por eso） | 目的（para eso） |
| 方式（por avión） | 接收者（para ti） |
| 时长（por dos horas） | 目的地（para Madrid） |
| 价格（por 10 euros） | 截止日期（para mañana） |
| 替代（por mí） | 用途（para cortar） |

## 英语

### 核心介词

| 介词 | 主要用法 | 例句 |
|---|---|---|
| in | 在……里 / 长时间 | **In** the room, **in** July |
| on | 在……上 / 日期 | **On** the table, **on** Monday |
| at | 具体点 / 时间 | **At** the door, **at** 3 o'clock |
| to | 方向 | Go **to** school |
| from | 来源 | Come **from** Spain |
| for | 目的 / 时长 | **For** you, **for** two hours |
| with | 伴随 / 工具 | **With** me, cut **with** a knife |
| without | 缺乏 | **Without** money |
| by | 施事 / 方式 / 旁边 | Written **by** him, **by** car, **by** the river |
| about | 主题 | Talk **about** politics |
| into | 进入 | Go **into** the room |
| through | 穿过 | Walk **through** the park |
| across | 横过 | Walk **across** the street |
| over | 越过 | Jump **over** the fence |
| under | 在……下 | **Under** the table |
| between | 两者之间 | **Between** you and me |
| among | 在……之中 | **Among** the crowd |
| during | 在……期间 | **During** the meeting |
| until / till | 直到 | Wait **until** tomorrow |
| since | 自从（时间） | **Since** 2010 |

### 介词悬空

| 正式 | 非正式 |
|---|---|
| With whom did you speak? | Who did you speak **with**? |
| To whom did you give it? | Who did you give it **to**? |
| In which city do you live? | Which city do you live **in**? |

**注**：正式英语避免以介词结尾；非正式英语允许。

### 短语动词

| 短语动词 | 意义 | 例句 |
|---|---|---|
| give up | 放弃 | Don't give **up** |
| look after | 照顾 | Look **after** the children |
| turn on | 打开 | Turn **on** the light |
| put up with | 忍受 | Put **up with** the noise |
| run out of | 用完 | Run **out of** time |

**注**：短语动词是习语的；不要逐字翻译。

## 快速比较

| 特征 | 西班牙语 | 英语 | 汉语 |
|---|---|---|---|
| 数量 | ~20个主要 | ~150+ | ~15个介词 |
| 与冠词结合 | 有 (al, del) | 无 | 无 |
| 介词悬空 | 无 | 有 | 不适用 |
| 短语动词 | 无 | 有 | 无 |
| 后置词 | 无 | 无 | 有 (上, 下, 里, 外) |
| por/para | 有区别 | 一个介词 (for) | 多个 |

## 常见错误

1. **英语母语者**：使用"for"时应该是"por"或"para" → 英语没有por/para的区别

2. **英语母语者**：汉语中省略介词 → "我北京" 而不是 "我**在**北京"

3. **西班牙语母语者**：正式英语中介词悬空 → "Who did you speak with?" 在正式语境中

4. **西班牙语母语者**：短语动词逐字翻译 → "dar arriba" 而不是 "rendirse"
