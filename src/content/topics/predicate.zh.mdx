## 概述

谓语是对主语的陈述。每种语言的构建方式不同。

- **西班牙语**：动词性谓语是裸的。形容词性和名词性需要ser/estar。区分固有属性和状态。
- **英语**：动词性谓语是裸的。所有非动词性需要be。使用become, seem, get作为伪系词。
- **汉语**：形容词性谓语是裸的（无系词）。名词性需要 是。进行体使用 在。结果性变化使用 了。

## 西班牙语

### ser vs estar

| 系词 | 用法 | 例句 |
|---|---|---|
| **ser** | 身份 / 固有属性 | Juan **es** alto（总是） |
| **ser** | 职业 | Juan **es** médico |
| **ser** | 来源 | Juan **es** de España |
| **estar** | 临时状态 | Juan **está** cansado（现在） |
| **estar** | 位置 | El libro **está** en la mesa |
| **estar** | 进行体 | Juan **está** comiendo |

**注**：ser = 固有 / 永久；estar = 临时 / 状态。

### 进行体

| 结构 | 例句 |
|---|---|
| estar + 动名词 | El perro **está ladrando** |
| ir + 动名词（强调进行） | **Va lloviendo** más fuerte |
| venir + 动名词 | **Viene diciendo** eso desde hace años |

### 无系词谓语

- **动词性**：*Llueve*（非人称）
- **存在性**：*Hay libros*（存在）
- **处所性**：*Aquí está Juan*（con estar）

### 反身结果性

- Se **puso** furioso（状态变化）
- Se **hizo** rico（意愿）
- Se **volvió** loco（过程）
- Se **quedó** dormido（结果）

## 英语

### be 作为系词

| 谓语类型 | 例句 |
|---|---|
| 形容词性 | John **is** tall |
| 名词性 | John **is** a teacher |
| 处所性 | The book **is** on the table |
| 进行体 | John **is** eating |
| 被动 | The book **was** written |

**注**：英语所有非动词性谓语都需要be。

### 进行体

| 结构 | 例句 |
|---|---|
| be + -ing | The dog **is barking** |
| get + -ing | **It's getting** dark |

### 伪系词

| 动词 | 用法 | 例句 |
|---|---|---|
| become | 状态变化 | He **became** angry |
| seem | 外表 | He **seems** happy |
| get | 变化（口语） | He **got** tired |
| turn | 突然变化 | He **turned** pale |
| grow |  gradual变化 | He **grew** old |
| remain | 持续 | He **remained** silent |

### 使役make

| 结构 | 例句 |
|---|---|
| make + 宾语 + 形容词 | She **made** him **happy** |
| make + 宾语 + 动词原形 | She **made** him **laugh** |

**注**：make后不用to-不定式。

### 虚主语it

- **It** rains（非人称）
- **It** seems that he is happy（提升）

**注**：英语需要语法主语；使用it作为占位符。

## 汉语

### 裸谓语（无系词）

| 类型 | 例句 | 说明 |
|---|---|---|
| 形容词性 | <ruby>他<rt>tā</rt></ruby><ruby>很<rt>hěn</rt></ruby><ruby>高<rt>gāo</rt></ruby> | 不用 是；很 = 程度 |
| 动词性 | <ruby>他<rt>tā</rt></ruby><ruby>吃<rt>chī</rt></ruby><ruby>饭<rt>fàn</rt></ruby> | 无系词 |
| 处所性 | <ruby>他<rt>tā</rt></ruby><ruby>在<rt>zài</rt></ruby><ruby>家<rt>jiā</rt></ruby> | 在 = 介词 |

### 是 用于名词性

| 结构 | 例句 |
|---|---|
| 主语 + 是 + 名词 | <ruby>他<rt>tā</rt></ruby><ruby>是<rt>shì</rt></ruby><ruby>老<rt>lǎo</rt></ruby><ruby>师<rt>shī</rt></ruby>（他是老师） |
| 否定：不是 | <ruby>他<rt>tā</rt></ruby><ruby>不<rt>bù</rt></ruby><ruby>是<rt>shì</rt></ruby><ruby>学<rt>xué</rt></ruby><ruby>生<rt>shēng</rt></ruby> |

**注**：是 只用于名词性谓语，从不用于形容词性。

### 在 表示进行体

| 结构 | 例句 |
|---|---|
| 主语 + 在 + 动词 | <ruby>狗<rt>gǒu</rt></ruby><ruby>在<rt>zài</rt></ruby><ruby>叫<rt>jiào</rt></ruby>（狗正在叫） |
| 主语 + 正在 + 动词 | <ruby>他<rt>tā</rt></ruby><ruby>正<rt>zhèng</rt></ruby><ruby>在<rt>zài</rt></ruby><ruby>吃<rt>chī</rt></ruby><ruby>饭<rt>fàn</rt></ruby>（他正在吃饭） |

### 了 表示结果性

| 结构 | 例句 |
|---|---|
| 动词 + 了 | <ruby>他<rt>tā</rt></ruby><ruby>生<rt>shēng</rt></ruby><ruby>气<rt>qì</rt></ruby><ruby>了<rt>le</rt></ruby>（他生气了） |
| 动词 + 了 + 宾语 | <ruby>他<rt>tā</rt></ruby><ruby>吃<rt>chī</rt></ruby><ruby>了<rt>le</rt></ruby><ruby>饭<rt>fàn</rt></ruby>（他吃饭了） |

**注**：了 表示状态变化或完成。

### 提升动词

| 动词 | 例句 |
|---|---|
| 好像 | <ruby>他<rt>tā</rt></ruby><ruby>好<rt>hǎo</rt></ruby><ruby>像<rt>xiàng</rt></ruby><ruby>很<rt>hěn</rt></ruby><ruby>高<rt>gāo</rt></ruby><ruby>兴<rt>xìng</rt></ruby>（他似乎很高兴） |
| 觉得 | <ruby>我<rt>wǒ</rt></ruby><ruby>觉<rt>jué</rt></ruby><ruby>得<rt>de</rt></ruby><ruby>很<rt>hěn</rt></ruby><ruby>好<rt>hǎo</rt></ruby>（我觉得很好） |

### 使役让

| 结构 | 例句 |
|---|---|
| 让 + 宾语 + 动词 / 形容词 | <ruby>他<rt>tā</rt></ruby><ruby>让<rt>ràng</rt></ruby><ruby>我<rt>wǒ</rt></ruby><ruby>很<rt>hěn</rt></ruby><ruby>生<rt>shēng</rt></ruby><ruby>气<rt>qì</rt></ruby>（他让我很生气） |

## 快速比较

| 谓语类型 | 西班牙语 | 英语 | 汉语 |
|---|---|---|---|
| 动词性 | 裸 | 裸 | 裸 |
| 形容词性 | ser/estar + 形容词 | be + 形容词 | 裸形容词（+ 很） |
| 名词性 | ser + 名词 | be + 名词 | 是 + 名词 |
| 进行体 | estar + 动名词 | be + -ing | 在 + 动词 |
| 处所性 | estar + 介词短语 | be + 介词短语 | 在 + 处所 |
| 结果性 | 反身（ponerse） | become/get + 形容词 | 动词 + 了 |
| 使役 | hacer + 宾语 + 形容词 | make + 宾语 + 形容词 | 让 + 宾语 + 形容词 |
| 非人称 | 裸 / hay | it + 动词 | 裸动宾 |

## 语境中的例子

- **ZH**: <ruby>约<rt>Yuē</rt></ruby><ruby>翰<rt>hàn</rt></ruby><ruby>很<rt>hěn</rt></ruby><ruby>高<rt>gāo</rt></ruby>（无系词）

## 常见错误

1. **西班牙语母语者**：汉语形容词使用 是 → <ruby>他<rt>tā</rt></ruby><ruby>是<rt>shì</rt></ruby><ruby>高<rt>gāo</rt></ruby> 错误

2. **西班牙语母语者**：英语省略系词 → "John tall" 而不是 "John is tall"

3. **英语母语者**：西班牙语进行体使用ser → "es ladrando" 而不是 "está ladrando"

4. **西班牙语母语者**：make后使用to-不定式 → "made me to be happy" 而不是 "made me happy"
