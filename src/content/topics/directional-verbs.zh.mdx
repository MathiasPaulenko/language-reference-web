## 概述

趋向动词表示朝向或离开参照点（通常是说话者）的移动。

- **西班牙语**：使用 *ir*（去）、*venir*（来）、*llevar*（带）、*traer*（拿），加介词（*a, hacia, desde*）。方向由动词选择和介词决定。
- **英语**：使用 *come/go* 和 *bring/take*，加短语动词（*come in, go out, get up, sit down*）。趋向小品词（*in, out, up, down, on, off*）与动词结合。
- **中文**：有独特的**趋向补语**系统：主要动词与趋向动词（*<ruby>来<rt>lái</rt></ruby>/<ruby>去<rt>qù</rt></ruby>/<ruby>上<rt>shàng</rt></ruby>/<ruby>下<rt>xià</rt></ruby>/<ruby>出<rt>chū</rt></ruby>/<ruby>进<rt>jìn</rt></ruby>/<ruby>回<rt>huí</rt></ruby>/<ruby>过<rt>guò</rt></ruby>*）结合，同时表示动作和方向。

## 西班牙语

### Ir vs venir

| | 方向 | 例子 |
|---|---|---|
| ir | 离说话者 | *Voy a tu casa.* (我去你家。) |
| venir | 向说话者 | *Ven a mi casa.* (你来我家。) |

**规则**：参照点通常是说话者。叙事中，可以是叙事中心。

### Llevar vs traer

| | 方向 | 例子 |
|---|---|---|
| llevar | 离说话者 | *Lleva el libro a la escuela.* (把书带到学校。) |
| traer | 向说话者 | *Trae el libro a casa.* (把书拿到家。) |

### 基本趋向动词

| 西班牙语 | 含义 | 例子 |
|---|---|---|
| entrar | 进入 | *Entra, por favor.* |
| salir | 出去/离开 | *Sal de aquí.* |
| subir | 上去/上车 | *Sube las escaleras.* |
| bajar | 下去/下车 | *Baja del autobús.* |
| acercarse | 接近 | *Acércate un poco.* |
| alejarse | 远离 | *Aléjate de mí.* |
| pasar | 经过/进来 | *Pasa, pasa.* |
| volver | 回来 | *Vuelve pronto.* |
| regresar | 返回 | *Regresa mañana.* |

### 方向介词

| 介词 | 含义 | 例子 |
|---|---|---|
| a | 向（方向）| *Voy a Madrid.* |
| hacia | 朝向 | *Camina hacia mí.* |
| desde | 从 | *Vengo desde Madrid.* |
| hasta | 直到/远至 | *Hasta luego.* |
| por | 通过/经 | *Pasa por aquí.* |
| de | 从 | *Sal de casa.* |

### 方向命令

- *Ven **aquí**.* (过来。)
- *Vete **de aquí**.* (离开这里。)
- *Pasa **adentro**.* (进来。)
- *Sal **fuera**.* (出去。)

### 反身趋向动词

- *Acercarse*（接近）：*Me acerqué.* (我靠近了。)
- *Alejarse*（远离）：*Se alejó.* (他走远了。)
- *Subirse*（上车）：*Me subí al tren.* (我上了火车。)
- *Bajarse*（下车）：*Me bajé del autobús.* (我下了公共汽车。)

## 英语

### Come vs go

| | 方向 | 例子 |
|---|---|---|
| come | 向说话者/中心 | *Come here.* |
| go | 离说话者/中心 | *Go there.* |

**重要**：方向相对于说话者或指示中心：

- *I'm at home. Please **come** to my house.* (向我)
- *I'm at your house. I need to **go** home now.* (离开这里)

### Bring vs take

| | 方向 | 例子 |
|---|---|---|
| bring | 向说话者/中心 | *Bring it here.* |
| take | 离说话者/中心 | *Take it there.* |

**例子**：

- *I'll **bring** snacks to the party.* (向聚会——注意中心)
- *Please **take** this to the kitchen.* (离开这里)

### 方向短语动词

英语大量使用**短语动词**表示方向：

| 动词+小品词 | 含义 | 例子 |
|---|---|---|
| come in | 进来（向说话者）| *Come in!* |
| go out | 出去（离开）| *Let's go out.* |
| come up | 上来（向）| *Come up here.* |
| go down | 下去（离开）| *Go down the stairs.* |
| get on | 上车 | *Get on the bus.* |
| get off | 下车 | *Get off at the next stop.* |
| come back | 回来（向）| *Come back soon.* |
| go away | 离开 | *Go away!* |
| come over | 来访 | *Come over tonight.* |
| go back | 回去（离开）| *Go back to work.* |

### 小品词及其含义

| 小品词 | 核心方向 |
|---|---|
| up | 向上/完成 | *get up, eat up* |
| down | 向下/记录 | *sit down, write down* |
| in | 向内/包含 | *come in, hand in* |
| out | 向外/排除 | *go out, find out* |
| on | 继续/附着 | *carry on, put on* |
| off | 停止/脱离 | *take off, cut off* |
| over | 横跨/访问 | *come over, think over* |
| back | 返回 | *give back, call back* |
| away | 远离 | *go away, take away* |

### 运动介词

| 介词 | 例子 |
|---|---|
| to | *Go to school.* |
| into | *Go into the room.* |
| out of | *Get out of here.* |
| through | *Go through the tunnel.* |
| across | *Walk across the street.* |
| along | *Walk along the beach.* |
| up | *Climb up the mountain.* |
| down | *Walk down the hill.* |
| toward(s) | *Walk toward me.* |
| away from | *Run away from danger.* |

## 中文

### 趋向补语：核心系统

中文将**主要动词**与**趋向补语**结合，同时表示动作和方向：

**简单趋向补语**（<ruby>来<rt>lái</rt></ruby>/<ruby>去<rt>qù</rt></ruby>）：

| 主要动词 | + <ruby>来<rt>lái</rt></ruby>（向）| + <ruby>去<rt>qù</rt></ruby>（离）|
|---|---|---|
| <ruby>走<rt>zǒu</rt></ruby>（走）| <ruby>走<rt>zǒu</rt></ruby><ruby>来<rt>lái</rt></ruby>（走来）| <ruby>走<rt>zǒu</rt></ruby><ruby>去<rt>qù</rt></ruby>（走去）|
| <ruby>跑<rt>pǎo</rt></ruby>（跑）| <ruby>跑<rt>pǎo</rt></ruby><ruby>来<rt>lái</rt></ruby>（跑来）| <ruby>跑<rt>pǎo</rt></ruby><ruby>去<rt>qù</rt></ruby>（跑去）|
| <ruby>拿<rt>ná</rt></ruby>（拿）| <ruby>拿<rt>ná</rt></ruby><ruby>来<rt>lái</rt></ruby>（拿来）| <ruby>拿<rt>ná</rt></ruby><ruby>去<rt>qù</rt></ruby>（拿去）|
| <ruby>送<rt>sòng</rt></ruby>（送）| <ruby>送<rt>sòng</rt></ruby><ruby>来<rt>lái</rt></ruby>（送来）| <ruby>送<rt>sòng</rt></ruby><ruby>去<rt>qù</rt></ruby>（送去）|

**复合趋向补语**（路径+方向）：

| 路径 | + <ruby>来<rt>lái</rt></ruby>（向）| + <ruby>去<rt>qù</rt></ruby>（离）|
|---|---|---|
| <ruby>上<rt>shàng</rt></ruby>（上）| <ruby>上<rt>shàng</rt></ruby><ruby>来<rt>lái</rt></ruby>（上来）| <ruby>上<rt>shàng</rt></ruby><ruby>去<rt>qù</rt></ruby>（上去）|
| <ruby>下<rt>xià</rt></ruby>（下）| <ruby>下<rt>xià</rt></ruby><ruby>来<rt>lái</rt></ruby>（下来）| <ruby>下<rt>xià</rt></ruby><ruby>去<rt>qù</rt></ruby>（下去）|
| <ruby>进<rt>jìn</rt></ruby>（进）| <ruby>进<rt>jìn</rt></ruby><ruby>来<rt>lái</rt></ruby>（进来）| <ruby>进<rt>jìn</rt></ruby><ruby>去<rt>qù</rt></ruby>（进去）|
| <ruby>出<rt>chū</rt></ruby>（出）| <ruby>出<rt>chū</rt></ruby><ruby>来<rt>lái</rt></ruby>（出来）| <ruby>出<rt>chū</rt></ruby><ruby>去<rt>qù</rt></ruby>（出去）|
| <ruby>回<rt>huí</rt></ruby>（回）| <ruby>回<rt>huí</rt></ruby><ruby>来<rt>lái</rt></ruby>（回来）| <ruby>回<rt>huí</rt></ruby><ruby>去<rt>qù</rt></ruby>（回去）|
| <ruby>过<rt>guò</rt></ruby>（过）| <ruby>过<rt>guò</rt></ruby><ruby>来<rt>lái</rt></ruby>（过来）| <ruby>过<rt>guò</rt></ruby><ruby>去<rt>qù</rt></ruby>（过去）|
| <ruby>起<rt>qǐ</rt></ruby>（起）| <ruby>起<rt>qǐ</rt></ruby><ruby>来<rt>lái</rt></ruby>（起来）| <ruby>起<rt>qǐ</rt></ruby><ruby>去<rt>qù</rt></ruby>（罕见）|

### 方向以说话者为参照

- <ruby>上<rt>shàng</rt></ruby><ruby>来<rt>lái</rt></ruby> = 上来（向说话者）
- <ruby>上<rt>shàng</rt></ruby><ruby>去<rt>qù</rt></ruby> = 上去（离说话者）
- <ruby>下<rt>xià</rt></ruby><ruby>来<rt>lái</rt></ruby> = 下来（向说话者）
- <ruby>下<rt>xià</rt></ruby><ruby>去<rt>qù</rt></ruby> = 下去（离说话者）

### 主要动词 + 复合补语

主要动词可在复合补语前：

- <ruby>走<rt>zǒu</rt></ruby><ruby>上<rt>shàng</rt></ruby><ruby>来<rt>lái</rt></ruby>（走上来）
- <ruby>跑<rt>pǎo</rt></ruby><ruby>出<rt>chū</rt></ruby><ruby>去<rt>qù</rt></ruby>（跑出去）
- <ruby>拿<rt>ná</rt></ruby><ruby>进<rt>jìn</rt></ruby><ruby>来<rt>lái</rt></ruby>（拿进来）
- <ruby>扔<rt>rēng</rt></ruby><ruby>出<rt>chū</rt></ruby><ruby>去<rt>qù</rt></ruby>（扔出去）

### <ruby>起<rt>qǐ</rt></ruby><ruby>来<rt>lái</rt></ruby> 作起始态

<ruby>起<rt>qǐ</rt></ruby><ruby>来<rt>lái</rt></ruby> 可表示动作开始：

- <ruby>唱<rt>chàng</rt></ruby><ruby>起<rt>qǐ</rt></ruby><ruby>来<rt>lái</rt></ruby>（唱起来 = 开始唱）
- <ruby>跳<rt>tiào</rt></ruby><ruby>起<rt>qǐ</rt></ruby><ruby>来<rt>lái</rt></ruby>（跳起来 = 跳起来/开始跳）
- <ruby>想<rt>xiǎng</rt></ruby><ruby>起<rt>qǐ</rt></ruby><ruby>来<rt>lái</rt></ruby>（想起来 = 记起）

### 趋向补语与宾语

有宾语时，宾语放在主要动词与补语之间或补语之后：

- <ruby>拿<rt>ná</rt></ruby><ruby>书<rt>shū</rt></ruby><ruby>出<rt>chū</rt></ruby><ruby>来<rt>lái</rt></ruby>（把书拿出来）

## 常见错误

1. **学中文的英语者**：省略趋向补语：<ruby>进<rt>jìn</rt></ruby> → <ruby>进<rt>jìn</rt></ruby><ruby>来<rt>lái</rt></ruby>（表示"进来"）

2. **学英语的汉语者**：逐字翻译：*Go up come* → *Come up*

3. **学西班牙语的英语者**：ir/venir 用错：在派对上说 *Voy a tu fiesta* → 应该说 *Vienes a mi fiesta*

4. **学英语的西班牙语者**：bring/take 混淆：在派对上说 *Take the wine to the party* → 应该说 *Bring the wine*
