## Resumen

Las prohibiciones expresan órdenes de no realizar una acción. Cada lengua tiene sus propias estrategias gramaticales para formar estas construcciones.

- **Español**: Usa **no + subjuntivo** para mandatos negativos. Las señales públicas usan formas impersonalizadas (**prohibido fumar**).
- **Inglés**: Usa **don't + imperativo** para casi todas las prohibiciones. Sin distinción de persona.
- **Chino**: Usa **别** para prohibiciones cotidianas, **不要** para las más enfáticas, **不许/禁止** para prohibiciones oficiales.

## Español

### Mandatos negativos

En español, los mandatos negativos usan el **presente de subjuntivo**:

| Infinitivo | Imperativo afirmativo | Mandato negativo (subjuntivo) |
|---|---|---|
| ir | ¡Ve! | ¡No **vayas**! |
| venir | ¡Ven! | ¡No **vengas**! |
| hablar | ¡Habla! | ¡No **hables**! |
| comer | ¡Come! | ¡No **comas**! |
| tener | ¡Ten! | ¡No **tengas**! |
| hacer | ¡Haz! | ¡No **hagas**! |

### Formas impersonales

Para señales públicas y prohibiciones generales:

| Expresión | Uso | Ejemplo |
|---|---|---|
| **Prohibido** + infinitivo | Señales formales | Prohibido **fumar** |
| **No** + infinitivo | Señales informales | No **tocar** |
| **Se prohíbe** + infinitivo | Prohibición oficial | Se prohíbe **el paso** |
| **No se permite** + infinitivo | Reglamentos | No se permite **el acceso** |

### Suavizado

Para hacer una prohibición menos directa:

| Expresión | Ejemplo |
|---|---|
| Será mejor que no... | Será mejor que no llegues tarde |
| Más te vale no... | Más te vale no tocar eso |
| Te aconsejo que no... | Te aconsejo que no salgas |
| Ojalá no... | Ojalá no llueva |

### Primera persona del plural (nosotros)

| Afirmativo | Negativo |
|---|---|
| ¡Vamos! | ¡No vayamos! |
| ¡Comamos! | ¡No comamos! |
| ¡Hagámoslo! | ¡No lo hagamos! |

**Nota**: El imperativo negativo de nosotros coincide con el presente de subjuntivo.

## Inglés

### Don't + imperativo

El inglés usa **don't + verbo base** para todas las prohibiciones:

| Persona | Forma | Ejemplo |
|---|---|---|
| 2ª persona | Don't + base | **Don't** go! |
| 1ª persona plural | Let's not + base | **Let's not** go! |
| 3ª persona | Don't let + sujeto + base | **Don't let** him go! |

**Nota**: No hay distinción de persona en las prohibiciones del inglés; todas usan don't.

### Señales públicas

| Español | Inglés |
|---|---|
| Prohibido fumar | **No smoking** |
| No tocar | **Do not touch** |
| Se prohíbe el paso | **No entry** |
| No se permite la entrada | **No admittance** |

**Nota**: Las señales en inglés usan gerundios o frases nominales, no oraciones completas.

### Suavizado

| Directo | Suavizado |
|---|---|
| Don't go there | **You might not want to** go there |
| Don't touch that | **I'd avoid** touching that if I were you |
| Don't be late | **Try not to** be late |

### Never + imperativo

Para prohibiciones más fuertes:

- **Never** go there alone.
- **Never** touch that button.
- **Never** be late again.

## Chino

### 别 (prohibición casual)

**别** es la forma más común y natural para prohibiciones cotidianas:

| Expresión | Significado |
|---|---|
| <ruby>别<rt>bié</rt></ruby><ruby>走<rt>zǒu</rt></ruby> | ¡No te vayas! |
| <ruby>别<rt>bié</rt></ruby><ruby>碰<rt>pèng</rt></ruby> | ¡No toques! |
| <ruby>别<rt>bié</rt></ruby><ruby>迟<rt>chí</rt></ruby><ruby>到<rt>dào</rt></ruby> | ¡No llegues tarde! |
| <ruby>别<rt>bié</rt></ruby><ruby>担<rt>dān</rt></ruby><ruby>心<rt>xīn</rt></ruby> | No te preocupes |

### 不要 (prohibición enfática)

**不要** es ligeramente más enfático o formal:

| Expresión | Significado |
|---|---|
| <ruby>不<rt>bù</rt></ruby><ruby>要<rt>yào</rt></ruby><ruby>走<rt>zǒu</rt></ruby> | ¡No te vayas! (más enfático) |
| <ruby>不<rt>bù</rt></ruby><ruby>要<rt>yào</rt></ruby><ruby>碰<rt>pèng</rt></ruby> | ¡No toques! (más enfático) |

### 不许 / 禁止 (prohibición oficial)

Para prohibiciones fuertes o oficiales:

| Expresión | Significado |
|---|---|
| <ruby>不<rt>bù</rt></ruby><ruby>许<rt>xǔ</rt></ruby><ruby>抽<rt>chōu</rt></ruby><ruby>烟<rt>yān</rt></ruby> | Prohibido fumar |

Para añadir urgencia o advertencia:

| Expresión | Significado |
|---|---|

## Comparación rápida

| Característica | Español | Inglés | Chino |
|---|---|---|---|
| Forma básica | no + subjuntivo | don't + base | 别 + verbo |
| Distinción de persona | Sí (6 personas) | No | No |
| Señales públicas | Prohibido + inf. / No + inf. | No + gerundio | 禁止 + verbo |
| Urgencia | Nunca + subjuntivo | Never + base | 千万别 + verbo |
| Suavizado | Será mejor que no... | You might not want to... | 最好别... |
| Primera persona plural | No vayamos... | Let's not... | 我们别... |

## Errores comunes

1. **Hispanohablantes**: Usar indicativo en mandatos negativos → "*No vas" en lugar de "No **vayas**"

2. **Angloparlantes**: Confundir 别 y 不要 en chino → 别 es más natural en prohibiciones cotidianas

3. **Angloparlantes**: Añadir "do" a prohibitivos de tercera persona en inglés → "*Don't he go" es incorrecto

4. **Chinos**: Traducir prohibiciones impersonales literalmente → "Se prohíbe fumar" = "No smoking", no "It is prohibited to smoke"
