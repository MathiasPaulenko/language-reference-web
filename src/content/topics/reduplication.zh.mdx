## 概述

重叠是重复一个词或词的一部分，以达到语法或表达目的。

- **西班牙语**：重叠极少。偶用于口语强调（*muy muy*）。小称（*-ito*）功能类似重叠，表达亲昵或小。
- **英语**：重叠非常有限。用于儿语（*choo-choo, wee-wee*）、某些复合词（*tick-tock, zig-zag*）、非正式强化（*so-so, goody-goody*）。
- **汉语**：广泛且语法上能产的重叠系统。动词、形容词、量词、名词均可重叠，各有含义。

## 西班牙语

### 口语重叠

| 表达 | 用法 | 例句 |
|---|---|---|
| muy muy | 口语强调 | Está **muy muy** lejos. |
| bien bien | 口语强调 | Lo hizo **bien bien**. |
| poco poco | 一点一点 | **Poco poco** aprende. |

**注**：西班牙语重叠有限且口语化。

### 小称作为重叠

小称表达亲昵或小，功能类似重叠：

| 形式 | 例句 | 含义 |
|---|---|---|
| -ito / -ita | cas**ita** | 小房子（亲昵/小） |
| -ico / -ica | moment**ico** | 一会儿（口语） |

### 强化词

西班牙语用强化词而非重叠：

| 强化词 | 例句 |
|---|---|
| -ísimo | rapidísimo, bellísimo |
| re-（口语） | rebueno, remalo |
| super-（口语） | superbueno |

## 快速比较

| 特征 | 西班牙语 | 英语 | 汉语 |
|---|---|---|---|
| 动词重叠 | 无 | 无 | 有（VV, V一V） |
| 形容词重叠 | 无（用 -ísimo） | 无（用 very） | 有（AA, AABB） |
| 名词重叠 | 无 | 无（例外） | 有（NN） |
| 量词重叠 | 无 | 无 | 有（M M） |
| 主要功能 | 口语/亲昵 | 儿语/复合词 | 语法/能产 |
| 软化功能 | 无 | 无 | 有（动词重叠） |

## 常见错误

1. **西班牙语母语者**：试图像汉语一样重叠 → 西班牙语不按语法重叠

2. **英语母语者**：英语中造 "look-look" → 不存在；用 "take a look"

3. **英语母语者**：汉语中所有形容词都用 AABB → 仅某些形容词允许

4. **汉语母语者**：动词不重叠以软化 → <ruby>看<rt>kàn</rt></ruby><ruby>看<rt>kàn</rt></ruby> 软化；<ruby>看<rt>kàn</rt></ruby> 听起来像命令
