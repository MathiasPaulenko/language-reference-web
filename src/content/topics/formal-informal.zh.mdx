## 概述

语体是说话者根据情境正式程度调整语言的方式。

- **西班牙语**：*Tú/usted*（单数）；*vosotros/ustedes*（复数，按地区）。动词变位不同。
- **英语**：*You* 通用。正式性通过词汇（*purchase* vs *buy*）、结构（间接请求）和语调标记。
- **中文**：*<ruby>你<rt>nǐ</rt></ruby>/<ruby>您<rt>nín</rt></ruby>*。*<ruby>请<rt>qǐng</rt></ruby>* 用于正式请求。*<ruby>麻<rt>má</rt></ruby><ruby>烦<rt>fán</rt></ruby><ruby>您<rt>nín</rt></ruby>* 用于请求帮忙。情态小品词（*吧ba, 吗ma*）。

## 西班牙语

### Tú vs Usted

| 代词 | 用法 | 动词 |
|---|---|---|
| Tú | 朋友、家人、孩子、同事 | 第二人称（hablas, comes） |
| Usted | 陌生人、长辈、权威、客户 | 第三人称（habla, come） |
| Vosotros | Vosotros（西班牙非正式复数） | 第二人称复数（habláis, coméis） |
| Ustedes | Ustedes（拉丁美洲复数/西班牙正式） | 第三人称复数（hablan, comen） |

### 问候

**非正式**：
- ¡Hola!（你好！）
- ¿Qué tal?（怎么样？）
- ¿Cómo estás?（你好吗？）
- ¡Buenas!（你好！）

**正式**：
- Buenos días / tardes（早上好/下午好）
- ¿Cómo está (usted)?（您好吗？）
- Encantado/a de conocerle（很高兴认识您）
- Es un placer（很荣幸）

### 请求

**非正式**：
- Dame el agua.（给我水。）
- Pásame el libro.（把书递给我。）
- ¿Me ayudas?（你能帮我吗？）

**正式**：
- ¿Podría darme el agua?（您能给我水吗？）
- ¿Me haría el favor de pasarme el libro?（您能帮我把书递过来吗？）
- ¿Le importaría ayudarme?（您介意帮我一下吗？）
- ¿Sería tan amable de...?（您能否好心...？）

### 道歉

**非正式**：
- Lo siento.（对不起。）
- Perdona.（抱歉。）
- Fue sin querer.（我不是故意的。）

**正式**：
- Disculpe.（请原谅。）
- Le pido disculpas.（我向您道歉。）
- Lamento mucho...（我非常遗憾...）
- Mis más sinceras disculpas.（我最诚挚的歉意。）

### 告别

**非正式**：
- Adiós.（再见。）
- Hasta luego.（回头见。）
- Nos vemos.（再见。）
- Chao.（拜拜。）

**正式**：
- Hasta luego / Hasta pronto.（回头见/不久见。）
- Ha sido un placer.（很荣幸。）
- Que tenga un buen día.（祝您有美好的一天。）

## 英语

### You通用

英语不在代词中区分正式程度。所有人都是*you*。

**非正式**：You guys / Y'all（地区性）
**正式**：You

### 问候

**非正式**：
- Hi / Hey（嗨/嘿）
- What's up?（怎么了？）
- How's it going?（最近怎么样？）

**正式**：
- Good morning / afternoon（早上好/下午好）
- How do you do?（您好？）
- Pleased to meet you.（很高兴认识你。）
- It's a pleasure.（很荣幸。）

### 请求

**非正式**：
- Give me the water.（给我水。）
- Pass the book.（把书递过来。）
- Can you help me?（你能帮我吗？）

**正式**：
- Could you give me the water?（您能给我水吗？）
- Would you mind passing the book?（您介意把书递过来吗？）
- I was wondering if you could help me.（我想知道您是否能帮我。）
- Would it be possible for you to...?（您是否可能...？）

### 道歉

**非正式**：
- Sorry.（对不起。）
- My bad.（我的错。）
- I didn't mean to.（我不是故意的。）

**正式**：
- I apologize.（我道歉。）
- Please accept my apologies.（请接受我的道歉。）
- I sincerely regret...（我真诚地后悔...）

### 正式与非正式词汇

| 非正式 | 正式 |
|---|---|
| buy | purchase |
| start | commence |
| end | terminate |
| help | assist |
| need | require |
| ask | inquire |
| tell | inform |
| use | utilize |
| get | obtain |
| thing | object / matter |

### 告别

**非正式**：
- Bye / See ya.（再见/回见。）
- Catch you later.（回头见。）
- Take care.（保重。）

**正式**：
- Goodbye.（再见。）
- It was a pleasure meeting you.（很高兴见到您。）
- Have a pleasant day.（祝您有愉快的一天。）

## 中文

| 代词 | 用法 |
|---|---|
| 你nǐ | 朋友、家人、同事、孩子 |

- <ruby>张<rt>zhāng</rt></ruby><ruby>老<rt>lǎo</rt></ruby><ruby>师<rt>shī</rt></ruby>（张老师）

## 常见错误

1. **学英语的西班牙语使用者**：在保守国家与陌生人使用tú

2. **学英语的西班牙语使用者**：认为缺少正式代词：*usted* → *you*（正确，但需调整词汇）

3. **学英语的中文使用者**：使用过于直接的词汇而没有调节语

4. **学中文的英语使用者**：对长辈或权威使用你nǐ

5. **学中文的西班牙语使用者**：混淆您nín与你nǐ
