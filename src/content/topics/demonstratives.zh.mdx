## 概述

指示词指向空间或话语中的事物：这本书、那座房子。

- **西班牙语**：三元区分 — 近说话者（*este*），近听话者（*ese*），远离双方（*aquel*）。完全的性和数一致。
- **英语**：二元区分 — 近（*this/these*），远（*that/those*）。无一致关系。
- **中文**：二元区分 — 近（*<ruby>这<rt>zhè</rt></ruby>*），远（*<ruby>那<rt>nà</rt></ruby>*）。通常需要量词或计量词。用 *<ruby>些<rt>xiē</rt></ruby>* 表示复数。

## 西班牙语

### 三元空间区分

| | 近说话者 | 近听话者 | 远离双方 |
|---|---|---|---|
| 阳性单数 | **este** libro | **ese** libro | **aquel** libro |
| 阴性单数 | **esta** mesa | **esa** mesa | **aquella** mesa |
| 阳性复数 | **estos** libros | **esos** libros | **aquellos** libros |
| 阴性复数 | **estas** mesas | **esas** mesas | **aquellas** mesas |

**记忆法**：*Este* = **e**stoy aquí（我在这里）。*Ese* = **e**stás ahí（你在那里）。*Aquel* = **a**llí（在那里）。

### 一致关系

指示词与名词保持性和数的一致：

- *me gusta **este** coche*（我喜欢这辆车 — 阳单）
- *prefiero **esta** casa*（我更喜欢这所房子 — 阴单）
- *quiero **estos** zapatos*（我想要这双鞋 — 阳复）
- *compro **estas** flores*（我买这些花 — 阴复）

### 中性形式（独立使用）

指抽象概念或未命名的事物时，使用中性形式：

- ***Esto** es importante.*（这很重要。）
- ***Eso** no me gusta.*（我不喜欢那个。）
- ***Aquello** fue increíble.*（那件事太不可思议了。）

**注意**：中性形式从不修饰名词。它们独立使用。

### 指示代词

西班牙语区分形容词形式和代词形式：

- *形容词*：***Este** libro es mío.*（这本书是我的。）
- *代词*：***Este** es mío.*（这个是我的。）

历史上，代词带重音符号（*éste, ése, aquél*），但西班牙皇家语言学院已不再要求。

## 英语

### 二元区分

| | 单数 | 复数 |
|---|---|---|
| 近说话者 | **this** | **these** |
| 远说话者 | **that** | **those** |

### 无一致关系

英语指示词是不变的：

- *this book / these books*
- *that car / those cars*

### 作为形容词和代词

英语使用相同形式作为形容词和代词：

- *形容词：**This** car is fast.*（这辆车很快。）
- *代词：**This** is fast.*（这个很快。）

### That 作为关系代词

英语使用 *that* 作为关系代词，西班牙语和中文则不这样使用：

- *The book **that** I read.*（我读的那本书。）

### This/that 在话语中

指示词可以指向话语中的前文或后文：

- *I didn't understand **that**.*（指前文陈述）
- ***This** is what I mean...*（引入即将出现的解释）

## 中文

### 带量词的二元区分

| | 单数 | 复数 |
|---|---|---|
| 近 | <ruby>这<rt>zhè</rt></ruby> + 量词 | <ruby>这<rt>zhè</rt></ruby><ruby>些<rt>xiē</rt></ruby> |
| 远 | <ruby>那<rt>nà</rt></ruby> + 量词 | <ruby>那<rt>nà</rt></ruby><ruby>些<rt>xiē</rt></ruby> |

例子：

- <ruby>这<rt>zhè</rt></ruby><ruby>本<rt>běn</rt></ruby><ruby>书<rt>shū</rt></ruby>（这本[量]书）
- <ruby>那<rt>nà</rt></ruby><ruby>辆<rt>liàng</rt></ruby><ruby>车<rt>chē</rt></ruby>（那辆[量]车）
- <ruby>这<rt>zhè</rt></ruby><ruby>些<rt>xiē</rt></ruby><ruby>人<rt>rén</rt></ruby>（这些人）
- <ruby>那<rt>nà</rt></ruby><ruby>些<rt>xiē</rt></ruby><ruby>猫<rt>māo</rt></ruby>（那些猫）

### 无量词

对于泛指或抽象指称，<ruby>这<rt>zhè</rt></ruby>/<ruby>那<rt>nà</rt></ruby> 可以独立使用：

- <ruby>这<rt>zhè</rt></ruby><ruby>很<rt>hěn</rt></ruby><ruby>好<rt>hǎo</rt></ruby>。(这很好。)
- <ruby>那<rt>nà</rt></ruby><ruby>怎<rt>zěn</rt></ruby><ruby>么<rt>me</rt></ruby><ruby>办<rt>bàn</rt></ruby>？(那怎么办？)

### <ruby>这<rt>zhè</rt></ruby><ruby>儿<rt>ér</rt></ruby> / <ruby>那<rt>nà</rt></ruby><ruby>里<rt>lǐ</rt></ruby> 表地点

中文使用 <ruby>这<rt>zhè</rt></ruby><ruby>儿<rt>ér</rt></ruby>/<ruby>这<rt>zhè</rt></ruby><ruby>里<rt>lǐ</rt></ruby>（这里）和 <ruby>那<rt>nà</rt></ruby><ruby>儿<rt>ér</rt></ruby>/<ruby>那<rt>nà</rt></ruby><ruby>里<rt>lǐ</rt></ruby>（那里）：

- <ruby>这<rt>zhè</rt></ruby><ruby>儿<rt>ér</rt></ruby> / <ruby>这<rt>zhè</rt></ruby><ruby>里<rt>lǐ</rt></ruby>（这里）
- <ruby>那<rt>nà</rt></ruby><ruby>儿<rt>ér</rt></ruby> / <ruby>那<rt>nà</rt></ruby><ruby>里<rt>lǐ</rt></ruby>（那里）

- <ruby>我<rt>wǒ</rt></ruby><ruby>在<rt>zài</rt></ruby><ruby>这<rt>zhè</rt></ruby><ruby>儿<rt>ér</rt></ruby>。(我在这里。)
- <ruby>他<rt>tā</rt></ruby><ruby>在<rt>zài</rt></ruby><ruby>那<rt>nà</rt></ruby><ruby>里<rt>lǐ</rt></ruby>。(他在那里。)

### <ruby>这<rt>zhè</rt></ruby>/<ruby>那<rt>nà</rt></ruby> 作为话语标记

与英语一样，<ruby>这<rt>zhè</rt></ruby>/<ruby>那<rt>nà</rt></ruby> 可以指话语：

- <ruby>这<rt>zhè</rt></ruby><ruby>就<rt>jiù</rt></ruby><ruby>是<rt>shì</rt></ruby><ruby>我<rt>wǒ</rt></ruby><ruby>想<rt>xiǎng</rt></ruby><ruby>说<rt>shuō</rt></ruby><ruby>的<rt>de</rt></ruby>。(这就是我想说的。)
- <ruby>那<rt>nà</rt></ruby><ruby>怎<rt>zěn</rt></ruby><ruby>么<rt>me</rt></ruby><ruby>办<rt>bàn</rt></ruby>？(那怎么办？)

## 常见错误

1. **学英语的西班牙语者**：只使用 este/ese：用 ese 表示 "that" 而不考虑距离 → 远处用 ***aquel***

2. **学英语的西班牙语者**：忘记一致关系：*este mesa* → *esta mesa*

3. **学英语的中文学习者**：省略量词：*<ruby>这<rt>zhè</rt></ruby><ruby>书<rt>shū</rt></ruby>* → *<ruby>这<rt>zhè</rt></ruby><ruby>本<rt>běn</rt></ruby><ruby>书<rt>shū</rt></ruby>*

4. **学西班牙语的中文学习者**：用 "this" 同时表示 este 和 ese，当指向听话者附近的东西时 → 用 **that**
