## 概述

禁止表达不做某事的命令。每种语言有自己的语法策略来构成这些结构。

- **西班牙语**：否定命令用 **no + 虚拟式**。公共标识用无人称形式（**prohibido fumar**）。
- **英语**：几乎所有禁止用 **don't + 命令式**。无人称区别。
- **汉语**：日常禁止用 **别**，更强调的用 **不要**，官方禁止用 **不许/禁止**。

## 西班牙语

### 否定命令

西班牙语否定命令使用 **现在时虚拟式**：

| 不定式 | 肯定命令 | 否定命令（虚拟式） |
|---|---|---|
| ir | ¡Ve! | ¡No **vayas**! |
| venir | ¡Ven! | ¡No **vengas**! |
| hablar | ¡Habla! | ¡No **hables**! |
| comer | ¡Come! | ¡No **comas**! |
| tener | ¡Ten! | ¡No **tengas**! |
| hacer | ¡Haz! | ¡No **hagas**! |

### 无人称形式

公共标识和一般禁止：

| 表达 | 用法 | 例句 |
|---|---|---|
| **Prohibido** + 不定式 | 正式标识 | Prohibido **fumar** |
| **No** + 不定式 | 非正式标识 | No **tocar** |
| **Se prohíbe** + 不定式 | 官方禁止 | Se prohíbe **el paso** |
| **No se permite** + 不定式 | 规章制度 | No se permite **el acceso** |

### 软化

使禁止不那么直接：

| 表达 | 例句 |
|---|---|
| Será mejor que no... | Será mejor que no llegues tarde |
| Más te vale no... | Más te vale no tocar eso |
| Te aconsejo que no... | Te aconsejo que no salgas |
| Ojalá no... | Ojalá no llueva |

### 第一人称复数（nosotros）

| 肯定 | 否定 |
|---|---|
| ¡Vamos! | ¡No vayamos! |
| ¡Comamos! | ¡No comamos! |
| ¡Hagámoslo! | ¡No lo hagamos! |

**注**：nosotros 否定命令与现在时虚拟式一致。

## 英语

### Don't + 命令式

英语用 **don't + 动词原形** 表示所有禁止：

| 人称 | 形式 | 例句 |
|---|---|---|
| 第二人称 | Don't + 原形 | **Don't** go! |
| 第一人称复数 | Let's not + 原形 | **Let's not** go! |
| 第三人称 | Don't let + 主语 + 原形 | **Don't let** him go! |

**注**：英语禁止无人称区别；都用 don't。

### 公共标识

| 西班牙语 | 英语 |
|---|---|
| Prohibido fumar | **No smoking** |
| No tocar | **Do not touch** |
| Se prohíbe el paso | **No entry** |
| No se permite la entrada | **No admittance** |

**注**：英语标识用动名词或名词短语，不用完整句子。

### 软化

| 直接 | 软化 |
|---|---|
| Don't go there | **You might not want to** go there |
| Don't touch that | **I'd avoid** touching that if I were you |
| Don't be late | **Try not to** be late |

### Never + 命令式

更强烈的禁止：

- **Never** go there alone.
- **Never** touch that button.
- **Never** be late again.

## 汉语

### 别（日常禁止）

**别** 是最常见、最自然的日常禁止形式：

| 表达 | 意义 |
|---|---|
| <ruby>别<rt>bié</rt></ruby><ruby>走<rt>zǒu</rt></ruby> | 别走！ |
| <ruby>别<rt>bié</rt></ruby><ruby>碰<rt>pèng</rt></ruby> | 别碰！ |
| <ruby>别<rt>bié</rt></ruby><ruby>迟<rt>chí</rt></ruby><ruby>到<rt>dào</rt></ruby> | 别迟到！ |
| <ruby>别<rt>bié</rt></ruby><ruby>担<rt>dān</rt></ruby><ruby>心<rt>xīn</rt></ruby> | 别担心 |

### 不要（强调禁止）

**不要** 稍显强调或正式：

| 表达 | 意义 |
|---|---|
| <ruby>不<rt>bù</rt></ruby><ruby>要<rt>yào</rt></ruby><ruby>走<rt>zǒu</rt></ruby> | 不要走！（更强调） |
| <ruby>不<rt>bù</rt></ruby><ruby>要<rt>yào</rt></ruby><ruby>碰<rt>pèng</rt></ruby> | 不要碰！（更强调） |

### 不许 / 禁止（官方禁止）

强烈或官方禁止：

| 表达 | 意义 |
|---|---|
| <ruby>不<rt>bù</rt></ruby><ruby>许<rt>xǔ</rt></ruby><ruby>抽<rt>chōu</rt></ruby><ruby>烟<rt>yān</rt></ruby> | 不许抽烟 |
| <ruby>禁<rt>jìn</rt></ruby><ruby>止<rt>zhǐ</rt></ruby><ruby>吸<rt>xī</rt></ruby><ruby>烟<rt>yān</rt></ruby> | 禁止吸烟 |
| <ruby>禁<rt>jìn</rt></ruby><ruby>止<rt>zhǐ</rt></ruby><ruby>通<rt>tōng</rt></ruby><ruby>行<rt>xíng</rt></ruby> | 禁止通行 |

### 千万别（紧急）

增加紧急感或警告：

| 表达 | 意义 |
|---|---|
| <ruby>千<rt>qiān</rt></ruby><ruby>万<rt>wàn</rt></ruby><ruby>别<rt>bié</rt></ruby><ruby>去<rt>qù</rt></ruby> | 千万别去！ |
| <ruby>千<rt>qiān</rt></ruby><ruby>万<rt>wàn</rt></ruby><ruby>别<rt>bié</rt></ruby><ruby>碰<rt>pèng</rt></ruby> | 千万别碰！ |

### 软化

| 直接 | 软化 |
|---|---|
| <ruby>别<rt>bié</rt></ruby><ruby>去<rt>qù</rt></ruby> | <ruby>最<rt>zuì</rt></ruby><ruby>好<rt>hǎo</rt></ruby><ruby>别<rt>bié</rt></ruby><ruby>去<rt>qù</rt></ruby>（最好别去） |
| <ruby>别<rt>bié</rt></ruby><ruby>碰<rt>pèng</rt></ruby> | <ruby>还<rt>hái</rt></ruby><ruby>是<rt>shì</rt></ruby><ruby>不<rt>bù</rt></ruby><ruby>要<rt>yào</rt></ruby><ruby>碰<rt>pèng</rt></ruby>（还是不要碰） |

## 快速比较

| 特征 | 西班牙语 | 英语 | 汉语 |
|---|---|---|---|
| 基本形式 | no + 虚拟式 | don't + 原形 | 别 + 动词 |
| 人称区别 | 有（6个人称） | 无 | 无 |
| 公共标识 | Prohibido + 不定式 / No + 不定式 | No + 动名词 | 禁止 + 动词 |
| 紧急 | Nunca + 虚拟式 | Never + 原形 | 千万别 + 动词 |
| 软化 | Será mejor que no... | You might not want to... | 最好别... |
| 第一人称复数 | No vayamos... | Let's not... | 我们别... |

## 常见错误

1. **西班牙语母语者**：否定命令用直陈式 → "*No vas" 应为 "No **vayas**"

2. **英语母语者**：汉语口语禁止混淆 别 和 不要 → 别 更自然

3. **英语母语者**：英语第三人称禁止加 "do" → "*Don't he go" 错误

4. **汉语母语者**：西班牙语无人称禁止直译 → "Se prohíbe fumar" = "No smoking"，不是 "It is prohibited to smoke"
