## 概述

假朋友是两种语言中形式相似但意义不同的词汇。

- **西班牙语-英语**：共享约30,000个拉丁同源词，但许多词已改变意义。
- **西班牙语-中文**：主要通过日语借词或现代仿译产生的假朋友。
- **英语-中文**：直接仿译有时因文化差异而失败。

## 语境中的例句

### Actually / Actualmente
- **EN**：Actually, I don't like coffee.（事实上，我不喜欢咖啡。）
- **ES**：Actualmente, vivo en Madrid.（目前，我住在马德里。）
- **ZH**：<ruby>实<rt>shí</rt></ruby><ruby>际<rt>jì</rt></ruby><ruby>上<rt>shàng</rt></ruby>，<ruby>我<rt>wǒ</rt></ruby><ruby>不<rt>bù</rt></ruby><ruby>喜<rt>xǐ</rt></ruby><ruby>欢<rt>huan</rt></ruby><ruby>咖<rt>kā</rt></ruby><ruby>啡<rt>fēi</rt></ruby>。

### Embarrassed / Embarazada
- **EN**：I'm so embarrassed.（我很尴尬。）
- **ES**：Estoy embarazada de cinco meses.（我怀孕五个月了。）
- **ZH**：<ruby>我<rt>wǒ</rt></ruby><ruby>很<rt>hěn</rt></ruby><ruby>尴<rt>gān</rt></ruby><ruby>尬<rt>gà</rt></ruby>。

### Library / Librería
- **EN**：I went to the library to study.（我去图书馆学习。）
- **ES**：Compré el libro en una librería.（我在书店买了这本书。）
- **ZH**：<ruby>我<rt>wǒ</rt></ruby><ruby>去<rt>qù</rt></ruby><ruby>图<rt>tú</rt></ruby><ruby>书<rt>shū</rt></ruby><ruby>馆<rt>guǎn</rt></ruby><ruby>学<rt>xué</rt></ruby><ruby>习<rt>xí</rt></ruby>。

## 常见错误

1. **英语使用者**：*Estoy excitado*（性方面）→ *Estoy emocionado*（热情）

2. **西班牙语使用者**：*I'm pregnant*（怀孕）当想说*I'm embarrassed*（尴尬）

3. **英语使用者**：*Actualmente*（目前）而不是*En realidad*（事实上）

4. **西班牙语使用者**：*I realized the project*（我执行了项目）→ *I **carried out** the project*

5. **所有人**：混淆*sensible*（敏感）与*sensible*（明智）
