## Resumen

Las oraciones de resultado expresan consecuencias: tan cansado que me quedé dormido, demasiado pequeño para alcanzar.

- **Español**: Usa *tan + adjective/adverbio + que* y *tanto + sustantivo + que*. El subjuntivo puede aparecer para resultados hipotéticos. *Demasiado...para* expresa "demasiado...para".
- **Inglés**: Usa *so + adjective/adverbio + that* y *such + (a/an) + adjective + sustantivo + that*. "Too...to" y "...enough to" también son comunes.
- **Chino**: Usa *<ruby>太<rt>tài</rt></ruby> + adjective + <ruby>了<rt>le</rt></ruby>* para "demasiado". El resultado se introduce con *<ruby>以<rt>yǐ</rt></ruby><ruby>至<rt>zhì</rt></ruby><ruby>于<rt>yú</rt></ruby>* o simplemente se yuxtapone. *<ruby>这<rt>zhè</rt></ruby><ruby>么<rt>mó</rt></ruby>/<ruby>那<rt>nà</rt></ruby><ruby>么<rt>mó</rt></ruby>* equivale a "so".

## Español

### Tan...que (adjetivo/adverbio)

| Estructura | Ejemplo | Significado |
|---|---|---|
| tan + adj. + que | Es **tan** alto **que** no entra. | Es tan alto que no entra. |
| tan + adv. + que | Corre **tan** rápido **que** nadie le alcanza. | Corre tan rápido que nadie le alcanza. |
| tanta + sust. fem. + que | Tiene **tanta** paciencia **que**... | Tiene tanta paciencia que... |
| tanto + sust. masc. + que | Tiene **tanto** dinero **que**... | Tiene tanto dinero que... |
| tantos + sust. pl. masc. + que | Tiene **tantos** amigos **que**... | Tiene tantos amigos que... |
| tantas + sust. pl. fem. + que | Tiene **tantas** opciones **que**... | Tiene tantas opciones que... |

### Demasiado...para (too...to)

| Estructura | Ejemplo | Significado |
|---|---|---|
| demasiado + adj. + para + inf. | Es **demasiado** alto **para** pasar. | Es demasiado alto para pasar. |
| demasiado + sust. + para + inf. | Tiene **demasiado** trabajo **para** salir. | Tiene demasiado trabajo para salir. |

### Lo suficientemente...para (...enough to)

| Estructura | Ejemplo | Significado |
|---|---|---|
| lo suficientemente + adj. + para + inf. | Es **lo suficientemente** alto **para** alcanzar. | Es lo suficientemente alto para alcanzar. |
| bastante + adj. + para + inf. | Es **bastante** alto **para** alcanzar. | Es bastante alto para alcanzar. |

### Subjuntivo en resultados hipotéticos

| Ejemplo | Significado |
|---|---|
| Es tan alto que no **quepa**. | Es tan alto que no quepa. |
| Habla tan rápido que no **entienda**. | Habla tan rápido que no entienda. |

**Nota**: El subjuntivo indica que el resultado es hipotético o no real.

## Inglés

### So...that (adjetivo/adverbio)

| Estructura | Ejemplo | Significado |
|---|---|---|
| so + adj. + that | He is **so** tall **that** he hits his head. | Es tan alto que se golpea la cabeza. |
| so + adv. + that | He runs **so** fast **that** no one catches him. | Corre tan rápido que nadie le alcanza. |

### Such...that (sustantivo)

| Estructura | Ejemplo | Significado |
|---|---|---|
| such + a/an + adj. + sust. + that | It is **such** a good book **that** I read it twice. | Es un libro tan bueno que lo leí dos veces. |
| such + adj. + sust. pl. + that | They are **such** good friends **that**... | Son tan buenos amigos que... |
| such + adj. + sust. incontable + that | It was **such** bad weather **that**... | Hacía tan mal tiempo que... |

### Too...to (demasiado...para)

| Estructura | Ejemplo | Significado |
|---|---|---|
| too + adj. + to + inf. | He is **too** tired **to** work. | Está demasiado cansado para trabajar. |
| too + much/little + sust. + to + inf. | There is **too** much work **to** finish. | Hay demasiado trabajo para terminar. |

### ...enough to (lo suficientemente...para)

| Estructura | Ejemplo | Significado |
|---|---|---|
| adj. + enough + to + inf. | He is tall **enough** to reach. | Es lo suficientemente alto para alcanzar. |
| enough + sust. + to + inf. | I have **enough** time **to** finish. | Tengo suficiente tiempo para terminar. |

## Errores comunes

1. **Hispanohablantes**: Very 代替 so → He was **so** tired **that** he fell asleep

2. **Angloparlantes**: Tan...que 后忘记虚拟式 → Es tan alto que no **quepa**（假设用虚拟式）

3. **Angloparlantes**: 汉语中太 不带 了 → **太**...**了** 是固定结构

4. **Chinos**: Such 直译为 这么 → such a good day = <ruby>这<rt>zhè</rt></ruby><ruby>么<rt>me</rt></ruby><ruby>好<rt>hǎo</rt></ruby><ruby>的<rt>de</rt></ruby><ruby>一<rt>yī</rt></ruby><ruby>天<rt>tiān</rt></ruby>
