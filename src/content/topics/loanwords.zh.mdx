## 概述

外来词是从一种语言借入另一种语言的词汇。不同语言对借词的开放程度和适应策略各不相同。

- **西班牙语**：历史上大量借自阿拉伯语（alcalde, azúcar, almohada）、美洲原住民语言（chocolate, tomate, canoa），以及越来越多来自英语（software, marketing, parking）。西班牙皇家语言学院监控并提出西班牙语化替代词。
- **英语**：大量借自法语（1066年后）、拉丁语、希腊语、北欧语，以及几乎所有接触过的语言。常保留原拼写，有时保留发音。
- **汉语**：偏好意译（语义翻译）而非音译。音译词存在，但常经过语义改造或缩小。方言差异显著（粤语vs普通话借词）。

## 西班牙语

### 阿拉伯语借词（西班牙语-阿拉伯语）

超过4,000个阿拉伯语词汇在安达卢斯时期进入西班牙语。许多以 al- 开头：

| 西班牙语 | 阿拉伯语来源 | 含义 |
|---|---|---|
| alcalde | al-qāḍī | 市长 |
| almohada | al-muẖādda | 枕头 |
| azúcar | as-sukkar | 糖 |
| arroz | ar-ruzz | 大米 |
| naranja | nāranj | 橙子 |
| aceite | az-zayt | 油 |
| ojalá | law šā' allāh | 但愿 |
| hasta | ḥatta | 直到 |

### 美洲原住民语言借词

| 西班牙语 | 来源 | 含义 |
|---|---|---|
| chocolate | 纳瓦特尔语 xocolātl | 巧克力 |
| tomate | 纳瓦特尔语 tomatl | 番茄 |
| patata | 泰诺语 batata | 土豆 |
| canoa | 泰诺语 kanoa | 独木舟 |
| huracán | 泰诺语 juracán | 飓风 |
| pampa | 克丘亚语 | 大草原 |
| llama | 克丘亚语 | 羊驼 |

### 英语借词（盎格鲁主义）

| 西班牙语 | 英语来源 | 备注 |
|---|---|---|
| software | software | 广泛接受 |
| marketing | marketing | 广泛接受 |
| parking | parking | 拉美；正式场合偏好 estacionamiento |
| jogging | jogging | RAE 偏好 trote |
| link | link | RAE 偏好 enlace |
| email / correo-e | email | 正式场合用 correo electrónico |
| fan | fan | aficionado 是本族对应词 |
| boss | boss | jefe 是本族对应词 |

### RAE 改编政策

西班牙皇家语言学院常提出西班牙语化形式：

- whisky → güisqui（未使用）
- parking → aparcamiento/estacionamiento
- email → correo electrónico
- blog → bitácora（罕见）

**注意**：尽管 RAE 建议，盎格鲁主义在口语和科技语境中广泛使用。

### 音译改编

英语词汇被改编以适应西班牙语音系：

- sándwich（非 sandwich——西班牙语重音和拼写）
- yogur（yogurt）
- fútbol（football）
- béisbol（baseball）
- champú（shampoo）
- mártir（martyr——来自拉丁语经希腊语）

## 常见错误

1. **西班牙语母语者学英语**：*I am very actual* → *I am very current / topical*（actual 在英语中 = real）

2. **英语母语者学西班牙语**：*Estoy embarazada* → **embarazada** = 怀孕（非 embarrassed = avergonzada）

3. **英语母语者学汉语**：正式邮件用伊妹儿 → 电子邮件（正式）vs 伊妹儿（俚语）

4. **汉语母语者学英语**：*I want to buy a sofa* — 正确，但可能不知道 "sofa" 本身已是经英语来自阿拉伯语的借词
