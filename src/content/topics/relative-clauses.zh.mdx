## 概述

关系从句通过添加额外信息来修饰名词。

- **西班牙语**：用 *que* 作默认关系代词。*Quien* 用于介词后指人。*El cual* 用于正式区分。*Cuyo* 表所有（与被拥有物一致）。先行词不确定时用虚拟式。
- **英语**：用 *who* 指人，*which* 指物，*that* 指两者皆可。*Whose* 表所有。*Whom* 用于介词宾语（正式）。限制性 vs 非限制性区分严格。
- **汉语**：无关系代词。关系从句直接放在名词前，用 *的* 连接。语境清楚时中心名词可省略。

## 西班牙语

### 关系代词

| 代词 | 用法 | 例句 |
|---|---|---|
| que | 通用（主语/宾语） | El hombre **que** vino |
| quien | 介词后人 | La mujer con **quien** hablé |
| el cual | 正式 / 区分 | El libro **el cual** leí |
| cuyo | 所有（一致） | El hombre **cuyo** hijo estudia |
| donde | 地点 | La casa **donde** vivo |
| cuando | 时间 | El día **cuando** llegué |

### Cuyo 的一致

| 被拥有物 | 形式 |
|---|---|
| 阳性单数 | cuyo |
| 阴性单数 | cuya |
| 阳性复数 | cuyos |
| 阴性复数 | cuyas |

**例句**：El hombre **cuyos** hijos estudian.

### 关系从句中的虚拟式

先行词不确定或未来时用虚拟式：

- Busco un médico **que hable** inglés.（虚拟——不确定）
- Conozco un médico **que habla** inglés.（直陈——确定）

### El que / el cual

| 代词 | 用法 | 例句 |
|---|---|---|
| el que | 替代名词 | El **que** vino fue Juan. |
| el cual | 正式，长介词后 | El estudiante, **el cual** aprobó... |

## 英语

### 关系代词

| 代词 | 用法 | 例句 |
|---|---|---|
| who | 人（主语） | The man **who** came |
| whom | 人（宾语/正式） | The man **whom** I met |
| which | 物 | The book **which** I read |
| that | 人或物 | The man/book **that** I saw |
| whose | 所有 | The man **whose** car was stolen |
| where | 地点 | The house **where** I live |
| when | 时间 | The day **when** I arrived |

### 限制性 vs 非限制性

| 类型 | 代词 | 逗号 | 例句 |
|---|---|---|---|
| 限制性 | that / which / who | 无 | The book **that** is red |
| 非限制性 | which / who | 有 | My car**, which is red,** is fast |

**注**：*That* 仅用于限制性从句（无逗号）。

### 介词 + 关系词

| 正式 | 非正式 |
|---|---|
| The man **to whom** I spoke | The man **(whom)** I spoke **to** |
| The book **in which** I read | The book **(which)** I read **in** |

**注**：正式英语将介词放在关系词前。

### 关系代词省略

宾语关系代词可省略：

- The book (that) I read.
- The man (whom) I met.

## 汉语

### 的 作关系连接词

汉语无关系代词。用 *的* 放在从句后：

| 类型 | 结构 | 例句 |
|---|---|---|
| 主语 | V + 的 + N | <ruby>来<rt>lái</rt></ruby>**<ruby>的<rt>de</rt></ruby>**<ruby>人<rt>rén</rt></ruby>（来的人） |
| 宾语 | S + V + 的 + N | <ruby>我<rt>wǒ</rt></ruby><ruby>读<rt>dú</rt></ruby>**<ruby>的<rt>de</rt></ruby>**<ruby>书<rt>shū</rt></ruby>（我读的书） |
| 所有格 | 所有者 + 的 + N | <ruby>我<rt>wǒ</rt></ruby><ruby>的<rt>de</rt></ruby><ruby>书<rt>shū</rt></ruby>（我的书） |

### 中心名词省略

语境清楚时中心名词可省略：

| 完整 | 省略 |
|---|---|
| <ruby>来<rt>lái</rt></ruby><ruby>的<rt>de</rt></ruby><ruby>人<rt>rén</rt></ruby> | <ruby>来<rt>lái</rt></ruby><ruby>的<rt>de</rt></ruby> |
| <ruby>红<rt>hóng</rt></ruby><ruby>的<rt>de</rt></ruby><ruby>车<rt>chē</rt></ruby> | <ruby>红<rt>hóng</rt></ruby><ruby>的<rt>de</rt></ruby> |

### 汉语关系从句中的介词

介词保留在动词旁：

- <ruby>我<rt>wǒ</rt></ruby><ruby>跟<rt>gēn</rt></ruby><ruby>他<rt>tā</rt></ruby><ruby>说<rt>shuō</rt></ruby><ruby>话<rt>huà</rt></ruby><ruby>的<rt>de</rt></ruby><ruby>人<rt>rén</rt></ruby>（我跟他说话的人）
- <ruby>我<rt>wǒ</rt></ruby><ruby>从<rt>cóng</rt></ruby><ruby>北<rt>běi</rt></ruby><ruby>京<rt>jīng</rt></ruby><ruby>来<rt>lái</rt></ruby><ruby>的<rt>de</rt></ruby><ruby>时<rt>shí</rt></ruby><ruby>候<rt>hou</rt></ruby>（我从北京来的时候）

### 例句

- <ruby>我<rt>wǒ</rt></ruby><ruby>认<rt>rèn</rt></ruby><ruby>识<rt>shi</rt></ruby><ruby>的<rt>de</rt></ruby><ruby>人<rt>rén</rt></ruby>（我认识的人）

## 快速比较

| 特征 | 西班牙语 | 英语 | 汉语 |
|---|---|---|---|
| 关系代词 | que, quien, el cual, cuyo | who, which, that, whose | 无 |
| 连接 | 代词 + 动词 | 代词 + 动词 | 从句 + 的 + 名词 |
| 关系化主语 | 保留 | 保留 | 省略 |
| 关系化宾语 | 省略 | 省略 | 省略 |
| 所有格 | cuyo（一致） | whose（不变） | 的 放在所有者后 |
| 介词 | 关系词前（正式） | 关系词前（正式） | 留在动词旁 |
| 限制性/非限制性 | 非限制性用逗号 | that vs which | 无正式区分 |

## 常见错误

1. **西班牙语母语者**：介词后忘记 quien → La mujer con **quien** hablé

2. **英语母语者**：限制性从句用 which → The book **that** I read（不用 which）

3. **英语母语者**：汉语中加关系代词 → 汉语无关系代词，用 的

4. **汉语母语者**：混淆 that/which → that = 限制性；which = 非限制性
