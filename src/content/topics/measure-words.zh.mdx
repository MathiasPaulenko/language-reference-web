## 概述

量词通过容器、部分或集合来表达数量。

- **西班牙语**：使用 un/una + 容器 + de + 名词。量词是可选的，相对较少。
- **英语**：使用 a + 量词 + of + 名词。对大多数名词可选，但对不可数数量必需。
- **汉语**：量词对所有可数名词都是强制性的。量词反映容器、形状或分组。不使用介词。

## 一览对比

| 特征 | 西班牙语 | 英语 | 汉语 |
|---------|---------|---------|---------|
| 所有名词都必须用 | 否 | 否 | 是 |
| 介词 | de | of | 无 |
| 顺序 | 数词 + 容器 + de + 名词 | 数词 + 量词 + of + 名词 | 数词 + 量词 + 名词 |
| 容器反映 | 部分 | 部分 | 广泛 |
| 部分量 | un poco de | some / a piece of | 些 / 点 |
| 集合量 | un par de / docena | a pair / a dozen | 双 / 对 / 群 / 堆 |
| 标准单位 | kilo / litro / metro | kg / liter / meter | 公斤 / 升 / 米 |

## 常见错误

1. **西班牙语/英语母语者学汉语**：省略量词：二咖啡 → 两**杯**咖啡

2. **英语母语者学汉语**：所有事物都用个：一个咖啡 → 一杯咖啡

3. **汉语母语者学英语**：不可数名词省略量词：I want two waters → I want two glasses of water

4. **汉语母语者学西班牙语**：不必要地添加量词：un pedazo de agua → un vaso de agua（水不是"切成块"的）
