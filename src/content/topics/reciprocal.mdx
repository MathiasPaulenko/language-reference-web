## Overview

Reciprocal constructions express mutual actions: they hug each other, we understand each other.

- **Spanish**: Uses the reflexive pronoun *se* (and its variants *nos, os*) with plural subjects. Reciprocal and reflexive are identical in form; context disambiguates.
- **English**: Uses the dedicated reciprocal pronouns *each other* and *one another*. These are grammatically distinct from reflexives (*themselves*).
- **Chinese**: Uses the pre-verbal adverbs *<ruby>互<rt>hù</rt></ruby><ruby>相<rt>xiāng</rt></ruby>* or *<ruby>相<rt>xiāng</rt></ruby><ruby>互<rt>hù</rt></ruby>*. There is no dedicated reciprocal pronoun. Reflexive (*<ruby>自<rt>zì</rt></ruby><ruby>己<rt>jǐ</rt></ruby>*) and reciprocal (*<ruby>互<rt>hù</rt></ruby><ruby>相<rt>xiāng</rt></ruby>*) are completely distinct.

## Spanish

### Se + plural verb

Spanish uses the same reflexive pronouns for reciprocity:

| Persons | Reflexive/Reciprocal pronoun |
|---|---|
| ellos/ellas/ustedes | se |
| nosotros/nosotras | nos |
| vosotros/vosotras | os |

**Reciprocal**:

- *Se quieren mucho.* (They love each other very much.)
- *Nos ayudamos.* (We help each other.)
- *Se escriben cada semana.* (They write to each other every week.)

### Disambiguation

When context is ambiguous, Spanish adds *el uno al otro* or *mutuamente*:

- *Se miraron **el uno al otro**.* (They looked at each other. — disambiguated)
- *Se odian **mutuamente**.* (They mutually hate each other.)

**Without clarification**:

- *Se lavaron.* (They washed themselves / each other. — ambiguous)
- *Se peinaron.* (They combed themselves / each other. — ambiguous)

### Verbs that are typically reciprocal

| Verb | Reciprocal meaning |
|---|---|
| abrazarse | hug each other |
| besarse | kiss each other |
| pelearse | fight each other |
| entenderse | understand each other |
| quererse | love each other |
| odiarse | hate each other |
| respetarse | respect each other |
| ayudarse | help each other |

### Preposition + el uno + al otro

When the verb requires a preposition:

- *Hablan **el uno con el otro**.* (They talk with each other.)
- *Se pelean **el uno contra el otro**.* (They fight against each other.)

### Mutual adjective

- *un acuerdo **mutuo*** (a mutual agreement)
- *el respeto **recíproco*** (mutual respect)

## English

### Each other / one another

English has dedicated reciprocal pronouns:

| | Usage |
|---|---|
| each other | general reciprocal (preferred for 2 people) |
| one another | general reciprocal (preferred for 3+ people) |

**Modern usage**: The distinction is largely ignored; both are interchangeable.

**Examples**:

- *They love **each other**.*
- *We respect **one another**.*
- *The team members support **each other**.*
- *They gave presents to **each other**.*

### Possessive form

- *They are **each other's** best friends.*
- *We borrowed **one another's** books.*

### Distinction from reflexive

| Reflexive | Reciprocal |
|---|---|
| They washed **themselves**. (each individually) | They washed **each other**. (mutually) |
| They saw **themselves** in the mirror. | They saw **each other** across the room. |
| They hurt **themselves**. (each individually) | They hurt **each other**. (mutually) |

### After prepositions

- *They talk **to each other**.*
- *They sat **next to each other**.*
- *They gave gifts **to one another**.*

### Mutual adjectives

- *a **mutual** friend*
- *a **shared** interest*
- *a **common** goal*
- *a **reciprocal** agreement*

### Together

- *They worked **together**.* (cooperative, not necessarily reciprocal)
- *They sang **together**.* (simultaneous, not necessarily mutual)

## Chinese

### <ruby>互<rt>hù</rt></ruby><ruby>相<rt>xiāng</rt></ruby> / <ruby>相<rt>xiāng</rt></ruby><ruby>互<rt>hù</rt></ruby> (pre-verbal adverbs)

<ruby>互<rt>hù</rt></ruby><ruby>相<rt>xiāng</rt></ruby> and <ruby>相<rt>xiāng</rt></ruby><ruby>互<rt>hù</rt></ruby> are adverbs placed before the verb:

- <ruby>他<rt>tā</rt></ruby><ruby>们<rt>men</rt></ruby><ruby>互<rt>hù</rt></ruby><ruby>相<rt>xiāng</rt></ruby><ruby>爱<rt>ài</rt></ruby>。(They love each other.)
- <ruby>我<rt>wǒ</rt></ruby><ruby>们<rt>men</rt></ruby><ruby>互<rt>hù</rt></ruby><ruby>相<rt>xiāng</rt></ruby><ruby>帮<rt>bāng</rt></ruby><ruby>助<rt>zhù</rt></ruby>。(We help each other.)
- <ruby>他<rt>tā</rt></ruby><ruby>们<rt>men</rt></ruby><ruby>相<rt>xiāng</rt></ruby><ruby>互<rt>hù</rt></ruby><ruby>理<rt>lǐ</rt></ruby><ruby>解<rt>jiě</rt></ruby>。(They understand each other.)

**<ruby>互<rt>hù</rt></ruby><ruby>相<rt>xiāng</rt></ruby> vs <ruby>相<rt>xiāng</rt></ruby><ruby>互<rt>hù</rt></ruby>**: Interchangeable in most contexts. <ruby>相<rt>xiāng</rt></ruby><ruby>互<rt>hù</rt></ruby> is slightly more formal.

### <ruby>彼<rt>bǐ</rt></ruby><ruby>此<rt>cǐ</rt></ruby> (each other / one another)

<ruby>彼<rt>bǐ</rt></ruby><ruby>此<rt>cǐ</rt></ruby> functions as a pronoun meaning "each other":

- <ruby>彼<rt>bǐ</rt></ruby><ruby>此<rt>cǐ</rt></ruby><ruby>了<rt>liǎo</rt></ruby><ruby>解<rt>jiě</rt></ruby> (mutual understanding)
- <ruby>彼<rt>bǐ</rt></ruby><ruby>此<rt>cǐ</rt></ruby><ruby>尊<rt>zūn</rt></ruby><ruby>重<rt>zhòng</rt></ruby> (mutual respect)
- <ruby>大<rt>dà</rt></ruby><ruby>家<rt>jiā</rt></ruby><ruby>彼<rt>bǐ</rt></ruby><ruby>此<rt>cǐ</rt></ruby><ruby>照<rt>zhào</rt></ruby><ruby>顾<rt>gù</rt></ruby> (everyone takes care of each other)

### No ambiguity with reflexive

Chinese strictly separates reflexive and reciprocal:

| Reflexive | Reciprocal |
|---|---|
| <ruby>他<rt>tā</rt></ruby><ruby>们<rt>men</rt></ruby><ruby>洗<rt>xǐ</rt></ruby><ruby>了<rt>le</rt></ruby><ruby>自<rt>zì</rt></ruby><ruby>己<rt>jǐ</rt></ruby>。(They washed themselves.) | <ruby>他<rt>tā</rt></ruby><ruby>们<rt>men</rt></ruby><ruby>互<rt>hù</rt></ruby><ruby>相<rt>xiāng</rt></ruby><ruby>洗<rt>xǐ</rt></ruby>。(They washed each other.) |

### <ruby>一<rt>yī</rt></ruby><ruby>起<rt>qǐ</rt></ruby> (together)

<ruby>一<rt>yì</rt></ruby><ruby>起<rt>qǐ</rt></ruby> means "together" and can imply cooperation:

- <ruby>我<rt>wǒ</rt></ruby><ruby>们<rt>men</rt></ruby><ruby>一<rt>yì</rt></ruby><ruby>起<rt>qǐ</rt></ruby><ruby>工<rt>gōng</rt></ruby><ruby>作<rt>zuò</rt></ruby>。(We work together.)
- <ruby>他<rt>tā</rt></ruby><ruby>们<rt>men</rt></ruby><ruby>一<rt>yì</rt></ruby><ruby>起<rt>qǐ</rt></ruby><ruby>吃<rt>chī</rt></ruby><ruby>饭<rt>fàn</rt></ruby>。(They eat together.)

**Note**: <ruby>一<rt>yī</rt></ruby><ruby>起<rt>qǐ</rt></ruby> does not necessarily imply reciprocity (giving and receiving), just joint activity.

### <ruby>共<rt>gòng</rt></ruby><ruby>同<rt>tóng</rt></ruby> / <ruby>共<rt>gòng</rt></ruby><ruby>享<rt>xiǎng</rt></ruby> (mutual/shared)

- <ruby>共<rt>gòng</rt></ruby><ruby>同<rt>tóng</rt></ruby><ruby>目<rt>mù</rt></ruby><ruby>标<rt>biāo</rt></ruby> (common goal)
- <ruby>共<rt>gòng</rt></ruby><ruby>享<rt>xiǎng</rt></ruby><ruby>资<rt>zī</rt></ruby><ruby>源<rt>yuán</rt></ruby> (shared resources)

## Common mistakes

1. **Spanish speakers learning English**: Omitting each other: *They help* → *They help **each other***

2. **English speakers learning Spanish**: Using explicit reciprocal where se suffices: *Se ayudan el uno al otro* → *Nos ayudamos* is sufficient

3. **English speakers learning Chinese**: Using <ruby>互<rt>hù</rt></ruby><ruby>相<rt>xiāng</rt></ruby> with singular subject: *<ruby>我<rt>wǒ</rt></ruby><ruby>互<rt>hù</rt></ruby><ruby>相<rt>xiāng</rt></ruby><ruby>帮<rt>bāng</rt></ruby><ruby>助<rt>zhù</rt></ruby>* → *<ruby>我<rt>wǒ</rt></ruby><ruby>们<rt>mén</rt></ruby>**<ruby>互<rt>hù</rt></ruby><ruby>相<rt>xiāng</rt></ruby>**<ruby>帮<rt>bāng</rt></ruby><ruby>助<rt>zhù</rt></ruby>*

4. **Chinese speakers learning English**: Confusing each other and together: *They work each other* → *They work **together*** / *They help **each other***
