## 概述

语域是语言随社交语境（包括正式性、媒介口语/书面、社交关系）而变化的方式。

- **西班牙语**：通过 *tú/usted*（动词形式区分）实现正式性的语法化。代词使用有地区差异（*vos* 在拉普拉塔河，*vosotros* 在西班牙）。
- **英语**：无 T-V 语法区分。正式性通过词汇选择、情态动词、头衔和间接表达实现。
- **汉语**：代词区分（*你/您*）。敬称头衔。正式语境有替代词汇。

## 西班牙语

### Tú vs usted

西班牙语有语法化的正式性区分：

| 人称 | 用法 | 例句 |
|---|---|---|
| **tú** | 朋友、家人、平辈 | ¿Cómo estás, **tú**? |
| **usted** | 陌生人、权威、长辈 | ¿Cómo está **usted**? |
| **vos** | 地区性（拉普拉塔河） | ¿Cómo andás, **vos**? |
| **vosotros** | 西班牙（复数非正式） | ¿Cómo estáis, **vosotros**? |

### 有正式性区分的动词

| 不定式 | tú | usted |
|---|---|---|
| venir | vienes | viene |
| estar | estás | está |
| poder | puedes | puede |
| tener | tienes | tiene |

### 地区差异

| 地区 | 用法 | 备注 |
|---|---|---|
| 西班牙 | tú/vosotros/usted/ustedes | Vosotros = 复数非正式 |
| 拉丁美洲 | tú/usted/ustedes | Ustedes = 通用复数 |
| 拉普拉塔河 | vos | 替代 tú |

### 书面 vs 口语

| 特征 | 口语 | 书面 |
|---|---|---|
| 缩约 | 有 | 无（正式） |
| 虚拟式 | 较少 | 较多 |
| 词汇 | 口语 | 正式 |

## 英语

### 无 T-V 区分

英语在代词上不区分正式性：

| 语境 | 用法 | 例句 |
|---|---|---|
| 非正式 | you + 缩约 | How **are you**? |
| 正式 | you + 头衔 | How **are you**, Professor Smith? |
| 非常正式 | Sir/Madam + would | **Would** you please... |

### 情态软化语

| 软化语 | 用法 | 例句 |
|---|---|---|
| would | 正式请求 | **Would** you help me? |
| could | 正式请求 | **Could** you open the door? |
| might | 正式建议 | You **might** want to... |
| please | 礼貌 | **Please** sit down. |

### 头衔

| 头衔 | 用法 | 例句 |
|---|---|---|
| Mr. / Ms. | 一般 | **Mr.** Smith |
| Mrs. / Miss | 已婚/未婚女性 | **Mrs.** Johnson |
| Dr. | 博士 | **Dr.** Williams |
| Professor | 大学教授 | **Professor** Lee |
| Sir / Madam | 正式 | Yes, **Sir**. |

### 书面 vs 口语

| 特征 | 口语 | 书面（正式） |
|---|---|---|
| 缩约 | 有（I'm, don't） | 无（I am, do not） |
| 词汇 | 口语 | 正式 |
| 句子 | 短 | 完整 |

## 汉语

### 你 vs 您

| 代词 | 用法 | 例句 |
|---|---|---|
| <ruby>你<rt>nǐ</rt></ruby> | 中性 / 非正式 | <ruby>你<rt>nǐ</rt></ruby><ruby>好<rt>hǎo</rt></ruby> |
| <ruby>您<rt>nín</rt></ruby> | 正式 / 尊敬 | <ruby>您<rt>nín</rt></ruby><ruby>好<rt>hǎo</rt></ruby> |

**注**：<ruby>您<rt>nín</rt></ruby> 用于长辈、老师、上司和陌生人。

### 请（qǐng）作为礼貌标记

| 表达 | 含义 |
|---|---|
| <ruby>请<rt>qǐng</rt></ruby><ruby>进<rt>jìn</rt></ruby> | 请进 |

### 书面 vs 口语

| 特征 | 口语 | 书面（正式） |
|---|---|---|
| 连接词 | 简单（<ruby>的<rt>de</rt></ruby>） | 正式（<ruby>之<rt>zhī</rt></ruby>, <ruby>其<rt>qí</rt></ruby>） |
| 代词 | <ruby>他<rt>tā</rt></ruby>, <ruby>的<rt>de</rt></ruby> | <ruby>其<rt>qí</rt></ruby>, <ruby>之<rt>zhī</rt></ruby> |
| 风格 | 直接 | 间接 / 繁复 |

## 快速比较

| 特征 | 西班牙语 | 英语 | 汉语 |
|---|---|---|---|
| T-V 区分 | 有（tú/usted） | 无（you 通用） | 有（你/您） |
| 动词变化 | 有 | 无 | 无 |
| 头衔 | 有 | 有 | 有 |
| 软化语 | Por favor, 虚拟式 | 情态动词, please | 请, 麻烦你 |
| 地区差异 | vos, vosotros | 无 | 无 |
| 书面 vs 口语 | 不同词汇 | 缩约 vs 完整 | 替代词汇 |

## 常见错误

1. **西班牙语母语者**：不恰当地用 tú → 对陌生人用 **usted**

2. **英语母语者**：汉语中不用 您 → 对长辈用 <ruby>您<rt>nín</rt></ruby><ruby>好<rt>hǎo</rt></ruby>

3. **汉语母语者**：认为英语仅靠语调就能正式化 → 用头衔：Professor Smith

4. **所有人**：正式写作中使用缩约 → do not（不用 don't）；cannot（不用 can't）
