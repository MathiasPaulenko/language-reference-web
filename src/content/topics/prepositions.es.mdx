## Resumen

Las preposiciones son palabras pequeñas que muestran relaciones — ubicación, tiempo, dirección, causa y más.

- **Español**: Menos preposiciones que el inglés, pero se combinan con artículos (*al, del*). Dos preposiciones problemáticas (*por* vs *para*) ambas se traducen como "for" pero tienen usos distintos.
- **Inglés**: Muchas preposiciones con distinciones sutiles. Los phrasal verbs (*give up, look after, turn on*) son un gran desafío. El preposition stranding está permitido en preguntas.
- **Chino**: Usa un pequeño conjunto de preposiciones centrales (<ruby>在<rt>zài</rt></ruby>, <ruby>从<rt>cóng</rt></ruby>, <ruby>到<rt>dào</rt></ruby>, <ruby>对<rt>duì</rt></ruby>). Las postposiciones (<ruby>上<rt>shàng</rt></ruby>, <ruby>下<rt>xià</rt></ruby>, <ruby>里<rt>lǐ</rt></ruby>, <ruby>外<rt>wài</rt></ruby>) a menudo siguen a los sustantivos para mostrar ubicación.

## Español

### Preposiciones centrales

| Preposición | Uso principal | Ejemplo |
|---|---|---|
| a | Dirección / tiempo | Voy **a** casa, **a** las tres |
| de | Posesión / origen | El libro **de** Juan, soy **de** España |
| en | Ubicación / tiempo | Estoy **en** casa, **en** verano |
| por | Causa / medio / precio | Lo hice **por** ti, paso **por** aquí |
| para | Propósito / destino | Es **para** ti, salgo **para** Madrid |
| con | Compañía / instrumento | Voy **con** María, corto **con** cuchillo |
| sin | Ausencia | Voy **sin** dinero |
| sobre | Encima / tema | El libro **sobre** la mesa, hablo **sobre** política |
| entre | Entre dos o más | **Entre** tú y yo |
| desde | Punto de origen | **Desde** Madrid hasta Barcelona |
| hasta | Punto de llegada | **Hasta** mañana |

### Contracciones con artículos

| Combinación | Forma | Ejemplo |
|---|---|---|
| a + el | **al** | Voy **al** cine |
| de + el | **del** | Vengo **del** trabajo |

**Nota**: No contraer con la: a + la = a la; de + la = de la.

### por vs para

| por | para |
|---|---|
| Causa (por eso) | Propósito (para eso) |
| Medio (por avión) | Destinatario (para ti) |
| Duración (por dos horas) | Destino (para Madrid) |
| Precio (por 10 euros) | Fecha límite (para mañana) |
| Sustitución (por mi) | Uso (para cortar) |

## Inglés

### Preposiciones centrales

| Preposición | Uso principal | Ejemplo |
|---|---|---|
| in | Dentro de / tiempo largo | **In** the room, **in** July |
| on | Sobre / días/fechas | **On** the table, **on** Monday |
| at | Punto específico / tiempo | **At** the door, **at** 3 o'clock |
| to | Dirección | Go **to** school |
| from | Origen | Come **from** Spain |
| for | Propósito / duración | **For** you, **for** two hours |
| with | Compañía / instrumento | **With** me, cut **with** a knife |
| without | Ausencia | **Without** money |
| by | Agente / medio / cerca | Written **by** him, **by** car, **by** the river |
| about | Tema | Talk **about** politics |
| into | Hacia dentro | Go **into** the room |
| through | A través de | Walk **through** the park |
| across | De un lado a otro | Walk **across** the street |
| over | Encima / por encima | Jump **over** the fence |
| under | Debajo | **Under** the table |
| between | Entre dos | **Between** you and me |
| among | Entre muchos | **Among** the crowd |
| during | Durante | **During** the meeting |
| until / till | Hasta | Wait **until** tomorrow |
| since | Desde (tiempo) | **Since** 2010 |

### Preposition stranding

| Formal | Informal |
|---|---|
| With whom did you speak? | Who did you speak **with**? |
| To whom did you give it? | Who did you give it **to**? |
| In which city do you live? | Which city do you live **in**? |

**Nota**: El inglés formal evita terminar con preposiciones; el informal las permite.

### Phrasal verbs

| Phrasal verb | Significado | Ejemplo |
|---|---|---|
| give up | Rendirse | Don't give **up** |
| look after | Cuidar | Look **after** the children |
| turn on | Encender | Turn **on** the light |
| put up with | Soportar | Put **up with** the noise |
| run out of | Quedarse sin | Run **out of** time |

**Nota**: Los phrasal verbs son idiomáticos; no traducir literalmente.

## Errores comunes

1. **Angloparlantes**: Usar "for" cuando debe ser "por" o "para" → La distinción por/para no existe en inglés

2. **Angloparlantes**: Olvidar la preposición en chino → "我北京" en lugar de "我**在**北京"

3. **Hispanohablantes**: Preposition stranding en inglés formal → "Who did you speak with?" en contextos formales

4. **Hispanohablantes**: Traducir phrasal verbs literalmente → "dar arriba" en lugar de "rendirse"
