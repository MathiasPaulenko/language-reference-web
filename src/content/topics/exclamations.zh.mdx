## 概述

感叹句表达强烈情感：惊讶、痛苦、喜悦、愤怒、难以置信。

- **西班牙语**：使用 *qué/cuánto* + 名词/形容词：¡Qué día! ¡Cuánto cuesta! 需要 ¡ 和 !.
- **英语**：使用 *what* + 名词短语和 *how* + 形容词/副词：What a surprise! How amazing! 无特殊标点。
- **中文**：使用句末小品词（*啊a/呀ya/呢ne/吧ba*），*<ruby>多<rt>duō</rt></ruby>* + 形容词，*<ruby>真<rt>zhēn</rt></ruby>* + 形容词，以及升调。

## 西班牙语

### Qué结构

**Qué + 名词**：
- ¡Qué día!（多忙的一天！）
- ¡Qué sorpresa!（真意外！）
- ¡Qué pena!（真遗憾！）

**Qué + 形容词**：
- ¡Qué hermoso!（真漂亮！）
- ¡Qué rico!（真好吃！）
- ¡Qué difícil!（真难！）

**Qué + tan + 形容词**：
- ¡Qué tan alto!（好高啊！）

### Cuánto结构

- ¡Cuánto cuesta!（好贵啊！）
- ¡Cuántos vienen!（来了这么多人！）
- ¡Cuánto te quiero!（我多么爱你！）

### 感叹词

| 感叹词 | 情感 |
|---|---|
| ¡Ay! | 痛苦、悲伤 |
| ¡Vaya! | 惊讶 |
| ¡Ojalá! | 愿望 |
| ¡Bravo! | 赞同 |
| ¡Vale! | 同意（西班牙） |
| ¡Olé! | 热情 |
| ¡Anda! | 惊讶、难以置信 |
| ¡Ostras! | 惊讶、尴尬（西班牙口语） |
| ¡Hostia! | 惊讶、生气（西班牙口语） |

### 否定感叹

- ¡Ni hablar!（想都别想！）
- ¡Ni en sueños!（做梦吧！）
- ¡Que no!（才不呢！）
- ¡Para nada!（绝不！）

### 动词感叹句

- ¡Que aproveche!（祝用餐愉快！）
- ¡Que te mejores!（祝你早日康复！）
- ¡Que aproveche la comida!（祝用餐愉快！）
- ¡Que viva la fiesta!（派对万岁！）

### 间接感叹句

- Me dijo ¡qué sorpresa!（他说真意外！）
- No sabes ¡qué calor hace!（你不知道有多热！）

## 英语

### What + 名词短语

- What a surprise!（真意外！）
- What a mess!（一团糟！）
- What a beautiful day!（多美好的一天！）
- What an idiot!（真是个白痴！）
- What nonsense!（胡说八道！）

### How + 形容词/副词

- How beautiful!（真漂亮！）
- How amazing!（太神奇了！）
- How wonderful!（太棒了！）
- How dare you!（你好大的胆子！）
- How come?（怎么会？）

### 感叹词

| 感叹词 | 情感 |
|---|---|
| Wow! | 惊叹 |
| Ouch! / Ow! | 疼痛 |
| Yikes! | 恐惧、担忧 |
| Yuck! / Eww! | 厌恶 |
| Yippee! / Yay! | 喜悦 |
| Uh-oh! | 担忧、麻烦 |
| Aww! | 可爱、同情 |
| Oh no! | 担忧、负面惊讶 |
| Hell yeah! | 热情（口语） |
| No way! | 难以置信 |
| Holy cow! | 惊讶（委婉语） |
| Bless you! | 健康（打喷嚏后） |

### 否定感叹

- What the hell!（见鬼！）
- What on earth!（到底怎么回事！）
- For crying out loud!（看在老天爷份上！）
- Oh for goodness' sake!（天哪！）
- Give me a break!（饶了我吧！）

### 呼语感叹

- You idiot!（你这个白痴！）
- What a fool I am!（我真是个傻瓜！）
- How stupid of me!（我真蠢！）

## 常见错误

1. **学西班牙语的英语使用者**：在书面感叹句中省略¡

2. **学英语的西班牙语使用者**：*¡What beautiful!* → *How beautiful!*（qué vs. how的混淆）

3. **学英语的中文使用者**：在what感叹句中省略冠词：*What surprise!* → *What **a** surprise!*

4. **学中文的英语使用者**：在感叹句中使用平调而非升调
