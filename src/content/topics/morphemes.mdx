## Overview

A morpheme is the smallest unit of language that carries meaning.

- **Free morpheme**: Can stand alone (book, read, happy).
- **Bound morpheme**: Must attach to another morpheme (*-s, un-,-ness, -ed*).
- **Inflectional morpheme**: Changes grammatical function without changing word class (*-s* for plural, *-ed* for past).
- **Derivational morpheme**: Creates a new word, often changing word class (*happy* → *happiness*).

- **Spanish**: Rich inflectional system (gender, number, person, tense, mood). Productive derivational morphology.
- **English**: Minimal inflection but productive derivational morphology. Compounding is extremely productive.
- **Chinese**: Almost no inflection. Most morphemes are free or bound roots. Compounding is the dominant word-formation strategy.

## Spanish

### Inflectional morphemes

| Category | Morpheme | Example |
|---|---|---|
| Plural (nouns/adjectives) | -s / -es | libro → libros, mes → meses |
| Feminine (nouns/adjectives) | -a | alto → alta |
| 1st person present | -o | hablar → hablo |
| 2nd person present | -as / -es | hablar → hablas |
| 3rd person present | -a / -e | hablar → habla |
| Past preterite | -é / -aste / -ó / -amos / -aron | hablar → hablé |
| Past imperfect | -aba / -ía | hablar → hablaba |
| Future | -é / -ás / -á / -emos / -án | hablar → hablaré |
| Conditional | -ía / -ías / -ía / -íamos / -ían | hablar → hablaría |
| Subjunctive | -e / -es / -e / -emos / -en | hable |
| Gerund | -ando / -iendo | hablando |
| Past participle | -ado / -ido | hablado |

### Derivational morphemes

| Morpheme | Function | Example |
|---|---|---|
| -idad / -tad | abstract noun | feliz → felicidad, libre → libertad |
| -mente | adverb | rápido → rápidamente |
| -ción / -sión | action noun | crear → creación |
| -dor / -dora | agent | cantar → cantador |
| -ero / -era | profession/place | pan → panadero |
| -oso / -osa | quality | peligro → peligroso |
| -able / -ible | capability | comer → comestible |
| des- | negation/opposite | hacer → deshacer |
| in- / im- / i- | negation | posible → imposible |
| re- | repetition | hacer → rehacer |

### Compound morphemes

Limited but productive in certain patterns:

- *limpiaparabrisas* = limpia + para + brisas
- *sacacorchos* = saca + corchos
- *rascacielos* = rasca + cielos

## English

### Inflectional morphemes (8 total)

| Morpheme | Function | Example |
|---|---|---|
| -s | 3rd person singular present | he speak**s** |
| -s | plural noun | cat**s** |
| -'s | possessive | John**'s** book |
| -ed | past tense / past participle | walk**ed** |
| -en | past participle (irregular) | eat**en** |
| -ing | present participle / gerund | read**ing** |
| -er | comparative | tall**er** |
| -est | superlative | tall**est** |

**Note**: English has only 8 inflectional morphemes — one of the least inflected Indo-European languages.

### Derivational morphemes

**Prefixes**:

| Prefix | Meaning | Example |
|---|---|---|
| un- | negation/opposite | unhappy, undo |
| re- | again | redo, rewrite |
| pre- | before | preview, predict |
| dis- | opposite | disagree, disconnect |
| mis- | wrong | misunderstand |
| over- | excessive | overcook |
| under- | insufficient | undercook |
| non- | not | nonfiction |
| anti- | against | antibiotic |
| co- | together | cooperate |

**Suffixes**:

| Suffix | Word class | Example |
|---|---|---|
| -ness | noun (abstract) | happiness, sadness |
| -tion / -sion | noun (action) | creation, decision |
| -ment | noun (result) | development |
| -er / -or | noun (agent) | teacher, actor |
| -ist | noun (profession) | artist, scientist |
| -ism | noun (doctrine) | capitalism |
| -ity | noun (quality) | reality |
| -ful | adjective | beautiful, helpful |
| -less | adjective (without) | homeless, careless |
| -ous | adjective | dangerous |
| -able / -ible | adjective | readable, edible |
| -ly | adverb | quickly |
| -en | verb | widen, strengthen |
| -ify | verb | simplify |
| -ize / -ise | verb | organize |

### Compound morphemes

Highly productive:

- *black + board = blackboard*
- *tooth + brush = toothbrush*
- *wash + ing + machine = washing machine*
- *note + book = notebook*

### Zero derivation (conversion)

- *email* (noun) → *to email* (verb)
- *bottle* (noun) → *to bottle* (verb)

## Chinese

### Character as morpheme

Each Chinese character is typically one morpheme:

- <ruby>人<rt>rén</rt></ruby> = person
- <ruby>大<rt>dà</rt></ruby> = big
- <ruby>学<rt>xué</rt></ruby> = study

**Free morphemes** (can stand alone):
- <ruby>书<rt>shū</rt></ruby> (book), <ruby>水<rt>shuǐ</rt></ruby> (water), <ruby>来<rt>lái</rt></ruby> (come)

**Bound morphemes** (must combine):
- <ruby>民<rt>mín</rt></ruby> (people — usually in compounds: <ruby>人<rt>rén</rt></ruby><ruby>民<rt>mín</rt></ruby>, <ruby>民<rt>mín</rt></ruby><ruby>族<rt>zú</rt></ruby>)
- <ruby>语<rt>yǔ</rt></ruby> (language — usually in compounds: <ruby>汉<rt>hàn</rt></ruby><ruby>语<rt>yǔ</rt></ruby>, <ruby>英<rt>yīng</rt></ruby><ruby>语<rt>yǔ</rt></ruby>)
- <ruby>化<rt>huà</rt></ruby> (-ize — in <ruby>现<rt>xiàn</rt></ruby><ruby>代<rt>dài</rt></ruby><ruby>化<rt>huà</rt></ruby>, <ruby>简<rt>jiǎn</rt></ruby><ruby>化<rt>huà</rt></ruby>)

### No inflectional morphemes

Chinese has no grammatical inflection:

- No plural: <ruby>书<rt>shū</rt></ruby> = book / books
- No tense: <ruby>读<rt>dú</rt></ruby> = read / reads / read (past)
- No person: <ruby>说<rt>shuō</rt></ruby> = I speak / he speaks
- No gender: <ruby>老<rt>lǎo</rt></ruby><ruby>师<rt>shī</rt></ruby> = teacher (male or female)
- No case: <ruby>我<rt>wǒ</rt></ruby> = I / me / my

### Aspect particles (not inflection)

| Particle | Function | Example |
|---|---|---|
| <ruby>了<rt>le</rt></ruby> | completion/change | <ruby>吃<rt>chī</rt></ruby><ruby>了<rt>le</rt></ruby> (ate) |
| <ruby>着<rt>zháo</rt></ruby> | ongoing state | <ruby>吃<rt>chī</rt></ruby><ruby>着<rt>zhe</rt></ruby> (eating) |
| <ruby>过<rt>guò</rt></ruby> | experience | <ruby>吃<rt>chī</rt></ruby><ruby>过<rt>guò</rt></ruby> (have eaten) |

**Note**: These are phrasal clitics, not word-bound inflections. They can appear after the object:
- <ruby>我<rt>wǒ</rt></ruby><ruby>吃<rt>chī</rt></ruby><ruby>饭<rt>fàn</rt></ruby><ruby>了<rt>le</rt></ruby> (I ate — <ruby>了<rt>le</rt></ruby> after object)

### Derivational suffixes (limited)

| Suffix | Function | Example |
|---|---|---|
| <ruby>者<rt>zhě</rt></ruby> | person | <ruby>读<rt>dú</rt></ruby><ruby>者<rt>zhě</rt></ruby> (reader) |
| <ruby>家<rt>jiā</rt></ruby> | expert | <ruby>科<rt>kē</rt></ruby><ruby>学<rt>xué</rt></ruby><ruby>家<rt>jiā</rt></ruby> (scientist) |
| <ruby>员<rt>yuán</rt></ruby> | worker | <ruby>演<rt>yǎn</rt></ruby><ruby>员<rt>yuán</rt></ruby> (actor) |
| <ruby>性<rt>xìng</rt></ruby> | quality | <ruby>可<rt>kě</rt></ruby><ruby>能<rt>néng</rt></ruby><ruby>性<rt>xìng</rt></ruby> (possibility) |
| <ruby>化<rt>huà</rt></ruby> | -ize | <ruby>现<rt>xiàn</rt></ruby><ruby>代<rt>dài</rt></ruby><ruby>化<rt>huà</rt></ruby> (modernize) |
| <ruby>学<rt>xué</rt></ruby> | -ology | <ruby>数<rt>shù</rt></ruby><ruby>学<rt>xué</rt></ruby> (mathematics) |

### Prefixes (very limited)

| Prefix | Meaning | Example |
|---|---|---|
| <ruby>不<rt>bù</rt></ruby> | negation | <ruby>不<rt>bù</rt></ruby><ruby>好<rt>hǎo</rt></ruby> (not good) |
| <ruby>非<rt>fēi</rt></ruby> | non- (formal) | <ruby>非<rt>fēi</rt></ruby><ruby>法<rt>fǎ</rt></ruby> (illegal) |
| <ruby>可<rt>kě</rt></ruby> | can/able | <ruby>可<rt>kě</rt></ruby><ruby>爱<rt>ài</rt></ruby> (lovable) |
| <ruby>再<rt>zài</rt></ruby> | re- | <ruby>再<rt>zài</rt></ruby><ruby>见<rt>jiàn</rt></ruby> (see again) |
| <ruby>第<rt>dì</rt></ruby> | ordinal | <ruby>第<rt>dì</rt></ruby><ruby>一<rt>yī</rt></ruby> (first) |
| <ruby>老<rt>lǎo</rt></ruby> | old / familiar | <ruby>老<rt>lǎo</rt></ruby><ruby>师<rt>shī</rt></ruby> (teacher), <ruby>老<rt>lǎo</rt></ruby><ruby>王<rt>Wáng</rt></ruby> (Old Wang) |
| <ruby>小<rt>xiǎo</rt></ruby> | small / affectionate | <ruby>小<rt>xiǎo</rt></ruby><ruby>王<rt>Wáng</rt></ruby> (Little Wang) |

### Compounding (dominant strategy)

- <ruby>电<rt>diàn</rt></ruby><ruby>脑<rt>nǎo</rt></ruby> (electric + brain = computer)
- <ruby>黑<rt>hēi</rt></ruby><ruby>板<rt>bǎn</rt></ruby> (black + board = blackboard)
- <ruby>手<rt>shǒu</rt></ruby><ruby>机<rt>jī</rt></ruby> (hand + machine = mobile phone)
- <ruby>洗<rt>xǐ</rt></ruby><ruby>衣<rt>yī</rt></ruby><ruby>机<rt>jī</rt></ruby> (wash + clothes + machine = washing machine)

## Comparison at a glance

| Feature | Spanish | English | Chinese |
|---------|---------|---------|---------|
| Inflectional morphemes | Very rich (gender, number, person, tense, mood) | Minimal (8 morphemes) | None |
| Derivational morphemes | Very productive | Very productive | Limited (<ruby>者<rt>zhě</rt></ruby>, <ruby>家<rt>jiā</rt></ruby>, <ruby>化<rt>huà</rt></ruby>, <ruby>性<rt>xìng</rt></ruby>) |
| Prefixes | Productive (des-, in-, re-) | Productive (un-, re-, dis-) | Very limited (<ruby>不<rt>bù</rt></ruby>, <ruby>非<rt>fēi</rt></ruby>, <ruby>可<rt>kě</rt></ruby>) |
| Compounding | Limited | Very productive | Extremely productive |
| Aspect marking | Verbal inflection | Auxiliary + participle | Particle (<ruby>了<rt>le</rt></ruby>, <ruby>着<rt>zháo</rt></ruby>, <ruby>过<rt>guò</rt></ruby>) |

## Examples in context

### Plural
- **ES**: *gato → gato**s*** (inflection)
- **EN**: *cat → cat**s*** (inflection)
- **ZH**: <ruby>猫<rt>māo</rt></ruby> (no change)

### Past tense
- **ES**: *habl**ar** → habl**é*** (inflection)
- **EN**: *walk → walk**ed*** (inflection)
- **ZH**: <ruby>走<rt>zǒu</rt></ruby><ruby>了<rt>le</rt></ruby> (particle, not inflection)

### Abstract noun
- **ES**: *feliz → felici**dad*** (derivation)
- **EN**: *happy → happi**ness*** (derivation)
- **ZH**: <ruby>幸<rt>xìng</rt></ruby><ruby>福<rt>fú</rt></ruby><ruby>感<rt>gǎn</rt></ruby> (compounding)

## Common mistakes

1. **English speakers learning Spanish**: *dos gato* → *dos gato**s*** (plural inflection)

2. **Spanish speakers learning English**: *He speak* → *He speak**s*** (3sg inflection)

3. **English speakers learning Chinese**: *<ruby>猫<rt>māo</rt></ruby><ruby>们<rt>mén</rt></ruby>* for "cats" → <ruby>几<rt>jǐ</rt></ruby><ruby>只<rt>zhī</rt></ruby><ruby>猫<rt>māo</rt></ruby> (classifier + numeral)

4. **Chinese speakers learning English**: *I have three book* → *I have three book**s*** (plural inflection)
