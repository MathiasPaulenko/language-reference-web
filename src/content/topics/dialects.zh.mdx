## 概述

方言是语言的区域或社会变体。三种语言都表现出大量的内部多样性，但性质和程度不同。

- **西班牙语**：方言差异主要在语音和词汇。所有方言仍可相互理解。
- **英语**：全球传播最广。变体从几乎相同（美/加）到相互挑战（强苏格兰语 vs Singlish）。
- **中文**："中文"一词涵盖多种汉藏语言。普通话是官方标准，但其他语言本身也是重要语言。

---

## 西班牙语方言

关键等语线（因地区而异的特点）：

**语音方面**：
- **不分/分/seseo/ceceo**：卡斯蒂利亚 c/z = /θ/（分）；美洲 c/z = /s/（不分）；安达卢西亚部分地区 s = /θ/（ceceo）
- **Yeísmo**：ll 与 y 合并为 /ʝ/，在世界上大部分地区
- **Voseo**：阿根廷、乌拉圭、巴拉圭、中美部分地区用 *vos* 代替 *tú*
- **-s 词尾送气化/脱落**：加勒比、秘鲁海岸、西班牙部分地区常见

**词汇**：*coche*（西班牙）vs *carro*（墨西哥）vs *auto*（阿根廷）表示"车"

## 英语方言

主要变体及其特点：

**英式英语**：
- 非 r 音（除某些地区）
- TRAP-BATH 分裂：*bath* /bɑːθ/ vs *trap* /træp/
- RP（标准发音）：有 prestige，但仅 `<3%` 英国人口使用

**美式英语**：
- 全 r 音
- 西部和加拿大 cot-caught 合并
- 东部 Mary-marry-merry 合并

**其他主要变体**：澳式、加式、印式、新加坡式（Singlish）、非裔美语（AAVE）

**AAVE** 是完全系统化、规则化的变体，特点包括：
- 习惯性 *be*：*She be working* = 她经常工作
- 远过去 *been*：*She been told him* = 她很久以前就告诉他了
- 系动词省略：*She smart* = 她很聪明

## 中文"方言"

中文情况独特。普通话（Putonghua）是官方语言，但其他变体语言学上截然不同：

| 变体 | 使用人口 | 与普通话的相互理解度 |
|------|---------|---------------------|
| 官话 | ~10 亿 | — |
| 粤语 | ~8000 万 | 低 |
| 吴语（上海话）| ~8000 万 | 很低 |
| 闽语（闽南语、台语）| ~7000 万 | 很低 |
| 客家话 | ~5000 万 | 低 |
| 湘语 | ~4000 万 | 中低 |
| 赣语 | ~3000 万 | 低 |
| 晋语 | ~6000 万 | 中等 |

**关键差异**：
- 粤语有 6-9 个声调；普通话有 4 个
- 闽语保留了普通话中已消失的古老声母区别
- 词汇分歧巨大

## 标准与政治

| 语言 | 标准化机构 | 政策 |
|------|-----------|------|
| 西班牙语 | RAE + ASALE | 描述性为主，部分规定性指导 |
| 英语 | 无（基于 prestige）| 媒体和教育的 de facto 标准 |
| 中文 | 国家语言文字工作委员会 | 推广普通话；其他变体文化上受保护但教育中不受重视 |
