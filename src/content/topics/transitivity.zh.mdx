## 概述

及物性指动词是否带直接宾语。语言在允许动词带或不带宾语以及如何标记这种转换方面存在差异。

- **西班牙语**：许多动词有及物和不及物版本。反身代词*se*使动词非及物化（*romper* → *romperse*）。宾语省略比英语更自由。
- **英语**：许多动词是兼类的（*I eat rice* / *I eat*）。有些需要宾语（*put the book down* — *I put*不完整）。无形态非及物化。
- **汉语**：动词在语境允许时自由省略宾语。许多常见动作是动词-名词复合词（<ruby>睡<rt>shuì</rt></ruby><ruby>觉<rt>jué</rt></ruby> = 睡觉）。无及物性转换的形态标记。

## 西班牙语

### 及物动词（需宾语）

这些动词需要显式宾语：

- *romper*（打破）：*Rompí el jarrón.*（我打碎了花瓶。）
- *ver*（看见）：*Veo la casa.*（我看见房子。）
- *poner*（放）：*Puse el libro aquí.*（我把书放在这里。）
- *dar*（给）：*Di un regalo.*（我送了礼物。）
- *hacer*（做）：*Hice la cena.*（我做了晚饭。）

### 不及物动词（无宾语）

这些动词不能带直接宾语：

- *llegar*（到达）：*Llegué.*（我到了。）
- *morir*（死亡）：*Murió.*（他死了。）
- *dormir*（睡觉）：*Durmió.*（他睡了。）
- *nacer*（出生）：*Nació.*（他出生了。）
- *caer*（落下）：*Cayó.*（它掉了。）

### 兼类动词

同一形式用于及物和不及物：

- *comer*（吃）：*Como arroz.* / *Como.*（我吃米饭。/ 我吃。）
- *leer*（读）：*Leo libros.* / *Leo.*（我读书。/ 我读。）
- *beber*（喝）：*Bebo agua.* / *Bebo.*（我喝水。/ 我喝。）
- *fumar*（抽烟）：*Fumo cigarrillos.* / *Fumo.*（我抽烟。/ 我抽。）

### Se + 动词（非及物化/起始体）

西班牙语使用*se*创造不及物/反使役对应形式：

| 及物 | 不及物（se） |
|---|---|
| romper（打破某物） | romperse（自己破碎） |
| caer（使某物落下） | caerse（落下） |
| quemar（烧某物） | quemarse（自燃） |
| abrir（打开某物） | abrirse（自己打开） |
| cerrar（关闭某物） | cerrarse（自己关闭） |
| mover（移动某物） | moverse（自己移动） |

**例句**：

- *Rompí el jarrón.*（我打碎了花瓶。——及物）
- *El jarrón **se** rompió.*（花瓶碎了。——不及物/反使役）
- *Abrí la puerta.*（我打开了门。）
- *La puerta **se** abrió.*（门自己开了。）

### Se表非故意

*Se*可标记非故意动作，主语为人：

- **Se** me cayó el vaso.（杯子从我手中掉了。）
- **Se** me olvidó.（我忘了。）
- **Se** me murió el perro.（我的狗死了。）

### 中动语态（se + 人）

- *Esta tela **se** lava fácilmente.*（这种布料容易洗。）
- *El libro **se** vende bien.*（这本书卖得好。）
- *La puerta **se** abre hacia afuera.*（门向外开。）

## 英语

### 及物动词（需宾语）

这些动词需要显式宾语：

- *put*: *Put the book down.*（不能说 *I put.*）
- *give*: *Give me the book.*（不能说 *I give.*）
- *tell*: *Tell me the truth.*（不能说 *I tell.*）
- *send*: *Send a letter.*（不能说 *I send.* — 孤立时）
- *make*: *Make dinner.*（不能说 *I make.* — 孤立时）

### 不及物动词（无宾语）

- *arrive*: *I arrived.*（我到了。）
- *die*: *He died.*（他死了。）
- *sleep*: *She slept.*（她睡了。）
- *fall*: *It fell.*（它掉了。）
- *happen*: *What happened?*（发生了什么？）

### 兼类动词（两者皆可）

**S=主语，O=宾语，同形**：

- *eat*: *I eat rice.* / *I eat.*（我吃米饭。/ 我吃。）
- *read*: *I read books.* / *I read.*（我读书。/ 我读。）
- *write*: *I write novels.* / *I write.*（我写小说。/ 我写。）
- *sing*: *I sing songs.* / *I sing.*（我唱歌。/ 我唱。）
- *drive*: *I drive a car.* / *I drive.*（我开车。/ 我开。）
- *cook*: *I cook dinner.* / *I cook.*（我做晚饭。/ 我做。）

**不同形式**：

- *lay*（及物） / *lie*（不及物）：*Lay the book down.* / *Lie down.*
- *raise*（及物） / *rise*（不及物）：*Raise your hand.* / *The sun rises.*
- *set*（及物） / *sit*（不及物）：*Set the table.* / *Sit down.*

### 兼类动词（同形，不同结构）

- *I **broke** the vase.*（及物）
- *The vase **broke**.*（不及物）
- *I **opened** the door.*（及物）
- *The door **opened**.*（不及物）
- *I **melted** the butter.*（及物）
- *The butter **melted**.*（不及物）

### Get-被动（反使役）

- *The vase **got** broken.*（花瓶被打碎了。）
- *The door **got** opened.*（门被打开了。）
- *He **got** hurt.*（他受伤了。）

### 使役make

- *I **made** him laugh.*（我让他笑了。）
- *She **made** me cry.*（她让我哭了。）
- *He **made** it work.*（他让它运作了。）

## 汉语

### 及物动词带宾语

- <ruby>我<rt>wǒ</rt></ruby><ruby>吃<rt>chī</rt></ruby><ruby>米<rt>mǐ</rt></ruby><ruby>饭<rt>fàn</rt></ruby>。（我吃米饭。）
- <ruby>他<rt>tā</rt></ruby><ruby>看<rt>kàn</rt></ruby><ruby>书<rt>shū</rt></ruby>。（他看书。）
- <ruby>我<rt>wǒ</rt></ruby><ruby>写<rt>xiě</rt></ruby><ruby>信<rt>xìn</rt></ruby>。（我写信。）

### 宾语省略（不及物用法）

汉语在语境清楚时自由省略宾语：

- <ruby>我<rt>wǒ</rt></ruby><ruby>吃<rt>chī</rt></ruby><ruby>了<rt>le</rt></ruby>。（我吃了[饭]。——宾语隐含）
- <ruby>他<rt>tā</rt></ruby><ruby>看<rt>kàn</rt></ruby><ruby>了<rt>le</rt></ruby>。（他看了[书]。——宾语隐含）
- <ruby>你<rt>nǐ</rt></ruby><ruby>写<rt>xiě</rt></ruby><ruby>完<rt>wán</rt></ruby><ruby>了<rt>le</rt></ruby><ruby>吗<rt>ma</rt></ruby>？（你写完[它]了吗？）

### 动词-名词复合词

许多汉语动词实际上是V+N复合词，其中N是动词的一部分：

| 复合词 | 字面义 | 含义 |
|---|---|---|
| <ruby>睡<rt>shuì</rt></ruby><ruby>觉<rt>jiào</rt></ruby> | 睡 + 觉（名词） | 睡觉 |
| <ruby>洗<rt>xǐ</rt></ruby><ruby>澡<rt>zǎo</rt></ruby> | 洗 + 澡 | 洗澡 |
| <ruby>唱<rt>chàng</rt></ruby><ruby>歌<rt>gē</rt></ruby> | 唱 + 歌 | 唱歌 |
| <ruby>跳<rt>tiào</rt></ruby><ruby>舞<rt>wǔ</rt></ruby> | 跳 + 舞 | 跳舞 |
| <ruby>跑<rt>pǎo</rt></ruby><ruby>步<rt>bù</rt></ruby> | 跑 + 步 | 跑步 |
| <ruby>散<rt>sàn</rt></ruby><ruby>步<rt>bù</rt></ruby> | 散 + 步 | 散步 |
| <ruby>吃<rt>chī</rt></ruby><ruby>饭<rt>fàn</rt></ruby> | 吃 + 饭 | 吃饭 |

**注意**：这些通常不能再带宾语：

- ❌ <ruby>我<rt>wǒ</rt></ruby><ruby>睡<rt>shuì</rt></ruby><ruby>觉<rt>jiào</rt></ruby><ruby>房<rt>fáng</rt></ruby><ruby>间<rt>jiān</rt></ruby>（错误）
- ✅ <ruby>我<rt>wǒ</rt></ruby><ruby>在<rt>zài</rt></ruby><ruby>房<rt>fáng</rt></ruby><ruby>间<rt>jiān</rt></ruby><ruby>睡<rt>shuì</rt></ruby><ruby>觉<rt>jiào</rt></ruby>（我在房间里睡觉。）

### 结果式复合词

汉语通过结果补语创造及物/不及物对：

- <ruby>打<rt>dǎ</rt></ruby><ruby>破<rt>pò</rt></ruby>（打-破 = 打破）：<ruby>我<rt>wǒ</rt></ruby><ruby>打<rt>dǎ</rt></ruby><ruby>破<rt>pò</rt></ruby><ruby>了<rt>le</rt></ruby><ruby>花<rt>huā</rt></ruby><ruby>瓶<rt>píng</rt></ruby>（我打碎了花瓶。）
- <ruby>破<rt>pò</rt></ruby>（破[状态]）：<ruby>花<rt>huā</rt></ruby><ruby>瓶<rt>píng</rt></ruby><ruby>破<rt>pò</rt></ruby><ruby>了<rt>le</rt></ruby>（花瓶破了。）

### <ruby>被<rt>bèi</rt></ruby>被动（标记及物性）

- <ruby>花<rt>huā</rt></ruby><ruby>瓶<rt>píng</rt></ruby><ruby>被<rt>bèi</rt></ruby><ruby>我<rt>wǒ</rt></ruby><ruby>打<rt>dǎ</rt></ruby><ruby>破<rt>pò</rt></ruby><ruby>了<rt>le</rt></ruby>（花瓶被我打破了。）
- <ruby>花<rt>huā</rt></ruby><ruby>瓶<rt>píng</rt></ruby><ruby>打<rt>dǎ</rt></ruby><ruby>破<rt>pò</rt></ruby><ruby>了<rt>le</rt></ruby>（花瓶破了。——无<ruby>被<rt>bèi</rt></ruby>，语境暗示）

### 使役<ruby>让<rt>ràng</rt></ruby>/<ruby>叫<rt>jiào</rt></ruby>/<ruby>使<rt>shǐ</rt></ruby>

- <ruby>让<rt>ràng</rt></ruby><ruby>他<rt>tā</rt></ruby><ruby>笑<rt>xiào</rt></ruby>（让他笑）
- <ruby>叫<rt>jiào</rt></ruby><ruby>他<rt>tā</rt></ruby><ruby>哭<rt>kū</rt></ruby>（叫他哭）
- <ruby>使<rt>shǐ</rt></ruby><ruby>他<rt>tā</rt></ruby><ruby>生<rt>shēng</rt></ruby><ruby>气<rt>qì</rt></ruby>（使他生气——正式）

## 概览对比

| 特征 | 西班牙语 | 英语 | 汉语 |
|---|---|---|---|
| 非及物化 | se + 动词 | 兼类动词（broke/broke） | 宾语省略 / 语境 |
| 使役 | hacer + 不定式 | make/let + 动词 | <ruby>让<rt>ràng</rt></ruby>/<ruby>叫<rt>jiào</rt></ruby>/<ruby>使<rt>shǐ</rt></ruby> + 人 + 动词 |
| 宾语省略 | 比英语自由 | 较不自由 | 非常自由 |
| 睡觉（不及物） | dormir | sleep | <ruby>睡<rt>shuì</rt></ruby><ruby>觉<rt>jué</rt></ruby>（V+N复合词） |
| 意外动作 | se me + 动词 | I dropped it | <ruby>打<rt>dǎ</rt></ruby><ruby>破<rt>pò</rt></ruby><ruby>了<rt>le</rt></ruby>（语境） |
| 中动语态 | se + 动词容易 | -s/-ing被动 | <ruby>容<rt>róng</rt></ruby><ruby>易<rt>yì</rt></ruby> + 动词 |
| V-N复合词 | 少 | 少 | 多（睡觉=<ruby>睡<rt>shuì</rt></ruby><ruby>觉<rt>jué</rt></ruby>） |

## 语境示例

### 我打碎了花瓶 / 花瓶破了
- **ES**：*Rompí el jarrón.* / *El jarrón **se** rompió.*
- **EN**：*I broke the vase.* / *The vase broke.*
- **ZH**：<ruby>我<rt>wǒ</rt></ruby><ruby>打<rt>dǎ</rt></ruby><ruby>破<rt>pò</rt></ruby><ruby>了<rt>le</rt></ruby><ruby>花<rt>huā</rt></ruby><ruby>瓶<rt>píng</rt></ruby>。 / <ruby>花<rt>huā</rt></ruby><ruby>瓶<rt>píng</rt></ruby><ruby>破<rt>pò</rt></ruby><ruby>了<rt>le</rt></ruby>。

### 杯子掉了（非故意）
- **ES**：**Se** me cayó el vaso.
- **EN**：I dropped the glass. / The glass slipped from my hand.
- **ZH**：<ruby>杯<rt>bēi</rt></ruby><ruby>子<rt>zi</rt></ruby><ruby>掉<rt>diào</rt></ruby><ruby>了<rt>le</rt></ruby>。（杯子掉了。——语境暗示非故意）

### 这本书卖得好
- **ES**：*El libro **se** vende bien.*
- **EN**：*The book sells well.*
- **ZH**：<ruby>这<rt>zhè</rt></ruby><ruby>本<rt>běn</rt></ruby><ruby>书<rt>shū</rt></ruby><ruby>很<rt>hěn</rt></ruby><ruby>好<rt>hǎo</rt></ruby><ruby>卖<rt>mài</rt></ruby>。（这本书很好卖。——<ruby>好<rt>hǎo</rt></ruby><ruby>卖<rt>mài</rt></ruby> = 容易卖）

## 常见错误

1. **英语学习者学西班牙语**：*El jarrón rompió* → *El jarrón **se** rompió*

2. **西班牙语学习者学英语**：*I dropped the glass* 当想说它自己掉落时 → *The glass **fell*** / *I **dropped** the glass*（暗示我的过错）

3. **英语学习者学汉语**：*<ruby>我<rt>wǒ</rt></ruby><ruby>睡<rt>shuì</rt></ruby>* → *<ruby>我<rt>wǒ</rt></ruby>**<ruby>睡<rt>shuì</rt></ruby><ruby>觉<rt>jué</rt></ruby>***（睡觉是复合词）

4. **汉语学习者学英语**：*I eat already* 无宾语 → *I **ate** / I **had dinner***
