## 概述

连词连接单词、短语或分句。每种语言都有核心并列连词（和、但是、或者）和从属连词（因为、如果、虽然）。

- **西班牙语**：区分 *pero* 和 *sino*。多个连词触发虚拟式。在 *i* 和 *o* 开头的词前分别使用 *e* 和 *u*。
- **英语**：允许自由使用句首连词。协调和从属使用相同连词。短语连词很常见（*as well as, in order that*）。
- **中文**：连词库存较小。当语境明确关系时常省略连词。使用配对结构（*<ruby>虽<rt>suī</rt></ruby><ruby>然<rt>rán</rt></ruby>...<ruby>但<rt>dàn</rt></ruby><ruby>是<rt>shì</rt></ruby>*, *<ruby>因<rt>yīn</rt></ruby><ruby>为<rt>wéi</rt></ruby>...<ruby>所<rt>suǒ</rt></ruby><ruby>以<rt>yǐ</rt></ruby>*）。

## 西班牙语

### 并列连词

| 连词 | 含义 | 例句 |
|---|---|---|
| y | 和 | *Juan **y** María* |
| e | 和（i-前） | *padre **e** hijo* |
| o | 或者 | * café **o** té* |
| u | 或者（o-/ho-前） | *siete **u** ocho* |
| pero | 但是 | *es caro, **pero** bueno* |
| sino | 而是 | *no es rojo, **sino** verde* |
| sino que | 而是（分句） | *no vino, **sino que** llamó* |

**Pero vs sino**：

- *Es caro, **pero** lo compro.* (很贵，但我买了。——对比)
- *No es caro, **sino** barato.* (不贵，而是便宜。——修正)
- *No vino, **sino que** se quedó.* (他没来，而是留下了。——动词修正)

**规则**：当第一个分句不是否定时用 *pero*。否定后用 *sino* 来修正。

### 从属连词

| 连词 | 含义 | 需要虚拟式？ |
|---|---|---|
| porque | 因为 | 否 |
| ya que / puesto que | 既然，鉴于 | 否 |
| aunque | 虽然 | 有时（不确定） |
| cuando | 当...时 | 有时（将来/不确定） |
| si | 如果 | 是（假设） |
| mientras | 当...时 | 有时 |
| a menos que | 除非 | 是 |
| con tal (de) que | 只要 | 是 |
| para que | 为了 | 是 |
| sin que | 没有 | 是 |
| antes (de) que | 在...之前 | 是 |
| hasta que | 直到 | 是（将来） |

**虚拟式触发**：当连词引入尚未实现或不确定的动作时：

- *Ven **cuando puedas**.* (你能来的时候就来。——虚拟式：未指定时间)
- *Venía **cuando podía**.* (我以前能来的时候就来。——陈述式：过去习惯)
- *Te llamaré **antes de que salgas**.* (你走之前我会给你打电话。——虚拟式：将来)

### 配对连词

- *no solo... **sino (también)**...* (不仅...而且...)
- ***tanto**... **como**...* (既...又... / 和...一样)
- ***ni**... **ni**...* (既不...也不...)

## 英语

### FANBOYS（并列连词）

| 连词 | 含义 | 例句 |
|---|---|---|
| **F**or | 原因（罕见） | *I stayed, **for** I was tired.* (正式) |
| **A**nd | 添加 | *coffee **and** tea* |
| **N**or | 否定添加 | *neither here **nor** there* |
| **B**ut | 对比 | *small **but** strong* |
| **O**r | 选择 | *tea **or** coffee* |
| **Y**et | 对比 | *expensive, **yet** worth it* |
| **S**o | 结果 | *tired, **so** I left* |

**重要**：FANBOYS 在连接两个独立分句时，前面需要逗号：

- *I came**,** and I saw.*
- *I was tired**,** so I left.*

### 从属连词

| 类别 | 连词 |
|---|---|
| 时间 | when, while, as, before, after, since, until, once |
| 条件 | if, unless, provided that, as long as |
| 原因 | because, since, as, now that |
| 对比 | although, though, even though, whereas, while |
| 目的 | so that, in order that |
| 结果 | so... that, such... that |
| 方式 | as if, as though, like |
| 地点 | where, wherever |

### 关联连词

- *either... **or**...* (要么...要么...)
- *neither... **nor**...* (既不...也不...)
- *not only... **but also**...* (不仅...而且...)
- *both... **and**...* (既...又...)
- *whether... **or**...* (是否...)
- *as... **as**...* (和...一样)
- *so... **that**...* (如此...以至于...)
- *such... **that**...* (如此...以至于...)

### That 省略

英语在名词性分句中常省略 *that*：

- *I think **(that)** he's right.*
- *She said **(that)** she would come.*

## 常见错误

1. **英语母语者学西班牙语**：否定修正后用 pero：*No es rojo, pero verde* → *No es rojo, **sino** verde*

2. **西班牙语母语者学英语**：sino 没有直接对应：*No es rojo, sino verde* → *It's not red, **but** green*（英语用 but 同时表示两者）

3. **英语母语者学中文**：在 虽然 分句中省略 但是：*虽然下雨，我还是去了* 可接受，但 *虽然下雨，**但是**我还是去了* 更完整

4. **英语母语者学西班牙语**：在表示怀疑的 aunque 后用陈述式：*Aunque **viene**, no importa* → *Aunque **venga**, no importa*（如果来是不确定的）
