## 概述

形容词描述名词。每种语言放置形容词的位置不同，标记一致性的方式也不同。

- **西班牙语**：大多数形容词跟在名词后，随性和数一致。名词前的位置改变含义（主观 vs 客观）。
- **英语**：形容词总是在名词前，按固定顺序。无一致性。
- **中文**：形容词在名词前，用 <ruby>的<rt>de</rt></ruby> 作为连接词。中文形容词还可以充当状态动词。

## 西班牙语

### 位置：名词后（默认）

大多数西班牙语形容词跟在名词后：

- *una casa **grande*** (大房子)
- *un coche **rojo*** (红汽车)
- *una mujer **alta*** (高个子女人)

### 位置：名词前（含义改变）

有些形容词根据位置改变含义：

| 名词后（客观） | 名词前（主观/强调） |
|---|---|
| *un hombre **grande*** (一个高大的人) | *un **gran** hombre* (一个伟大的人) |
| *una casa **antigua*** (一座老房子) | *una **antigua** casa* (一座以前的房子) |

**规则**：名词前 = 主观、强调或惯用表达。名词后 = 字面、描述性。

### 一致性

形容词与名词保持性和数一致：

| | 阳性 | 阴性 |
|---|---|---|
| 单数 | el libro **rojo** | la mesa **roja** |
| 复数 | los libros **rojos** | las mesas **rojas** |

以 -e 或辅音结尾的形容词不随性别变化：

- *un coche **verde*** / *una mesa **verde***
- *un hombre **joven*** / *una mujer **joven***

但仍然需要变复数：

- *los coches **verdes*** / *las mesas **verdes***

### 比较级和最高级

**比较级**（更/更少）：

- *más **alto** que* (比...高)
- *menos **caro** que* (比...便宜)
- *tan **rápido** como* (和...一样快)

**最高级**（最）：

- *el **más** alto* (最高的)
- *la **menos** cara* (最便宜的)

**绝对最高级**（-ísimo）：

- *altísimo* (极高)
- *rapidísimo* (极快)

### 指小词和指大词

西班牙语通过加后缀构成：

- *pequeño* → *pequeñ**ito*** (很小，可爱)
- *grande* → *grand**ote*** (巨大)

## 英语

### 位置：总是在名词前

英语形容词总是在名词前：

- *a **big** house* (大房子)
- *a **red** car* (红汽车)
- *a **beautiful** woman* (美丽的女人)

### 多个形容词的顺序

英语有严格的形容词顺序：

**观点 → 大小 → 年龄 → 形状 → 颜色 → 来源 → 材料 → 用途 → 名词**

- *a **lovely little old rectangular brown French oak writing desk***

实际上，超过三个形容词连用很少见。

### 无一致性

英语形容词**不变**：

- *a big house / two big houses*
- *the red car / the red cars*

### 比较级和最高级

**短形容词**（单音节，或以 -y 结尾的双音节）：

| 原级 | 比较级 | 最高级 |
|---|---|---|
| tall | tall**er** | tall**est** |
| fast | fast**er** | fast**est** |
| happy | happi**er** | happi**est** |

**长形容词**（两个或更多音节）：

| 原级 | 比较级 | 最高级 |
|---|---|---|
| beautiful | **more** beautiful | **the most** beautiful |

**不规则**：

| 原级 | 比较级 | 最高级 |
|---|---|---|
| good | better | the best |
| bad | worse | the worst |

### 形容词还是副词？

有些词既是形容词又是副词（形式相同）：

- *a **fast** car* (形容词) / *He drives **fast**.* (副词)

其他需要加 -ly 变为副词：

- *a **quick** meal* / *He ate **quickly**.*

## 中文

### 位置：名词前 + <ruby>的<rt>de</rt></ruby>

中文形容词在名词前，用 <ruby>的<rt>de</rt></ruby> 连接：

- <ruby>大<rt>dà</rt></ruby><ruby>的<rt>de</rt></ruby><ruby>房<rt>fáng</rt></ruby><ruby>子<rt>zi</rt></ruby> (大房子)
- <ruby>红<rt>hóng</rt></ruby><ruby>的<rt>de</rt></ruby><ruby>车<rt>chē</rt></ruby> (红汽车)
- <ruby>漂<rt>piào</rt></ruby><ruby>亮<rt>liang</rt></ruby><ruby>的<rt>de</rt></ruby><ruby>女<rt>nǚ</rt></ruby><ruby>人<rt>rén</rt></ruby> (漂亮女人)

### 省略 <ruby>的<rt>de</rt></ruby>

关系近或常见表达中，<ruby>的<rt>de</rt></ruby> 可以省略：

- <ruby>好<rt>hǎo</rt></ruby><ruby>朋<rt>péng</rt></ruby><ruby>友<rt>yǒu</rt></ruby> (好朋友) — <ruby>的<rt>de</rt></ruby> 省略
- <ruby>老<rt>lǎo</rt></ruby><ruby>师<rt>shī</rt></ruby> (老师) — 不用 <ruby>的<rt>de</rt></ruby>

**规则**：单音节形容词 + 亲密关系名词 → <ruby>的<rt>de</rt></ruby> 可选。否则保留 <ruby>的<rt>de</rt></ruby>。

### 状态动词

中文形容词可以充当动词：

- <ruby>他<rt>tā</rt></ruby><ruby>很<rt>hěn</rt></ruby><ruby>高<rt>gāo</rt></ruby>。(他很高。)
- <ruby>这<rt>zhè</rt></ruby><ruby>个<rt>ge</rt></ruby><ruby>苹<rt>píng</rt></ruby><ruby>果<rt>guǒ</rt></ruby><ruby>很<rt>hěn</rt></ruby><ruby>红<rt>hóng</rt></ruby>。(这个苹果很红。)

**重要**：<ruby>很<rt>hěn</rt></ruby> 通常是语法填充词，不是强调的"很"。要强调，用 <ruby>非<rt>fēi</rt></ruby><ruby>常<rt>cháng</rt></ruby> 或 <ruby>特<rt>tè</rt></ruby><ruby>别<rt>bié</rt></ruby>：

- <ruby>非<rt>fēi</rt></ruby><ruby>常<rt>cháng</rt></ruby><ruby>好<rt>hǎo</rt></ruby> (非常好)
- <ruby>特<rt>tè</rt></ruby><ruby>别<rt>bié</rt></ruby><ruby>大<rt>dà</rt></ruby> (特别大)

### 比较级和最高级

**比较级**：

- <ruby>更<rt>gèng</rt></ruby> + 形容词: <ruby>更<rt>gèng</rt></ruby><ruby>高<rt>gāo</rt></ruby> (更高)
- <ruby>比<rt>bǐ</rt></ruby> + 名词 + 形容词: <ruby>我<rt>wǒ</rt></ruby><ruby>比<rt>bǐ</rt></ruby><ruby>你<rt>nǐ</rt></ruby><ruby>高<rt>gāo</rt></ruby> (我比你高。)

**最高级**：

- <ruby>最<rt>zuì</rt></ruby> + 形容词: <ruby>最<rt>zuì</rt></ruby><ruby>高<rt>gāo</rt></ruby> (最高)
- <ruby>最<rt>zuì</rt></ruby><ruby>好<rt>hǎo</rt></ruby><ruby>的<rt>de</rt></ruby><ruby>学<rt>xué</rt></ruby><ruby>生<rt>shēng</rt></ruby> (最好的学生)

### 重叠

中文形容词可以重叠表示强调或亲昵：

- <ruby>慢<rt>màn</rt></ruby><ruby>慢<rt>màn</rt></ruby><ruby>的<rt>de</rt></ruby> (慢慢地)
- <ruby>漂<rt>piào</rt></ruby><ruby>漂<rt>piào</rt></ruby><ruby>亮<rt>liang</rt></ruby><ruby>亮<rt>liang</rt></ruby><ruby>的<rt>de</rt></ruby> (漂漂亮亮的)

## 一览对比

| 特征 | 西班牙语 | 英语 | 中文 |
|---------|---------|---------|---------|
| 位置 | 通常在名词后 | 在名词前 | 在名词前 + <ruby>的<rt>de</rt></ruby> |
| 一致性 | 性 + 数 | 无 | 无 |
| 多个形容词 | 有限 | 固定严格顺序 | 灵活 |
| 比较级 | más / menos + adj | more / -er + adj | <ruby>更<rt>gēng</rt></ruby> + adj / <ruby>比<rt>bǐ</rt></ruby> |
| 最高级 | el/la más + adj | the most / -est | <ruby>最<rt>zuì</rt></ruby> + adj |
| 状态动词 | 否（需要 ser/estar） | 否（需要 be） | 是（形容词作谓语） |
| 重叠 | 否 | 否 | 是 |

## 常见错误

1. **英语母语者学西班牙语**：形容词放在名词前：*un grande casa* → *una casa **grande***

2. **英语母语者学西班牙语**：忘记一致性：*las mesas rojo* → *las mesas **rojas***

3. **英语母语者学中文**：省略 <ruby>的<rt>de</rt></ruby>：应使用 <ruby>大<rt>dà</rt></ruby><ruby>的<rt>de</rt></ruby><ruby>红<rt>hóng</rt></ruby><ruby>色<rt>sè</rt></ruby><ruby>的<rt>de</rt></ruby><ruby>房<rt>fáng</rt></ruby><ruby>子<rt>zi</rt></ruby>

4. **中文母语者学英语**：形容词顺序错误：*a red big house* → *a **big red** house*
