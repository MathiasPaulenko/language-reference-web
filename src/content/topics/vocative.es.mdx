## Resumen

El vocativo es la forma o estrategia usada para dirigirse directamente a alguien.

- **Español**: No hay morfología vocativa especial. Los nombres y títulos se usan directamente. La entonación (tono ascendente, pausa con coma) marca el vocativo. *Oye, mira, venga* son llamadas de atención comunes.
- **Inglés**: No hay morfología vocativa. La dirección directa usa nombres, títulos o palabras de llamada de atención (*hey, excuse me*). La entonación separa el vocativo del resto de la oración.
- **Chino**: No hay morfología vocativa. Se usan partículas (*<ruby>喂<rt>wèi</rt></ruby>, <ruby>哎<rt>āi</rt></ruby>*) y títulos. En contextos públicos, *<ruby>不<rt>bù</rt></ruby><ruby>好<rt>hǎo</rt></ruby><ruby>意<rt>yì</rt></ruby><ruby>思<rt>sī</rt></ruby>* o *<ruby>麻<rt>má</rt></ruby><ruby>烦<rt>fán</rt></ruby><ruby>一<rt>yī</rt></ruby><ruby>下<rt>xià</rt></ruby>* suavizan la intrusión.

## Español

### Dirección directa con nombres

Los nombres se usan sin cambio:

- *María, ven aquí.*
- *Juan, escúchame.*
- *Pedro, ¿dónde estás?*

**Entonación**: Tono ascendente en el nombre, pausa ligera.

### Títulos como vocativos

- *Señor García, un momentito.*
- *Señora López, por favor.*
- *Doctor Martínez, ¿puedo hablar con usted?*
- *Don Juan, buenos días.*
- *Doña María, ¿cómo está?*
- *Profesora, tengo una pregunta.*

### Llamadas de atención

| Expresión | Uso | Registro |
|---|---|---|
| Oiga | atención (formal, a desconocido) | formal |
| Oye | atención (informal) | informal |
| Mira | mira / escucha | informal |
| Venga | vamos / oye | informal |
| Eh / Ey | oye | muy informal |
| Ea | oye (España) | informal |
| Chico / Chica | oye chico/chica | informal |
| Muchacho / Muchacha | oye chaval | informal |

### Atención educada

- *Disculpe, señor...*
- *Perdone, señora...*
- *Con permiso...*
- *Perdón...*

### Íntimo / familiar

- *Mamá, ¿dónde estás?*
- *Papá, ven.*
- *Hijo, escucha.*
- *Mi amor, te quiero.*
- *Cariño, ven aquí.*

### Cartas

- *Querido Juan:*
- *Estimado señor García:*
- *Querida familia:*

## Inglés

### Dirección directa con nombres

- *María, come here.*
- *John, listen to me.*
- *Sarah, where are you?*

**Nota**: No hay cambio de entonación en la coma en escritura. La coma marca el vocativo.

### Títulos

| Título | Uso |
|---|---|
| Mr. + apellido | hombre formal |
| Mrs. + apellido | mujer casada formal |
| Ms. + apellido | mujer formal (neutral) |
| Miss + apellido | mujer soltera formal (anticuado) |
| Dr. + apellido | doctor, PhD |
| Professor + apellido | profesor universitario |
| sir | hombre (servicio, formal) |
| ma'am | mujer (servicio, formal) |
| officer | policía |
| waiter / waitress | restaurante (en declive) |
| nurse | médico |
| Your Honor | juez |

### Llamadas de atención

| Expresión | Uso | Registro |
|---|---|---|
| Excuse me | atención, disculpa, pasar | neutral |
| Hey | atención | informal |
| Hi / Hello | saludo | neutral |
| Listen | atención | informal |
| Look | atención | informal |
| Pardon me | disculpa, atención | formal |
| Sorry | disculpa | neutral |
| Yo | atención (EE.UU.) | muy informal |

### Fórmulas educadas

- *Excuse me, sir...*
- *Pardon me, ma'am...*
- *I'm sorry to bother you...*
- *Could I have your attention, please?*

### Íntimo / familiar

- *Mom, where are you?*
- *Dad, come here.*
- *Honey, listen to me.*
- *Babe / Baby, I love you.*
- *Sweetie, come here.*
- *Guys, listen up.*

### Advertencias / urgencia

### Cartas

- *Dear John,*
- *Dear Mr. Smith,*
- *To whom it may concern,*
- *Dear Sir or Madam,*
- *Hello Maria,* (email informal)

## Errores comunes

1. **Angloparlantes aprendiendo español**: *Hola* a desconocidos para llamar atención → *Oiga* (formal) / *Oye* (informal)

2. **Hispanohablantes aprendiendo inglés**: *Oiga* en todos los contextos → *Excuse me* (formal) / *Hey* (informal)

3. **Angloparlantes aprendiendo chino**: *<ruby>你<rt>nǐ</rt></ruby><ruby>好<rt>hǎo</rt></ruby>* para toda atención → *<ruby>不<rt>bù</rt></ruby><ruby>好<rt>hǎo</rt></ruby><ruby>意<rt>yì</rt></ruby><ruby>思<rt>sī</rt></ruby>* (público) / *<ruby>喂<rt>wèi</rt></ruby>* (teléfono)

4. **Chinos aprendiendo inglés**: *<ruby>喂<rt>wèi</rt></ruby>* en persona → *Hey* / *Excuse me*
