## 概述

非人称结构表达事件、状态或义务，而不指明特定施事者。

- **西班牙语**：使用 *se*（非人称/反身被动）、第三人称复数（*dicen que*）、非人称动词（*llueve, hace frío*）和 *hay* 表示存在。主语常被完全省略。
- **英语**：使用虚主语（*it, there*）、被动语态、泛指代词（*you, one, they*）和名词化。不能省略主语。
- **汉语**：在语境允许时完全省略主语。对天气、时间和一般陈述使用隐含主语。无虚代词。

## 西班牙语

### Se + 动词（非人称）

非人称 *se* 取代主语：

- *Se habla español.* (西班牙语被使用。)
- *Se vende coches.* (汽车出售。)
- *Se dice que...* (据说……)
- *No se puede fumar.* (禁止吸烟。)

**注意**：当宾语置于动词前时，*se* 成为被动标记：

- *El español **se** habla aquí.* (西班牙语在这里被使用。)

### 非人称动词

仅存在于第三人称的西班牙语动词：

| 动词 | 含义 |
|---|---|
| llover | 下雨 |
| nevar | 下雪 |
| tronar | 打雷 |
| amanecer | 天亮 |
| anochecer | 天黑 |
| hacer + 名词 | （天气）…… |

**天气表达**：

- *Llueve.* (在下雨。)
- *Nieva.* (在下雪。)
- *Hace frío.* (天冷。)
- *Hace calor.* (天热。)
- *Hace viento.* (刮风。)
- *Hace sol.* (晴天。)

### 第三人称复数表"人们"

- *Dicen que...* (人们说…… / 据说……)
- *En mi país comen tarde.* (在我的国家，人们吃得晚。)
- *Aquí bailan mucho.* (这里人们经常跳舞。)

### Hay / estar 表存在

- *Hay un problema.* (有一个问题。)
- *Hay mucha gente.* (有很多人。)
- *No hay solución.* (没有办法。)

### 非人称表达

| 表达 | 含义 |
|---|---|
| es necesario que | 必须…… |
| es importante que | 重要的是…… |
| es posible que | 可能…… |
| hay que + 不定式 | 必须…… |
| conviene que | 最好…… |
| falta / sobra | 缺少 / 多余 |

### Se hace / se está haciendo

- *Se hace tarde.* (天晚了。)
- *Se está haciendo tarde.* (天正在变晚。)

## 英语

### 虚it

英语要求 *it* 用于天气、时间和距离：

- *It is raining.* (在下雨。)
- *It is snowing.* (在下雪。)
- *It is cold.* (天冷。)
- *It is 5 o'clock.* (五点了。)
- *It is Tuesday.* (星期二。)
- *It is a long way to Tipperary.* (到蒂珀雷里路很长。)

### 虚there

*There* 引导存在：

- *There is a problem.* (有一个问题。)
- *There are many people.* (有很多人。)
- *There is no solution.* (没有办法。)
- *There seems to be a mistake.* (似乎有个错误。)
- *There appears to be an error.* (似乎有个错误。)

### 泛指 you / one / they

- *You can't smoke here.* (一般规则)
- *One must be careful.* (正式，泛指)
- *They say he's rich.* (未指明的人)
- *People say he's rich.* (明确的泛指)

### 被动语态删去施事

- *Spanish is spoken here.* (西班牙语在这里被使用。——由未指明的人)
- *The car was stolen.* (车被偷了。——由未知的人)
- *It is believed that...* (人们相信……)
- *It is known that...* (众所周知……)

### It + 形容词 + to-不定式

- *It is important to study.* (学习很重要。)
- *It is necessary to leave.* (有必要离开。)
- *It is easy to learn.* (学起来容易。)
- *It is difficult to say.* (很难说。)

### It seems / appears / happens

- *It seems that he left.* (似乎他离开了。)
- *It appears that we were wrong.* (似乎我们错了。)
- *It happens that I know him.* (碰巧我认识他。)

### 分裂句it

- *It was John who broke the window.* (是约翰打破了窗户。)
- *It is here that we met.* (我们是在这里见面的。)

## 常见错误

1. **英语母语者学习西班牙语**：用ello/esto指天气：*Ello llueve* → *Llueve*

2. **英语母语者学习汉语**：用<ruby>它<rt>tā</rt></ruby>指天气：*<ruby>它<rt>tā</rt></ruby><ruby>下<rt>xià</rt></ruby><ruby>雨<rt>yǔ</rt></ruby><ruby>了<rt>le</rt></ruby>* → *<ruby>下<rt>xià</rt></ruby><ruby>雨<rt>yǔ</rt></ruby><ruby>了<rt>le</rt></ruby>*

3. **西班牙语母语者学习英语**：省略it：*Is raining* → *It is raining*

4. **汉语母语者学习英语**：省略there/it：*Has a problem* → *There is a problem*
