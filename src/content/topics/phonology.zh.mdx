## 概述

音系学是研究语言中声音系统的学科。

- **西班牙语**：简单的元音系统（5个纯元音）。辅音大多规则。重音可从拼写预测。无复杂辅音丛。
- **英语**：复杂的元音系统（约20个元音）。许多辅音丛。重音不可预测且是词汇性的。无声调。
- **汉语**：简单的元音系统但有词汇声调（4个声调+轻声）。无复杂辅音丛。韵尾辅音限于-n, -ng, -r。

## 一览比较

| 特征 | 西班牙语 | 英语 | 汉语 |
|---|---|---|---|
| 元音 | 5个纯元音 | ~20个（纯元音+双元音） | ~10个+声调 |
| 辅音 | ~19个 | ~24个 | ~21个 |
| 音节 | (C)V(C) | CCCVCCCC | (C)(G)V(N) |
| 声调 | 无 | 无 | 有（4个+轻声） |
| 重音 | 可预测 | 不可预测 | 无词汇重音 |
| r音 | 闪音/颤音/ɾ, r/ | 近音/ɹ/ | 卷舌音/ʂ, ʈʂ, ʐ/ |
| 塞擦音 | /tʃ/ | /tʃ, dʒ/ | /ts, tʂ, tɕ/系列 |

## 语境中的例子

### 语音系统
- **ES**：5个元音，19个辅音，可预测的重音
- **EN**：~20个元音，24个辅音，不可预测的重音
- **ZH**：~10个元音+4个声调，21个声母，简单音节结构

### 最小对立对
- **ES**：*pasa*（经过） / *paza*（非词） / *pasa*（葡萄干）——仅重音
- **EN**：*ship* / *sheep*; *bad* / *bed*; *beat* / *bit*
- **ZH**：<ruby>妈<rt>mā</rt></ruby> / <ruby>麻<rt>má</rt></ruby> / <ruby>马<rt>mǎ</rt></ruby> / <ruby>骂<rt>mà</rt></ruby> ——声调区分意义

### 重音
- **ES**：*término*（我结束） / *terminó*（他结束了）
- **EN**：*ˈrecord*（名词） / *reˈcord*（动词）
- **ZH**：无词汇重音；用声调代替

## 常见错误

1. **英语母语者学西班牙语**：双元音化元音 → 需要单元音

2. **西班牙语母语者学英语**：使用5元音系统 → 需要学习约20个元音

3. **英语母语者学汉语**：忽视声调 → <ruby>妈<rt>mā</rt></ruby> vs <ruby>骂<rt>mà</rt></ruby> 是不同的词

4. **汉语母语者学英语**：语调平淡 → 使用重音和音高变化
