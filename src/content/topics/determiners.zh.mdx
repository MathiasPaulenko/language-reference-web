## 概述

限定词是引入名词并表达指称、数量或远近的词语。

- **西班牙语**：冠词 (*el, la, un, una*) 有性数一致。指示词区分三个距离等级 (*este, ese, aquel*)。领属形容词有完整代词形式。
- **英语**：冠词 (*the, a, an*) 不变格。指示词分两个等级 (*this, that*)。量词区分可数与不可数名词 (*many, much, few, little*)。
- **中文**：没有冠词。指示词 (*<ruby>这<rt>zhè</rt></ruby>, <ruby>那<rt>nà</rt></ruby>*) 需要量词。数词需要量词。领属关系用 *<ruby>的<rt>de</rt></ruby>* 标记。

## 西班牙语

### 冠词

| | 阳性 | 阴性 |
|---|---|---|
| 定指单数 | el | la |
| 定指复数 | los | las |
| 不定指单数 | un | una |
| 不定指复数 | unos | unas |

**例子**：
- *el libro, la mesa, los niños, las niñas*
- *un problema, una mano, unos días, unas cosas*

**以重音 a- 开头的阴性名词用 el**：
- *el agua, el hacha, el águila* (但: *las aguas frías*)

### 指示词

三个距离等级：

| | 阳性 | 阴性 | 含义 |
|---|---|---|---|
| este | este | esta | 这个（近说话者）|
| ese | ese | esa | 那个（近听话者）|
| aquel | aquel | aquella | 那个（远处）|

**复数**：
- *estos/estas, esos/esas, aquellos/aquellas*

**中性**（用于抽象概念）：
- *esto, eso, aquello*
- *Esto es importante.* (这很重要。)

### 领属形容词

| | 名词前 | 独立使用 |
|---|---|---|
| 我的 | mi(s) | mío(s)/mía(s) |
| 你的 | tu(s) | tuyo(s)/tuya(s) |
| 他/她/它的 | su(s) | suyo(s)/suya(s) |
| 我们的 | nuestro(s)/nuestra(s) | nuestro(s)/nuestra(s) |
| 你们的 | vuestro(s)/vuestra(s) | vuestro(s)/vuestra(s) |
| 他们的 | su(s) | suyo(s)/suya(s) |

**名词前**：
- *mi libro, tus amigos, su casa, nuestra familia*

**独立使用**：
- *El libro es mío.* (这本书是我的。)
- *Esas son nuestras.* (那些是我们的。)

### 量词

| | 含义 | 例子 |
|---|---|---|
| todo(s)/toda(s) | 全部 | *todos los libros* |
| cada | 每个 | *cada día* |
| algunos/algunas | 一些 | *algunos libros* |
| ningún/ninguna | 没有一个 | *ningún problema* |
| muchos/muchas | 许多 | *muchos amigos* |
| pocos/pocas | 很少 | *pocas veces* |
| varios/varias | 几个 | *varias personas* |

**注意**：*Cada* 总是单数；*ningún* 在阳性单数前缩短。

### 数词

- *un libro, dos libros, tres libros, diez libros, cien libros*
- *primero* → *el primer libro* (在阳性单数前缩短)

## 英语

### 冠词

| 冠词 | 用法 | 例子 |
|---|---|---|
| the | 定指（特定）| the book |
| a | 不定指（单数可数，辅音开头）| a book |
| an | 不定指（单数可数，元音开头）| an apple |
| 零冠词 | 不定指（复数、不可数、抽象）| books, water, love |

**注意**：英语冠词不与名词一致。

### 指示词

| | 单数 | 复数 |
|---|---|---|
| 近指 | this | these |
| 远指 | that | those |

**例子**：
- *this book, these books, that idea, those people*

### 领属形容词

| 名词前 | 独立使用 |
|---|---|
| my | mine |
| your | yours |
| his | his |
| her | hers |
| its | (独立使用罕见) |
| our | ours |
| their | theirs |

**名词前**：
- *my book, your friend, her house*

**独立使用**：
- *The book is mine.*
- *That house is ours.*

### 量词

**与可数名词连用**：

| 量词 | 例子 |
|---|---|
| many | many books |
| a few | a few friends |
| few | few problems |
| several | several people |
| both | both students |
| each/every | each book, every day |
| all (of) | all books / all of the books |

**与不可数名词连用**：

| 量词 | 例子 |
|---|---|
| much | much water |
| a little | a little time |
| little | little money |
| some | some water |
| any | any water (否定/疑问) |
| no | no water |
| all (of) | all water / all of the water |

**两者皆可**：

| 量词 | 例子 |
|---|---|
| some | some books / some water |
| any | any books / any water |
| no | no books / no water |
| all | all books / all water |
| enough | enough time / enough books |
| plenty of | plenty of time / plenty of books |
| a lot of / lots of | a lot of books / a lot of water |

### 数词

- *one book, two books, three books, ten books, a hundred books*
- *the first book, the second chapter, the third time*

## 中文

### 没有冠词

中文没有冠词：

- <ruby>书<rt>shū</rt></ruby> = 书 / 这本书 / 一本书（语境决定）
- <ruby>我<rt>wǒ</rt></ruby><ruby>买<rt>mǎi</rt></ruby><ruby>书<rt>shū</rt></ruby> = 我买了书 / 我买了书们 / 我买了那本书

**定指通过以下方式表达**：
- 指示词：<ruby>这<rt>zhè</rt></ruby><ruby>本<rt>běn</rt></ruby><ruby>书<rt>shū</rt></ruby> (这本书)
- 前文语境
- 主语-述语结构

### 指示词

| | 带量词 | 不带（抽象）|
|---|---|---|
| 这 | <ruby>这<rt>zhè</rt></ruby> + 量词 | <ruby>这<rt>zhè</rt></ruby> |
| 那 | <ruby>那<rt>nà</rt></ruby> + 量词 | <ruby>那<rt>nà</rt></ruby> |
| 这些 | <ruby>这<rt>zhè</rt></ruby><ruby>些<rt>xiē</rt></ruby> | — |
| 那些 | <ruby>那<rt>nà</rt></ruby><ruby>些<rt>xiē</rt></ruby> | — |

**例子**：
- <ruby>这<rt>zhè</rt></ruby><ruby>本<rt>běn</rt></ruby><ruby>书<rt>shū</rt></ruby> (这本书)
- <ruby>那<rt>nà</rt></ruby><ruby>个<rt>ge</rt></ruby><ruby>人<rt>rén</rt></ruby> (那个人)
- <ruby>这<rt>zhè</rt></ruby><ruby>些<rt>xiē</rt></ruby><ruby>书<rt>shū</rt></ruby> (这些书)
- <ruby>那<rt>nà</rt></ruby><ruby>些<rt>xiē</rt></ruby><ruby>人<rt>rén</rt></ruby> (那些人)

### 领属关系

**领属 + <ruby>的<rt>de</rt></ruby> + 名词**：

- <ruby>我<rt>wǒ</rt></ruby><ruby>的<rt>de</rt></ruby><ruby>书<rt>shū</rt></ruby> (我的书)
- <ruby>你<rt>nǐ</rt></ruby><ruby>的<rt>de</rt></ruby><ruby>朋<rt>péng</rt></ruby><ruby>友<rt>yǒu</rt></ruby> (你的朋友)
- <ruby>他<rt>tā</rt></ruby><ruby>的<rt>de</rt></ruby><ruby>家<rt>jiā</rt></ruby> (他的家)

**注意**：亲属关系中 <ruby>的<rt>de</rt></ruby> 可省略：
- <ruby>我<rt>wǒ</rt></ruby><ruby>妈<rt>mā</rt></ruby> (我妈 — 不说 <ruby>我<rt>wǒ</rt></ruby><ruby>的<rt>de</rt></ruby><ruby>妈<rt>mā</rt></ruby>)
- <ruby>我<rt>wǒ</rt></ruby><ruby>爸<rt>bà</rt></ruby> (我爸)
- <ruby>我<rt>wǒ</rt></ruby><ruby>们<rt>men</rt></ruby><ruby>学<rt>xué</rt></ruby><ruby>校<rt>xiào</rt></ruby> (我们学校)

### 量词

| | 含义 | 例子 |
|---|---|---|
| <ruby>每<rt>měi</rt></ruby> | 每 | <ruby>每<rt>měi</rt></ruby><ruby>天<rt>tiān</rt></ruby> (每天) |
| <ruby>每<rt>měi</rt></ruby> + 量词 | 每（可数）| <ruby>每<rt>měi</rt></ruby><ruby>本<rt>běn</rt></ruby><ruby>书<rt>shū</rt></ruby> (每本书) |
| <ruby>所<rt>suǒ</rt></ruby><ruby>有<rt>yǒu</rt></ruby> | 所有 | <ruby>所<rt>suǒ</rt></ruby><ruby>有<rt>yǒu</rt></ruby><ruby>的<rt>de</rt></ruby><ruby>书<rt>shū</rt></ruby> (所有的书) |
| <ruby>一<rt>yī</rt></ruby><ruby>切<rt>qiē</rt></ruby> | 一切（抽象）| <ruby>一<rt>yí</rt></ruby><ruby>切<rt>qiè</rt></ruby> (一切) |
| <ruby>一<rt>yī</rt></ruby><ruby>些<rt>xiē</rt></ruby> | 一些 | <ruby>一<rt>yì</rt></ruby><ruby>些<rt>xiē</rt></ruby><ruby>人<rt>rén</rt></ruby> (一些人) |
| <ruby>很<rt>hěn</rt></ruby><ruby>多<rt>duō</rt></ruby> | 很多 | <ruby>很<rt>hěn</rt></ruby><ruby>多<rt>duō</rt></ruby><ruby>书<rt>shū</rt></ruby> (很多书) |
| <ruby>几<rt>jī</rt></ruby><ruby>个<rt>gè</rt></ruby> | 几个 | <ruby>几<rt>jǐ</rt></ruby><ruby>个<rt>ge</rt></ruby><ruby>人<rt>rén</rt></ruby> (几个人) |
| <ruby>没<rt>méi</rt></ruby><ruby>有<rt>yǒu</rt></ruby> | 没有 | <ruby>没<rt>méi</rt></ruby><ruby>有<rt>yǒu</rt></ruby><ruby>书<rt>shū</rt></ruby> (没有书) |
| <ruby>别<rt>bié</rt></ruby><ruby>的<rt>de</rt></ruby> | 别的 | <ruby>别<rt>bié</rt></ruby><ruby>的<rt>de</rt></ruby><ruby>书<rt>shū</rt></ruby> (别的书) |

### 数词

- <ruby>一<rt>yì</rt></ruby><ruby>本<rt>běn</rt></ruby><ruby>书<rt>shū</rt></ruby> (一本书)
- <ruby>两<rt>liǎng</rt></ruby><ruby>本<rt>běn</rt></ruby><ruby>书<rt>shū</rt></ruby> (两本书 — 用 <ruby>两<rt>liǎng</rt></ruby>, 不用 <ruby>二<rt>èr</rt></ruby>)
- <ruby>三<rt>sān</rt></ruby><ruby>本<rt>běn</rt></ruby><ruby>书<rt>shū</rt></ruby> (三本书)
- <ruby>十<rt>shí</rt></ruby><ruby>本<rt>běn</rt></ruby><ruby>书<rt>shū</rt></ruby> (十本书)

**注意**：量词前用 <ruby>两<rt>liǎng</rt></ruby>；不用量词时用 <ruby>二<rt>èr</rt></ruby>。

|---|---|---|---|

| 大量 | mucho/mucha | much (不可数) | <ruby>很<rt>hěn</rt></ruby><ruby>多<rt>duō</rt></ruby> (相同) |

## 常见错误

1. **学英语的西班牙语者**：I bought book → I bought **a** book（需要冠词）

2. **学西班牙语的英语者**：el problema → 正确！但 la problema → el **problema**（阳性）

3. **学中文的英语者**：<ruby>这<rt>zhè</rt></ruby><ruby>是<rt>shì</rt></ruby><ruby>一<rt>yī</rt></ruby><ruby>书<rt>shū</rt></ruby> → <ruby>这<rt>zhè</rt></ruby>**<ruby>一<rt>yī</rt></ruby>**<ruby>本<rt>běn</rt></ruby>**<ruby>书<rt>shū</rt></ruby>**（需要量词）

4. **学英语的西班牙语者**：I have many waters → I have **a lot of** water（many 与 much 的区别）
