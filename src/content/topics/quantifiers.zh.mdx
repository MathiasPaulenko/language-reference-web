## 概述

量词表达数量：所有、一些、没有、许多、很少。它们以复杂方式与否定互动。

- **西班牙语**：Alguno（一些/任何）在否定语境变为 ninguno（没有）。Todo 有性数一致。否定一致意味着多个否定互相强化。
- **英语**：Some/any 区分取决于极性（肯定、否定、疑问）。Few/a few 和 little/a little 含义相反。无否定一致。
- **汉语**：都 是全称量化副词。什么...都/也 构成全称否定。无数/量区别。否定用 不/没 取决于体。

## 西班牙语

### 全称量词

| 量词 | 用法 | 例句 |
|---|---|---|
| todo / toda | 整个集合 | **Todo** el mundo vino |
| todos / todas | 整个集合（复数） | **Todos** los días |
| cada | 每个元素 | **Cada** persona |

### 存在量词

| 量词 | 用法 | 例句 |
|---|---|---|
| algún / alguna | 某（肯定） | Tengo **algún** problema |
| algunos / algunas | 某些（肯定） | **Algunos** vinieron |
| algo | 某物（不可数） | Quiero **algo** de agua |
| alguien | 某人 | **Alguien** llamó |

### 否定量词

| 量词 | 用法 | 例句 |
|---|---|---|
| ningún / ninguna | 没有（与 no 连用） | No tengo **ningún** problema |
| nadie | 没人 | No vino **nadie** |
| nada | 没有 | No hizo **nada** |

### 否定一致

西班牙语中，多个否定互相强化：

- No vino **nadie**.
- No hizo **nada**.
- No tengo **ningún** problema.
- **Nunca** dijo **nada** a **nadie**.

**注**：与英语不同，西班牙语允许且偏好同一句话中多个否定。

### Mucho / poco

| 量词 | 含义 | 例句 |
|---|---|---|
| mucho / muchos | 很多 | Tengo **mucho** dinero |
| poco / pocos | 很少 | Tengo **poco** dinero |
| bastante / bastantes | 足够 | Tengo **bastante** tiempo |

## 英语

### Some vs any

| 极性 | 使用 | 例句 |
|---|---|---|
| 肯定 | some | I have **some** money. |
| 否定 | any | I don't have **any** money. |
| 疑问 | any | Do you have **any** money? |
| 提供 | some | Would you like **some** tea? |

**注**：当期待肯定回答时，some 也可用于疑问句。

### Few / a few

| 量词 | 含义 | 蕴含 |
|---|---|---|
| **few** | 很少 | 负面（几乎没有） |
| **a few** | 一些 | 正面（足够） |
| **little** | 很少 | 负面（几乎没有） |
| **a little** | 一点 | 正面（足够） |

**例句**：
- I have **few** friends. (= 几乎没有，感到孤独)
- I have **a few** friends. (= 有一些，很好)

### All / every / each

| 量词 | 用法 | 例句 |
|---|---|---|
| **all** | 整个集合 | **All** students passed. |
| **every** | 每个元素（单数） | **Every** student passed. |
| **each** | 每个元素（个体） | **Each** student passed. |

### Neither / either / both

| 量词 | 含义 | 例句 |
|---|---|---|
| **both** | 两者都 | **Both** books are good. |
| **either** | 两者任一 | **Either** book is fine. |
| **neither** | 两者都不 | **Neither** book is good. |

## 快速比较

| 特征 | 西班牙语 | 英语 | 汉语 |
|---|---|---|---|
| 全称 | todo(s) / toda(s) | all / every / each | 都 / 所有 |
| 存在 | algún / algunos / algo | some / a few / several | 一些 / 几个 |
| 任何 | alguno / ninguno | any | 任何 / 什么 |
| 没有 | ninguno / nadie / nada | none / nobody / nothing | 没有 / 什么都不 |
| 两者 | ambos / los dos | both / either / neither | 两个都 / 两边 |
| 否定一致 | 有 | 无 | 无（用 都） |
| 可数/不可数 | 有（muchos/mucho） | 有（many/much） | 无 |

## 常见错误

1. **西班牙语母语者**：英语否定中用 "some" → I don't have **any** money（不用 some）

2. **英语母语者**：西班牙语否定中用 "alguno" → No tengo **ninguno**（不用 alguno）

3. **英语母语者**：汉语全称中省略 <ruby>都<rt>dū</rt></ruby> → <ruby>大<rt>dà</rt></ruby><ruby>家<rt>jiā</rt></ruby>**<ruby>都<rt>dōu</rt></ruby>**<ruby>来<rt>lái</rt></ruby><ruby>了<rt>le</rt></ruby>（不能只说 大家来了）

4. **汉语母语者**：将 什么 直译为 "any" → <ruby>什<rt>shén</rt></ruby><ruby>么<rt>me</rt></ruby> + <ruby>都<rt>dū</rt></ruby> + Neg = "nada / ninguno"
