## 概述

书写系统将口语以视觉形式编码。

- **西班牙语**：音位字母（拉丁文字）。拼写高度规则，音形对应一致。变音符号标记重音并区分同音词。
- **英语**：音位字母（拉丁文字）但深层正字法。历史拼写保留词源。大量不规则和静音字母。
- **汉语**：表意文字。字符代表语素，不直接表音。拼音提供语音转写。简体和繁体。

### 关键拼写规则

| 规则 | 例词 | 说明 |
|---|---|---|
| ca, co, cu | casa, cosa, cuna | c = /k/ 在 a, o, u 前 |
| que, qui | queso, quince | qu = /k/ 在 e, i 前 |
| ga, go, gu | gato, goma, gusto | g = /g/ 在 a, o, u 前 |
| gue, gui | guerra, guitarra | gu = /g/ 在 e, i 前 |
| güe, güi | vergüenza, pingüino | ü 标记 /w/ |
| z | zapato, zona | z = /θ/ 或 /s/ |

| 功能 | 例词 |
|---|---|
| 音位重音 | término, terminó, término |

| 疑问/感叹词 | qué, cuál, cómo, cuánto |

- 2010：取消 ch 和 ll 作为独立字母

## 常见错误

1. **英语母语者**：西班牙语大写 → lunes, martes（不用 Lunes, Martes）；español（不用 Español）

2. **所有人**：数字格式 → 西班牙语：1.234,56（不用 1,234.56）

3. **所有人**：汉字 = 单词 → <ruby>计<rt>jì</rt></ruby><ruby>算<rt>suàn</rt></ruby><ruby>机<rt>jī</rt></ruby> = 三个字符，一个词

4. **英语母语者**：省略西班牙语重音 → término（我结束）vs terminó（他结束）
