## Overview

Quantifiers express quantity: all, some, none, many, few. They interact with negation in complex ways.

- **Spanish**: *Alguno* (some/any) changes to *ninguno* (none) in negative contexts. *Todo* agrees in gender and number. Negative concord means multiple negatives reinforce each other.
- **English**: The some/any distinction depends on polarity (affirmative, negative, interrogative). Few/a few and little/a little have opposite implications. No negative concord.
- **Chinese**: *<ruby>都<rt>dū</rt></ruby>* is the universal quantifier adverb. *<ruby>什<rt>shí</rt></ruby><ruby>么<rt>mó</rt></ruby>...<ruby>都<rt>dū</rt></ruby>/<ruby>也<rt>yě</rt></ruby>* creates universal negation. No count/mass distinction. Negation uses *<ruby>不<rt>bù</rt></ruby>/<ruby>没<rt>méi</rt></ruby>* depending on aspect.

## Spanish

### Universal (todo / todos)

*Todo* agrees and can be adjective, pronoun, or noun:

- *Todo el mundo* (everyone — masculine singular)
- *Todas las personas* (all the people — feminine plural)
- *Todo está bien.* (Everything is fine.)
- *Lo dijo todo.* (He said it all.)

### Existential (alguno / algunos)

| Form | Usage | Example |
|---|---|---|
| algún | before masc sing noun | *algún día* (some day) |
| alguno | pronoun or after noun | *alguno de ellos* (some of them) |
| algunos | plural noun or pronoun | *algunos libros* (some books) |
| algo | thing (invariable) | *algo interesante* (something interesting) |
| alguien | person (invariable) | *alguien vino* (someone came) |

### Negative (ninguno / nadie / nada)

| Form | Meaning | Example |
|---|---|---|
| ningún / ninguno | none / no (thing) | *ningún problema* (no problem) |
| nadie | nobody | *No vino nadie.* (Nobody came.) |
| nada | nothing | *No dijo nada.* (He said nothing.) |
| nunca | never | *Nunca viene.* (He never comes.) |
| tampoco | neither / not either | *Yo tampoco.* (Me neither.) |

**Negative concord**: Multiple negatives reinforce:

- *No vi **nada**.* (I saw nothing. — lit. "I didn't see nothing")
- *No vino **nadie** nunca.* (Nobody ever came.)
- *No dijo **nada** a **nadie**.* (He said nothing to nobody.)

**Note**: This is grammatical in Spanish but would be double negative (≠ positive) in English.

### Mucho / poco

| | Count plural | Mass singular |
|---|---|---|
| much | muchos libros | mucho dinero |
| little/few | pocos libros | poco dinero |
| a lot of | muchos | mucho |
| too much | demasiados | demasiado |

### Ambos / los dos (both)

- *Ambos vinieron.* (Both came.)
- *Los dos están aquí.* (Both are here.)
- *A ambos les gusta.* (Both like it.)

## English

### All / every / each

| | Usage | Example |
|---|---|---|
| all | whole group, plural | *all the books* |
| every | individual in group, sing | *every book* |
| each | individual emphasis, sing | *each book* |

**Note**: *Every* and *each* take singular verbs:

- *Every student **has** a book.*
- *Each of them **is** here.*

### Some / any

| | Affirmative | Negative | Question |
|---|---|---|---|
| some | ✅ I have some. | ❌ | ✅ Would you like some? |
| any | ✅ Any student can. | ✅ I don't have any. | ✅ Do you have any? |

**Any** in affirmative (free-choice):

- *Any student can solve this.* (= every)
- *Take any book you want.* (= whichever)

**Some** in question (offer/expectation):

- *Would you like some coffee?* (offer)
- *Did someone call?* (expectation of yes)

### Few / a few / little / a little

| | Negative implication | Positive implication |
|---|---|---|
| count | few | a few |
| mass | little | a little |

- *Few people came.* (disappointing — not many)
- *A few people came.* (positive — some did)
- *There's little water.* (worrying — not much)
- *There's a little water.* (reassuring — some exists)

### None / nobody / nothing

- *None of them **is** / **are** here.* (singular or plural verb)
- *Nobody knows.*
- *Nothing happened.*

**Note**: English uses any-word + not, not double negative:

- ✅ *I don't have anything.*
- ❌ *I don't have nothing.* (non-standard)

### Both / either / neither

| | Meaning | Example |
|---|---|---|
| both | the two, together | *Both are good.* |
| either | one or the other | *Either is fine.* |
| neither | not either | *Neither is good.* |

**Neither...nor**:

- *Neither John nor Mary came.*
- *I like neither tea nor coffee.*

**Verb agreement**: closest subject determines agreement

- *Neither the teacher nor the students **were** ready.*

### Many / much / a lot of

| | Count plural | Mass singular |
|---|---|---|
| formal many/much | many books | much water |
| informal a lot of / lots of | a lot of books | a lot of water |

## Chinese

### Universal (<ruby>都<rt>dū</rt></ruby> / <ruby>全<rt>quán</rt></ruby> / <ruby>所<rt>suǒ</rt></ruby><ruby>有<rt>yǒu</rt></ruby>)

<ruby>都<rt>dōu</rt></ruby> is an adverb that quantifies over a preceding noun phrase:

- <ruby>大<rt>dà</rt></ruby><ruby>家<rt>jiā</rt></ruby><ruby>都<rt>dōu</rt></ruby><ruby>来<rt>lái</rt></ruby><ruby>了<rt>le</rt></ruby>。(Everyone came.)
- <ruby>每<rt>měi</rt></ruby><ruby>个<rt>ge</rt></ruby><ruby>人<rt>rén</rt></ruby><ruby>都<rt>dōu</rt></ruby><ruby>知<rt>zhī</rt></ruby><ruby>道<rt>dào</rt></ruby>。(Every person knows. — <ruby>都<rt>dū</rt></ruby> required)
- <ruby>我<rt>wǒ</rt></ruby><ruby>们<rt>men</rt></ruby><ruby>全<rt>quán</rt></ruby><ruby>去<rt>qù</rt></ruby>。(We all go.)
- <ruby>所<rt>suǒ</rt></ruby><ruby>有<rt>yǒu</rt></ruby><ruby>的<rt>de</rt></ruby><ruby>书<rt>shū</rt></ruby> (all the books)

**Pattern**: Universal NP + <ruby>都<rt>dū</rt></ruby> + predicate

### Existential (<ruby>一<rt>yī</rt></ruby><ruby>些<rt>xiē</rt></ruby> / <ruby>有<rt>yǒu</rt></ruby><ruby>些<rt>xiē</rt></ruby> / <ruby>几<rt>jī</rt></ruby><ruby>个<rt>gè</rt></ruby>)

| | Usage | Example |
|---|---|---|
| <ruby>一<rt>yì</rt></ruby><ruby>些<rt>xiē</rt></ruby> | some | <ruby>一<rt>yì</rt></ruby><ruby>些<rt>xiē</rt></ruby><ruby>人<rt>rén</rt></ruby> (some people) |
| <ruby>有<rt>yǒu</rt></ruby><ruby>些<rt>xiē</rt></ruby> | some (topic-comment) | <ruby>有<rt>yǒu</rt></ruby><ruby>些<rt>xiē</rt></ruby><ruby>事<rt>shì</rt></ruby> (some things) |
| <ruby>几<rt>jǐ</rt></ruby><ruby>个<rt>gè</rt></ruby> | a few | <ruby>几<rt>jǐ</rt></ruby><ruby>个<rt>ge</rt></ruby><ruby>人<rt>rén</rt></ruby> (a few people) |
| <ruby>一<rt>yì</rt></ruby><ruby>点<rt>diǎn</rt></ruby> | a little | <ruby>一<rt>yì</rt></ruby><ruby>点<rt>diǎn</rt></ruby><ruby>水<rt>shuǐ</rt></ruby> (a little water) |

### Negative quantification

**<ruby>什<rt>shí</rt></ruby><ruby>么<rt>mó</rt></ruby>...<ruby>都<rt>dū</rt></ruby>/<ruby>也<rt>yě</rt></ruby> + Neg** (universal negative):

- <ruby>我<rt>wǒ</rt></ruby><ruby>什<rt>shén</rt></ruby><ruby>么<rt>me</rt></ruby><ruby>都<rt>dōu</rt></ruby><ruby>不<rt>bù</rt></ruby><ruby>知<rt>zhī</rt></ruby><ruby>道<rt>dào</rt></ruby>。(I don't know anything.)
- <ruby>他<rt>tā</rt></ruby><ruby>什<rt>shén</rt></ruby><ruby>么<rt>me</rt></ruby><ruby>也<rt>yě</rt></ruby><ruby>没<rt>méi</rt></ruby><ruby>说<rt>shuō</rt></ruby>。(He didn't say anything.)
- <ruby>哪<rt>nǎ</rt></ruby><ruby>儿<rt>er</rt></ruby><ruby>都<rt>dōu</rt></ruby><ruby>不<rt>bù</rt></ruby><ruby>去<rt>qù</rt></ruby>。(Not going anywhere.)

**<ruby>谁<rt>shuí</rt></ruby> / <ruby>什<rt>shí</rt></ruby><ruby>么<rt>mó</rt></ruby> / <ruby>哪<rt>nǎ</rt></ruby><ruby>儿<rt>ér</rt></ruby> / <ruby>什<rt>shí</rt></ruby><ruby>么<rt>mó</rt></ruby><ruby>时<rt>shí</rt></ruby><ruby>候<rt>hòu</rt></ruby> + <ruby>都<rt>dū</rt></ruby>/<ruby>也<rt>yě</rt></ruby> + Neg**:

- <ruby>谁<rt>shéi</rt></ruby><ruby>都<rt>dōu</rt></ruby><ruby>不<rt>bù</rt></ruby><ruby>来<rt>lái</rt></ruby>。(Nobody is coming.)
- <ruby>什<rt>shén</rt></ruby><ruby>么<rt>me</rt></ruby><ruby>时<rt>shí</rt></ruby><ruby>候<rt>hou</rt></ruby><ruby>都<rt>dōu</rt></ruby><ruby>可<rt>kě</rt></ruby><ruby>以<rt>yǐ</rt></ruby>。(Anytime is fine. — positive free choice)

### <ruby>不<rt>bù</rt></ruby> / <ruby>没<rt>méi</rt></ruby> as negative quantifier

- <ruby>没<rt>méi</rt></ruby><ruby>有<rt>yǒu</rt></ruby><ruby>人<rt>rén</rt></ruby> (nobody)
- <ruby>没<rt>méi</rt></ruby><ruby>有<rt>yǒu</rt></ruby><ruby>东<rt>dōng</rt></ruby><ruby>西<rt>xi</rt></ruby> (nothing)
- <ruby>什<rt>shén</rt></ruby><ruby>么<rt>me</rt></ruby><ruby>都<rt>dōu</rt></ruby><ruby>没<rt>méi</rt></ruby><ruby>有<rt>yǒu</rt></ruby> (have nothing at all)

### <ruby>任<rt>rèn</rt></ruby><ruby>何<rt>hé</rt></ruby> (free-choice any)

<ruby>任<rt>rèn</rt></ruby><ruby>何<rt>hé</rt></ruby> = any (free choice, usually in positive or conditional):

- <ruby>任<rt>rèn</rt></ruby><ruby>何<rt>hé</rt></ruby><ruby>人<rt>rén</rt></ruby><ruby>都<rt>dōu</rt></ruby><ruby>可<rt>kě</rt></ruby><ruby>以<rt>yǐ</rt></ruby>。(Anyone can.)
- <ruby>任<rt>rèn</rt></ruby><ruby>何<rt>hé</rt></ruby><ruby>时<rt>shí</rt></ruby><ruby>候<rt>hou</rt></ruby> (anytime)

### <ruby>俩<rt>liǎng</rt></ruby> (two of them)

<ruby>俩<rt>liǎ</rt></ruby> = two (colloquial contraction of <ruby>两<rt>liǎng</rt></ruby><ruby>个<rt>gè</rt></ruby>):

- <ruby>我<rt>wǒ</rt></ruby><ruby>们<rt>men</rt></ruby><ruby>俩<rt>liǎ</rt></ruby> (the two of us)
- <ruby>这<rt>zhè</rt></ruby><ruby>俩<rt>liǎ</rt></ruby> (these two)

## Common mistakes

1. **Spanish speakers learning English**: Double negative: *I don't have nothing* → *I don't have **anything***

2. **English speakers learning Spanish**: Using some in negative: *No tengo some* → *No tengo **ninguno***

3. **English speakers learning Chinese**: Omitting <ruby>都<rt>dū</rt></ruby>: *<ruby>每<rt>měi</rt></ruby><ruby>个<rt>gè</rt></ruby><ruby>人<rt>rén</rt></ruby><ruby>来<rt>lái</rt></ruby>* → *<ruby>每<rt>měi</rt></ruby><ruby>个<rt>gè</rt></ruby><ruby>人<rt>rén</rt></ruby>**<ruby>都<rt>dū</rt></ruby>**<ruby>来<rt>lái</rt></ruby>*

4. **Chinese speakers learning English**: Using any in affirmative: *I have any* → *I have **some***
