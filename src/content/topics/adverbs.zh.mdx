## 概述

副词修饰动词、形容词、其他副词或整个句子。它们表达方式、时间、地点、频率、程度和确定性。

- **西班牙语**：-mente 构成的副词很常见，但很多高频副词是词汇性的。位置相对灵活。
- **英语**：大多数副词由 -ly 构成。位置比西班牙语更严格，尤其是方式副词。
- **中文**：形容词经常不作改变就充当副词。地 dì 粒子可以添加以强调或用于正式书面语。

## 西班牙语

### 用 -mente 构成

西班牙语通过在形容词阴性单数形式后加 -mente 来构成副词：

- *rápido* → *rápida* → *rápidamente* (快速地)
- *lento* → *lenta* → *lentamente* (慢慢地)
- *claro* → *clara* → *claramente* (清楚地)

两个形容词结合时，只有第二个加 -mente：

- *lenta y cuidadosamente* (缓慢而小心地)

### 词汇副词（不用 -mente）

许多西班牙语常用副词不用 -mente 构成：

| 副词 | 含义 |
|---|---|
| ya | 已经 |
| ahora | 现在 |
| hoy | 今天 |
| mañana | 明天 |
| siempre | 总是 |
| nunca | 从不 |
| también | 也 |
| tampoco | 也不 |
| bastante | 相当，足够 |
| demasiado | 太 |
| muy | 很 |
| poco | 少 |
| mucho | 多 |

### 位置

西班牙语副词位置相对灵活：

- *Habla **rápidamente**.* (他说得很快。)
- ***Rápidamente** habla.* (很快地，他说。)
- *Habla español **bien**.* (他西班牙语说得很好。)

频率副词通常在主要动词前：

- ***Siempre** llega tarde.* (他总是迟到。)
- ***Nunca** como carne.* (我从不吃肉。)

但在 ser 和助动词后：

- *Es **siempre** puntual.* (他总是准时。)

### 修饰形容词的副词

副词在被修饰的形容词前：

- *Es **muy** alto.* (他很高。)
- *Está **bastante** bien.* (他相当好。)
- *Es **demasiado** caro.* (太贵了。)

## 英语

### 用 -ly 构成

英语通过在形容词后加 -ly 来构成副词：

- *quick* → *quick**ly*** (快速地)
- *slow* → *slow**ly*** (缓慢地)
- *careful* → *careful**ly*** (小心地)

例外（扁平副词）：

- *fast* → *fast* (不是 fastly)
- *hard* → *hard* (不是 hardly — 那意思是"几乎不")
- *late* → *late* (不是 lately — 那意思是"最近")
- *early* → *early* (形容词和副词相同)

### 形容词 vs 副词混淆

有些词既是形容词又是副词，但含义不同：

| 词 | 形容词 | 副词 |
|---|---|---|
| hard | 困难的 | 努力地 |
| hardly | — | 几乎不 |
| late | 迟到的 | 迟到地 |
| lately | — | 最近 |
| near | 近的 | 在附近 |
| nearly | — | 几乎 |

### 位置

**方式副词**通常在动词或宾语后：

- *She sings **beautifully**.* (她唱得很美。)
- *He speaks Spanish **well**.* (他西班牙语说得很好。)

**频率副词**在主要动词前：

- *I **always** eat breakfast.* (我总是吃早餐。)
- *She **never** arrives late.* (她从不迟到。)

但在 be 动词后：

- *He **is always** late.* (他总是迟到。)

**情态副词**在主要动词前，但在 be 后：

- *It **will probably** rain.* (可能会下雨。)
- *He **is certainly** wrong.* (他肯定是错的。)

### 句子副词

修饰整个句子的副词放在句首：

- ***Honestly**, I don't know.* (老实说，我不知道。)
- ***Surprisingly**, he passed.* (令人惊讶的是，他通过了。)

## 中文

### 形容词作副词

中文形容词经常不作改变就充当副词：

- <ruby>他<rt>tā</rt></ruby><ruby>跑<rt>pǎo</rt></ruby><ruby>得<rt>de</rt></ruby><ruby>很<rt>hěn</rt></ruby><ruby>快<rt>kuài</rt></ruby>。(他跑得很快。—— <ruby>快<rt>kuài</rt></ruby> 是形容词作副词)
- <ruby>她<rt>tā</rt></ruby><ruby>说<rt>shuō</rt></ruby><ruby>得<rt>de</rt></ruby><ruby>很<rt>hěn</rt></ruby><ruby>清<rt>qīng</rt></ruby><ruby>楚<rt>chu</rt></ruby>。(她说得很清楚。)

### <ruby>地<rt>dì</rt></ruby> 用于方式副词

<ruby>地<rt>dì</rt></ruby> 可以加在形容词后构成方式副词，尤见于正式/书面中文：

- <ruby>快<rt>kuài</rt></ruby><ruby>地<rt>de</rt></ruby><ruby>跑<rt>pǎo</rt></ruby> (快速地跑)
- <ruby>慢<rt>màn</rt></ruby><ruby>慢<rt>màn</rt></ruby><ruby>地<rt>de</rt></ruby><ruby>说<rt>shuō</rt></ruby> (慢慢地说)
- <ruby>认<rt>rèn</rt></ruby><ruby>真<rt>zhēn</rt></ruby><ruby>地<rt>de</rt></ruby><ruby>学<rt>xué</rt></ruby><ruby>习<rt>xí</rt></ruby> (认真地学习)

**注意**：<ruby>地<rt>dì</rt></ruby> 读音为 de（与 <ruby>的<rt>de</rt></ruby>、<ruby>得<rt>dé</rt></ruby> 相同），但写法不同。

### 位置

中文副词几乎总是在动词前：

- <ruby>他<rt>tā</rt></ruby><ruby>很<rt>hěn</rt></ruby><ruby>快<rt>kuài</rt></ruby><ruby>跑<rt>pǎo</rt></ruby>。(他跑得很快。)
- <ruby>我<rt>wǒ</rt></ruby><ruby>经<rt>jīng</rt></ruby><ruby>常<rt>cháng</rt></ruby><ruby>去<rt>qù</rt></ruby>。(我经常去。)
- <ruby>她<rt>tā</rt></ruby><ruby>从<rt>cóng</rt></ruby><ruby>来<rt>lái</rt></ruby><ruby>不<rt>bù</rt></ruby><ruby>抽<rt>chōu</rt></ruby><ruby>烟<rt>yān</rt></ruby>。(她从来不抽烟。)

### <ruby>得<rt>dé</rt></ruby> 作结果补语标记

<ruby>得<rt>de</rt></ruby> 跟在动词后，引入描述结果或方式的补语：

- <ruby>跑<rt>pǎo</rt></ruby><ruby>得<rt>de</rt></ruby><ruby>很<rt>hěn</rt></ruby><ruby>快<rt>kuài</rt></ruby> (跑得很快)

## 一览对比

| 特征 | 西班牙语 | 英语 | 中文 |
|---------|---------|---------|---------|
| 构成 | 形容词 + -mente | 形容词 + -ly | 形容词无变化（可选 + <ruby>地<rt>dì</rt></ruby>） |
| 方式位置 | 灵活 | 动词/宾语后 | 动词前 |
| 频率位置 | 主要动词前 / ser后 | 主要动词前 / be后 | 动词前 |
| 时间位置 | 灵活 | 灵活 | 句首或动词前 |
| 强化词 | muy, bastante, demasiado | very, really, extremely | <ruby>很<rt>hěn</rt></ruby>, <ruby>非<rt>fēi</rt></ruby><ruby>常<rt>cháng</rt></ruby> |
| 比较级 | más + 副词 | more / -er | <ruby>更<rt>gēng</rt></ruby> |
| 最高级 | el más + 副词 | the most | <ruby>最<rt>zuì</rt></ruby> |
| 扁平副词 | 无 | fast, hard, late, early | 所有形容词都可作副词 |

## 常见错误

1. **西班牙语母语者学英语**：形容词作副词：*He drives quick* → *He drives **quickly***

2. **英语母语者学西班牙语**：过度使用 -mente：*hablo rápido* 在口语中更自然（很少用 rápidamente）

3. **英语母语者学中文**：到处加<ruby>地<rt>dì</rt></ruby>：*<ruby>快<rt>kuài</rt></ruby><ruby>快<rt>kuài</rt></ruby><ruby>地<rt>dì</rt></ruby><ruby>跑<rt>pǎo</rt></ruby>* 正确但 *<ruby>快<rt>kuài</rt></ruby><ruby>跑<rt>pǎo</rt></ruby>* 口语中更自然

4. **中文母语者学英语**：位置错误：*He quickly runs* → *He runs **quickly***
