## Resumen

El predicado es lo que se dice del sujeto. Cada idioma lo estructura de manera diferente.

- **Español**: Los predicados verbales son desnudos. Los adjetivales y nominales requieren ser/estar. Distingue inherente vs estado.
- **Inglés**: Los predicados verbales son desnudos. Todos los no verbales requieren be. Usa become, seem, get como pseudo-copulares.
- **Chino**: Los predicados adjetivales son desnudos (sin cópula). Los nominales requieren 是. El progresivo usa 在. El cambio resultativo usa 了.

## Español

### ser vs estar

| Cópula | Uso | Ejemplo |
|---|---|---|
| **ser** | Identidad / propiedad inherente | Juan **es** alto (siempre) |
| **ser** | Profesión | Juan **es** médico |
| **ser** | Origen | Juan **es** de España |
| **estar** | Estado temporal | Juan **está** cansado (ahora) |
| **estar** | Ubicación | El libro **está** en la mesa |
| **estar** | Progresivo | Juan **está** comiendo |

**Nota**: ser = inherente / permanente; estar = temporal / estado.

### Progresivo

| Estructura | Ejemplo |
|---|---|
| estar + gerundio | El perro **está ladrando** |
| ir + gerundio (progresivo enfático) | **Va lloviendo** más fuerte |
| venir + gerundio | **Viene diciendo** eso desde hace años |

### Predicados sin cópula

- **Verbales**: *Llueve* (impersonal)
- **Existenciales**: *Hay libros* (existencia)
- **Locativos**: *Aquí está Juan* (con estar)

### Reflexivos resultativos

- Se **puso** furioso (cambio de estado)
- Se **hizo** rico (voluntad)
- Se **volvió** loco (proceso)
- Se **quedó** dormido (resultado)

## Inglés

### be como cópula

| Tipo de predicado | Ejemplo |
|---|---|
| Adjetival | John **is** tall |
| Nominal | John **is** a teacher |
| Locativo | The book **is** on the table |
| Progresivo | John **is** eating |
| Pasivo | The book **was** written |

**Nota**: El inglés requiere be para TODOS los predicados no verbales.

### Progresivo

| Estructura | Ejemplo |
|---|---|
| be + -ing | The dog **is barking** |
| get + -ing | **It's getting** dark |

### Pseudo-copulares

| Verbo | Uso | Ejemplo |
|---|---|---|
| become | Cambio de estado | He **became** angry |
| seem | Apariencia | He **seems** happy |
| get | Cambio (coloquial) | He **got** tired |
| turn | Cambio repentino | He **turned** pale |
| grow | Cambio gradual | He **grew** old |
| remain | Continuación | He **remained** silent |

### make causativo

| Estructura | Ejemplo |
|---|---|
| make + objeto + adjetivo | She **made** him **happy** |
| make + objeto + verbo base | She **made** him **laugh** |

**Nota**: No se usa to-infinitive después de make.

### Expletivo it

- **It** rains (impersonal)
- **It** seems that he is happy (elevación)

**Nota**: El inglés requiere un sujeto gramatical; usa it como marcador de lugar.

## Chino

### Predicados desnudos (sin cópula)

| Tipo | Ejemplo | Nota |
|---|---|---|
| Adjetival | <ruby>他<rt>tā</rt></ruby><ruby>很<rt>hěn</rt></ruby><ruby>高<rt>gāo</rt></ruby> | Sin 是; 很 = grado |
| Verbal | <ruby>他<rt>tā</rt></ruby><ruby>吃<rt>chī</rt></ruby><ruby>饭<rt>fàn</rt></ruby> | Sin cópula |
| Locativo | <ruby>他<rt>tā</rt></ruby><ruby>在<rt>zài</rt></ruby><ruby>家<rt>jiā</rt></ruby> | 在 = preposición |

### 是 para nominales

| Estructura | Ejemplo |
|---|---|
| Sujeto + 是 + sustantivo | <ruby>他<rt>tā</rt></ruby><ruby>是<rt>shì</rt></ruby><ruby>老<rt>lǎo</rt></ruby><ruby>师<rt>shī</rt></ruby> (Él es profesor) |
| Negación: 不是 | <ruby>他<rt>tā</rt></ruby><ruby>不<rt>bù</rt></ruby><ruby>是<rt>shì</rt></ruby><ruby>学<rt>xué</rt></ruby><ruby>生<rt>shēng</rt></ruby> |

**Nota**: 是 solo se usa con predicados nominales, NUNCA con adjetivales.

### Progresivo con 在

| Estructura | Ejemplo |
|---|---|
| Sujeto + 在 + verbo | <ruby>狗<rt>gǒu</rt></ruby><ruby>在<rt>zài</rt></ruby><ruby>叫<rt>jiào</rt></ruby> (El perro está ladrando) |
| Sujeto + 正在 + verbo | <ruby>他<rt>tā</rt></ruby><ruby>正<rt>zhèng</rt></ruby><ruby>在<rt>zài</rt></ruby><ruby>吃<rt>chī</rt></ruby><ruby>饭<rt>fàn</rt></ruby> (Él está comiendo ahora mismo) |

### Resultativo con 了

| Estructura | Ejemplo |
|---|---|
| Verbo + 了 | <ruby>他<rt>tā</rt></ruby><ruby>生<rt>shēng</rt></ruby><ruby>气<rt>qì</rt></ruby><ruby>了<rt>le</rt></ruby> (Se enfureció) |
| Verbo + 了 + objeto | <ruby>他<rt>tā</rt></ruby><ruby>吃<rt>chī</rt></ruby><ruby>了<rt>le</rt></ruby><ruby>饭<rt>fàn</rt></ruby> (Comió) |

**Nota**: 了 indica cambio de estado o completitud.

### Verbos de elevación

| Verbo | Ejemplo |
|---|---|
| 好像 | <ruby>他<rt>tā</rt></ruby><ruby>好<rt>hǎo</rt></ruby><ruby>像<rt>xiàng</rt></ruby><ruby>很<rt>hěn</rt></ruby><ruby>高<rt>gāo</rt></ruby><ruby>兴<rt>xìng</rt></ruby> (Parece feliz) |
| 觉得 | <ruby>我<rt>wǒ</rt></ruby><ruby>觉<rt>jué</rt></ruby><ruby>得<rt>de</rt></ruby><ruby>很<rt>hěn</rt></ruby><ruby>好<rt>hǎo</rt></ruby> (Creo que es bueno) |

### Causativo con 让

| Estructura | Ejemplo |
|---|---|
| 让 + objeto + verbo / adjetivo | <ruby>他<rt>tā</rt></ruby><ruby>让<rt>ràng</rt></ruby><ruby>我<rt>wǒ</rt></ruby><ruby>很<rt>hěn</rt></ruby><ruby>生<rt>shēng</rt></ruby><ruby>气<rt>qì</rt></ruby> (Me hizo enfadar) |

## Comparación rápida

| Tipo de predicado | Español | Inglés | Chino |
|---|---|---|---|
| Verbal | Desnudo | Desnudo | Desnudo |
| Adjetival | ser/estar + adjetivo | be + adjetivo | Adjetivo desnudo (+ 很) |
| Nominal | ser + sustantivo | be + sustantivo | 是 + sustantivo |
| Progresivo | estar + gerundio | be + -ing | 在 + verbo |
| Locativo | estar + PP | be + PP | 在 + ubicación |
| Resultativo | Reflexivo (ponerse) | become/get + adjetivo | Verbo + 了 |
| Causativo | hacer + objeto + adjetivo | make + objeto + adjetivo | 让 + objeto + adjetivo |
| Impersonal | Desnudo / hay | it + verbo | Verbo-objeto desnudo |

## Errores comunes

1. **Hispanohablantes**: Usar 是 con adjetivos en chino → <ruby>他<rt>tā</rt></ruby><ruby>是<rt>shì</rt></ruby><ruby>高<rt>gāo</rt></ruby> es incorrecto

2. **Hispanohablantes**: Omitir cópula en inglés → "John tall" en lugar de "John is tall"

3. **Angloparlantes**: Usar ser para progresivo en español → "es ladrando" en lugar de "está ladrando"

4. **Hispanohablantes**: Usar to-infinitive después de make → "made me to be happy" en lugar de "made me happy"
