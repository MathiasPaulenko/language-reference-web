## Visión general

Las palabras de medida expresan cantidad a través de contenedores, porciones o colecciones.

- **Español**: Usa *un/una + contenedor + de + sustantivo*. Las palabras de medida son opcionales y relativamente pocas.
- **Inglés**: Usa *a + medida + of + sustantivo*. Opcional para la mayoría de los sustantivos pero requerida para cantidades incontables.
- **Chino**: Las palabras de medida son **obligatorias** para todos los sustantivos contados. La palabra de medida refleja el contenedor, la forma o el agrupamiento. No se usa preposición.

## Errores comunes

1. **Hispanohablantes/angloparlantes aprendiendo chino**: Omitir la palabra de medida: *<ruby>二<rt>èr</rt></ruby><ruby>咖<rt>kā</rt></ruby><ruby>啡<rt>fēi</rt></ruby>* → *<ruby>两<rt>liǎng</rt></ruby>**<ruby>杯<rt>bēi</rt></ruby>**<ruby>咖<rt>kā</rt></ruby><ruby>啡<rt>fēi</rt></ruby>*

2. **Angloparlantes aprendiendo chino**: Usar <ruby>个<rt>gè</rt></ruby> para todo: *<ruby>一<rt>yī</rt></ruby><ruby>个<rt>gè</rt></ruby><ruby>咖<rt>kā</rt></ruby><ruby>啡<rt>fēi</rt></ruby>* → *<ruby>一<rt>yī</rt></ruby><ruby>杯<rt>bēi</rt></ruby><ruby>咖<rt>kā</rt></ruby><ruby>啡<rt>fēi</rt></ruby>*

3. **Sinohablantes aprendiendo inglés**: Omitir medidas para incontables: *I want two waters* → *I want two **glasses of** water*

4. **Sinohablantes aprendiendo español**: Añadir medidas innecesariamente: *un pedazo de agua* → *un **vaso de** agua* (el agua no se "corta en pedazos")
