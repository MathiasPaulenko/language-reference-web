## 概述

词源学研究词汇的历史 — 它们从何而来，形式和意义如何变化，以及如何在语言间传播。

- **西班牙语**：直接从伊比利亚半岛的口语拉丁语演变而来。后来吸收了阿拉伯语、美洲原住民和英语词汇。
- **英语**：一种日耳曼语言，在诺曼征服（1066年）后经历了大规模词汇转变，获得了数千个法语和拉丁语词汇。
- **中文**：世界上持续使用时间最久的书写系统。汉字同时编码语音和语义信息。

## 西班牙语词源

**拉丁语遗产**：约75-80%的西班牙语词汇源自拉丁语：
- *agua* < 拉丁语 *aqua*
- *corazón* < 拉丁语 *cor* + 指小词
- *noche* < 拉丁语 *noctem*

**阿拉伯语影响**：711-1492年间超过4,000个词汇：
- *alcohol* < 阿拉伯语 *al-kuḥl*
- *aceite* < 阿拉伯语 *al-zayt*
- *azúcar* < 阿拉伯语 *al-sukkar*
- *ojalá* < 阿拉伯语 *in sha'Allah*（如果真主愿意）

**美洲原住民语言**：
- *tomate* < 纳瓦特尔语 *tomatl*
- *chocolate* < 纳瓦特尔语 *xocolātl*
- *llama* < 克丘亚语
- *papa* < 克丘亚语/泰诺语

## 英语词源

英语由于诺曼征服而拥有著名的双重词汇：

**日耳曼核心**（日常、非正式）：
- house, eat, drink, live, work, make, come, go

**罗曼语层**（正式、学术、法律）：
- residence, consume, beverage, reside, labor, manufacture, arrive, depart

**有微妙区别的配对**：
- *cow*（日耳曼语，动物）vs *beef*（法语，肉类）
- *pig* vs *pork*
- *sheep* vs *mutton*
- *deer* vs *venison*

这种分裂反映了社会等级：盎格鲁-撒克逊农民饲养动物；诺曼贵族食用它们。

## 中文词源

汉字在视觉上编码词源：

**象形字**（约4%）：直接绘画
- <ruby>日<rt>rì</rt></ruby>（太阳）, <ruby>月<rt>yuè</rt></ruby>（月亮）, <ruby>山<rt>shān</rt></ruby>（山）, <ruby>水<rt>shuǐ</rt></ruby>（水）

**形声字**（约80-90%）：意符 + 声符
- <ruby>妈<rt>mā</rt></ruby>（母亲）：<ruby>女<rt>nǚ</rt></ruby>（女人，意符） + <ruby>马<rt>mǎ</rt></ruby>（mǎ，声符）
- <ruby>河<rt>hé</rt></ruby>（河流）：<ruby>水<rt>shuǐ</rt></ruby>（水） + <ruby>可<rt>kě</rt></ruby>（kě，声符）

**会意字**（约2%）：组合含义
- <ruby>明<rt>míng</rt></ruby>（明亮）：<ruby>日<rt>rì</rt></ruby>（太阳） + <ruby>月<rt>yuè</rt></ruby>（月亮）
- <ruby>休<rt>xiū</rt></ruby>（休息）：<ruby>人<rt>rén</rt></ruby>（人） + <ruby>木<rt>mù</rt></ruby>（树 → 倚靠在树上）

**中文中的外来词**：
- 佛教术语：<ruby>世<rt>shì</rt></ruby><ruby>界<rt>jiè</rt></ruby>（世界）, <ruby>因<rt>yīn</rt></ruby><ruby>果<rt>guǒ</rt></ruby>（因果/业力）
- 现代西方词：<ruby>电<rt>diàn</rt></ruby><ruby>话<rt>huà</rt></ruby>（电 + 话 = 电话）, <ruby>电<rt>diàn</rt></ruby><ruby>脑<rt>nǎo</rt></ruby>（电 + 脑 = 电脑）
