## Overview

Directional verbs express movement toward or away from a reference point, usually the speaker.

- **Spanish**: Uses *ir* (go), *venir* (come), *llevar* (take), *traer* (bring), plus prepositions (*a, hacia, desde*). Direction is determined by verb choice and preposition.
- **English**: Uses *come/go* and *bring/take*, plus phrasal verbs (*come in, go out, get up, sit down*). Directional particles (*in, out, up, down, on, off*) combine with verbs.
- **Chinese**: Has a unique system of **directional complements**: a main verb combines with a directional verb (<ruby>来<rt>lái</rt></ruby>/<ruby>去<rt>qù</rt></ruby>/<ruby>上<rt>shàng</rt></ruby>/<ruby>下<rt>xià</rt></ruby>/<ruby>出<rt>chū</rt></ruby>/<ruby>进<rt>jìn</rt></ruby>/<ruby>回<rt>huí</rt></ruby>/<ruby>过<rt>guò</rt></ruby>) to indicate both action and direction.

## Spanish

### Ir vs venir

| | Direction | Example |
|---|---|---|
| ir | Away from speaker | *Voy a tu casa.* (I'm going to your house.) |
| venir | Toward speaker | *Ven a mi casa.* (Come to my house.) |

**Rule**: The reference point is usually the speaker. In storytelling, it can be the narrative center.

### Llevar vs traer

| | Direction | Example |
|---|---|---|
| llevar | Away from speaker | *Lleva el libro a la escuela.* (Take the book to school.) |
| traer | Toward speaker | *Trae el libro a casa.* (Bring the book home.) |

### Basic directional verbs

| Spanish | Meaning | Example |
|---|---|---|
| entrar | enter | *Entra, por favor.* |
| salir | exit / leave | *Sal de aquí.* |
| subir | go up / get on | *Sube las escaleras.* |
| bajar | go down / get off | *Baja del autobús.* |
| acercarse | approach | *Acércate un poco.* |
| alejarse | move away | *Aléjate de mí.* |
| pasar | pass / come in | *Pasa, pasa.* |
| volver | return | *Vuelve pronto.* |
| regresar | return | *Regresa mañana.* |

### Prepositions of direction

| Preposition | Meaning | Example |
|---|---|---|
| a | to (direction) | *Voy a Madrid.* |
| hacia | toward | *Camina hacia mí.* |
| desde | from | *Vengo desde Madrid.* |
| hasta | until / as far as | *Hasta luego.* |
| por | through / by | *Pasa por aquí.* |
| de | from | *Sal de casa.* |

### Commands with direction

- *Ven **aquí**.* (Come here.)
- *Vete **de aquí**.* (Go away from here.)
- *Pasa **adentro**.* (Come inside.)
- *Sal **fuera**.* (Go outside.)

### Reflexive directional verbs

- *Acercarse* (approach): *Me acerqué.* (I approached.)
- *Alejarse* (move away): *Se alejó.* (He moved away.)
- *Subirse* (get on): *Me subí al tren.* (I got on the train.)
- *Bajarse* (get off): *Me bajé del autobús.* (I got off the bus.)

## English

### Come vs go

| | Direction | Example |
|---|---|---|
| come | Toward speaker/center | *Come here.* |
| go | Away from speaker/center | *Go there.* |

**Important**: Direction is relative to the speaker or the deictic center:

- *I'm at home. Please **come** to my house.* (toward me)
- *I'm at your house. I need to **go** home now.* (away from here)

### Bring vs take

| | Direction | Example |
|---|---|---|
| bring | Toward speaker/center | *Bring it here.* |
| take | Away from speaker/center | *Take it there.* |

**Examples**:

- *I'll **bring** snacks to the party.* (toward the party — the center of attention)
- *Please **take** this to the kitchen.* (away from here)

### Phrasal verbs for direction

English uses **phrasal verbs** extensively for direction:

| Verb + Particle | Meaning | Example |
|---|---|---|
| come in | enter toward speaker | *Come in!* |
| go out | exit away | *Let's go out.* |
| come up | ascend toward | *Come up here.* |
| go down | descend away | *Go down the stairs.* |
| get on | board | *Get on the bus.* |
| get off | disembark | *Get off at the next stop.* |
| come back | return toward | *Come back soon.* |
| go away | leave | *Go away!* |
| come over | visit | *Come over tonight.* |
| go back | return away | *Go back to work.* |

### Particles and their meanings

| Particle | Core direction |
|---|---|
| up | upward / completion | *get up, eat up* |
| down | downward / recording | *sit down, write down* |
| in | inward / inclusion | *come in, hand in* |
| out | outward / exclusion | *go out, find out* |
| on | continuation / attachment | *carry on, put on* |
| off | cessation / detachment | *take off, cut off* |
| over | across / visitation | *come over, think over* |
| back | return | *give back, call back* |
| away | distance | *go away, take away* |

### Prepositions of movement

| Preposition | Example |
|---|---|
| to | *Go to school.* |
| into | *Go into the room.* |
| out of | *Get out of here.* |
| through | *Go through the tunnel.* |
| across | *Walk across the street.* |
| along | *Walk along the beach.* |
| up | *Climb up the mountain.* |
| down | *Walk down the hill.* |
| toward(s) | *Walk toward me.* |
| away from | *Run away from danger.* |

## Chinese

### Directional complements: the core system

Chinese combines a **main verb** with a **directional complement** to show both action and direction:

**Simple directional complements** (<ruby>来<rt>lái</rt></ruby>/<ruby>去<rt>qù</rt></ruby>):

| Main verb | + <ruby>来<rt>lái</rt></ruby> (toward) | + <ruby>去<rt>qù</rt></ruby> (away) |
|---|---|---|
| <ruby>走<rt>zǒu</rt></ruby> (walk) | <ruby>走<rt>zǒu</rt></ruby><ruby>来<rt>lái</rt></ruby> (walk here) | <ruby>走<rt>zǒu</rt></ruby><ruby>去<rt>qù</rt></ruby> (walk away) |
| <ruby>跑<rt>pǎo</rt></ruby> (run) | <ruby>跑<rt>pǎo</rt></ruby><ruby>来<rt>lái</rt></ruby> (run here) | <ruby>跑<rt>pǎo</rt></ruby><ruby>去<rt>qù</rt></ruby> (run away) |
| <ruby>拿<rt>ná</rt></ruby> (take) | <ruby>拿<rt>ná</rt></ruby><ruby>来<rt>lái</rt></ruby> (bring here) | <ruby>拿<rt>ná</rt></ruby><ruby>去<rt>qù</rt></ruby> (take away) |
| <ruby>送<rt>sòng</rt></ruby> (deliver) | <ruby>送<rt>sòng</rt></ruby><ruby>来<rt>lái</rt></ruby> (send here) | <ruby>送<rt>sòng</rt></ruby><ruby>去<rt>qù</rt></ruby> (send away) |

**Compound directional complements** (path + direction):

| Path | + <ruby>来<rt>lái</rt></ruby> (toward) | + <ruby>去<rt>qù</rt></ruby> (away) |
|---|---|---|
| <ruby>上<rt>shàng</rt></ruby> (up) | <ruby>上<rt>shàng</rt></ruby><ruby>来<rt>lái</rt></ruby> (come up) | <ruby>上<rt>shàng</rt></ruby><ruby>去<rt>qù</rt></ruby> (go up) |
| <ruby>下<rt>xià</rt></ruby> (down) | <ruby>下<rt>xià</rt></ruby><ruby>来<rt>lái</rt></ruby> (come down) | <ruby>下<rt>xià</rt></ruby><ruby>去<rt>qù</rt></ruby> (go down) |
| <ruby>进<rt>jìn</rt></ruby> (in) | <ruby>进<rt>jìn</rt></ruby><ruby>来<rt>lái</rt></ruby> (come in) | <ruby>进<rt>jìn</rt></ruby><ruby>去<rt>qù</rt></ruby> (go in) |
| <ruby>出<rt>chū</rt></ruby> (out) | <ruby>出<rt>chū</rt></ruby><ruby>来<rt>lái</rt></ruby> (come out) | <ruby>出<rt>chū</rt></ruby><ruby>去<rt>qù</rt></ruby> (go out) |
| <ruby>回<rt>huí</rt></ruby> (back) | <ruby>回<rt>huí</rt></ruby><ruby>来<rt>lái</rt></ruby> (come back) | <ruby>回<rt>huí</rt></ruby><ruby>去<rt>qù</rt></ruby> (go back) |
| <ruby>过<rt>guò</rt></ruby> (over/across) | <ruby>过<rt>guò</rt></ruby><ruby>来<rt>lái</rt></ruby> (come over) | <ruby>过<rt>guò</rt></ruby><ruby>去<rt>qù</rt></ruby> (go over) |
| <ruby>起<rt>qǐ</rt></ruby> (up) | <ruby>起<rt>qǐ</rt></ruby><ruby>来<rt>lái</rt></ruby> (get up) | <ruby>起<rt>qǐ</rt></ruby><ruby>去<rt>qù</rt></ruby> (rare) |

### Direction is speaker-relative

- <ruby>上<rt>shàng</rt></ruby><ruby>来<rt>lái</rt></ruby> = come up (toward speaker)
- <ruby>上<rt>shàng</rt></ruby><ruby>去<rt>qù</rt></ruby> = go up (away from speaker)
- <ruby>下<rt>xià</rt></ruby><ruby>来<rt>lái</rt></ruby> = come down (toward speaker)
- <ruby>下<rt>xià</rt></ruby><ruby>去<rt>qù</rt></ruby> = go down (away from speaker)

### Main verb + compound complement

The main verb can precede the compound:

- <ruby>走<rt>zǒu</rt></ruby><ruby>上<rt>shàng</rt></ruby><ruby>来<rt>lái</rt></ruby> (walk up here)
- <ruby>跑<rt>pǎo</rt></ruby><ruby>出<rt>chū</rt></ruby><ruby>去<rt>qù</rt></ruby> (run out there)
- <ruby>拿<rt>ná</rt></ruby><ruby>进<rt>jìn</rt></ruby><ruby>来<rt>lái</rt></ruby> (bring in here)
- <ruby>扔<rt>rēng</rt></ruby><ruby>出<rt>chū</rt></ruby><ruby>去<rt>qù</rt></ruby> (throw out there)

### <ruby>起<rt>qǐ</rt></ruby><ruby>来<rt>lái</rt></ruby> as inceptive

<ruby>起<rt>qǐ</rt></ruby><ruby>来<rt>lái</rt></ruby> can indicate the beginning of an action:

- <ruby>唱<rt>chàng</rt></ruby><ruby>起<rt>qǐ</rt></ruby><ruby>来<rt>lái</rt></ruby> (start singing)
- <ruby>跳<rt>tiào</rt></ruby><ruby>起<rt>qǐ</rt></ruby><ruby>来<rt>lái</rt></ruby> (jump up / start dancing)
- <ruby>想<rt>xiǎng</rt></ruby><ruby>起<rt>qǐ</rt></ruby><ruby>来<rt>lái</rt></ruby> (remember — "think up")

### Directional complements with objects

When an object is present, it goes between the main verb and the complement or after the complement:

- <ruby>拿<rt>ná</rt></ruby><ruby>书<rt>shū</rt></ruby><ruby>出<rt>chū</rt></ruby><ruby>来<rt>lái</rt></ruby> (take the book out here)
- <ruby>拿<rt>ná</rt></ruby><ruby>出<rt>chū</rt></ruby><ruby>一<rt>yì</rt></ruby><ruby>本<rt>běn</rt></ruby><ruby>书<rt>shū</rt></ruby><ruby>来<rt>lái</rt></ruby> (take out a book here)

## Common mistakes

1. **English speakers learning Chinese**: Omitting directional complement: *<ruby>进<rt>jìn</rt></ruby>* → *<ruby>进<rt>jìn</rt></ruby><ruby>来<rt>lái</rt></ruby>* (for "come in")

2. **Chinese speakers learning English**: Literal translation: *Go up come* → *Come up*

3. **English speakers learning Spanish**: Using ir where venir is needed: *Voy a tu fiesta* when you're at the party → *Vienes a mi fiesta*

4. **Spanish speakers learning English**: Confusing bring/take: *Take the wine to the party* (where you are) → *Bring the wine*
