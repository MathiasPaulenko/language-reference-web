## 概述

虚拟式表达非客观现实的动作——愿望、怀疑、情感、可能性和假设。

- **西班牙语**：虚拟式**无处不在**。它出现在怀疑、愿望、情感、无人称表达和不确定先行词关系从句等触发词后的从句中。西班牙语学习者必须掌握虚拟式才能自然表达。
- **英语**：虚拟式**几乎消亡**。仅存几个僵化形式：*If I **were**...*, *I demand that he **be**...*, *It is important that she **speak**...*。现代英语用情态动词和迂回表达替代。
- **汉语**：**完全没有虚拟式**。相同句法结构表达真实与非真实。语境、副词和助词作区分。

## 西班牙语

### 虚拟式触发词

| 类别 | 触发词 | 例句 |
|---|---|---|
| 怀疑 | dudar, no creer, es posible que | Es posible que **llueva**. |
| 愿望 | querer, desear, esperar que | Quiero que **vengas**. |
| 情感 | alegrarse, temer, sorprenderse | Me alegro de que **vengas**. |
| 无人称 | es necesario, es importante | Es necesario que **estudies**. |
| 否定 | no pensar, no creer | No creo que **sea** verdad. |
| 关系（不确定） | buscar, necesitar | Busco un libro que **sea** interesante. |

### 虚拟式时态

| 时态 | 构成 | 例句 |
|---|---|---|
| 现在时 | 词干 + 词尾 | hable, comas, vivamos |
| 过去未完成时 | 词干 + -ra / -se | hablara/hablase, comieras/comieses |
| 将来时（极少用） | 词干 + -re | hablare, comiere |
| 完成时 | haya + 分词 | haya hablado |
| 过去完成时 | hubiera/hubiese + 分词 | hubiera hablado |

### 虚拟式 vs. 直陈式

| 语境 | 式 | 例句 |
|---|---|---|
| Creo que（肯定） | 直陈式 | Creo que **es** verdad. |
| No creo que（否定） | 虚拟式 | No creo que **sea** verdad. |
| Es cierto que | 直陈式 | Es cierto que **viene**. |
| Es posible que | 虚拟式 | Es posible que **venga**. |
| Conozco a alguien que | 直陈式 | Conozco a alguien que **habla** chino. |
| Busco a alguien que | 虚拟式 | Busco a alguien que **hable** chino. |

## 英语

### 虚拟式僵化形式

| 语境 | 例句 | 说明 |
|---|---|---|
| 反事实 | If I **were** you... | 各人称均用 were |
| 命令 | I demand that he **be** present. | be 替代 is |
| 建议 | It is important that she **speak** up. | speak 无 -s |
| 祝颂 | Long **live** the king! | 古虚拟式 |
| 祝颂 | God **save** the Queen! | 古虚拟式 |

### 现代替代

| 语境 | 古形式 | 现代替代 |
|---|---|---|
| 假设 | If I were rich | If I was rich（非正式） |
| 建议 | I suggest he go | I suggest he **should** go |
| 必要 | It is vital he be | It is vital that he **is**（口语） |

### 情态动词替代虚拟式

| 意义 | 情态 | 例句 |
|---|---|---|
| 可能 | may, might | It **may** rain. |
| 必要 | must, have to | You **must** go. |
| 建议 | should, ought to | You **should** study. |
| 愿望 | would like | I **would like** to go. |

## 汉语

### 无虚拟式标记

汉语不标记动词式。同一动词形式表达现实、可能、愿望和反事实。

| 语境 | 例句 | 说明 |
|---|---|---|
| 怀疑 | <ruby>可<rt>kě</rt></ruby><ruby>能<rt>néng</rt></ruby><ruby>会<rt>huì</rt></ruby><ruby>下<rt>xià</rt></ruby><ruby>雨<rt>yǔ</rt></ruby> | 可能 = 可能 |
| 愿望 | <ruby>我<rt>wǒ</rt></ruby><ruby>想<rt>xiǎng</rt></ruby><ruby>让<rt>ràng</rt></ruby><ruby>你<rt>nǐ</rt></ruby><ruby>来<rt>lái</rt></ruby> | 想 = 想 |
| 反事实 | <ruby>如<rt>rú</rt></ruby><ruby>果<rt>guǒ</rt></ruby><ruby>我<rt>wǒ</rt></ruby><ruby>有<rt>yǒu</rt></ruby><ruby>钱<rt>qián</rt></ruby>... | 如果 = 如果（条件） |
| 情感 | <ruby>我<rt>wǒ</rt></ruby><ruby>很<rt>hěn</rt></ruby><ruby>高<rt>gāo</rt></ruby><ruby>兴<rt>xìng</rt></ruby><ruby>你<rt>nǐ</rt></ruby><ruby>来<rt>lái</rt></ruby> | 无动词形式变化 |

| 副词/助词 | 功能 | 例句 |
|---|---|---|
| 如果 | 条件 | <ruby>如<rt>rú</rt></ruby><ruby>果<rt>guǒ</rt></ruby>...就... |
| 要是 | 假设 | <ruby>要<rt>yào</rt></ruby><ruby>是<rt>shì</rt></ruby>...就... |
| 假如 | 假设 | <ruby>假<rt>jiǎ</rt></ruby><ruby>如<rt>rú</rt></ruby>... |
| 的话 | 条件标记 | ...<ruby>的<rt>de</rt></ruby><ruby>话<rt>huà</rt></ruby> |
| 会 | 将来/可能 | <ruby>会<rt>huì</rt></ruby><ruby>下<rt>xià</rt></ruby><ruby>雨<rt>yǔ</rt></ruby> |
| 了 | 状态变化 | <ruby>下<rt>xià</rt></ruby><ruby>雨<rt>yǔ</rt></ruby><ruby>了<rt>le</rt></ruby> |

- 真实：<ruby>他<rt>tā</rt></ruby><ruby>来<rt>lái</rt></ruby>。（他来。）

## 快速比较

| 语境 | 西班牙语 | 英语 | 汉语 |
|---|---|---|---|
| 怀疑 | 虚拟式 | 情态（may） | 可能 + 动词 |
| 愿望 | 虚拟式 | 不定式（want to） | 想/希望 + 动词 |
| 反事实 | 虚拟式 | were/should | 如果...就... |
| 情感 | 虚拟式 | 直陈式 | 直陈式（无变化） |
| 否定命令 | 虚拟式 | Do + not | 别 + 动词 |
| 关系（不确定） | 虚拟式 | 直陈式 | 的 修饰 |
| 频率 | 日常 | 罕见（僵化） | 不适用 |
| 式标记 | 动词词尾 | 僵化形式 | 语境、副词 |

## 常见错误

1. **英语母语者**：西班牙语触发词后用直陈式 → Es posible que **llueva**（虚拟式）

2. **西班牙语母语者**：虚拟式直译英语 → I want you **to come**（不说 I want that you come）

3. **所有人**：认为汉语有虚拟式 → 汉语没有虚拟式

4. **英语母语者**：假设用 was → If I **were**（标准）
