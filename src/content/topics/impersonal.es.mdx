## Visión general

Las construcciones impersonales expresan eventos, estados u obligaciones sin identificar un agente específico.

- **Español**: Usa *se* (pasiva refleja/impersonal), tercera persona del plural (*dicen que*), verbos impersonales (*llueve, hace frío*) y *hay* para la existencia. El sujeto a menudo se omite por completo.
- **Inglés**: Usa sujetos vacíos (*it, there*), voz pasiva, pronombres genéricos (*you, one, they*) y nominalizaciones. No puede omitir el sujeto.
- **Chino**: Omite el sujeto por completo cuando el contexto lo permite. Usa sujetos implícitos para el tiempo, la hora y los enunciados generales. Sin pronombres vacíos.

## Español

### Se + verbo (impersonal)

El *se* impersonal reemplaza el sujeto:

- *Se habla español.* (Se habla español.)
- *Se vende coches.* (Se venden coches.)
- *Se dice que...* (Se dice que...)
- *No se puede fumar.* (No se puede fumar.)

**Nota**: Cuando el objeto precede al verbo, *se* se convierte en marcador pasivo:

- *El español **se** habla aquí.* (El español se habla aquí.)

### Verbos impersonales

Verbos españoles que solo existen en tercera persona:

| Verbo | Significado |
|---|---|
| llover | llover |
| nevar | nevar |
| tronar | tronar |
| amanecer | amanecer |
| anochecer | anochecer |
| hacer + sustantivo | hacer (tiempo) |

**Expresiones del tiempo**:

- *Llueve.* (Está lloviendo.)
- *Nieva.* (Está nevando.)
- *Hace frío.* (Hace frío.)
- *Hace calor.* (Hace calor.)
- *Hace viento.* (Hace viento.)
- *Hace sol.* (Hace sol.)

### Tercera persona del plural para "la gente"

- *Dicen que...* (Dicen que... / Se dice que...)
- *En mi país comen tarde.* (En mi país se cena tarde.)
- *Aquí bailan mucho.* (Aquí se baila mucho.)

### Hay / estar para existencia

- *Hay un problema.* (Hay un problema.)
- *Hay mucha gente.* (Hay mucha gente.)
- *No hay solución.* (No hay solución.)

### Expresiones impersonales

| Expresión | Significado |
|---|---|
| es necesario que | es necesario que |
| es importante que | es importante que |
| es posible que | es posible que |
| hay que + infinitivo | hay que |
| conviene que | conviene que |
| falta / sobra | falta / sobra |

### Se hace / se está haciendo

- *Se hace tarde.* (Se hace tarde.)
- *Se está haciendo tarde.* (Se está haciendo tarde.)

## Inglés

### Dummy it

El inglés requiere *it* para el tiempo, la hora y la distancia:

- *It is raining.* (Está lloviendo.)
- *It is snowing.* (Está nevando.)
- *It is cold.* (Hace frío.)
- *It is 5 o'clock.* (Son las cinco.)
- *It is Tuesday.* (Es martes.)
- *It is a long way to Tipperary.* (Es un largo camino a Tipperary.)

### Dummy there

*There* introduce la existencia:

- *There is a problem.* (Hay un problema.)
- *There are many people.* (Hay mucha gente.)
- *There is no solution.* (No hay solución.)
- *There seems to be a mistake.* (Parece que hay un error.)
- *There appears to be an error.* (Parece que hay un error.)

### You / one / they genéricos

- *You can't smoke here.* (regla general)
- *One must be careful.* (formal, genérico)
- *They say he's rich.* (gente no especificada)
- *People say he's rich.* (genérico explícito)

### Pasiva para omisión del agente

- *Spanish is spoken here.* (por gente no especificada)
- *The car was stolen.* (por alguien desconocido)
- *It is believed that...* (la gente cree que...)
- *It is known that...* (todo el mundo sabe que...)

### It + adjetivo + to-infinitivo

- *It is important to study.* (Es importante estudiar.)
- *It is necessary to leave.* (Es necesario irse.)
- *It is easy to learn.* (Es fácil aprender.)
- *It is difficult to say.* (Es difícil decir.)

### It seems / appears / happens

- *It seems that he left.* (Parece que se fue.)
- *It appears that we were wrong.* (Parece que nos equivocamos.)
- *It happens that I know him.* (Resulta que lo conozco.)

### Cleft it

- *It was John who broke the window.* (Fue John quien rompió la ventana.)
- *It is here that we met.* (Es aquí donde nos conocimos.)

## Errores comunes

1. **Angloparlantes aprendiendo español**: Usar ello/esto para el tiempo: *Ello llueve* → *Llueve*

2. **Angloparlantes aprendiendo chino**: Usar <ruby>它<rt>tā</rt></ruby> para el tiempo: *<ruby>它<rt>tā</rt></ruby><ruby>下<rt>xià</rt></ruby><ruby>雨<rt>yǔ</rt></ruby><ruby>了<rt>le</rt></ruby>* → *<ruby>下<rt>xià</rt></ruby><ruby>雨<rt>yǔ</rt></ruby><ruby>了<rt>le</rt></ruby>*

3. **Hispanohablantes aprendiendo inglés**: Omitir it: *Is raining* → *It is raining*

4. **Sinohablantes aprendiendo inglés**: Omitir there/it: *Has a problem* → *There is a problem*
