## 概述

Wh-移位是将疑问词置于从句开头的句法过程。

- **西班牙语**：疑问词强制移到句首。无 do-support 助动词。Qué vs cuál（什么 vs 哪个）区分。回声问句用正常语序。
- **英语**：疑问词强制移到句首。简单时态需要 do-support。主语疑问词不触发倒装。
- **汉语**：疑问词留在原位（原位置）。无移位。无助动词。词序与陈述句完全相同。

## 西班牙语

### 疑问词

| 西班牙语 | 英语 | 例句 |
|---|---|---|
| qué | what | ¿Qué quieres? |
| quién | who | ¿Quién vino? |
| dónde | where | ¿Dónde está? |
| adónde | where to | ¿Adónde vas? |
| cuándo | when | ¿Cuándo llega? |
| por qué | why | ¿Por qué lloras? |
| cuál | which | ¿Cuál prefieres? |
| cuánto | how much | ¿Cuánto cuesta? |
| cómo | how | ¿Cómo estás? |

### Qué vs cuál

**Qué** = 什么（开放，定义性）：
- *¿Qué es esto?*
- *¿Qué hora es?*

**Cuál** = 哪个（选择集合）：
- *¿Cuál prefieres?*
- *¿Cuál es tu favorito?*

### 移到句首

- *¿Qué **lees**?*
- *¿Quién **vino**?*
- *¿Dónde **está**?*

**注意**：无 do-support。动词-主语直接顺序。

### 回声问句

- *¿Qué dijiste?*（你说了什么？—— qué 高调）
- *¿A quién viste?*（你见了谁？）

**注意**：疑问词留在句首，不后置。

### 间接问句

- *No sé **qué** quiere.*
- *Me preguntó **quién** vino.*

## 英语

### 疑问词

| 英语 | 例句 |
|---|---|
| what | What do you want? |
| who | Who came? |
| where | Where is he? |
| when | When did he arrive? |
| why | Why do you cry? |
| which | Which do you prefer? |
| how | How are you? |
| how much/many | How much does it cost? |
| whose | Whose book is this? |

### 移位 + 倒装

**宾语疑问句**（需要倒装）：
- *What **are** you reading?*
- *Where **did** he go?*
- *Why **do** you cry?*

**主语疑问句**（不倒装）：
- *Who **came**?* (who = 主语)
- *What **happened**?* (what = 主语)

### Do-support

- *What **did** you **say**?* (NOT What said you?)
- *Where **does** he **live**?*

### 回声问句

- *You said **WHAT**?* (疑问词后置，高调)
- *He went **WHERE**?*

**注意**：英语是唯一允许疑问词后置在回声问句中的语言。

### Pied-piping

疑问词与其介词一起移动：
- *To **whom** did you speak?*（正式）
- *Who did you speak **to**?*（非正式，介词滞后）

### 间接问句

- *I don't know **what** he wants.*
- *He asked me **who** came.*

**注意**：间接问句中不倒装。

## 汉语

### 疑问词

| 汉语 | 英语 |
|---|---|
| <ruby>什<rt>shén</rt></ruby><ruby>么<rt>me</rt></ruby> | what |
| <ruby>谁<rt>shéi</rt></ruby> | who |
| <ruby>哪<rt>nǎ</rt></ruby><ruby>儿<rt>er</rt></ruby> / <ruby>哪<rt>nǎ</rt></ruby><ruby>里<rt>lǐ</rt></ruby> | where |
| <ruby>什<rt>shén</rt></ruby><ruby>么<rt>me</rt></ruby><ruby>时<rt>shí</rt></ruby><ruby>候<rt>hou</rt></ruby> | when |
| <ruby>为<rt>wèi</rt></ruby><ruby>什<rt>shén</rt></ruby><ruby>么<rt>me</rt></ruby> | why |
| <ruby>哪<rt>nǎ</rt></ruby> | which |
| <ruby>多<rt>duō</rt></ruby><ruby>少<rt>shǎo</rt></ruby> | how much/many |
| <ruby>怎<rt>zěn</rt></ruby><ruby>么<rt>me</rt></ruby> | how |
| <ruby>怎<rt>zěn</rt></ruby><ruby>么<rt>me</rt></ruby><ruby>样<rt>yàng</rt></ruby> | how / what kind |

### 留在原位（不移位）

汉语疑问词留在原位置：

- <ruby>你<rt>nǐ</rt></ruby><ruby>读<rt>dú</rt></ruby><ruby>什<rt>shén</rt></ruby><ruby>么<rt>me</rt></ruby>？(你读什么？= 你在读什么？)
- <ruby>谁<rt>shéi</rt></ruby><ruby>来<rt>lái</rt></ruby><ruby>了<rt>le</rt></ruby>？(谁来了？)
- <ruby>他<rt>tā</rt></ruby><ruby>去<rt>qù</rt></ruby><ruby>哪<rt>nǎ</rt></ruby><ruby>儿<rt>er</rt></ruby>？(他去哪儿？= 他去哪里了？)
- <ruby>你<rt>nǐ</rt></ruby><ruby>为<rt>wèi</rt></ruby><ruby>什<rt>shén</rt></ruby><ruby>么<rt>me</rt></ruby><ruby>哭<rt>kū</rt></ruby>？(你为什么哭？)

**注意**：词序与陈述句完全相同。

### 疑问词与量词

- <ruby>哪<rt>nǎ</rt></ruby><ruby>本<rt>běn</rt></ruby><ruby>书<rt>shū</rt></ruby>？(哪本书？—— <ruby>本<rt>běn</rt></ruby> = 量词)
- <ruby>哪<rt>nǎ</rt></ruby><ruby>个<rt>ge</rt></ruby><ruby>人<rt>rén</rt></ruby>？(哪个人？—— <ruby>个<rt>gè</rt></ruby> = 量词)

### 回声问句

- <ruby>你<rt>nǐ</rt></ruby><ruby>说<rt>shuō</rt></ruby><ruby>什<rt>shén</rt></ruby><ruby>么<rt>me</rt></ruby>？(你说什么？)
- <ruby>他<rt>tā</rt></ruby><ruby>去<rt>qù</rt></ruby><ruby>哪<rt>nǎ</rt></ruby><ruby>儿<rt>er</rt></ruby>？(他去哪儿？)

**注意**：汉语回声问句在结构上与常规 wh-问句完全相同。

### 间接问句

- <ruby>我<rt>wǒ</rt></ruby><ruby>不<rt>bù</rt></ruby><ruby>知<rt>zhī</rt></ruby><ruby>道<rt>dào</rt></ruby><ruby>他<rt>tā</rt></ruby><ruby>想<rt>xiǎng</rt></ruby><ruby>要<rt>yào</rt></ruby><ruby>什<rt>shén</rt></ruby><ruby>么<rt>me</rt></ruby>。(我不知道他想要什么。)
- <ruby>他<rt>tā</rt></ruby><ruby>问<rt>wèn</rt></ruby><ruby>谁<rt>shéi</rt></ruby><ruby>来<rt>lái</rt></ruby><ruby>了<rt>le</rt></ruby>。(他问谁来了。)

### Wh-不定代词

疑问词在某些语境中可表示不定指：

- <ruby>谁<rt>shéi</rt></ruby><ruby>都<rt>dōu</rt></ruby><ruby>知<rt>zhī</rt></ruby><ruby>道<rt>dào</rt></ruby>。(谁都知道。—— 谁 + <ruby>都<rt>dū</rt></ruby> = 所有人)
- <ruby>什<rt>shén</rt></ruby><ruby>么<rt>me</rt></ruby><ruby>都<rt>dōu</rt></ruby><ruby>不<rt>bù</rt></ruby><ruby>要<rt>yào</rt></ruby>。(我什么都不要。—— 什么 + <ruby>都<rt>dū</rt></ruby> = 什么都没有)

## 快速对比

| 特征 | 西班牙语 | 英语 | 汉语 |
|---|---|---|---|
| Wh-移位 | 有（强制） | 有（强制） | 无（原位） |
| 助动词倒装 | 无 | 有（do-support） | 无 |
| 回声问句 | 句首（正常语序） | 句末（唯一） | 原位（与常规相同） |

| Wh-不定代词 | alguien/algo | someone/something | <ruby>谁<rt>shuí</rt></ruby><ruby>都<rt>dū</rt></ruby> / <ruby>什<rt>shí</rt></ruby><ruby>么<rt>mó</rt></ruby><ruby>都<rt>dū</rt></ruby> |

## 常见错误

1. **英语者学汉语**：<ruby>什<rt>shí</rt></ruby><ruby>么<rt>mó</rt></ruby><ruby>你<rt>nǐ</rt></ruby><ruby>读<rt>dú</rt></ruby>？ → <ruby>你<rt>nǐ</rt></ruby><ruby>读<rt>dú</rt></ruby>**<ruby>什<rt>shí</rt></ruby><ruby>么<rt>mó</rt></ruby>**？（wh-in-situ）

2. **汉语者学英语**：What you are reading? → What **are** you reading?（倒装）

3. **英语者学西班牙语**：Qué quieres tú? → ¿Qué **quieres**?（无 do-support，无额外代词）

4. **西班牙语者学英语**：What you want? → What **do** you want?（do-support）
