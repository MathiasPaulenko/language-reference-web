## 概述

所有语言都允许比较物体属性，但使用不同形态和句法策略：

- **西班牙语**：用 *más...que, menos...que, tan...como*。不规则比较级：*mejor/peor/mayor/menor*。相对最高级用 *el más...de*。
- **英语**：用 *-er/more...than, -est/most...in/of, as...as*。不规则比较级：*good→better→best, bad→worse→worst*。
- **中文**：用 *<ruby>比<rt>bǐ</rt></ruby>* 表比较级，*<ruby>最<rt>zuì</rt></ruby>* 表最高级，*<ruby>跟<rt>gēn</rt></ruby>...<ruby>一<rt>yí</rt></ruby><ruby>样<rt>yàng</rt></ruby>* 表同级。无屈折变化，形容词不变。

## 西班牙语

### 较高级比较

**más + 形容词 + que**：

- *Juan es **más alto que** Pedro.* (胡安比佩德罗高。)
- *Este libro es **más interesante que** el otro.* (这本书比那本有趣。)

**动词**：

- *María **come más que** Juan.* (玛丽亚比胡安吃得多。)
- *Yo **trabajo más que** tú.* (我比你工作得多。)

### 较低级比较

**menos + 形容词 + que**：

- *Pedro es **menos alto que** Juan.* (佩德罗不如胡安高。)
- *Este problema es **menos difícil que** el otro.* (这个问题不如那个难。)

### 相对最高级

**el/la/los/las + más/menos + 形容词 + de**：

- *Es **el más alto de** la clase.* (他是班里最高的。)
- *Es **la más inteligente de** todas.* (她是所有人中最聪明的。)

### 绝对最高级

**-ísimo**（后缀）：

- *altísimo* (非常高)
- *rapidísimo* (非常快)
- *fácilísimo* (非常容易)

### 不规则比较级

| 原级 | 比较级 | 最高级 |
|---|---|---|
| bueno | mejor | el mejor |
| malo | peor | el peor |
| grande | mayor（指人） | el mayor |
| pequeño | menor（指人） | el menor |
| mucho | más | - |
| poco | menos | - |

**注意**：*mayor/menor* 用于人（年龄），不用于物理大小：

- *Mi hermano **mayor*** (我的哥哥——年龄)
- *Una casa **más grande*** (一所更大的房子——大小)

### 同级

**tan + 形容词 + como**：

- *Juan es **tan alto como** Pedro.* (胡安和佩德罗一样高。)
- *No es **tan difícil como** parece.* (它不像看起来那么难。)

**tanto + 名词 + como**：

- *Tiene **tanto dinero como** yo.* (他和我一样有钱。)
- *No leo **tanto como** tú.* (我不像你读得那么多。)

### 渐进

**cada vez más / menos + 形容词**：

- *Está **cada vez más** alto.* (他越来越高。)
- *Es **cada vez menos** interesante.* (它越来越没意思。)

## 英语

### 较高级比较

**-er + than**（短形容词）或 **more + 形容词 + than**（长形容词）：

- *John is **taller than** Peter.*
- *This book is **more interesting than** the other.*
- *She runs **faster than** me.*

**单音节规则**：始终 -er：

- tall → taller, fast → faster, big → bigger

**双音节规则**：以 -y, -er, -er, -le 结尾用 -er：

- happy → happier, simple → simpler

**长形容词**（3+音节）：more：

- beautiful → more beautiful, expensive → more expensive

### 最高级

**-est** 或 **most + 形容词**：

- *He is **the tallest** in the class.*
- *She is **the most intelligent** of all.*
- *This is **the fastest** car.*

### 不规则比较级

| 原级 | 比较级 | 最高级 |
|---|---|---|
| good | better | the best |
| bad | worse | the worst |
| far | farther / further | the farthest / furthest |
| little | less | the least |
| much / many | more | the most |
| old | older / elder | the oldest / eldest |

**Elder/eldest** 只用于家庭成员语境：

- *My **elder** brother* (我的哥哥)
- *The **eldest** daughter* (大女儿)

### 同级

**as + 形容词 + as**：

- *John is **as tall as** Peter.*
- *This isn't **as difficult as** it seems.*

### 同级否定

**not as/so + 形容词 + as**：

- *This car isn't **as fast as** mine.*
- *I'm not **so tall as** you.*

### 渐进

**comparative + and + comparative**：

- *It's getting **taller and taller**.*
- *She's becoming **more and more** beautiful.*

**The + comparative, the + comparative**：

- *The **more** you practice, the **better** you get.*
- *The **sooner**, the **better**.*

## 常见错误

1. **英语母语者学西班牙语**：*más bueno* → *mejor*，*más malo* → *peor*

2. **西班牙语母语者学英语**：省略 than：She is taller me → She is taller **than** me

3. **英语母语者学中文**：*<ruby>比<rt>bǐ</rt></ruby>...<ruby>更<rt>gèng</rt></ruby>* → 只用 *<ruby>比<rt>bǐ</rt></ruby>...*（冗余）

4. **所有人**：西班牙语混淆 *mayor*（年龄）与 *más grande*（大小）
