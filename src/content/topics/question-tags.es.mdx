## Visión general

El inglés tiene etiquetas de pregunta gramaticalizadas que invierten polaridad. El español usa verdad/que no. El chino usa <ruby><ruby>吗<rt>ma</rt></ruby><rt>ma</rt></ruby>/<ruby><ruby>吧<rt>bā</rt></ruby><rt>bā</rt></ruby>/<ruby><ruby>对<rt>duì</rt></ruby><rt>duì</rt></ruby><ruby><ruby>不<rt>bù</rt></ruby><rt>bù</rt></ruby><ruby><ruby>对<rt>duì</rt></ruby><rt>duì</rt></ruby>/<ruby><ruby>是<rt>shì</rt></ruby><rt>shì</rt></ruby><ruby><ruby>不<rt>bù</rt></ruby><rt>bù</rt></ruby><ruby><ruby>是<rt>shì</rt></ruby><rt>shì</rt></ruby>/<ruby><ruby>是<rt>shì</rt></ruby><rt>shì</rt></ruby><ruby><ruby>吧<rt>bā</rt></ruby><rt>bā</rt></ruby>.

Los tres idiomas abordan este concepto gramatical de maneras distintas que reflejan sus sistemas subyacentes. A continuación se presenta una comparación detallada.

## Resumen

Las etiquetas de pregunta buscan confirmación o expresan duda sobre un enunciado. Cada lengua tiene su propio sistema.

- **Español**: Usa etiquetas invariables (*¿verdad?*, *¿no?*, *¿cierto?*) que no invierten el verbo.
- **Inglés**: Tiene un sistema gramaticalizado donde la etiqueta copia el auxiliar e invierte la polaridad.
- **Chino**: Usa partículas finales (*<ruby>吧<rt>bā</rt></ruby>*, *<ruby>吗<rt>ma</rt></ruby>*) o estructuras reduplicativas (*<ruby>对<rt>duì</rt></ruby><ruby>不<rt>bù</rt></ruby><ruby>对<rt>duì</rt></ruby>*, *<ruby>是<rt>shì</rt></ruby><ruby>不<rt>bù</rt></ruby><ruby>是<rt>shì</rt></ruby>*).

## Español

### Etiquetas invariables

El español no invierte el verbo para formar etiquetas de pregunta. En su lugar, usa palabras invariables:

| Etiqueta | Uso | Ejemplo |
|---|---|---|
| ¿verdad? | General | Vienes, **¿verdad?** |
| ¿no? | General | Está aquí, **¿no?** |
| ¿cierto? | Confirmación | Es así, **¿cierto?** |
| ¿o sí? | Después de negativo | No viene, **¿o sí?** |
| ¿o no? | Después de afirmativo | Viene, **¿o no?** |
| ¿a que sí? | Retórica | Es bonito, **¿a que sí?** |

### Ejemplos

- Vienes mañana, **¿verdad?**
- No está aquí, **¿no?**
- Lo entiendes, **¿cierto?**
- Hace calor, **¿no?**

**Nota**: Las etiquetas españolas son invariables; no cambian según el tiempo o el verbo.

### Etiquetas regionales

| Región | Etiqueta | Ejemplo |
|---|---|---|
| España | ¿vale? | Vienes, **¿vale?** |
| Rioplatense | ¿o? | Vienes, **¿o?** |
| México | ¿verdad? | Vienes, **¿verdad?** |

## Inglés

### Inversión de polaridad

En inglés, la etiqueta invierte la polaridad del enunciado principal:

| Enunciado | Etiqueta | Ejemplo |
|---|---|---|
| Positivo | Negativo | You're coming, **aren't you**? |
| Negativo | Positivo | He's not here, **is he**? |

### Copia del auxiliar

La etiqueta copia el auxiliar (o forma de be) del enunciado principal:

| Auxiliar | Ejemplo |
|---|---|
| be | She is here, **isn't she**? |
| do | You like it, **don't you**? |
| have | They've left, **haven't they**? |
| modales | He can swim, **can't he**? |

### Contracciones negativas

| Completo | Contraído |
|---|---|
| are not | aren't |
| is not | isn't |
| do not | don't |
| does not | doesn't |
| did not | didn't |
| have not | haven't |
| has not | hasn't |
| will not | won't |
| would not | wouldn't |
| can not | can't |
| could not | couldn't |

**Nota**: Las etiquetas negativas SIEMPRE usan la forma contraída.

### Preguntas de eco

Las preguntas de eco repiten un elemento con énfasis para expresar sorpresa:

- You did **WHAT**?
- You went **WHERE**?
- She paid **HOW MUCH**?

**Nota**: La palabra cuestionada recibe énfasis prosódico fuerte.

### Etiquetas universales

| Etiqueta | Uso | Registro |
|---|---|---|
| right? | General | Neutral |
| yeah? | General | Informal |
| innit? | General | No estándar (Británico) |

## Errores comunes

1. **Hispanohablantes**: Invertir el verbo en etiquetas → Vienes, **¿verdad?** (NO ¿no vienes?)

2. **Angloparlantes**: Confundir <ruby>吗<rt>ma</rt></ruby> y <ruby>吧<rt>bā</rt></ruby> en chino → <ruby>对<rt>duì</rt></ruby><ruby>吧<rt>ba</rt></ruby>（espera acuerdo）vs <ruby>对<rt>duì</rt></ruby><ruby>吗<rt>ma</rt></ruby>（neutral）

3. **Angloparlantes**: Olvidar contracción en etiquetas negativas → He's here, **isn't** he? (NO is not he?)

4. **Chinos**: No usar entonación ascendente en preguntas de eco del inglés → You did **WHAT?**
