## Overview

Numbers seem universal, but each language has quirks in formation, counting, and agreement.

- **Spanish**: Numbers 1-15 are unique. 16+ follow a pattern but have contractions. Numbers agree with nouns they precede. Ordinals are adjectives that agree.
- **English**: Teens are mostly regular (thir-teen, four-teen). Irregular teens: eleven, twelve. Ordinals mostly add -th. No agreement.
- **Chinese**: The most logical system. 11 = 10+1, 20 = 2×10, 100 = 1×100. No irregular forms. Uses <ruby>两<rt>liǎng</rt></ruby> instead of <ruby>二<rt>èr</rt></ruby> before classifiers. <ruby>万<rt>wàn</rt></ruby> is a separate unit (10,000).

### Agreement of uno

*Uno* shortens before masculine nouns and changes to *una* before feminine:

- *un libro* (a book)
- *una mesa* (a table)
- *veintiún años* (twenty-one years — contracted)
- *veintiuna personas* (twenty-one people)

### Hundreds and thousands

- *doscientos, trescientos... novecientos* (agree in gender: *doscientas mesas*)
- *mil* (invariable: *mil libros*)
- *dos mil, tres mil...*
- *un millón / dos millones* (takes *de*: *dos millones de personas*)

| Context | Use |
|---|---|
| Counting 1, 2, 3... | <ruby>二<rt>èr</rt></ruby> |
| Math, phone numbers, dates | <ruby>二<rt>èr</rt></ruby> |

- <ruby>二<rt>èr</rt></ruby><ruby>加<rt>jiā</rt></ruby><ruby>二<rt>èr</rt></ruby><ruby>等<rt>děng</rt></ruby><ruby>于<rt>yú</rt></ruby><ruby>四<rt>sì</rt></ruby>。(2 + 2 = 4)

## Comparison at a glance

| Feature | Spanish | English | Chinese |
|---------|---------|---------|---------|
| Teens (11-19) | Irregular (11-15), then regular | Irregular (11, 12), then regular | Completely regular (10+1) |
| Tens formation | -enta ending | -ty ending | number + <ruby>十<rt>shí</rt></ruby> |
| Number agreement | Yes (uno shortens, ordinals agree) | No | No |
| Classifier needed | No | No | Yes |
| Separate 10,000 unit | No | No | Yes (<ruby>万<rt>wàn</rt></ruby>) |
| Ordinals | primero/segundo... | first/second... | <ruby>第<rt>dì</rt></ruby> + number |
| 2 forms | No | No | Yes (<ruby>二<rt>èr</rt></ruby> vs <ruby>两<rt>liǎng</rt></ruby>) |
| Large number system | mil, millón | thousand, million | <ruby>万<rt>wàn</rt></ruby>, <ruby>亿<rt>yì</rt></ruby> |

## Common mistakes

1. **English speakers learning Chinese**: Using <ruby>二<rt>èr</rt></ruby> before classifiers: *<ruby>二<rt>èr</rt></ruby><ruby>个<rt>gè</rt></ruby><ruby>人<rt>rén</rt></ruby>* → *<ruby>两<rt>liǎng</rt></ruby><ruby>个<rt>gè</rt></ruby><ruby>人<rt>rén</rt></ruby>*

2. **Chinese speakers learning Spanish**: Not shortening uno: *uno libro* → *un libro*

3. **English speakers learning Spanish**: Forgetting agreement in ordinals: *el primera vez* → *la **primera** vez*

4. **English speakers learning Chinese**: Translating 20,000 as <ruby>二<rt>èr</rt></ruby><ruby>十<rt>shí</rt></ruby><ruby>千<rt>qiān</rt></ruby> → <ruby>两<rt>liǎng</rt></ruby><ruby>万<rt>wàn</rt></ruby>
