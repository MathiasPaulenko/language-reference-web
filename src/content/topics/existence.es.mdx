## Visión general

La existencia trata sobre si algo existe, está presente, o se encuentra en algún lugar.

- **Español**: *Haber* (*hay, había, hubo*) para existencia impersonal. *Estar* para localización. *Ser* para identidad.
- **Inglés**: *There is/are* para existencia presentacional. *Be* para localización. *Exist* para filosofía/abstracción.
- **Chino**: *<ruby>有<rt>yǒu</rt></ruby>* para existencia. *<ruby>在<rt>zài</rt></ruby>* para localización. *<ruby>是<rt>shì</rt></ruby>* para identidad. No hay marcadores de definitud.

## Inglés

### Existencia: There is/are

**Singular**:
- There is a problem. (Hay un problema.)
- There is no time. (No hay tiempo.)
- Is there a doctor? (¿Hay un doctor?)

**Plural**:
- There are many books. (Hay muchos libros.)
- There are no solutions. (No hay soluciones.)
- Are there any questions? (¿Hay alguna pregunta?)

**Pasado**:
- There was a fire. (Hubo un incendio.)
- There were people waiting. (Había gente esperando.)

**Futuro**:
- There will be time. (Habrá tiempo.)
- There will be problems. (Habrá problemas.)

### Localización: Be

- The book is on the table. (El libro está en la mesa.)
- Where are you? (¿Dónde estás?)
- I'm at home. (Estoy en casa.)
- The store is nearby. (La tienda está cerca.)

### Identidad: Be

- It's a book. (Es un libro.)
- London is the capital. (Londres es la capital.)
- I'm a doctor. (Soy médico.)

### Exist (formal/filosófico)

- God exists. (Dios existe.)
- There is no such thing. (No existe tal cosa.)
- Love does not exist. (El amor no existe.)

### Negación

- There isn't any money. (No hay dinero.)
- There is no justice. (No hay justicia.)
- He isn't here. (Él no está aquí.)
- It isn't true. (No es verdad.)

### Contracciones en existenciales

| Completo | Contracción |
|---|---|

| There is not | There isn't / There's not |
| There are not | There aren't |
| There was not | There wasn't |
| There were not | There weren't |

- I have a book. (Tengo un libro. — posesión)

## Errores comunes

1. **Angloparlantes aprendiendo español**: *Ser* para localización: *El libro es en la mesa* → *El libro **está** en la mesa*

2. **Hispanohablantes aprendiendo inglés**: Omitir *there*: *A problem is* → *There is a problem*

3. **Sinohablantes aprendiendo inglés**: *Is a problem* → *There **is** a problem*

4. **Angloparlantes aprendiendo chino**: *<ruby>是<rt>shì</rt></ruby>* para existencia: *<ruby>是<rt>shì</rt></ruby>yǒu* → *<ruby>有<rt>yǒu</rt></ruby>*

5. **Hispanohablantes aprendiendo chino**: *<ruby>没<rt>méi</rt></ruby><ruby>有<rt>yǒu</rt></ruby>méi yǒu* para identidad: *<ruby>不<rt>bù</rt></ruby><ruby>是<rt>shì</rt></ruby>*
