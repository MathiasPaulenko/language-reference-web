## 概述

正字法是一套书写语言的惯例。

- **西班牙语**：表音字母表，音形对应规则。变音符号标记重音并区分同音词。大小写有限。
- **英语**：深层正字法，大量不规则现象。不发音字母，同一音多种拼写。大小写广泛。
- **汉语**：表意文字系统。汉字为原子单位。无空格，无大小写。拼音提供语音转写。

## 西班牙语

### 字母与发音

西班牙语正字法高度规则：

| 字母 | 发音 | 例词 |
|---|---|---|
| a | /a/ | c**a**s**a** |
| e | /e/ | m**e**s**a** |
| i | /i/ | l**i**bro |
| o | /o/ | p**o**rta |
| u | /u/ | l**u**na |
| ñ | /ɲ/ | **ñ**o**ñ**o |

### 二合字母

| 二合字母 | 发音 | 例词 |
|---|---|---|
| ch | /tʃ/ | **ch**ico |
| ll | /ʝ/ 或 /ʎ/ | ca**ll**e |
| rr | /r/ | pe**rr**o |
| qu | /k/ | **qu**eso |
| gu + e,i | /g/ | **gu**erra |
| gü | /gw/ | lin**gü**ista |

### 重音规则与书写重音

| 词尾 | 重音位置 | 例词 |
|---|---|---|
| 元音, -n, -s | 倒数第二音节 | habl**a**n, mes**a**s |
| 辅音（非 -n, -s） | 最后音节 | habl**ar**, pape**l** |
| 例外 | 用重音符号标记 | habl**ó**, term**í**n**o** |

### 大小写

**大写**：
- 句首单词
- 专有名词（人名、地名、机构名）
- 首字母缩略词

**小写**：
- 月份、星期几
- 语言、国籍
- 宗教、头衔（除非作为名称的一部分）

## 英语

### 字母与发音

英语有深层正字法：

| 拼写 | 发音 | 例词 |
|---|---|---|
| a | /æ/, /eɪ/, /ɑː/ | c**a**t, c**a**ke, f**a**ther |
| e | /e/, /iː/ | b**e**d, s**ee** |
| i | /ɪ/, /aɪ/ | s**i**t, f**i**ne |
| o | /ɒ/, /oʊ/, /uː/ | h**o**t, g**o**, d**o** |
| u | /ʌ/, /juː/ | c**u**t, c**u**te |
| gh | 不发音, /f/, /g/ | ni**gh**t, lau**gh**, **gh**ost |
| th | /θ/, /ð/ | **th**ink, **th**is |

### 大小写

**大写**：
- 句首单词
- 专有名词
- I（总是大写）
- 月份、星期
- 语言、国籍
- 头衔（Professor, President）

### 拼写模式

| 模式 | 规则 | 例词 |
|---|---|---|
| 不发音e | 拉长元音 | m**a**ke, l**i**ke |
| 双辅音 | 短元音 | s**i**tter, b**a**gger |
| -tion | /ʃən/ | na**tion** |
| -sion | /ʒən/ | vi**sion** |

## 汉语

### 汉字类型

| 类型 | 描述 | 例字 |
|---|---|---|
| 象形 | 物体图形 | <ruby>日<rt>rì</rt></ruby> 太阳, <ruby>月<rt>yuè</rt></ruby> 月亮, <ruby>山<rt>shān</rt></ruby> 山 |
| 指事 | 抽象概念 | <ruby>上<rt>shàng</rt></ruby> 上, <ruby>下<rt>xià</rt></ruby> 下, <ruby>本<rt>běn</rt></ruby> 根 |
| 会意 | 组合意义 | <ruby>明<rt>míng</rt></ruby> 明亮（日+月） |
| 形声 | 声旁+意旁 | <ruby>妈<rt>mā</rt></ruby> mā（母：<ruby>女<rt>nǚ</rt></ruby>；声：<ruby>马<rt>mǎ</rt></ruby>） |

### 笔顺

汉字按特定顺序书写：
1. 从上到下
2. 从左到右
3. 先横后竖
4. 先外后内
5. 先中间后两边

### 简体与繁体

| 简体 | 繁体 |
|---|---|
| <ruby>国<rt>guó</rt></ruby> | <ruby>國<rt>guó</rt></ruby> |
| <ruby>学<rt>xué</rt></ruby> | <ruby>學<rt>xué</rt></ruby> |
| <ruby>爱<rt>ài</rt></ruby> | <ruby>愛<rt>ài</rt></ruby> |
| <ruby>书<rt>shū</rt></ruby> | <ruby>書<rt>shū</rt></ruby> |
| <ruby>龙<rt>lóng</rt></ruby> | <ruby>龍<rt>lóng</rt></ruby> |

### 拼音

语音转写：
- <ruby>中<rt>zhōng</rt></ruby><ruby>国<rt>guó</rt></ruby> = Zhōngguó
- 声调：ā á ǎ à

### 词间无空格

汉语书写词间无空格：
- <ruby>我<rt>wǒ</rt></ruby><ruby>是<rt>shì</rt></ruby><ruby>一<rt>yī</rt></ruby><ruby>个<rt>gè</rt></ruby><ruby>学<rt>xué</rt></ruby><ruby>生<rt>shēng</rt></ruby>（我是一名学生）
- 读者根据上下文切分

### 无大小写

- <ruby>我<rt>wǒ</rt></ruby> = <ruby>我<rt>wǒ</rt></ruby>（无大小写区别）
- 专有名词不大写

## 对比一览

| 特征 | 西班牙语 | 英语 | 汉语 |
|---|---|---|---|
| 文字 | 字母 | 字母 | 表意 |
| 音形对应 | 高度规则 | 不规则 | 不适用（拼音=表音） |
| 字符数 | 27个字母 | 26个字母 | ~50,000个（常用3,500） |
| 重音标记 | 有（书写重音符号） | 无 | 无（拼音有声调符号） |
| 大小写 | 有限 | 广泛 | 无 |
| 空格 | 有 | 有 | 无 |
| 断字 | 有（按音节） | 有（按音节/词素） | 无 |
| 方向 | 左到右 | 左到右 | 左到右（现代）或上到下（传统） |

## 常见错误

1. **英语者学西班牙语**：将月份/星期大写 → 西班牙语小写

2. **西班牙语者学英语**：忘记大写 I、语言、国籍

3. **英语者学汉语**：假设字=词 → 字=语素

4. **汉语者学英语**：专有名词大小写不一致
