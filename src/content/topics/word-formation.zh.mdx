## 概述

构词法是从现有词创造新词的方式。

- **西班牙语**：广泛使用派生词缀（*-idad, -mente, -ción, -ero, -dor*）。复合不如英语常见，透明度也更低。
- **英语**：前缀、后缀、复合和转换（零派生）都极具能产性。新复合词和转换不断出现。
- **汉语**：几乎完全依赖复合。词缀极其有限。大多数词是由两个汉字组成的复合词，每个字贡献意义。

## 西班牙语

### 前缀

| 前缀 | 意义 | 例词 |
|---|---|---|
| in-/im-/i- | 否定 | *imposible, ilegal, injusto* |
| des- | 相反，去除 | *desconocer, deshacer, desgracia* |
| re- | 再次 | *rehacer, repetir, regresar* |
| pre- | 之前 | *prever, predecir, previo* |
| sub- | 下面 | *subterráneo, subrayar* |
| super-/sobre- | 上面 | *superficial, sobrepasar* |
| anti- | 反对 | *antiguo, antipatía* |
| contra- | 反对 | *contradecir, contrario* |
| ex- | 前- | *exmarido, exministro* |
| auto- | 自动 | *autobús, automático* |

### 后缀

**名词化**：

| 后缀 | 例词 | 意义 |
|---|---|---|
| -dad / -tad | felicidad, libertad | 抽象名词 |
| -ción / -sión | creación, decisión | 动作/结果 |
| -miento | descubrimiento | 动作结果 |
| -ura | escritura, pintura | 动作/结果 |
| -ero / -era | panadero, zapatero | 职业/场所 |
| -dor / -dora | cantador, cantadora | 施事 |
| -ista | artista, turista | 职业/信奉者 |
| -aje | paisaje, equipaje | 集合/结果 |
| -ble | amable, posible | 能够 |

**形容词化**：

| 后缀 | 例词 |
|---|---|
| -oso / -osa | hermoso, peligroso |
| -able / -ible | comestible, amable |

**副词化**：

| 后缀 | 例词 |
|---|---|
| -mente | rápidamente, felizmente |

**注意**：*-mente* 与阴性形式结合：

- *rápido* → *rápida* → *rápidamente*
- *feliz* → *feliz* → *felizmente*

### 复合词（有限）

西班牙语复合词存在但透明度较低：

- *limpiaparabrisas* (limpiaparabrisas — limpia-parabrisas)
- *sacacorchos* (sacacorchos — saca-corchos)
- *parabrisas* (parabrisas — para-brisas)
- *lavaplatos* (lavaplatos — lava-platos)
- *rascacielos* (rascacielos — rasca-cielos)

**注意**：这些通常是动词-宾语复合词。

### 转换

与英语相比有限：

- *el presidente* (名词) vs *presidente* (头衔，无冠词)

### 指小词和指大词

| 后缀 | 意义 | 例词 |
|---|---|---|
| -ito / -ita | 指小 | *casita*, *perrito* |
| -illo / -illa | 指小（轻微贬义） | *casilla* |
| -ón / -ona | 指大 | *casón*, *perrona* |
| -ote / -ota | 指大（贬义） | *casote* |

| -ery / -ry | bakery, bravery | 场所/性质 |

**形容词化**：

| 后缀 | 例词 |
|---|---|

| 后缀 | 例词 |
|---|---|
| -ly | quickly, happily |
| -ward(s) | forward, homeward |

**动词化**：

| 后缀 | 例词 |
|---|---|
| -en | widen, strengthen |
| -ify | simplify, beautify |

### 复合词

英语复合词极具能产性：

**名词 + 名词**：
- *blackboard, bookshelf, bedroom, toothbrush*

**形容词 + 名词**：
- *blackbird, heavyweight, blueprint*

**动词 + 名词**：
- *scarecrow, pickpocket, breakfast*

**名词 + 动词**：
- *sunrise, snowfall, rainfall*

**副词/介词 + 名词**：
- *overcoat, underground, afterthought*

### 转换（零派生）

英语自由地在词类间转换：

| 名词 → 动词 | 动词 → 名词 | 形容词 → 动词 |
|---|---|---|
| email (to email) | run (a run) | empty (to empty) |
| bottle (to bottle) | walk (a walk) | clean (to clean) |
| chair (to chair) | talk (a talk) | dry (to dry) |
| knife (to knife) | look (a look) | better (to better) |
| Google (to google) | guess (a guess) | lower (to lower) |

### 混成词

- *smoke + fog = smog*
- *breakfast + lunch = brunch*
- *web + log = blog*
- *emotion + icon = emoji*（来自日语）

### 首字母缩略词

- *NASA, UNESCO, laser, radar, scuba*

### 逆构词

- *editor → edit*（最初 editor 来自拉丁语，后来逆构出 edit）
- *television → televise*
- *babysitter → babysit*

## 汉语

汉语几乎通过组合汉字来构成所有新词：

**修饰语 + 中心语**：

- <ruby>黑<rt>hēi</rt></ruby><ruby>板<rt>bǎn</rt></ruby> (黑 + 板 = 黑板)
- <ruby>火<rt>huǒ</rt></ruby><ruby>车<rt>chē</rt></ruby> (火 + 车 = 火车)

- <ruby>飞<rt>fēi</rt></ruby><ruby>机<rt>jī</rt></ruby> (飞 + 机 = 飞机)
- <ruby>洗<rt>xǐ</rt></ruby><ruby>手<rt>shǒu</rt></ruby><ruby>间<rt>jiān</rt></ruby> (洗 + 手 + 间 = 洗手间)

**动词 + 宾语**：

- <ruby>看<rt>kàn</rt></ruby><ruby>书<rt>shū</rt></ruby> (看 + 书 = 读书/学习)
- <ruby>吃<rt>chī</rt></ruby><ruby>饭<rt>fàn</rt></ruby> (吃 + 饭 = 吃饭)
- <ruby>睡<rt>shuì</rt></ruby><ruby>觉<rt>jiào</rt></ruby> (睡 + 觉 = 睡觉)

**动词 + 补语**：

- <ruby>打<rt>dǎ</rt></ruby><ruby>破<rt>pò</rt></ruby> (打 + 破 = 打破)

## 快速对比

| 策略 | 西班牙语 | 英语 | 汉语 |
|------|---------|------|------|
| 前缀 | 极有能产性 | 极有能产性 | 有限（<ruby>不<rt>bù</rt></ruby>, <ruby>非<rt>fēi</rt></ruby>, <ruby>可<rt>kě</rt></ruby>, <ruby>再<rt>zài</rt></ruby>） |

| 混成 | 罕见 | 有能产性 | 无 |
| 重叠 | 仅指小词 | 仅非正式 | 极度能产性 |
| 指小词 | -ito/-illa | -y/-ie | 重叠 / <ruby>小<rt>xiǎo</rt></ruby> |

## 常见错误

1. **英语者学西班牙语**：Casa-coche 表示车库 → *garaje*（西班牙语不像英语那样自由复合）

2. **英语者学汉语**：<ruby>快<rt>kuài</rt></ruby><ruby>乐<rt>lè</rt></ruby>-ness → <ruby>快<rt>kuài</rt></ruby><ruby>乐<rt>lè</rt></ruby><ruby>感<rt>gǎn</rt></ruby>（<ruby>感<rt>gǎn</rt></ruby> 添加抽象性质；-ness 不直接对应）

3. **汉语者学英语**：I am very happiness → I am very **happy**（英语用形容词，不用名词）

4. **西班牙语者学英语**：I am agree → I **agree**（英语转换：形容词 → 动词，不是西班牙语式动词结构）
