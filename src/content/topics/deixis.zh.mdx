## 概述

指示是指那些相对于说话者位置、时间或身份进行指向的词语。

- **西班牙语**：在指示词和地点副词中都有三元空间区分（近我、近你、远离双方）。时间指示包括 *ahora, entonces, antes, después*。
- **英语**：二元空间体系（*this/that, here/there*）。时间指示使用 *now, then, today, yesterday, tomorrow*。存在一些古体形式（*yon, yonder*）。
- **中文**：二元体系（*<ruby>这<rt>zhè</rt></ruby>/<ruby>那<rt>nà</rt></ruby>*）。地点副词使用相同指示词 + *<ruby>里<rt>lǐ</rt></ruby>/<ruby>儿<rt>ér</rt></ruby>/<ruby>边<rt>biān</rt></ruby>*。时间指示简单且依赖语境。

## 西班牙语

### 三元指示词

| | 近说话者 | 近听话者 | 远离双方 |
|---|---|---|---|
| 阳性 | este | ese | aquel |
| 阴性 | esta | esa | aquella |
| 中性 | esto | eso | aquello |
| 复数（阳） | estos | esos | aquellos |
| 复数（阴） | estas | esas | aquellas |

**例子**：

- *Este libro es mío.*（这本书[近我]是我的。）
- *Ese libro es tuyo.*（那本书[近你]是你的。）
- *Aquel libro es de Juan.*（那本书[远处]是胡安的。）

### 地点副词

| 近说话者 | 近听话者 | 远离双方 |
|---|---|---|
| aquí | ahí | allí / allá |

- *Ven **aquí**.*（来这儿[找我]。）
- *Ponlo **ahí**.*（放在那儿[近你处]。）
- *Allá en el campo...*（在那边的乡下……）

**注意**：*Allá* 比 *allí* 稍微更远/更强调。

### 时间指示

| 词 | 含义 |
|---|---|
| ahora | 现在 |
| hoy | 今天 |
| ayer | 昨天 |
| mañana | 明天 |
| entonces | 那时 |
| antes | 之前 |
| después | 之后 |
| ahora mismo | 就在此刻 |
| en aquel entonces | 在那个时候 |

### 人称指示

| 人称 | 主语 | 宾语 | 反身 |
|---|---|---|---|
| 我 | yo | me | me |
| 你 | tú | te | te |
| 您 | usted | lo/la | se |
| 他/她 | él/ella | lo/la | se |
| 我们 | nosotros | nos | nos |
| 你们/诸位 | vosotros/ustedes | os/los/las | os/se |
| 他们 | ellos | los/las | se |

### 动词指示（来/去，带来/带去）

- *venir*（向说话者来）
- *ir*（离说话者去）
- *traer*（向说话者带来）
- *llevar*（离说话者带去）

## 英语

### 二元指示词

| 近说话者 | 远说话者 |
|---|---|
| this | that |
| these（复数） | those（复数） |

**例子**：

- *This book is mine.*（近我）
- *That book is yours.*（远我，近你或远处）

### 地点副词

| 近说话者 | 远说话者 |
|---|---|
| here | there |

- *Come **here**.*（来我这儿）
- *Go **there**.*（去那儿）
- *Over **there***（更远）
- *Right **here** / Right **there***（强调）

**古体**：*yonder*（yon mountain = 那边的山）

### 时间指示

| 词 | 含义 |
|---|---|
| now | 当前时刻 |
| then | 过去或未来的参照点 |
| today | 今天 |
| yesterday | 昨天 |
| tomorrow | 明天 |
| tonight | 今晚 |
| this morning | 今天早上 |
| last week | 上周 |
| next week | 下周 |
| ago | 以前（3 days ago）|
| hence | 以后（3 days hence — 古体）|

### 人称指示

| 人称 | 主语 | 宾语 | 反身 |
|---|---|---|---|
| 我 | I | me | myself |
| 你 | you | you | yourself |
| 他/她/它 | he/she/it | him/her/it | himself/herself/itself |
| 我们 | we | us | ourselves |
| 你们 | you | you | yourselves |
| 他们 | they | them | themselves |

### 社会指示

英语使用头衔、敬语和语体：

- *Mr., Mrs., Ms., Dr.*
- *sir, ma'am*（服务场合）
- *first names*（非正式）
- *Would you mind...?*（正式疏远）

### 动词指示（come/go, bring/take）

- *come*（向说话者/中心来）
- *go*（离说话者/中心去）
- *bring*（向说话者/中心带来）
- *take*（离说话者/中心带去）

## 常见错误

1. **学英语的西班牙语者**：用 ese 指远处物体：*Ese montaña* → *Aquel montaña*（如果远离双方）

2. **学西班牙语的英语者**：过度指定距离：*That there* → *That* 或 *That over there*

3. **学中文的西班牙语者**：期待三元区分：中文没有 aquí/ahí/allí，只有 <ruby>这<rt>zhè</rt></ruby><ruby>儿<rt>ér</rt></ruby>/<ruby>那<rt>nà</rt></ruby><ruby>儿<rt>ér</rt></ruby>

4. **学西班牙语的中文者**：用 ese 表示所有 "那个"：必须区分 ese（近你）和 aquel（远）
