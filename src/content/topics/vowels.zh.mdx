## 概述

元音是声道中没有明显阻塞发出的语音。三种语言在元音复杂性上差异巨大。

- **西班牙语**：五个纯元音，音质一致。每个字母代表一个音。不向schwa弱化。
- **英语**： notoriously complex 的元音系统，包含长短、音质和r色彩的区别。大量音位变体。
- **汉语（普通话）**：六个简单韵母，但声调为每个元音音质增加了一整套额外的音位对立。

---

## 西班牙语元音

西班牙语有五个纯元音，总是清晰一致地发音：

| 字母 | IPA | 例词 |
|------|-----|------|
| a | /a/ | casa |
| e | /e/ | mesa |
| i | /i/ | misa |
| o | /o/ | mosca |
| u | /u/ | musa |

**关键规则**：
- 无松紧对立
- 无长短对立
- 不向schwa弱化
- 双元音由弱元音（i, u）与强元音组合形成

---

## 英语元音

英语大约有20个元音音位，取决于方言：

**单元音**：
- 紧元音：/iː/ (fleece), /uː/ (goose), /eɪ/ (face), /oʊ/ (goat), /aɪ/ (price), /aʊ/ (mouth), /ɔɪ/ (choice)
- 松元音：/ɪ/ (kit), /ʊ/ (foot), /ɛ/ (dress), /ʌ/ (strut), /ɒ/ (lot), /æ/ (trap), /ə/ (comma)

**r色彩元音**：/ɝ/ (nurse), /ɚ/ (letter), /ɑr/ (start), /ɔr/ (north), /ɪr/ (near), /ɛr/ (square), /ʊr/ (cure)

**Schwa /ə/**：英语最常见的元音，出现在非重读音节中。

## 汉语（普通话）元音（韵母）

普通话有六个单韵母（<ruby>单<rt>dān</rt></ruby><ruby>韵<rt>yùn</rt></ruby><ruby>母<rt>mǔ</rt></ruby>）：

| 韵母 | IPA | 例词 |
|------|-----|------|
| a | /a/ | <ruby>妈<rt>mā</rt></ruby> mā |
| o | /o/ | <ruby>波<rt>bō</rt></ruby> bō |
| e | /ɤ/ | <ruby>哥<rt>gē</rt></ruby> gē |
| i | /i/ | <ruby>衣<rt>yī</rt></ruby> yī |
| u | /u/ | <ruby>乌<rt>wū</rt></ruby> wū |
| ü | /y/ | <ruby>女<rt>nǚ</rt></ruby> nǚ |

**复合韵母**包括双元音和三元音（iao, uai等）。

**ü 的挑战**：普通话的 ü 需要发 /i/ 的同时圆唇。英语和西班牙语完全没有这个元音。

## 元音系统对比

| 特征 | 西班牙语 | 英语 | 汉语 |
|------|---------|------|------|
| 简单元音 | 5 | 12+ | 6 |
| 双元音 | 10+ | 8+ | 8+ |
| 长短对立 | 无 | 有 | 无 |
| 松紧对立 | 无 | 有 | 无 |
| Schwa | 无 | 有 | 无 |
| r色彩 | 无 | 有 | 无 |
| 鼻化 | 无 | 无 | 无 |
| 声调叠加 | 无 | 无 | 有 |
