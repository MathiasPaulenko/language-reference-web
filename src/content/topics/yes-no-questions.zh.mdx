## 概述

是非问句请求确认或否定。

- **西班牙语**：是非问句使用隐含倒装（主语在动词前后）或仅用语调。无 do-support 助动词。
- **英语**：是非问句对简单动词要求 do-support，对系动词和情态动词要求倒装。
- **汉语**：是非问句在陈述句句末添加助词 <ruby>吗<rt>ma</rt></ruby>。语序不变。V-不-V 形式是常见替代。

## 西班牙语

### 倒装 + 语调

- *¿Vienes?*
- *¿Es esto un libro?*
- *¿Te gusta?*

**规则**：主语可在动词前后：

- *¿Tú vienes?* / *¿Vienes tú?*
- *¿Esto es un libro?* / *¿Es esto un libro?*

### 仅用语调（口语非正式）

- *¿Vienes?* (vienes 升调)
- *¿Esto es un libro?* (升调)

**注意**：西班牙语口语中，仅语调就可构成问句。

### A o no A 形式

- *¿Vienes o no?*
- *¿Lo sabes o no lo sabes?*
- *¿Es verdad o no?*

### 附加问句

- *Vienes, ¿verdad?*
- *Es tarde, ¿no?*
- *Ella está aquí, ¿no es así?*

### 回答

| 回答 | 意义 |
|---|---|
| Sí | 确认 |
| No | 否定 |
| Claro / Por supuesto | 强调确认 |
| No, no es / No lo creo | 带语境的否定 |
| Sí, claro | 强调确认 |

## 英语

### Do-support（简单动词）

- *Do you come?*
- *Does she like it?*
- *Did they eat?*

**规则**：一般现在/过去时，用 do/does/did + 主语 + 原形。

### 系动词 (be)

- *Is this a book?*
- *Are you happy?*
- *Was he here?*

**规则**：直接倒装主语-系动词。

### 情态动词

- *Can he come?*
- *Will you go?*
- *Should I leave?*

**规则**：直接倒装主语-情态动词。

### 完成时

- *Have you eaten?*
- *Has she arrived?*
- *Had they left?*

**规则**：直接倒装主语-have。

### 进行时

- *Are you coming?*
- *Was she sleeping?*

**规则**：直接倒装主语-be。

### A or not A 形式

- *Are you coming or not?*
- *Do you like it or not?*
- *Is it true or not?*

### 附加问句

- *You come, don't you?*
- *She likes it, doesn't she?*
- *He's here, isn't he?*

**规则**：附加问句与主句极性相反。

### 回答

| 回答 | 意义 |
|---|---|
| Yes | 确认 |
| No | 否定 |
| Sure / Of course | 确认 |
| I think so / I don't think so | 谨慎 |
| Yes, I do / No, I don't | 带助动词的强调回答 |

## 汉语

### 句末加 吗 ma

在陈述句后加 <ruby>吗<rt>ma</rt></ruby>：

- <ruby>你<rt>nǐ</rt></ruby><ruby>来<rt>lái</rt></ruby>。(你来。) → <ruby>你<rt>nǐ</rt></ruby><ruby>来<rt>lái</rt></ruby>**<ruby>吗<rt>ma</rt></ruby>**？(你来吗？)
- <ruby>这<rt>zhè</rt></ruby><ruby>是<rt>shì</rt></ruby><ruby>书<rt>shū</rt></ruby>。(这是书。) → <ruby>这<rt>zhè</rt></ruby><ruby>是<rt>shì</rt></ruby><ruby>书<rt>shū</rt></ruby>**<ruby>吗<rt>ma</rt></ruby>**？(这是书吗？)
- <ruby>他<rt>tā</rt></ruby><ruby>喜<rt>xǐ</rt></ruby><ruby>欢<rt>huan</rt></ruby>。(他喜欢。) → <ruby>他<rt>tā</rt></ruby><ruby>喜<rt>xǐ</rt></ruby><ruby>欢<rt>huan</rt></ruby>**<ruby>吗<rt>ma</rt></ruby>**？(他喜欢吗？)

**注意**：没有 <ruby>吗<rt>ma</rt></ruby>，句子是陈述句。仅靠语调不构成问句。

### V-不-V 形式

<ruby>吗<rt>ma</rt></ruby> 的常见替代：

- <ruby>你<rt>nǐ</rt></ruby><ruby>来<rt>lái</rt></ruby>**<ruby>不<rt>bù</rt></ruby><ruby>来<rt>lái</rt></ruby>**？(你来不来？)
- <ruby>你<rt>nǐ</rt></ruby><ruby>喜<rt>xǐ</rt></ruby><ruby>欢<rt>huan</rt></ruby>**<ruby>不<rt>bù</rt></ruby><ruby>喜<rt>xǐ</rt></ruby><ruby>欢<rt>huan</rt></ruby>**？(你喜不喜欢？)
- <ruby>这<rt>zhè</rt></ruby><ruby>是<rt>shì</rt></ruby>**<ruby>不<rt>bù</rt></ruby><ruby>是<rt>shì</rt></ruby>**？(这是不是？)

**注意**：V-不-V 在口语中比 <ruby>吗<rt>ma</rt></ruby> 更自然。

### 与 了 le 连用

- <ruby>你<rt>nǐ</rt></ruby><ruby>吃<rt>chī</rt></ruby><ruby>了<rt>le</rt></ruby>**<ruby>吗<rt>ma</rt></ruby>**？(你吃了吗？)
- <ruby>他<rt>tā</rt></ruby><ruby>走<rt>zǒu</rt></ruby><ruby>了<rt>le</rt></ruby>**<ruby>吗<rt>ma</rt></ruby>**？(他走了吗？)

### 与情态动词连用

- <ruby>你<rt>nǐ</rt></ruby><ruby>能<rt>néng</rt></ruby><ruby>来<rt>lái</rt></ruby>**<ruby>吗<rt>ma</rt></ruby>**？(你能来吗？)
- <ruby>他<rt>tā</rt></ruby><ruby>会<rt>huì</rt></ruby><ruby>说<rt>shuō</rt></ruby><ruby>中<rt>zhōng</rt></ruby><ruby>文<rt>wén</rt></ruby>**<ruby>吗<rt>ma</rt></ruby>**？(他会说中文吗？)

### 回答

| 回答 | 意义 |
|---|---|
| <ruby>是<rt>shì</rt></ruby> / <ruby>对<rt>duì</rt></ruby> | 确认 |
| <ruby>好<rt>hǎo</rt></ruby> / <ruby>行<rt>xíng</rt></ruby> | 同意 |
| <ruby>不<rt>bù</rt></ruby> / <ruby>不<rt>bù</rt></ruby><ruby>是<rt>shì</rt></ruby> / <ruby>不<rt>bù</rt></ruby><ruby>对<rt>duì</rt></ruby> | 否定 |
| <ruby>没<rt>méi</rt></ruby><ruby>有<rt>yǒu</rt></ruby> | 否定 有/了 |

## 快速对比

| 策略 | 西班牙语 | 英语 | 汉语 |
|---|---|---|---|
| 倒装 | 隐含（主语灵活） | Do-support / 系动词倒装 | 无（陈述语序） |
| 问句助词 | 无（语调） | 无 | <ruby>吗<rt>ma</rt></ruby> 在句末 |
| A-不-A 形式 | A o no A | A or not | V-<ruby>不<rt>bù</rt></ruby>-V |
| 附加问句 | ¿verdad? / ¿no? | don't you? / isn't it? | <ruby>是<rt>shì</rt></ruby><ruby>不<rt>bù</rt></ruby><ruby>是<rt>shì</rt></ruby> |
| 肯定回答 | Sí | Yes | <ruby>是<rt>shì</rt></ruby> / <ruby>对<rt>duì</rt></ruby> / <ruby>好<rt>hǎo</rt></ruby> |
| 否定回答 | No | No | <ruby>不<rt>bù</rt></ruby> / <ruby>没<rt>méi</rt></ruby><ruby>有<rt>yǒu</rt></ruby> |

## 常见错误

1. **英语者学西班牙语**：Do you come? → ¿**Vienes**? (无 do-support)

2. **西班牙语者学英语**：You come? → **Do** you come? (需要 do-support)

3. **英语者学汉语**：<ruby>吗<rt>ma</rt></ruby><ruby>你<rt>nǐ</rt></ruby><ruby>来<rt>lái</rt></ruby>？ → <ruby>你<rt>nǐ</rt></ruby><ruby>来<rt>lái</rt></ruby><ruby>吗<rt>ma</rt></ruby>？(<ruby>吗<rt>ma</rt></ruby> 在句末，不在句首)

4. **汉语者学英语**：You like it? (升调) → **Do** you like it? (书面；口语非正式中升调可接受)
