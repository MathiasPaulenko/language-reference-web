## Visión general

El español y el inglés añaden sufijos para formar plurales, con muchas excepciones irregulares. Los sustantivos chinos son generalmente invariables; el plural se muestra a través del contexto, numerales o <ruby><ruby>们<rt>mén</rt></ruby><rt>mén</rt></ruby>/<ruby><ruby>些<rt>xiē</rt></ruby><rt>xiē</rt></ruby>.

Los tres idiomas abordan este concepto gramatical de maneras distintas que reflejan sus sistemas subyacentes. A continuación se presenta una comparación detallada.

## Resumen

La pluralización marca más de uno. Cada idioma lo maneja de manera diferente.

- **Español**: Añade -s o -es. Los artículos y adjetivos deben concordar en género y número. Muchos sustantivos terminados en consonante añaden -es.
- **Inglés**: Añade -s o -es. Muchos plurales irregulares. Los sustantivos incontables no se pluralizan.
- **Chino**: Los sustantivos no cambian de forma. El plural se muestra a través de numerales + clasificadores, contexto, o <ruby>们<rt>mén</rt></ruby>mén.

## Español

### Plurales regulares

| Regla | Singular | Plural |
|---|---|---|
| -s para vocal | libro | libros |
| -es para consonante | mes | meses |
| -es para -z | lápiz | lápices |
| -es para -ión (con tilde) | nación | naciones |
| Invariable (algunos) | el lunes | los lunes |

### Plurales irregulares

| Singular | Plural | Nota |
|---|---|---|
| el padre | los padres | Cambio de artículo |
| el rey | los reyes | Añade -es |
| el pie | los pies | Cambio interno |
| la mujer | las mujeres | -es |
| el café | los cafés | Conserva tilde |

### Concordancia

- **Artículos**: el → los; la → las; un → unos; una → unas
- **Adjetivos**: rojo → rojos; alta → altas; bueno → buenos
- **Verbos**: Los verbos no cambian por número del sujeto (a diferencia del inglés)

### Sustantivos colectivos

- *la gente piensa* → concordancia verbal en plural (people think)
- *la familia es grande* → concordancia verbal en singular

## Inglés

### Plurales regulares

| Regla | Singular | Plural |
|---|---|---|
| -s | book | books |
| -es (s, x, z, ch, sh) | box | boxes |
| -ies (consonante + y) | baby | babies |
| -ves (f, fe) | knife | knives |

### Plurales irregulares

| Singular | Plural |
|---|---|
| man | men |
| woman | women |
| child | children |
| foot | feet |
| tooth | teeth |
| mouse | mice |
| goose | geese |
| person | people |
| sheep | sheep (invariable) |
| deer | deer (invariable) |

### Sustantivos incontables

| Categoría | Ejemplos |
|---|---|
| Líquidos | water, milk, coffee |
| Materiales | wood, metal, cotton |
| Abstractos | advice, information, knowledge |
| Actividades | homework, research, work |

**Nota**: No se pluralizan. Se cuantifican con "some", "much", "a piece of".

### Sustantivos colectivos

- The team **is** playing well (enfoque en el grupo)
- The team **are** arguing (enfoque en los individuos)

## Chino

### Sin cambio de forma

| Singular / Plural | Contexto |
|---|---|
| <ruby>书<rt>shū</rt></ruby> | libro o libros (depende del contexto) |
| <ruby>人<rt>rén</rt></ruby> | persona o personas |
| <ruby>桌<rt>zhuō</rt></ruby><ruby>子<rt>zi</rt></ruby> | mesa o mesas |

### Marcador <ruby>们<rt>mén</rt></ruby> (para personas)

| Pronombre | Plural |
|---|---|
| <ruby>我<rt>wǒ</rt></ruby> → | <ruby>我<rt>wǒ</rt></ruby><ruby>们<rt>men</rt></ruby> (nosotros) |
| <ruby>你<rt>nǐ</rt></ruby> → | <ruby>你<rt>nǐ</rt></ruby><ruby>们<rt>men</rt></ruby> (vosotros) |
| <ruby>他<rt>tā</rt></ruby> → | <ruby>他<rt>tā</rt></ruby><ruby>们<rt>men</rt></ruby> (ellos) |
| <ruby>她<rt>tā</rt></ruby> → | <ruby>她<rt>tā</rt></ruby><ruby>们<rt>men</rt></ruby> (ellas) |

**Nota**: <ruby>们<rt>mén</rt></ruby> solo se usa con pronombres y algunos sustantivos humanos: <ruby>朋<rt>péng</rt></ruby><ruby>友<rt>yǒu</rt></ruby><ruby>们<rt>men</rt></ruby> ✓, <ruby>书<rt>shū</rt></ruby><ruby>们<rt>men</rt></ruby> ✗

### Clasificadores + numerales

| Número | Clasificador | Sustantivo |
|---|---|---|
| <ruby>三<rt>sān</rt></ruby> | <ruby>本<rt>běn</rt></ruby> | <ruby>书<rt>shū</rt></ruby> (libros) |
| <ruby>五<rt>wǔ</rt></ruby> | <ruby>个<rt>gè</rt></ruby> | <ruby>人<rt>rén</rt></ruby> (personas) |

### <ruby>些<rt>xiē</rt></ruby> para "algunos"

| Expresión | Significado |
|---|---|

| <ruby>那<rt>nà</rt></ruby><ruby>些<rt>xiē</rt></ruby><ruby>人<rt>rén</rt></ruby> | esas personas |

## Comparación rápida

| Característica | Español | Inglés | Chino |
|---|---|---|---|
| Cambio de forma | Sí (-s / -es) | Sí (-s / -es / irregular) | No |
| Concordancia artículo | Sí (el → los) | No (the) | No (sin artículos) |
| Concordancia adjetivo | Sí (rojo → rojos) | No | No |
| Marcador personas | -s / -es | -s / -es | <ruby>们<rt>mén</rt></ruby> (limitado) |
| Con numerales | Sí (tres libros) | Sí (three books) | No (<ruby>三<rt>sān</rt></ruby><ruby>本<rt>běn</rt></ruby><ruby>书<rt>shū</rt></ruby>) |
| Colectivos | Plural verbal | Singular o plural | Contexto |
| Incontables | Algunos | Muchos | Pocas restricciones |

## Errores comunes

1. **Hispanohablantes**: Añadir -s a sustantivos chinos → <ruby>书<rt>shū</rt></ruby><ruby>们<rt>men</rt></ruby> es incorrecto

2. **Angloparlantes**: Forzar concordancia de plural en chino → los adjetivos chinos no cambian

3. **Hispanohablantes**: Usar irregularidades del inglés en español → "hiji" en lugar de "hijos"

4. **Angloparlantes**: Tratar sustantivos incontables del español como contables → "el agua" no se pluraliza como "las aguas" en todos los contextos
