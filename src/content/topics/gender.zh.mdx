## 概述

语法性别是名词的一种分类，影响冠词、形容词和代词。

- **西班牙语**：所有名词强制语法性别。阳性（*-o, -e, 辅音*）vs 阴性（*-a, -ción, -dad*）。阳性作为通称。
- **英语**：名词无语法性别。代词 *he/she/it/they* 基于生物性别或身份。they单数作为通称日益增长。
- **中文**：无语法性别。性别代词（*<ruby>他<rt>tā</rt></ruby>/<ruby>她<rt>tā</rt></ruby>*）是20世纪发明。性别差异通过前缀（*<ruby>公<rt>gōng</rt></ruby>/<ruby>母<rt>mǔ</rt></ruby>, <ruby>男<rt>nán</rt></ruby>/<ruby>女<rt>nǚ</rt></ruby>*）表达。

## 西班牙语

### 语法性别

**阳性**：冠词 *el/un*，形容词 *-o*
- el niño, un médico, el libro, el problema
- El niño es **alto**.（男孩很高。）
- Un médico **bueno**.（一个好医生。）

**阴性**：冠词 *la/una*，形容词 *-a*
- la niña, una médica, la mesa, la ciudad
- La niña es **alta**.（女孩很高。）
- Una médica **buena**.（一个好医生。）

### 一般规则

| 词尾 | 性别 | 例子 |
|---|---|---|
| -o | 阳性 | libro, niño, auto |
| -a | 阴性 | mesa, niña, casa |
| -ción, -sión | 阴性 | nación, división, televisión |
| -dad, -tad, -tud | 阴性 | ciudad, libertad, actitud |
| -e | 阳性/阴性 | coche (masc.), calle (fem.) |
| 辅音 | 阳性 | papel, reloj, árbol |
| -z | 阴性 | luz, paz, vez |
| -ma, -pa, -ta (希腊语) | 阳性 | problema, mapa, poeta |

### 双性名词

- el/la mar（el mar / la mar 诗歌体）
- el/la azúcar（el azúcar 通用 / la azúcar 某些国家）
- el/la sartén（地区性）

### 性别与生物性别

- **一致**：niño/niña, médico/médica, perro/perra, gato/gata
- **不一致**：mesa (fem.), problema (masc.), persona (fem.), víctima (fem.)

### 阳性作为通称

- *Los niños* = 男孩和女孩
- *Los doctores* = 男医生和女医生
- *Todos* = 所有男性和女性
- *Ninguno* = 没有任何男性和女性

**现代替代形式**：
- Niños y niñas
- -os/as: doctoras/os
- -e: todes
- @符号：niñ@s（应避免）

### 中性冠词 lo

- *Lo bueno* es que...（好处是...）
- *Lo difícil* es...（困难的是...）
- *Lo importante* es...（重要的是...）

### 形容词

形容词与名词性一致：

- El chico **alto** / La chica **alta**（高个子男孩 / 高个子女孩）
- Un libro **interesante** / Una película **interesante**（一本有趣的书 / 一部有趣的电影 — -e 不变）
- El problema **grave** / La situación **grave**（严重的问题 / 严峻的形势 — 不变）

## 英语

### 名词无语法性别

- The boy / The girl（男孩 / 女孩 — 生物性别，非语法）
- The table / The chair / The sun / The moon（桌子 / 椅子 / 太阳 / 月亮 — 全部中性）
- A doctor / A teacher（医生 / 教师 — 中性）

### 性别代词

| 代词 | 用法 |
|---|---|
| he | 阳性（人，雄性动物） |
| she | 阴性（人，雌性动物，传统上船只） |
| it | 无生命，性别未知的动物 |
| they | 复数（所有人）/ 现代单数通称 |

### 单数they

- *Someone left their book.*（有人落下了他们的书。）
- *The doctor said they would call.*（医生说他们会打电话。）
- *Alex said they are tired.*（Alex说他们累了。）

### 包容性语言的演变

- *Firefighter* 代替 *fireman*
- *Flight attendant* 代替 *stewardess*
- *Police officer* 代替 *policeman*
- *They* 作为非二元性别代词

### 性别词汇差异（已过时）

| 阳性 | 阴性 | 现代用法 |
|---|---|---|
| actor | actress | actor（通用） |
| waiter | waitress | server |
| steward | stewardess | flight attendant |
| chairman | chairwoman | chair / chairperson |
| mailman | mailwoman | mail carrier |
| policeman | policewoman | police officer |

## 中文

### 无语法性别

中文名词、形容词或冠词没有语法性别。

- <ruby>书<rt>shū</rt></ruby>（书）— 无性别
- <ruby>桌<rt>zhuō</rt></ruby><ruby>子<rt>zǐ</rt></ruby>（桌子）— 无性别
- <ruby>太<rt>tài</rt></ruby><ruby>阳<rt>yáng</rt></ruby>（太阳）— 无性别
- <ruby>月<rt>yuè</rt></ruby><ruby>亮<rt>liàng</rt></ruby>（月亮）— 无性别

### 通过前缀区分性别

当性别相关时，添加前缀：

| 前缀 | 含义 | 例词 |
|---------|---------|---------|
| <ruby>男<rt>nán</rt></ruby> | 阳性/男性 | <ruby>男<rt>nán</rt></ruby><ruby>孩<rt>hái</rt></ruby>（男孩）, <ruby>男<rt>nán</rt></ruby><ruby>医<rt>yī</rt></ruby><ruby>生<rt>shēng</rt></ruby>（男医生） |
| <ruby>女<rt>nǚ</rt></ruby> | 阴性/女性 | <ruby>女<rt>nǚ</rt></ruby><ruby>孩<rt>hái</rt></ruby>（女孩）, <ruby>女<rt>nǚ</rt></ruby><ruby>医<rt>yī</rt></ruby><ruby>生<rt>shēng</rt></ruby>（女医生） |
| <ruby>公<rt>gōng</rt></ruby> | 雄性（动物） | <ruby>公<rt>gōng</rt></ruby><ruby>猫<rt>māo</rt></ruby>（公猫） |
| <ruby>母<rt>mǔ</rt></ruby> | 雌性（动物） | <ruby>母<rt>mǔ</rt></ruby><ruby>猫<rt>māo</rt></ruby>（母猫） |
| <ruby>雄<rt>xióng</rt></ruby> | 雄性（正式/技术） | <ruby>雄<rt>xióng</rt></ruby><ruby>狮<rt>shī</rt></ruby>（雄狮） |
| <ruby>雌<rt>cí</rt></ruby> | 雌性（正式/技术） | <ruby>雌<rt>cí</rt></ruby><ruby>狮<rt>shī</rt></ruby>（雌狮） |

### 性别代词：现代发明

**历史**：女性代词<ruby>她<rt>tā</rt></ruby>由刘半农在1920年代发明，受欧洲语言影响。原本，<ruby>他<rt>tā</rt></ruby>服务于所有性别。

| 代词 | 历史 | 现代用法 |
|---|---|---|
| <ruby>他<rt>tā</rt></ruby> | 原为通称 | 阳性 / 通称 |
| <ruby>她<rt>tā</rt></ruby> | ~1920年发明 | 阴性 |
| <ruby>它<rt>tā</rt></ruby> | 古代 | 无生命 / 动物 |
| <ruby>牠<rt>tā</rt></ruby> | 已废弃 | 动物（传统变体） |
| <ruby>祂<rt>tā</rt></ruby> | 宗教 | 神性 |

**注意**：全部读作*tā*（第一声）。

### 用们men表示复数

- <ruby>他<rt>tā</rt></ruby><ruby>们<rt>men</rt></ruby>（他们/混合）
- <ruby>她<rt>tā</rt></ruby><ruby>们<rt>men</rt></ruby>（她们）
- <ruby>它<rt>tā</rt></ruby><ruby>们<rt>men</rt></ruby>（它们）

### 无性别职业

- <ruby>医<rt>yī</rt></ruby><ruby>生<rt>shēng</rt></ruby>（医生 — 中性）
- <ruby>老<rt>lǎo</rt></ruby><ruby>师<rt>shī</rt></ruby>（老师 — 中性）
- <ruby>警<rt>jǐng</rt></ruby><ruby>察<rt>chá</rt></ruby>（警察 — 中性）

要指定性别，添加<ruby>男<rt>nán</rt></ruby>或<ruby>女<rt>nǚ</rt></ruby>：
- <ruby>男<rt>nán</rt></ruby><ruby>医<rt>yī</rt></ruby><ruby>生<rt>shēng</rt></ruby> / <ruby>女<rt>nǚ</rt></ruby><ruby>医<rt>yī</rt></ruby><ruby>生<rt>shēng</rt></ruby>（男医生 / 女医生）

## 常见错误

1. **学英语的西班牙语使用者**：使用自然性别代替语法性别：*la problema* → *el problema*

2. **学英语的西班牙语使用者**：*The table, she is...* → *The table, **it** is...*

3. **学西班牙语的中文使用者**：忘记形容词性一致：*la mesa bonito* → *la mesa **bonita***

4. **学中文的英语使用者**：假设他/她历史悠久：实际上她是20世纪发明

5. **学中文的西班牙语使用者**：在不需要的地方添加性别前缀：*女医生* 当 *医生* 已经是中性
