## 概述

助词是表达语法特征的小品词。汉语以其助词系统闻名，但每种语言都有助词。

- **西班牙语**：通过动词变位编码体、时和语气。助词极少——主要是连词和话语标记。
- **英语**：使用助动词（have、be、will、do）加语序和语调。助词存在（to、up、out 在短语动词中）但与汉语不同。
- **汉语**：依赖丰富的助词系统，跟在动词、名词或整句后表达体、语气、领属和程度。

## 西班牙语

### 动词上的体与时

西班牙语通过动词词尾编码大部分语法信息：

- *habl**é***（我说了——简单过去时，已完成）
- *habl**aba***（我过去说——未完成时，持续）
- *he habl**ado***（我已经说了——完成时，经历）
- *estoy habl**ando***（我正在说——进行时）

### 话语助词

| 助词 | 功能 | 例句 |
|---|---|---|
| ¿...? | 问号 | *¿Vienes?* |
| pues | 嗯，那么 | *Pues, no sé.* |
| o sea | 也就是说 | *O sea, no me gusta.* |
| vamos | 来吧 | *¡Vamos, apúrate!* |
| hombre / mujer | 嘿（感叹词） | *Hombre, ¡qué sorpresa!* |
| ¿verdad? / ¿no? | 附加问句 | *Es tarde, ¿verdad?* |
| eh | 犹豫 | *Voy a, eh, comprar pan.* |

### 介词助词

- *acabar **de*** + 不定式（刚刚）
- *volver **a*** + 不定式（再次）
- *dejar **de*** + 不定式（停止）

## 英语

### 助动词

| 功能 | 助动词 | 例句 |
|---|---|---|
| 过去完成 | did | *I **did** see him.* |
| 进行 | be + -ing | *I **am** going.* |

| 被动 | be + -ed | *I **was** seen.* |
| 将来 | will / going to | *I **will** call.* |
| 否定 | do + not | *I **do** not know.* |
| 疑问 | do / be / have + 主语 | ***Do** you know?* |

|---|---|---|

| off | fall off | take off（离开） |
| on | put on | carry on（继续） |
| out | go out | figure out（理解） |
| away | run away | put away（收好） |
| back | come back | take back（收回） |

| 助词 | 功能 | 例句 |
|---|---|---|

## 汉语

汉语助词是其语法的支柱。它们表达其他语言通过变位或助动词编码的内容。

### 体助词

| 助词 | 功能 | 例句 |
|---|---|---|
| <ruby>了<rt>le</rt></ruby> | 完成/变化 | <ruby>我<rt>wǒ</rt></ruby><ruby>吃<rt>chī</rt></ruby><ruby>了<rt>le</rt></ruby>。（我吃了。/我吃过了。） |
| <ruby>过<rt>guò</rt></ruby> | 过去经历 | <ruby>我<rt>wǒ</rt></ruby><ruby>去<rt>qù</rt></ruby><ruby>过<rt>guò</rt></ruby><ruby>北<rt>běi</rt></ruby><ruby>京<rt>jīng</rt></ruby>。（我去过北京。） |
| <ruby>在<rt>zài</rt></ruby> | 进行动作 | <ruby>我<rt>wǒ</rt></ruby><ruby>在<rt>zài</rt></ruby><ruby>吃<rt>chī</rt></ruby>。（我正在吃。） |
| <ruby>着<rt>zhe</rt></ruby> | 持续状态 | <ruby>门<rt>mén</rt></ruby><ruby>开<rt>kāi</rt></ruby><ruby>着<rt>zhe</rt></ruby>。（门开着。） |

**<ruby>了<rt>le</rt></ruby>** 的细微差别：
- 动词 + <ruby>了<rt>le</rt></ruby> = 完成的动作：<ruby>吃<rt>chī</rt></ruby><ruby>了<rt>le</rt></ruby>（吃了）
- 句尾 <ruby>了<rt>le</rt></ruby> = 状态变化：<ruby>下<rt>xià</rt></ruby><ruby>雨<rt>yǔ</rt></ruby><ruby>了<rt>le</rt></ruby>（下雨了。——变化）
- 两者都用：<ruby>吃<rt>chī</rt></ruby><ruby>了<rt>le</rt></ruby><ruby>饭<rt>fàn</rt></ruby><ruby>了<rt>le</rt></ruby>（我吃完饭了。）

### 句末语气助词

| 助词 | 功能 | 例句 |
|---|---|---|
| <ruby>吗<rt>ma</rt></ruby> | 是非问 | <ruby>你<rt>nǐ</rt></ruby><ruby>好<rt>hǎo</rt></ruby><ruby>吗<rt>ma</rt></ruby>？（你好吗？） |
| <ruby>呢<rt>ne</rt></ruby> | 承接问 | <ruby>你<rt>nǐ</rt></ruby><ruby>呢<rt>ne</rt></ruby>？（你呢？） |
| <ruby>吧<rt>ba</rt></ruby> | 建议、缓和 | <ruby>走<rt>zǒu</rt></ruby><ruby>吧<rt>ba</rt></ruby>！（走吧！） |
| <ruby>啊<rt>a</rt></ruby> | 感叹、情感 | <ruby>好<rt>hǎo</rt></ruby><ruby>啊<rt>a</rt></ruby>！（好啊！） |
| <ruby>嘛<rt>ma</rt></ruby> | 显然、劝说 | <ruby>来<rt>lái</rt></ruby><ruby>嘛<rt>ma</rt></ruby>！（来嘛！） |
| <ruby>呗<rt>bei</rt></ruby> | 无可奈何、显然 | <ruby>随<rt>suí</rt></ruby><ruby>便<rt>biàn</rt></ruby><ruby>呗<rt>bei</rt></ruby>。（随便吧。） |
| <ruby>呀<rt>ya</rt></ruby> | 缓和（<ruby>啊<rt>ā</rt></ruby> 的变体） | <ruby>快<rt>kuài</rt></ruby><ruby>来<rt>lái</rt></ruby><ruby>呀<rt>ya</rt></ruby>！（快来呀！） |

### 结构助词：<ruby>的<rt>de</rt></ruby>、<ruby>地<rt>dì</rt></ruby>、<ruby>得<rt>dé</rt></ruby>

| 字 | 读音 | 功能 | 例句 |
|---|---|---|---|
| <ruby>的<rt>de</rt></ruby> | de | 领属、修饰 | <ruby>我<rt>wǒ</rt></ruby><ruby>的<rt>de</rt></ruby><ruby>书<rt>shū</rt></ruby>（我的书） |
| <ruby>地<rt>dì</rt></ruby> | de | 方式状语标记 | <ruby>慢<rt>màn</rt></ruby><ruby>慢<rt>màn</rt></ruby><ruby>地<rt>de</rt></ruby><ruby>走<rt>zǒu</rt></ruby>（慢慢地走） |
| <ruby>得<rt>dé</rt></ruby> | de | 程度补语标记 | <ruby>跑<rt>pǎo</rt></ruby><ruby>得<rt>de</rt></ruby><ruby>很<rt>hěn</rt></ruby><ruby>快<rt>kuài</rt></ruby>（跑得很快） |

### <ruby>把<rt>bǎ</rt></ruby> 字句

<ruby>把<rt>bǎ</rt></ruby> 标记直接宾语并将其提前到动词前：

- <ruby>我<rt>wǒ</rt></ruby><ruby>把<rt>bǎ</rt></ruby><ruby>书<rt>shū</rt></ruby><ruby>放<rt>fàng</rt></ruby><ruby>在<rt>zài</rt></ruby><ruby>桌<rt>zhuō</rt></ruby><ruby>子<rt>zi</rt></ruby><ruby>上<rt>shàng</rt></ruby>。（我把书放在桌子上了。）
- <ruby>请<rt>qǐng</rt></ruby><ruby>把<rt>bǎ</rt></ruby><ruby>门<rt>mén</rt></ruby><ruby>关<rt>guān</rt></ruby><ruby>上<rt>shàng</rt></ruby>。（请把门关上。）

**规则**：<ruby>把<rt>bǎ</rt></ruby> 需要特定、明确的宾语和结果状态。

### <ruby>被<rt>bèi</rt></ruby> 字句

<ruby>被<rt>bèi</rt></ruby> 标记被动：

- <ruby>他<rt>tā</rt></ruby><ruby>被<rt>bèi</rt></ruby><ruby>老<rt>lǎo</rt></ruby><ruby>师<rt>shī</rt></ruby><ruby>批<rt>pī</rt></ruby><ruby>评<rt>píng</rt></ruby><ruby>了<rt>le</rt></ruby>。（他被老师批评了。）

## 对比一览

| 特征 | 西班牙语 | 英语 | 汉语 |
|---------|---------|---------|---------|
| 完成 | 动词词尾（-é） | 一般过去时（-ed） | <ruby>了<rt>le</rt></ruby> |
| 过去经历 | 完成时（he hablado） | 现在完成时（have spoken） | <ruby>过<rt>guò</rt></ruby> |
| 进行动作 | 进行时（estoy hablando） | 进行时（am speaking） | <ruby>在<rt>zài</rt></ruby> / <ruby>着<rt>zháo</rt></ruby> |
| 是非问 | ¿...? + 语调 | 助动词倒装 / 语调 | <ruby>吗<rt>ma</rt></ruby> |
| 建议 | Vamos a / ¿Por qué no? | Let's / Why don't we | <ruby>吧<rt>bā</rt></ruby> |
| 领属 | de / 形容词 | 's / of | <ruby>的<rt>de</rt></ruby> |
| 程度 | muy + 形容词 | very + 形容词 | <ruby>得<rt>dé</rt></ruby> + 补语 |
| 方式状语 | 形容词 + -mente | 形容词 + -ly | 形容词 + <ruby>地<rt>dì</rt></ruby> |
| 被动 | ser + 分词 | be + 分词 | <ruby>被<rt>bèi</rt></ruby> |

## 常见错误

1. **英语者学汉语**：给每个过去事件加 <ruby>了<rt>le</rt></ruby>：*<ruby>昨<rt>zuó</rt></ruby><ruby>天<rt>tiān</rt></ruby><ruby>我<rt>wǒ</rt></ruby><ruby>去<rt>qù</rt></ruby><ruby>了<rt>le</rt></ruby><ruby>商<rt>shāng</rt></ruby><ruby>店<rt>diàn</rt></ruby><ruby>了<rt>le</rt></ruby>* → *<ruby>昨<rt>zuó</rt></ruby><ruby>天<rt>tiān</rt></ruby><ruby>我<rt>wǒ</rt></ruby><ruby>去<rt>qù</rt></ruby><ruby>商<rt>shāng</rt></ruby><ruby>店<rt>diàn</rt></ruby>*（简单过去叙述，不需要 <ruby>了<rt>le</rt></ruby>）

2. **英语者学汉语**：在 <ruby>为<rt>wéi</rt></ruby><ruby>什<rt>shí</rt></ruby><ruby>么<rt>mó</rt></ruby> 后用 <ruby>吗<rt>ma</rt></ruby>：*<ruby>你<rt>nǐ</rt></ruby><ruby>为<rt>wéi</rt></ruby><ruby>什<rt>shí</rt></ruby><ruby>么<rt>mó</rt></ruby><ruby>来<rt>lái</rt></ruby><ruby>吗<rt>ma</rt></ruby>？* → *<ruby>你<rt>nǐ</rt></ruby><ruby>为<rt>wéi</rt></ruby><ruby>什<rt>shí</rt></ruby><ruby>么<rt>mó</rt></ruby><ruby>来<rt>lái</rt></ruby>？*

3. **汉语者学英语**：省略助动词：*I go now* → *I **am** going now*

4. **英语者学汉语**：混淆 <ruby>的<rt>de</rt></ruby>/<ruby>地<rt>dì</rt></ruby>/<ruby>得<rt>dé</rt></ruby>：*<ruby>我<rt>wǒ</rt></ruby><ruby>慢<rt>màn</rt></ruby><ruby>慢<rt>màn</rt></ruby><ruby>的<rt>de</rt></ruby><ruby>走<rt>zǒu</rt></ruby>* → *<ruby>我<rt>wǒ</rt></ruby><ruby>慢<rt>màn</rt></ruby><ruby>慢<rt>màn</rt></ruby><ruby>地<rt>dì</rt></ruby><ruby>走<rt>zǒu</rt></ruby>*（状语需要 <ruby>地<rt>dì</rt></ruby>）
