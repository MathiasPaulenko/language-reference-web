## Resumen

Los sistemas de escritura codifican el lenguaje hablado visualmente.

- **Español**: Alfabeto fonémico (escritura latina). Ortografía altamente regular con correspondencia consistente entre sonido y letra. Los diacríticos marcan acento y distinguen homófonos.
- **Inglés**: Alfabeto fonémico (escritura latina) pero con ortografía profunda. La ortografía histórica preserva la etimología. Muchas irregularidades y letras mudas.
- **Chino**: Escritura logográfica. Los caracteres representan morfemas, no sonidos directamente. El Pinyin proporciona transcripción fonética. Caracteres simplificados y tradicionales.

| Regla | Ejemplo | Explicación |

| Función | Ejemplo |

## Chino

### Sistema de escritura

| Tipo | Descripción | Ejemplo |
|---|---|---|
| Logográfico | Cada carácter = un morfema | <ruby>人<rt>rén</rt></ruby> = persona |
| Compuestos | Múltiples morfemas forman palabras | <ruby>计<rt>jì</rt></ruby><ruby>算<rt>suàn</rt></ruby><ruby>机<rt>jī</rt></ruby> = computadora |
| Simplificados | Forma moderna | <ruby>国<rt>guó</rt></ruby> |
| Tradicionales | Forma completa | 國 |

### Tipos de caracteres

| Tipo | Descripción | Ejemplo |
|---|---|---|
| Pictográficos | Dibujos de objetos | <ruby>日<rt>rì</rt></ruby> (sol), <ruby>月<rt>yuè</rt></ruby> (luna) |
| Ideográficos | Símbolos abstractos | <ruby>上<rt>shàng</rt></ruby> (arriba), <ruby>下<rt>xià</rt></ruby> (abajo) |
| Compuestos ideográficos | Combinación de significados | <ruby>明<rt>míng</rt></ruby> (sol + luna = brillante) |
| Fonosemánticos | Radical semántico + componente fonético | <ruby>妈<rt>mā</rt></ruby> (mujer + má) |

| Función | Ejemplo |
|---|---|

## Errores comunes

1. **Angloparlantes**: Mayúsculas en español → lunes, martes (NO Lunes, Martes); español (NO Español)

2. **Todos**: Formato numérico → Español: 1.234,56 (NO 1,234.56)

3. **Todos**: Caracteres chinos = palabras → <ruby>计<rt>jì</rt></ruby><ruby>算<rt>suàn</rt></ruby><ruby>机<rt>jī</rt></ruby> = tres caracteres, una palabra

4. **Angloparlantes**: Omitir tildes españolas → término (yo termino) vs terminó (él terminó)
