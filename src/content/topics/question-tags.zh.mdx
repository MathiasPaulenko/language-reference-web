## 概述

附加疑问句寻求确认或对陈述表达怀疑。每种语言有自己的系统。

- **西班牙语**：用不变的附加语（*¿verdad?*、*¿no?*、*¿cierto?*），不倒置动词。
- **英语**：有语法化系统，附加语复制助动词并反转极性。
- **汉语**：用句末助词（*吧*、*吗*）或重叠结构（*对不对*、*是不是*）。

## 西班牙语

### 不变附加语

西班牙语不为附加疑问句倒置动词，而用不变词：

| 附加语 | 用法 | 例句 |
|---|---|---|
| ¿verdad? | 通用 | Vienes, **¿verdad?** |
| ¿no? | 通用 | Está aquí, **¿no?** |
| ¿cierto? | 确认 | Es así, **¿cierto?** |
| ¿o sí? | 否定后 | No viene, **¿o sí?** |
| ¿o no? | 肯定后 | Viene, **¿o no?** |
| ¿a que sí? | 修辞 | Es bonito, **¿a que sí?** |

### 例句

- Vienes mañana, **¿verdad?**
- No está aquí, **¿no?**
- Lo entiendes, **¿cierto?**
- Hace calor, **¿no?**

**注**：西班牙语附加语不变；不随时态或动词变化。

### 地区附加语

| 地区 | 附加语 | 例句 |
|---|---|---|
| 西班牙 | ¿vale? | Vienes, **¿vale?** |
| 拉普拉塔河 | ¿o? | Vienes, **¿o?** |
| 墨西哥 | ¿verdad? | Vienes, **¿verdad?** |

## 英语

### 极性反转

英语中，附加语反转主句的极性：

| 主句 | 附加语 | 例句 |
|---|---|---|
| 肯定 | 否定 | You're coming, **aren't you**? |
| 否定 | 肯定 | He's not here, **is he**? |

### 复制助动词

附加语复制主句的助动词（或 be 形式）：

| 助动词 | 例句 |
|---|---|
| be | She is here, **isn't she**? |
| do | You like it, **don't you**? |
| have | They've left, **haven't they**? |
| 情态 | He can swim, **can't he**? |

### 否定缩约

| 完整 | 缩约 |
|---|---|
| are not | aren't |
| is not | isn't |
| do not | don't |
| does not | doesn't |
| did not | didn't |
| have not | haven't |
| has not | hasn't |
| will not | won't |
| would not | wouldn't |
| can not | can't |
| could not | couldn't |

**注**：否定附加语总是用缩约形式。

### 回声问句

回声问句重复某成分，重读以表惊讶：

- You did **WHAT**?
- You went **WHERE**?
- She paid **HOW MUCH**?

**注**：被问词接受强韵律重音。

### 通用附加语

| 附加语 | 用法 | 语域 |
|---|---|---|
| right? | 通用 | 中性 |
| yeah? | 通用 | 非正式 |
| innit? | 通用 | 非标准（英式） |

## 快速比较

| 特征 | 西班牙语 | 英语 | 汉语 |
|---|---|---|---|
| 附加语类型 | 不变 | 极性反转 | 句末助词 |
| 动词倒置 | 无 | 有 | 无 |
| 助动词一致 | 无 | 有 | 无 |
| 缩约否定 | 不适用 | 有 | 不适用 |
| 回声问句 | 重复+语调 | wh-词重音 | 重复+语调 |
| 通用附加语 | ¿no? / ¿vale? | right? / innit? | 对吧？ |

## 常见错误

1. **西班牙语母语者**：附加语中倒置动词 → Vienes, **¿verdad?**（不说 ¿no vienes?）

2. **英语母语者**：汉语中混淆 吗 和 吧 → <ruby>对<rt>duì</rt></ruby><ruby>吧<rt>ba</rt></ruby>（期待同意）vs <ruby>对<rt>duì</rt></ruby><ruby>吗<rt>ma</rt></ruby>（中性）

3. **英语母语者**：否定附加语中忘记缩约 → He's here, **isn't** he?（不说 is not he?）

4. **汉语母语者**：英语回声问句中未使用升调 → You did **WHAT?**
