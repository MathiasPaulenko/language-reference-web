## Resumen

Cada idioma tiene su sistema para expresar tiempo y fechas.

- **Español**: Usa ser para la hora (Son las tres). Usa estar para la condición (Estoy tarde). Orden: día de mes de año.
- **Inglés**: Usa at para horas, on para días, in para meses/años/periodos. Orden: mes día, año.
- **Chino**: Usa 点 para horas, 分 para minutos. Orden: año-mes-día. Usa 两 en lugar de 二 antes de clasificadores.

## Errores comunes

1. **Angloparlantes**: Estar para la hora → **Son** las tres（ser）

2. **Hispanohablantes**: Preposiciones inglesas → **at** 3 o'clock, **on** Monday, **in** March

3. **Todos**: Formato chino → año-mes-día: 2024年3月15日

4. **Todos**: 二点 → <ruby>两<rt>liǎng</rt></ruby><ruby>点<rt>diǎn</rt></ruby>（chino usa 两 para horas）
