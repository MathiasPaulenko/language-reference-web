## Overview

Honorifics encode social hierarchy, respect, and distance between speaker and hearer.

- **Spanish**: Grammatical distinction between *tú* (familiar) and *usted* (formal), with corresponding verb forms. Some dialects add *vos*. Titles (*Don/Doña*) exist but are less central than in East Asian languages.
- **English**: No grammatical honorifics. Respect is encoded pragmatically through modal remoteness (*would/could*), lexical choice, and titles (*Mr./Ms.*). First-name basis signals intimacy.
- **Chinese**: Extensive system. Pronoun *<ruby>您<rt>nín</rt></ruby>* marks respect. Kinship terms (*<ruby>哥<rt>gē</rt></ruby>, <ruby>姐<rt>jiě</rt></ruby>, <ruby>阿<rt>ā</rt></ruby><ruby>姨<rt>yí</rt></ruby>*) are used as social honorifics. *<ruby>贵<rt>guì</rt></ruby>* elevates the other's possessions. Archaic forms (*<ruby>鄙<rt>bǐ</rt></ruby><ruby>人<rt>rén</rt></ruby>, <ruby>在<rt>zài</rt></ruby><ruby>下<rt>xià</rt></ruby>*) persist in formal writing.

## Spanish

### Tú vs usted

| Feature | Tú | Usted |
|---|---|---|
| Usage | Friends, family, children, peers | Strangers, elders, authority, service |
| Verb form | 2nd person singular | 3rd person singular |
| Object pronoun | te / ti | lo/la / le / se |
| Possessive | tu / tuyo | su / suyo |

**Examples**:

- *¿Cómo **estás**, María?* (How are you, María? — familiar)
- *¿Cómo **está**, señor García?* (How are you, Mr. García? — formal)
- ***Te** ayudo.* (I'll help you [familiar].)
- ***Le** ayudo.* (I'll help you [formal].)

### Voseo

Some dialects use *vos* instead of *tú*:

| | Tú standard | Voseo (Rioplatense) |
|---|---|---|
| you are | tú **eres** | vos **sos** |
| you have | tú **tienes** | vos **tenés** |
| come! | ¡**Ven**! | ¡**Vení**! |

**Regions**: Argentina, Uruguay, Paraguay, parts of Central America.

### Titles

- *Don / Doña* + first name (traditional respect)
  - *Don Quijote*
  - *Doña María*
- *Señor / Señora* + last name
  - *Señor García*
  - *Señora López*

### Formal request strategies

- *Le ruego que...* (I beg you to...)
- *Tenga la bondad de...* (Be so kind as to...)
- *Sería tan amable de...* (Would you be so kind as to...)
- *¿Podría usted...?* (Could you [formal]...)

### Historical elevated forms (obsolete)

- *Vuecencia* (your excellency)
- *Vuesa merced* → *usted* (origin of usted)

## English

### No grammatical honorifics

English has no grammatical distinction for social distance:

- *You* is used for everyone (king or child)
- Verbs do not change for formality

### Pragmatic honorifics

**Modal remoteness**:

- *Can you help?* (direct)
- *Could you help?* (more polite)
- *Would you mind helping?* (very polite)
- *I was wondering if you could help.* (most distant/polite)

**Lexical softening**:

- *I'm afraid I can't.*
- *With respect, I disagree.*
- *If I may say so...*
- *If you don't mind my saying...*

### Titles

| Title | Usage |
|---|---|
| Mr. + last name | Adult male (default) |
| Mrs. + last name | Married woman |
| Ms. + last name | Woman (marital status neutral) |
| Miss + last name | Young unmarried woman (dated) |
| Dr. + last name | Doctor, PhD |
| Professor + last name | University professor |
| Sir | Male (service/formal) |
| Ma'am / Madam | Female (service/formal) |
| Your Honor | Judge |
| Your Majesty | Monarch |
| Your Excellency | Ambassador |

### First-name basis

- Using first names signals intimacy
- *Please, call me John.* (invitation to drop formality)
- In business: start with last names, switch to first names when invited

### Historical thou/you

English once had *thou* (familiar) and *you* (formal/plural). *Thou* disappeared, leaving *you* for all.

## Chinese

### <ruby>你<rt>nǐ</rt></ruby> vs <ruby>您<rt>nín</rt></ruby>

| | Pinyin | Usage |
|---|---|---|
| <ruby>你<rt>nǐ</rt></ruby> | nǐ | Neutral/informal you |
| <ruby>您<rt>nín</rt></ruby> | nín | Respectful/formal you |

- <ruby>你<rt>nǐ</rt></ruby><ruby>好<rt>hǎo</rt></ruby> (hello — neutral)
- <ruby>您<rt>nín</rt></ruby><ruby>好<rt>hǎo</rt></ruby> (hello — respectful)
- <ruby>请<rt>qǐng</rt></ruby><ruby>问<rt>wèn</rt></ruby><ruby>您<rt>nín</rt></ruby><ruby>贵<rt>guì</rt></ruby><ruby>姓<rt>xìng</rt></ruby>？(May I ask your honorable surname?)

**Note**: <ruby>您<rt>nín</rt></ruby> is used with elders, superiors, customers, and strangers.

### <ruby>贵<rt>guì</rt></ruby> as honorific prefix

<ruby>贵<rt>guì</rt></ruby> (honorable/expensive) elevates the other's things:

- <ruby>贵<rt>guì</rt></ruby><ruby>姓<rt>xìng</rt></ruby> (your honorable surname)
- <ruby>贵<rt>guì</rt></ruby><ruby>公<rt>gōng</rt></ruby><ruby>司<rt>sī</rt></ruby> (your honorable company)
- <ruby>贵<rt>guì</rt></ruby><ruby>国<rt>guó</rt></ruby> (your honorable country)
- <ruby>贵<rt>guì</rt></ruby><ruby>校<rt>xiào</rt></ruby> (your honorable school)

### Humble self-reference (archaic/formal)

- <ruby>鄙<rt>bǐ</rt></ruby><ruby>人<rt>rén</rt></ruby> (this humble person)
- <ruby>在<rt>zài</rt></ruby><ruby>下<rt>xià</rt></ruby> (this one below)
- <ruby>敝<rt>bì</rt></ruby><ruby>人<rt>rén</rt></ruby> (this unworthy person)
- <ruby>拙<rt>zhuō</rt></ruby><ruby>荆<rt>jīng</rt></ruby> (my humble wife — archaic)
- <ruby>犬<rt>quǎn</rt></ruby><ruby>子<rt>zǐ</rt></ruby> (my worthless son — archaic)

**Modern usage**: Mostly replaced by neutral <ruby>我<rt>wǒ</rt></ruby>, <ruby>我<rt>wǒ</rt></ruby><ruby>们<rt>mén</rt></ruby>.

### Kinship terms as social honorifics

Chinese uses family terms for non-relatives:

| Term | Used for |
|---|---|
| <ruby>大<rt>dà</rt></ruby><ruby>爷<rt>yé</rt></ruby> | Older male (respectful) |
| <ruby>阿<rt>ā</rt></ruby><ruby>姨<rt>yí</rt></ruby> | Middle-aged woman |
| <ruby>大<rt>dà</rt></ruby><ruby>哥<rt>gē</rt></ruby> | Older male peer |
| <ruby>大<rt>dà</rt></ruby><ruby>姐<rt>jiě</rt></ruby> | Older female peer |
| <ruby>小<rt>xiǎo</rt></ruby><ruby>弟<rt>dì</rt></ruby> | Younger male |
| <ruby>小<rt>xiǎo</rt></ruby><ruby>妹<rt>mèi</rt></ruby> | Younger female |
| <ruby>爷<rt>yé</rt></ruby><ruby>爷<rt>ye</rt></ruby> | Very old man |
| <ruby>奶<rt>nǎi</rt></ruby><ruby>奶<rt>nǎi</rt></ruby> | Very old woman |

### Humble verbs (formal)

- <ruby>拜<rt>bài</rt></ruby><ruby>读<rt>dú</rt></ruby> (humbly read — your writing)
- <ruby>拜<rt>bài</rt></ruby><ruby>访<rt>fǎng</rt></ruby> (humbly visit — you)
- <ruby>请<rt>qǐng</rt></ruby><ruby>教<rt>jiào</rt></ruby> (humbly ask for instruction)
- <ruby>赐<rt>cì</rt></ruby><ruby>教<rt>jiào</rt></ruby> (graciously teach — your action)
- <ruby>莅<rt>lì</rt></ruby><ruby>临<rt>lín</rt></ruby> (graciously arrive — your arrival)

### Sentence-final politeness

- <ruby>吧<rt>ba</rt></ruby> (soft suggestion): <ruby>走<rt>zǒu</rt></ruby><ruby>吧<rt>ba</rt></ruby> (let's go)
- <ruby>吗<rt>ma</rt></ruby> (question): <ruby>好<rt>hǎo</rt></ruby><ruby>吗<rt>ma</rt></ruby> (okay?)

## Common mistakes

1. **English speakers learning Spanish**: Using tú with everyone → use **usted** with strangers/elders

2. **Spanish speakers learning English**: Expecting grammatical honorific → English uses **pragmatic** strategies

3. **English speakers learning Chinese**: Using <ruby>你<rt>nǐ</rt></ruby> in all contexts → use **<ruby>您<rt>nín</rt></ruby>** for elders/superiors

4. **Chinese speakers learning English**: Calling strangers "auntie" or "brother" → English uses **first names** or **Mr./Ms.**
