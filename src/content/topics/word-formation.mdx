## Overview

Word formation is how new words are created from existing ones.

- **Spanish**: Uses derivational affixes extensively (*-idad, -mente, -ción, -ero, -dor*). Compounding is less common and often less transparent than in English or Chinese.
- **English**: Uses prefixes, suffixes, compounding, and conversion (zero-derivation) with great productivity. New compounds and conversions appear constantly.
- **Chinese**: Relies almost entirely on compounding. Affixation is extremely limited. Most words are two-character compounds where each character contributes meaning.

## Spanish

### Prefixes

| Prefix | Meaning | Example |
|---|---|---|
| in-/im-/i- | negation | *imposible, ilegal, injusto* |
| des- | opposite, undo | *desconocer, deshacer, desgracia* |
| re- | again | *rehacer, repetir, regresar* |
| pre- | before | *prever, predecir, previo* |
| sub- | under | *subterráneo, subrayar* |
| super-/sobre- | over | *superficial, sobrepasar* |
| anti- | against | *antiguo, antipatía* |
| contra- | against | *contradecir, contrario* |
| ex- | former | *exmarido, exministro* |
| auto- | self | *autobús, automático* |

### Suffixes

**Noun-forming**:

| Suffix | Example | Meaning |
|---|---|---|
| -dad / -tad | felicidad, libertad | abstract noun |
| -ción / -sión | creación, decisión | action/result |
| -miento | descubrimiento | result of action |
| -ura | escritura, pintura | action/result |
| -ero / -era | panadero, zapatero | profession/place |
| -dor / -dora | cantador, cantadora | agent |
| -ista | artista, turista | profession/adherent |
| -aje | paisaje, equipaje | collective/result |
| -ble | amable, posible | capable of |

**Adjective-forming**:

| Suffix | Example |
|---|---|
| -oso / -osa | hermoso, peligroso |
| -able / -ible | comestible, amable |

**Adverb-forming**:

| Suffix | Example |
|---|---|
| -mente | rápidamente, felizmente |

**Note**: *-mente* attaches to the feminine form:

- *rápido* → *rápida* → *rápidamente*
- *feliz* → *feliz* → *felizmente*

### Compounds (limited)

Spanish compounds exist but are less transparent:

- *limpiaparabrisas* (windshield wiper — clean-windshields)
- *sacacorchos* (corkscrew — pull-corks)
- *parabrisas* (windshield — stop-winds)
- *lavaplatos* (dishwasher — wash-dishes)
- *rascacielos* (skyscraper — scrape-skies)

**Note**: These are often verb-object compounds.

### Conversion

Limited compared to English:

- *el presidente* (noun) vs *presidente* (title, no article)

### Diminutives and augmentatives

| Suffix | Meaning | Example |
|---|---|---|
| -ito / -ita | diminutive | *casita* (little house), *perrito* (puppy) |
| -illo / -illa | diminutive (slight pejorative) | *casilla* (small box) |
| -ón / -ona | augmentative | *casón* (big house), *perrona* (big dog) |
| -ote / -ota | augmentative (pejorative) | *casote* (huge/ugly house) |

### Compounding

English compounds are highly productive:

**Noun + Noun**:
- *blackboard, bookshelf, bedroom, toothbrush*

**Adjective + Noun**:
- *blackbird, heavyweight, blueprint*

**Verb + Noun**:
- *scarecrow, pickpocket, breakfast*

**Noun + Verb**:
- *sunrise, snowfall, rainfall*

**Adverb/Preposition + Noun**:
- *overcoat, underground, afterthought*

### Conversion (zero-derivation)

English freely converts words between categories:

| Noun → Verb | Verb → Noun | Adjective → Verb |
|---|---|---|
| email (to email) | run (a run) | empty (to empty) |
| bottle (to bottle) | walk (a walk) | clean (to clean) |
| chair (to chair) | talk (a talk) | dry (to dry) |
| knife (to knife) | look (a look) | better (to better) |
| Google (to google) | guess (a guess) | lower (to lower) |

### Blending

- *smoke + fog = smog*
- *breakfast + lunch = brunch*
- *web + log = blog*
- *emotion + icon = emoji* (from Japanese)

### Acronyms

- *NASA, UNESCO, laser, radar, scuba*

### Back-formation

- *editor → edit* (originally editor from Latin, later back-formed edit)
- *television → televise*
- *babysitter → babysit*

## Chinese

### Compounding (primary strategy)

Chinese forms almost all new words by compounding characters:

**Modifier + Head**:

- <ruby>黑<rt>hēi</rt></ruby><ruby>板<rt>bǎn</rt></ruby> (black + board = blackboard)
- <ruby>火<rt>huǒ</rt></ruby><ruby>车<rt>chē</rt></ruby> (fire + car = train)
- <ruby>电<rt>diàn</rt></ruby><ruby>话<rt>huà</rt></ruby> (electric + speech = telephone)
- <ruby>飞<rt>fēi</rt></ruby><ruby>机<rt>jī</rt></ruby> (fly + machine = airplane)
- <ruby>洗<rt>xǐ</rt></ruby><ruby>手<rt>shǒu</rt></ruby><ruby>间<rt>jiān</rt></ruby> (wash + hand + room = restroom)

**Verb + Object**:

- <ruby>看<rt>kàn</rt></ruby><ruby>书<rt>shū</rt></ruby> (read + book = study)
- <ruby>吃<rt>chī</rt></ruby><ruby>饭<rt>fàn</rt></ruby> (eat + meal = have a meal)
- <ruby>睡<rt>shuì</rt></ruby><ruby>觉<rt>jiào</rt></ruby> (sleep + sleep = sleep)

**Verb + Complement**:

- <ruby>打<rt>dǎ</rt></ruby><ruby>破<rt>pò</rt></ruby> (hit + break = break)
- <ruby>看<rt>kàn</rt></ruby><ruby>见<rt>jiàn</rt></ruby> (look + see = see)
- <ruby>听<rt>tīng</rt></ruby><ruby>懂<rt>dǒng</rt></ruby> (listen + understand = understand by listening)

**Coordinate**:

- <ruby>朋<rt>péng</rt></ruby><ruby>友<rt>yǒu</rt></ruby> (friend + friend = friend)
- <ruby>学<rt>xué</rt></ruby><ruby>习<rt>xí</rt></ruby> (study + practice = study)

## Common mistakes

1. **English speakers learning Spanish**: *Casa-coche* for garage → *garaje* (Spanish doesn't compound freely like English)

2. **English speakers learning Chinese**: *<ruby>快<rt>kuài</rt></ruby><ruby>乐<rt>lè</rt></ruby>-ness* → *<ruby>快<rt>kuài</rt></ruby><ruby>乐<rt>lè</rt></ruby><ruby>感<rt>gǎn</rt></ruby>* (<ruby>感<rt>gǎn</rt></ruby> adds abstract quality; -ness doesn't translate directly)

3. **Chinese speakers learning English**: *I am very happiness* → *I am very **happy*** (English uses adjective, not noun)

4. **Spanish speakers learning English**: *I am agree* → *I **agree*** (English conversion: adjective → verb, not Spanish agreement verb structure)
