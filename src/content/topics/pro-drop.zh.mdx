## 概述

空主语语言允许在上下文可恢复时省略主语代词。这减少冗余，使话语更高效。

- **西班牙语**：一致的空主语语言。动词变位标记人称和数，使主语在大多数情况下冗余。
- **英语**：非空主语语言。主语在几乎所有限定句中都必须出现，除了祈使句和某些并列结构。
- **汉语**：语篇空主语语言。主语在可从上下文或前文语篇恢复时自由省略。

## 西班牙语

### 西班牙语是空主语语言

西班牙语中主语默认省略。动词变位明确标记人称和数：

| 人称 | 动词 | 含义 |
|---|---|---|
| 1sg | habl**o** | (我)说 |
| 2sg | habl**as** | (你)说 |
| 3sg | habl**a** | (他/她/您)说 |
| 1pl | habl**amos** | (我们)说 |
| 2pl | habl**áis** | (你们)说 |
| 3pl | habl**an** | (他们/她们/您们)说 |

### 何时使用显式代词

显式主语代词用于：

1. **强调**：*Yo* hablo español（不是你）
2. **对比**：Tú vienes, pero *yo* me quedo
3. **清晰**：Él habla inglés, pero ella habla español
4. **转换主语**：Juan vino y *él* se fue（区分另一可能主语）

### 天气表达

西班牙语天气表达无主语：

| 西班牙语 | 英语 |
|---|---|
| Llueve | It is raining |
| Nieva | It is snowing |
| Hace frío | It is cold |
| Es tarde | It is late |

**注**：不用 "ello" 作天气主语。

## 英语

### 英语是非空主语语言

英语中主语在几乎所有限定句中都必须出现：

| 错误 | 正确 |
|---|---|
| *Speak English | **I** speak English |
| *Is raining | **It** is raining |
| *Went to school | **He** went to school |

### 例外

1. **祈使句**：*Come here!*（主语 "you" 省略）
2. **并列结构**：*John came and **he** left*（"he" 在某些语境中可选）
3. **简短回答**：*Where are you going? — Home.*（回答中省略主语）
4. **日记/标题**：*Went to the store. Bought milk.*（电报体）

### 虚主语 "it"

英语在无语义主语的结构中需要形式主语（虚主语）：

| 西班牙语 | 英语 |
|---|---|
| Es tarde | **It** is late |
| Hace frío | **It** is cold |
| Llueve | **It** is raining |
| Parece que sí | **It** seems so |

**注**：虚主语 "it" 无意义；仅履行主语的语法功能。

## 汉语

### 汉语是语篇空主语语言

汉语中主语在可从上下文或前文语篇恢复时省略：

| 完整 | 省略 | 语境 |
|---|---|---|
| <ruby>我<rt>wǒ</rt></ruby><ruby>吃<rt>chī</rt></ruby><ruby>饭<rt>fàn</rt></ruby> | <ruby>吃<rt>chī</rt></ruby><ruby>饭<rt>fàn</rt></ruby> | 我吃饭（语境清楚） |
| <ruby>他<rt>tā</rt></ruby><ruby>去<rt>qù</rt></ruby><ruby>了<rt>le</rt></ruby> | <ruby>去<rt>qù</rt></ruby><ruby>了<rt>le</rt></ruby> | 他去了（语境清楚） |
| <ruby>你<rt>nǐ</rt></ruby><ruby>好<rt>hǎo</rt></ruby><ruby>吗<rt>ma</rt></ruby>？ | — | 你好吗？ |

### 省略主语的条件

1. **即时语境**：主语在前一句中已提及
2. **共享情境**：双方都知道在谈论什么
3. **泛指主语**：适用于任何人
4. **回答**：回答通常省略主语

### 例句

- **A**: <ruby>你<rt>nǐ</rt></ruby><ruby>去<rt>qù</rt></ruby><ruby>哪<rt>nǎ</rt></ruby><ruby>儿<rt>ér</rt></ruby>？（你去哪儿？）
- **B**: <ruby>回<rt>huí</rt></ruby><ruby>家<rt>jiā</rt></ruby>。（回家。）[省略主语]

- **A**: <ruby>你<rt>nǐ</rt></ruby><ruby>吃<rt>chī</rt></ruby><ruby>了<rt>le</rt></ruby><ruby>吗<rt>ma</rt></ruby>？（你吃了吗？）
- **B**: <ruby>吃<rt>chī</rt></ruby><ruby>了<rt>le</rt></ruby>。（吃了。）[省略主语]

## 快速比较

| 特征 | 西班牙语 | 英语 | 汉语 |
|---|---|---|---|
| 空主语类型 | 是（形态学） | 否（除祈使句外） | 是（语篇性） |
| 动词标记 | 丰富（6个人称） | 极少（3种形式） | 无 |
| 主语必须 | 否（默认省略） | 是 | 否（依赖语境） |
| 虚主语 | 无 | 有（it/there） | 无 |
| 祈使句 | 省略主语 | 省略主语 | 省略主语 |

## 常见错误

1. **英语母语者**：英语中省略主语 → "*Speak English" 而不是 "**I** speak English"

2. **英语母语者**：西班牙语天气表达用 "it" → "*Ello llueve" 而不是 "Llueve"

3. **西班牙语母语者**：西班牙语中不必要地使用显式代词 → "*Yo hablo español" 在语境中性时

4. **西班牙语母语者**：英语中忘记 "it" → "*Is raining" 而不是 "**It** is raining"

5. **汉语母语者**：英语中省略主语 → 英语需要显式主语
