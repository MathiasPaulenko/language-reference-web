## 概述

语法格是一种系统，其中名词或代词的形式根据其句子中的语法功能而变化。

- **西班牙语**：保留了人称代词的完整格系统，区分主格、宾格、与格和介词格。名词没有格标记。
- **英语**：几乎完全丧失了格。仅人称代词保留不同形式（I/me, he/him, she/her, we/us, they/them）。's 属格是名词格的唯一残留。
- **汉语**：完全没有格系统。语法角色完全由语序和类介词助词表示。

---

## 西班牙语

西班牙语代词保持清晰的格区别：

| 功能 | 形式 | 例句 |
|----------|------|---------|
| 主语（主格） | yo, tú, él, ella, nosotros, vosotros, ellos | Yo canto (我唱歌) |
| 直接宾语（宾格） | me, te, lo/la, nos, os, los/las | Me vio (他看见我) |
| 间接宾语（与格） | me, te, le, nos, os, les | Le di el libro (我给他一本书) |
| 介词后 | mí, ti, él, ella | Para mí (为了我) |
| 所有格 | mío, tuyo, suyo, nuestro | Es mío (这是我的) |

**重身代词**：西班牙语经常重复间接宾语代词：*Le di el libro a María* (我把书给了玛丽亚)。

---

## 英语

英语的格标记极少：

- **代词**：I/me, he/him, she/her, we/us, they/them, who/whom
- **名词**：仅's属格（the boy's book）
- **无与格/宾格区别**："I gave him the book" — him 既是与格也是宾格

**Whom 正在消失**：大多数英语母语者现在在所有位置都使用 "who"，即使作为宾语。

---

## 汉语

汉语通过语序和助词表达语法角色：

- **主语**：句子中的第一个位置
- **宾语**：动词之后
- **所有关系**：领有者 + <ruby>的<rt>de</rt></ruby> + 被领有者
- **接受者**：<ruby>给<rt>gěi</rt></ruby> + 接受者（功能上相当于与格）

<div class="chinese-section">

- <ruby>我<rt>wǒ</rt></ruby><ruby>给<rt>gěi</rt></ruby><ruby>了<rt>le</rt></ruby><ruby>他<rt>tā</rt></ruby><ruby>一<rt>yī</rt></ruby><ruby>本<rt>běn</rt></ruby><ruby>书<rt>shū</rt></ruby>。(我给了他一本书。)
- <ruby>这<rt>zhè</rt></ruby><ruby>是<rt>shì</rt></ruby><ruby>我<rt>wǒ</rt></ruby><ruby>的<rt>de</rt></ruby><ruby>书<rt>shū</rt></ruby>。(这是我的书。)
- <ruby>他<rt>tā</rt></ruby><ruby>从<rt>cóng</rt></ruby><ruby>北<rt>běi</rt></ruby><ruby>京<rt>jīng</rt></ruby><ruby>来<rt>lái</rt></ruby>。(他从北京来。)

</div>

汉语完全没有形态格。每个名词和代词无论其语法功能如何，都保持相同的形式。

## 主要区别

| 特征 | 西班牙语 | 英语 | 汉语 |
|---------|---------|---------|---------|
| 代词格 | 广泛 | 极少（6对） | 无 |
| 名词格 | 无 | 仅's属格 | 无 |
| 与格/宾格区别 | 有 | 无（合并） | 不适用 |
| 介词格 | 有（mí vs yo） | 无 | 不适用 |
| 角色表达 | 代词形式 + 语序 | 主要语序 | 语序 + 助词 |
