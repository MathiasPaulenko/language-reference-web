## Overview

Modal verbs express attitudes toward an action — possibility, necessity, permission, ability, and advice.

- **Spanish**: Uses conjugated verbs (*poder, deber, querer*) that inflect for person and tense. The subjunctive is required for many modal expressions.
- **English**: Uses a small, closed set of invariable modals (*can, could, may, might, must, shall, should, will, would*). They never change form.
- **Chinese**: Uses auxiliary verbs (<ruby>会<rt>huì</rt></ruby>, <ruby>能<rt>néng</rt></ruby>, <ruby>可<rt>kě</rt></ruby><ruby>以<rt>yǐ</rt></ruby>, <ruby>要<rt>yào</rt></ruby>, <ruby>应<rt>yīng</rt></ruby><ruby>该<rt>gāi</rt></ruby>) placed before the main verb. They do not conjugate.

## Spanish

Spanish modals are **fully conjugated verbs** that require specific grammar depending on meaning.

### Poder (can, may)

| Tense | Form | Example |
|---|---|---|
| Present | puedo, puedes, puede | *No **puedo** nadar.* (I can't swim.) |
| Past (pret) | pude, pudiste, pudo | *No **pude** venir.* (I couldn't come.) |
| Past (imp) | podía | *No **podía** verlo.* (I couldn't see him. — ongoing) |
| Subjunctive | pueda, puedas, pueda | *Espero que **pueda** venir.* (I hope I can come.) |
| Conditional | podría | *¿**Podrías** ayudarme?* (Could you help me?) |

### Deber (must, should, ought to)

- *Debes estudiar.* (You must study. — strong obligation)
- *Deberías descansar.* (You should rest. — advice)
- *No debes fumar.* (You must not smoke. — prohibition)

**Important**: *Deber de* + infinitive indicates probability, not obligation:

- *Debe de ser tarde.* (It must be late. — deduction)
- *Debe ser honesto.* (He must be honest. — obligation)

### Querer (want, will)

- *Quiero ir.* (I want to go.)
- *No quiero ir.* (I don't want to go.)
- *Querría un café.* (I would like a coffee.)

### Future with ir a

Spanish often uses *ir a* (going to) for near future:

- *Voy a estudiar.* (I'm going to study.)
- *Va a llover.* (It's going to rain.)

### Subjunctive in modal expressions

Many Spanish modals trigger the subjunctive:

- *Es posible que **llueva**.* (It may rain.)
- *Quiero que **vengas**.* (I want you to come.)
- *Es necesario que **estudies**.* (It's necessary that you study.)

## English

English has **nine core modal verbs**: can, could, may, might, must, shall, should, will, would. They are **invariable** — they never change for person or tense.

### Can / could (ability, permission, possibility)

- **Ability**: *I **can** swim.* (present) / *I **could** swim when I was young.* (past)
- **Permission**: **Can** I go? (informal) / **May** I go? (formal)
- **Possibility**: *It **can** get cold at night.* / *It **could** rain tomorrow.*

### Must / have to / should (obligation, advice)

| | Strength | Example |
|---|---|---|
| must | Strong obligation (internal) | *I **must** finish this.* |
| have to | Strong obligation (external) | *I **have to** pay taxes.* |
| should | Advice / recommendation | *You **should** rest.* |
| ought to | Advice (more formal) | *You **ought to** apologize.* |
| need to | Necessity | *You **need to** see a doctor.* |

### Will / would (future, volition, habit)

- **Prediction**: *It **will** rain.*
- **Willingness**: *I **will** help you.*
- **Habitual past**: *He **would** always sing.*
- **Polite request**: **Would** you help me?

### No conjugation

English modals **never** add -s, -ed, or -ing:

- ✅ *He **can** swim.*
- ❌ *He **cans** swim.*
- ❌ *He **canning** swim.*

### Negative modals

Negation is added directly to the modal:

- *I **cannot** swim.* / *I **can't** swim.*
- *You **must not** smoke.* / *You **mustn't** smoke.*
- *He **should not** leave.* / *He **shouldn't** leave.*

## Chinese

Chinese modals are **auxiliary verbs** placed before the main verb. They do not conjugate.

### <ruby>会<rt>huì</rt></ruby> (learned skill / prediction)

- <ruby>我<rt>wǒ</rt></ruby><ruby>会<rt>huì</rt></ruby><ruby>说<rt>shuō</rt></ruby><ruby>中<rt>zhōng</rt></ruby><ruby>文<rt>wén</rt></ruby>。(I can speak Chinese. — learned ability)
- <ruby>明<rt>míng</rt></ruby><ruby>天<rt>tiān</rt></ruby><ruby>会<rt>huì</rt></ruby><ruby>下<rt>xià</rt></ruby><ruby>雨<rt>yǔ</rt></ruby>。(It will rain tomorrow. — prediction)

### <ruby>能<rt>néng</rt></ruby> (physical ability / possibility)

- <ruby>我<rt>wǒ</rt></ruby><ruby>能<rt>néng</rt></ruby><ruby>吃<rt>chī</rt></ruby><ruby>十<rt>shí</rt></ruby><ruby>个<rt>ge</rt></ruby><ruby>包<rt>bāo</rt></ruby><ruby>子<rt>zi</rt></ruby>。(I can eat ten dumplings. — physical capacity)
- <ruby>现<rt>xiàn</rt></ruby><ruby>在<rt>zài</rt></ruby><ruby>能<rt>néng</rt></ruby><ruby>走<rt>zǒu</rt></ruby><ruby>了<rt>le</rt></ruby>。(Now I can walk. — possibility after injury)

**Rule**: <ruby>能<rt>néng</rt></ruby> implies circumstances allow it; <ruby>会<rt>huì</rt></ruby> implies skill or prediction.

### <ruby>可<rt>kě</rt></ruby><ruby>以<rt>yǐ</rt></ruby> (permission / possibility)

- <ruby>我<rt>wǒ</rt></ruby><ruby>可<rt>kě</rt></ruby><ruby>以<rt>yǐ</rt></ruby><ruby>进<rt>jìn</rt></ruby><ruby>来<rt>lái</rt></ruby><ruby>吗<rt>ma</rt></ruby>？(May I come in? — permission)
- <ruby>这<rt>zhè</rt></ruby><ruby>个<rt>ge</rt></ruby><ruby>可<rt>kě</rt></ruby><ruby>以<rt>yǐ</rt></ruby><ruby>吃<rt>chī</rt></ruby>。(This is edible. — possibility)

### <ruby>要<rt>yào</rt></ruby> (want / need / future intention)

- <ruby>我<rt>wǒ</rt></ruby><ruby>要<rt>yào</rt></ruby><ruby>喝<rt>hē</rt></ruby><ruby>水<rt>shuǐ</rt></ruby>。(I want to drink water.)
- <ruby>我<rt>wǒ</rt></ruby><ruby>要<rt>yào</rt></ruby><ruby>去<rt>qù</rt></ruby><ruby>北<rt>běi</rt></ruby><ruby>京<rt>jīng</rt></ruby>。(I'm going to Beijing. — intention)
- <ruby>你<rt>nǐ</rt></ruby><ruby>要<rt>yào</rt></ruby><ruby>小<rt>xiǎo</rt></ruby><ruby>心<rt>xīn</rt></ruby>！(You need to be careful!)

### <ruby>应<rt>yīng</rt></ruby><ruby>该<rt>gāi</rt></ruby> (should / ought to)

- <ruby>你<rt>nǐ</rt></ruby><ruby>应<rt>yīng</rt></ruby><ruby>该<rt>gāi</rt></ruby><ruby>早<rt>zǎo</rt></ruby><ruby>点<rt>diǎn</rt></ruby><ruby>睡<rt>shuì</rt></ruby><ruby>觉<rt>jiào</rt></ruby>。(You should go to bed early.)
- <ruby>应<rt>yīng</rt></ruby><ruby>该<rt>gāi</rt></ruby><ruby>不<rt>bù</rt></ruby><ruby>会<rt>huì</rt></ruby><ruby>下<rt>xià</rt></ruby><ruby>雨<rt>yǔ</rt></ruby>。(It probably won't rain. — deduction)

### <ruby>必<rt>bì</rt></ruby><ruby>须<rt>xū</rt></ruby> (must)

<ruby>必<rt>bì</rt></ruby><ruby>须<rt>xū</rt></ruby> expresses strong obligation:

- <ruby>你<rt>nǐ</rt></ruby><ruby>必<rt>bì</rt></ruby><ruby>须<rt>xū</rt></ruby><ruby>完<rt>wán</rt></ruby><ruby>成<rt>chéng</rt></ruby><ruby>作<rt>zuò</rt></ruby><ruby>业<rt>yè</rt></ruby>。(You must finish your homework.)

### <ruby>可<rt>kě</rt></ruby><ruby>能<rt>néng</rt></ruby> (possible / may)

<ruby>可<rt>kě</rt></ruby><ruby>能<rt>néng</rt></ruby> expresses possibility, placed before the verb:

- <ruby>他<rt>tā</rt></ruby><ruby>可<rt>kě</rt></ruby><ruby>能<rt>néng</rt></ruby><ruby>不<rt>bù</rt></ruby><ruby>来<rt>lái</rt></ruby>。(He might not come.)

## Common mistakes

1. **Spanish speakers learning English**: Conjugating modals: *He cans swim* → *He can swim*

2. **English speakers learning Spanish**: Forgetting subjunctive: *Es posible que llueve* → *Es posible que llueva*

3. **English speakers learning Chinese**: Using <ruby>会<rt>huì</rt></ruby> for all ability: *<ruby>我<rt>wǒ</rt></ruby><ruby>会<rt>huì</rt></ruby><ruby>吃<rt>chī</rt></ruby><ruby>十<rt>shí</rt></ruby><ruby>个<rt>gè</rt></ruby><ruby>包<rt>bāo</rt></ruby><ruby>子<rt>zǐ</rt></ruby>* → *<ruby>我<rt>wǒ</rt></ruby><ruby>能<rt>néng</rt></ruby><ruby>吃<rt>chī</rt></ruby><ruby>十<rt>shí</rt></ruby><ruby>个<rt>gè</rt></ruby><ruby>包<rt>bāo</rt></ruby><ruby>子<rt>zǐ</rt></ruby>* (physical capacity needs <ruby>能<rt>néng</rt></ruby>)

4. **Chinese speakers learning English**: Using 'will' for everything: *I will can* → *I will be able to*
