## 概述

句子按交际目的分类。

- **西班牙语**：疑问句用主谓倒装，开头用倒置符号（¿ ¡），否定命令用虚拟式。
- **英语**：疑问句用 do-support，助动词倒装，附加问句用代词。
- **汉语**：所有句类保持 SVO 语序。用助词（吗、呢、吧）和语调标记疑问、命令和感叹。

## 西班牙语

### 陈述句

| 类型 | 例句 | 特征 |
|---|---|---|
| 肯定 | Juan **lee** un libro. | 主语 + 动词 + 宾语 |
| 否定 | Juan **no lee** un libro. | No + 动词 |
| 被动 | El libro **fue leído** por Juan. | Ser + 过去分词 |

### 疑问句

| 类型 | 例句 | 特征 |
|---|---|---|
| 是非问 | **¿**Juan lee**?** | 倒装 + ¿? |
| 特指问 | **¿Qué** lee Juan**?** | Wh- 前置 + 倒装 |
| 选择问 | **¿**Lees o escribes**?** | ¿A o B? |
| 附加问 | Juan lee, **¿verdad?** | 陈述句 + 附加语 |

### 命令句

| 类型 | 例句 | 特征 |
|---|---|---|
| 肯定 | **¡Lee!** | 命令式动词 |
| 否定 | **¡No leas!** | No + 虚拟式 |
| 反身 | **¡Levántate!** | 命令式 + 代词 |

### 感叹句

| 类型 | 例句 | 特征 |
|---|---|---|
| Qué | **¡Qué** libro**!** | ¡Qué + 名词！ |
| Cómo | **¡Cómo** corre**!** | ¡Cómo + 动词！ |
| Cuánto | **¡Cuánto** trabajo**!** | ¡Cuánto + 名词！ |

## 英语

### 陈述句

| 类型 | 例句 | 特征 |
|---|---|---|
| 肯定 | John **reads** a book. | 主语 + 动词 + 宾语 |
| 否定 | John **does not read**. | Do/does + not + 动词 |
| 被动 | The book **is read** by John. | Be + 过去分词 |

### 疑问句

| 类型 | 例句 | 特征 |
|---|---|---|
| 是非问 | **Does** John read? | Do/does + 主语 + 动词 |
| 特指问 | **What** is John reading? | Wh- + 助动词 + 主语 |
| 选择问 | Do you read **or** write? | A or B |
| 附加问 | John reads, **doesn't he?** | 助动词 + 代词 |

### 命令句

| 类型 | 例句 | 特征 |
|---|---|---|
| 肯定 | **Read!** | 裸动词 |
| 否定 | **Don't read!** | Do not + 动词 |
| 加副词 | **Please** read. | Please + 动词 |

### 感叹句

| 类型 | 例句 | 特征 |
|---|---|---|
| What | **What** an interesting book! | What a/an + 形容词 + 名词 |
| How | **How** fast! | How + 形容词/副词 |
| So | **So** interesting! | So + 形容词 |

| 类型 | 西班牙语 | 英语 | 汉语 |

## 常见错误

1. **西班牙语母语者**：英语中忘记 do-support → **Does** John read?（不说 John reads?）

2. **英语母语者**：汉语中倒装 → 汉语不倒装；用助词

3. **汉语母语者**：西班牙语中省略倒置符号 → **¿**Cómo estás**?** / **¡**Qué bien**!**

4. **英语母语者**：西班牙语否定命令用直陈式 → **¡No leas!**（虚拟式）
