## 概述

名词命名人、地点、事物或概念。不同语言分类和修饰名词的方式各不相同。

- **西班牙语**：所有名词有性（阳/阴）和数（单/复）。冠词一致。领属用 de。
- **英语**：无语法性。可数/不可数区分严格。领属用 -'s。冠词不一致。
- **汉语**：无性无数。计数时量词必需。领属用 的。无冠词。

## 西班牙语

### 性

所有西班牙语名词都是**阳性或阴性**：

| 一般规则 | 阳性 | 阴性 |
|---|---|---|
| 词尾 | -o, -e, 辅音 | -a, -d, -z, -ión |
| 冠词 | el / un / los / unos | la / una / las / unas |

**例子**：

- el perro / la perra（公狗/母狗）
- el gato / la gata（公猫/母猫）
- el libro / la mesa（书/桌子）
- el problema / la mano（问题/手——例外）

**注意**：性是语法性的，非自然性的：

- la mesa（阴性——桌子）
- el mapa（阳性——地图）
- la mano（阴性——手）
- el día（阳性——天）

### 数

| 单数 | 复数 |
|---|---|
| el perro | los perros |
| la mesa | las mesas |
| el mes | los meses |
| la ciudad | las ciudades |
| el lápiz | los lápices |
| la mujer | las mujeres |
| el inglés | los ingleses |
| la ley | las leyes |

### 冠词

| | 阳性单数 | 阴性单数 | 阳性复数 | 阴性复数 |
|---|---|---|---|---|
| 定冠词 | el | la | los | las |
| 不定冠词 | un | una | unos | unas |

### 领属

西班牙语用 de 表示领属：

- el libro de María（玛丽亚的书）
- la casa de mi padre（我父亲的房子）
- un amigo de mi hermano（我兄弟的一个朋友）

### 可数与不可数

西班牙语区分不如英语严格：

- una información（可能，虽然 información 无冠词更常见）
- un consejo（一个建议——可数）
- dinero（钱——不可数，但口语中 un dinero 可能）

### 专有名词

- 仅专有名词大写：España, Madrid, Juan
- 人名不加冠词：María（非 la María）
- 地名有时加冠词：el Perú, la India, los Estados Unidos

## 英语

### 无语法性

英语**没有**名词的语法性：

- the dog（公狗/母狗）
- the table（桌子）
- the book（书）

只有人称代词有自然性：he, she, it。

### 复数

| 规则 | 单数 | 复数 |
|---|---|---|
| 加 -s | cat | cats |
| s, x, z, ch, sh 后加 -es | box | boxes |
| -y 变 -ies | city | cities |
| 不规则 | man | men |
| 不规则 | child | children |
| 不规则 | foot | feet |
| 不规则 | tooth | teeth |
| 不变 | sheep | sheep |

### 冠词

| 类型 | 用法 | 例子 |
|---|---|---|
| the | 定冠词 | the dog（这只狗） |
| a / an | 不定冠词（可数单数） | a dog（一只狗） |
| some | 不定冠词（复数/不可数） | some dogs, some water |
| （无） | 泛指 | Dogs bark.（狗会叫。） |

### 可数与不可数

| 可数 | 不可数 |
|---|---|
| a dog, two dogs | water, information, advice |
| a book, some books | rice, furniture, luggage |
| a chair, many chairs | money, bread, weather |

**关键规则**：

- 不可数名词**不用** a/an 也不加复数 -s
- ❌ an information → ✅ some information / a piece of information
- ❌ two waters → ✅ two glasses of water

### 领属

- 's（单数和复数不规则）：Maria's book, children's toys
- s'（规则复数）：the dogs' owner
- of（正式/无生命）：the roof of the house

### 专有名词

- 专有名词大写：Spain, London, John
- I（我）大写：I am here.
- 专有名词不加冠词：Spain（非 the Spain），Maria（非 the Maria）
- 例外：the United States, the Philippines, the Netherlands

### 集合名词

- family（家庭——单数）
- team（团队——单数）
- police（警察——复数）
- cattle（牛——复数）

## 汉语

### 无性无数

汉语**没有**语法性，也没有数：

- 狗 = 狗/母狗/狗们/狗们
- 桌子 = 桌子/桌子们
- 书 = 书/书们

### 无冠词

汉语没有定冠词或不定冠词：

- 我看书。（我看书/我读书。）
- 狗在跑。（狗在跑。）

**注意**：语境和指示词（这 这个，那 那个）常起定冠词作用。

### 量词必需

汉语计数名词时需要**量词**：

| 名词 | 量词 | 例子 |
|---|---|---|
| 人（人） | 个（通用） | 一个人（一个人） |
| 猫（猫） | 只（动物） | 三只猫（三只猫） |
| 书（书） | 本（册） | 两本书（两本书） |
| 笔（笔） | 支（细长） | 一支笔（一支笔） |
| 车（车） | 辆（车辆） | 五辆车（五辆车） |

**规则**：数词 + 量词 + 名词

### 领属

汉语用 的 表示领属：

- 玛丽亚**的**书（玛丽亚的书）
- 我**的**爸爸（我的父亲）
- 学校**的**老师（学校的老师）

**注意**：的 在亲近亲属代词中可省略：

- 我妈（我妈——比 我的妈妈 更常见）

### 可数与不可数

汉语不正式标记区分，但不可数名词常需要量词：

- 三杯水（三杯水）
- 一碗米（一碗米）
- 一些建议（一些建议）

### 专有名词

- 仅专有名词大写：北京（北京），中国（中国）
- 无形式变化：李先生（李先生）
- 姓在前：王小明（王小明——姓王）

## 一览对比

| 特征 | 西班牙语 | 英语 | 汉语 |
|---------|---------|---------|---------|
| 语法性 | 是（阳/阴） | 否 | 否 |
| 复数标记 | 是（-s / -es） | 是（-s / -es / -ies） | 否（语境） |
| 定冠词 | el/la/los/las | the | 无 |
| 不定冠词 | un/una/unos/unas | a/an | 无（一/语境） |
| 可数/不可数 | 较松 | 严格 | 较松 |
| 量词 | 无 | 无 | **必需** |
| 领属 | de + 名词 | -'s / of | 的 |
| 大写（专有） | 是 | 是（+ I） | 是 |

## 语境中的例句

### 狗
- **ES**: el perro（阳性，定冠词）
- **EN**: the dog（无性，定冠词）
- **ZH**: 狗（无性，无冠词）

### 三只猫
- **ES**: tres gatos（复数 -s）
- **EN**: three cats（复数 -s）
- **ZH**: 三只猫（量词必需，无复数）

### 玛丽亚的书
- **ES**: el libro de María
- **EN**: Maria's book
- **ZH**: 玛丽亚的书

### 一个建议
- **ES**: un consejo（可数）
- **EN**: some advice（不可数——无 an advice）
- **ZH**: 一个建议（可数，带量词）

## 常见错误

1. **英语母语者学西班牙语**：期望性与生物性别一致：el mesa → la mesa

2. **西班牙语母语者学英语**：不可数名词用冠词：an information → some information

3. **西班牙语/英语母语者学汉语**：省略量词：三猫 → 三只猫

4. **汉语母语者学英语**：不可数名词加复数：informations → information
