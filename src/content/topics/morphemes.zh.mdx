## 概述

词素是词语中最小的意义单位。不同语言组合词素的方式差异巨大。

- **西班牙语**：屈折形态（性、数、人称、时态、式）和派生形态。拉丁/希腊语源前缀后缀能产。词素常为黏着的。
- **英语**：屈折形态极少（复数 -s，过去时 -ed，进行时 -ing）但派生广泛（拉丁/希腊/日耳曼前缀后缀）。复合词常见。
- **汉语**：无屈折。所有词语都是自由词素。复合是主要策略。功能助词（的、了、着、过）是独立词素，非黏着词缀。

## 西班牙语

### 屈折形态

西班牙语对动词和名词/形容词进行**屈折变化**：

| 范畴 | 例子 | 词素 |
|---|---|---|
| 复数 | gato → gatos | -s |
| 性 | gato / gata | -o / -a |
| 人称（动词） | hablo, hablas, habla | -o, -as, -a |
| 时态（动词） | hablé, hablaba, hablaré | -é, -aba, -aré |
| 式（动词） | hable（虚拟式） | -e |
| 过去分词 | hablado | -ado |
| 副动词 | hablando | -ando |

### 前缀派生

| 前缀 | 含义 | 例子 |
|---|---|---|
| in-/im- | 否定 | imposible, incompleto |
| des- | 否定/反转 | descontento, deshacer |
| re- | 再次 | rehacer, reescribir |
| anti- | 反对 | antiguo, antinatural |
| super- | 超过 | supermercado, superhombre |
| sub- | 下面 | subterráneo, submarino |
| pre- | 之前 | prehistoria, prever |
| post- | 之后 | postguerra, posmoderno |
| auto- | 自己 | automóvil, autobiografía |

### 后缀派生

| 后缀 | 含义 | 例子 |
|---|---|---|
| -oso | 性质 | peligroso, hermoso |
| -ico/-ica | 相关 | histórico, político |
| -al | 相关 | musical, cultural |
| -mente | 副词 | rápidamente, felizmente |
| -ción/-sión | 动作名词 | educación, decisión |
| -dor/-dora | 施事者 | trabajador, cantadora |
| -ble | 能力 | comible, legible |
| -ito/-ita | 指小 | gatito, mesita |
| -ote/-ota | 指大 | casona, mujerota |
| -ismo/-ista | 主义/支持者 | socialismo, pianista |

### 复合构词

- limpiaparabrisas（擦+挡风玻璃）
- sacacorchos（拔+软木塞）
- abrelatas（开+罐头）
- paraguas（挡+雨）
- rascacielos（刮+天）
- ferrocarril（铁+轨道——来自拉丁语）
- parabrisas（挡+风）

### 自由词素与黏着词素

| 自由词素 | 黏着词素 |
|---|---|
| gato, casa, alto | -oso, -ción, in-, des- |
| caminar, beber, leer | -ar, -er, -ir（单独无意义） |

西班牙语两者都有，但派生主要使用黏着词缀。

## 英语

### 最小屈折形态

英语只有约8个屈折词素：

| 范畴 | 例子 | 词素 |
|---|---|---|
| 复数 | cat → cats | -s / -es / -ies |
| 所有格 | cat → cat's | -'s |
| 第三人称单数 | walk → walks | -s |
| 过去时 | walk → walked | -ed |
| 进行时 | walk → walking | -ing |
| 过去分词 | walk → walked | -ed |
| 比较级 | tall → taller | -er |
| 最高级 | tall → tallest | -est |

### 前缀派生

| 前缀 | 含义 | 例子 |
|---|---|---|
| un- | 否定（日耳曼） | unhappy, unkind |
| in-/im-/ir-/il- | 否定（拉丁） | impossible, illegal, irregular |
| dis- | 否定/反转 | dislike, disconnect |
| re- | 再次 | rewrite, rebuild |
| anti- | 反对 | antiviral, antifreeze |
| super- | 超过 | superhuman, supermarket |
| sub- | 下面 | submarine, subzero |
| pre- | 之前 | prehistoric, preview |
| post- | 之后 | postwar, postgraduate |
| auto- | 自己 | automobile, automatic |
| non- | 非 | nonstop, nonviolent |
| over- / under- | 过度/不足 | overeat, undercooked |
| mis- | 错误 | misunderstand, misplace |
| co- | 共同 | cooperate, coauthor |
| ex- | 前任 | ex-president, ex-wife |
| self- | 自己 | self-aware, self-control |

### 后缀派生

| 后缀 | 含义 | 例子 |
|---|---|---|
| -ful | 充满 | beautiful, helpful |
| -less | 无 | homeless, fearless |
| -ness | 状态/性质 | happiness, kindness |
| -ment | 结果/动作 | government, movement |
| -tion/-sion | 动作名词 | education, decision |
| -er/-or | 施事者 | worker, actor |
| -able/-ible | 能力 | readable, legible |
| -ous | 性质 | dangerous, nervous |
| -ive | 倾向 | active, creative |
| -ly | 副词 | quickly, happily |
| -en | 使成为 | darken, strengthen |
| -ize/-ise | 使 | modernize, apologize |
| -hood | 集体状态 | childhood, neighborhood |
| -ship | 状态/关系 | friendship, leadership |
| -let | 指小 | piglet, booklet |
| -y / -ie | 亲昵指小 | doggy, daddy |

### 复合构词

- blackboard（black+board）
- toothbrush（tooth+brush）
- notebook（note+book）
- sunrise（sun+rise）
- handwriting（hand+writing）
- bedroom（bed+room）
- dishwasher（dish+washer）
- snowman（snow+man）
- raincoat（rain+coat）
- overlook（over+look——也是复合动词）

### 英语双式词

同一概念，不同来源：

| 本土/日耳曼 | 拉丁/法语 |
|---|---|
| kingly | royal |
| time | temporal |
| holy | sacred |
| ask | inquire |
| freedom | liberty |

### 形态分析

- **un-happi-ness**：un-（否定）+ happy（词根）+ -ness（名词化）
- **re-write-able**：re-（再次）+ write（词根）+ -able（能力）
- **co-oper-ate**：co-（共同）+ oper（工作——来自拉丁语）+ -ate（动词）

## 汉语

### 无屈折

汉语**不**变化名词、动词或形容词：

- 猫 = 猫/猫们（无变化）
- 走 = 走/走了/走过（无变化）
- 大 = 大（无性、无数）

### 复合构词（主要策略）

| 复合词 | 成分 | 含义 |
|---|---|---|
| 电脑 | 电+脑 | 电脑 |
| 电话 | 电+话 | 电话 |
| 火车 | 火+车 | 火车 |
| 飞机 | 飞+机 | 飞机 |
| 洗手间 | 洗+手+间 | 洗手间 |
| 超市 | 超+市 | 超市 |
| 地铁 | 地+铁 | 地铁 |
| 公共汽车 | 公+共+汽+车 | 公共汽车 |
| 北京 | 北+京 | 北京 |

### 自由词素

每个汉字都是具有意义的自由词素：

- 人 = 人
- 大 = 大
- 水 = 水
- 走 = 走
- 来 = 来
- 上 = 上
- 下 = 下

### 功能助词（非词缀）

| 助词 | 用途 | 例子 |
|---|---|---|
| 的 | 形容词/领属 | 美丽**的**花 |
| 了 | 状态变化/完成 | 吃**了**（吃了/吃完了） |
| 着 | 持续进行 | 走**着**（正在走） |
| 过 | 过去经历 | 去**过**（去过） |
| 地 | 副词 | 快**地**走（快走） |
| 得 | 程度补语 | 走**得**很快（走得很快） |

**重要**：这些助词是**独立词**，非黏着词缀。不与词根融合。

### 重叠

汉语使用重叠表达多种效果：

- 看看（看一看——委婉）
- 人人（每个人——分配）
- 妈妈（妈妈——亲昵）
- 爸爸（爸爸——亲昵）

### 词缀（非常有限）

- 子：桌子，椅子——轻微名词化
- 儿：花儿（小花——北方方言）
- 们：我们，朋友们——人称复数

### 比较：复数词素

- 3 cats（英语：后缀）
- 3 gatos（西班牙语：后缀）
- 三只猫（汉语：猫不变——数词+量词表达数量）

## 一览对比

| 策略 | 西班牙语 | 英语 | 汉语 |
|----------|---------|---------|---------|
| 屈折 | 广泛（动词、性、数） | 极少（复数、过去、-ing） | 无 |
| 派生（前缀） | 拉丁/希腊语源能产 | 日耳曼/拉丁/希腊语源 | 无（自由词素） |
| 派生（后缀） | 广泛（-oso, -ción, -mente） | 广泛（-ness, -ful, -ly, -tion） | 非常有限（-子, -儿, -们） |
| 复合 | 常见（limpiaparabrisas） | 非常常见（blackboard） | **主要策略**（电脑） |
| 自由词素 | 一些 | 一些 | **所有** |
| 功能助词 | 黏着附着词 | 分离助动词 | **独立助词**（的, 了, 着, 过） |

## 语境中的例子

### 小猫
- **ES**: gatito（-ito，指小）
- **EN**: kitty / kitten（-y，本土词）
- **ZH**: 小猫（小+猫——复合）

### 不可能
- **ES**: imposible（in- + posible）
- **EN**: impossible（im- + possible）
- **ZH**: 不可能（不+可+能——复合）

### 快乐地
- **ES**: felizmente（feliz + -mente）
- **EN**: happily（happy + -ly）
- **ZH**: 开心地（开心+副词助词地——独立助词）

## 常见错误

1. **西班牙语/英语母语者学汉语**：期待动词屈折变化：他走了过去 → 他走过去**了**（了 跟在动词后，不融合）

2. **汉语母语者学英语**：忘记复数：three book → three book**s**

3. **汉语母语者学西班牙语**：忘记性一致：un persona → una persona

4. **英语母语者学西班牙语**：到处加 -s：la problema → el problema（不规则阳性）
