## Resumen

La sintaxis estudia cómo las palabras se combinan en unidades mayores.

- **Español**: Orden SVO básico pero con orden de palabras relativamente libre gracias a la rica flexión. La estructura constituyente paralela al inglés. Los adjetivos usualmente postnominales.
- **Inglés**: Orden SVO rígido. La estructura constituyente está bien definida: SN, SV, SP, SAdj, SAdv. Las reglas de movimiento dependen de estructura jerárquica.
- **Chino**: Orden SVO básico pero con extensa topicalización. Los constituyentes se marcan por partículas (的, 地, 得) en lugar de flexión. Las relativas prenominales.

## Español

### Constituyentes principales

| Constituyente | Ejemplo | Abreviatura |
|---|---|---|
| Sustantivo | el libro | SN |
| Verbo | lee el libro | SV |
| Preposicional | en la mesa | SP |
| Adjetivo | muy alto | SAdj |
| Adverbial | muy rápidamente | SAdv |
| Complementizador | que vino | CP |
| Infinitivo | leer el libro | SInf |

### Orden flexible

| Orden | Ejemplo | Contexto |
|---|---|---|
| SVO (neutral) | El estudiante lee un libro. | Orden básico |
| VSO (interrogativo) | ¿Lee el estudiante un libro? | Pregunta |
| OVS (marcado) | Un libro lo lee el estudiante. | Énfasis en objeto |
| VS (inicial) | Llegó Juan. | Énfasis en verbo |

### Posición del adjetivo

| Posición | Significado | Ejemplo |
|---|---|---|
| Prenominal | Subjetivo/emocional | **viejo** amigo (querido) |
| Postnominal | Objetivo/descriptivo | amigo **viejo** (de edad avanzada) |

| Constituyente | Ejemplo | Abreviatura |
|---|---|---|

|---|---|---|

- **EN**: The student reads a book.

## Errores comunes

1. **Todos**: Asumir que el chino = inglés → El chino es tema-comentario

2. **Angloparlantes**: Adjetivos prenominales en español → estudiante **alto**（NO alto estudiante）

3. **Chinos**: Auxiliares ingleses como cláusulas separadas → [VP has been reading]

4. **Todos**: Pensar que sin morfología = sin estructura → El chino usa partículas
