## Resumen

Las oraciones de relativo modifican sustantivos añadiendo información extra.

- **Español**: Usa *que* como pronombre relativo por defecto. *Quien* para personas después de preposiciones. *El cual* para desambiguación formal. *Cuyo* para posesión (concuerda con el sustantivo poseído). El subjuntivo se usa cuando el antecedente es incierto.
- **Inglés**: Usa *who* para personas, *which* para cosas, *that* para cualquiera. *Whose* para posesión. *Whom* para objeto de preposición (formal). La distinción restrictivo vs no restrictivo es estricta.
- **Chino**: No tiene pronombres relativos. La cláusula de relativo simplemente precede al sustantivo, enlazada por *<ruby>的<rt>de</rt></ruby>*. El sustantivo cabeza puede omitirse si el contexto es claro.

## Español

### Pronombres relativos

| Pronombre | Uso | Ejemplo |
|---|---|---|
| que | General (sujeto/objeto) | El hombre **que** vino |
| quien | Personas después de prep. | La mujer con **quien** hablé |
| el cual | Formal / desambiguación | El libro **el cual** leí |
| cuyo | Posesión (concuerda) | El hombre **cuyo** hijo estudia |
| donde | Lugar | La casa **donde** vivo |
| cuando | Tiempo | El día **cuando** llegué |

### Concordancia de cuyo

| Poseído | Forma |
|---|---|
| Masculino singular | cuyo |
| Femenino singular | cuya |
| Masculino plural | cuyos |
| Femenino plural | cuyas |

**Ejemplo**: El hombre **cuyos** hijos estudian.

### Subjuntivo en relativos

Se usa el subjuntivo cuando el antecedente es incierto o futuro:

- Busco un médico **que hable** inglés. (subjuntivo — incierto)
- Conozco un médico **que habla** inglés. (indicativo — cierto)

### El que / el cual

| Pronombre | Uso | Ejemplo |
|---|---|---|
| el que | Reemplaza sustantivo | El **que** vino fue Juan. |
| el cual | Formal, tras preposición larga | El estudiante, **el cual** aprobó... |

## Inglés

### Pronombres relativos

| Pronombre | Uso | Ejemplo |
|---|---|---|
| who | Personas (sujeto) | The man **who** came |
| whom | Personas (objeto/formal) | The man **whom** I met |
| which | Cosas | The book **which** I read |
| that | Personas o cosas | The man/book **that** I saw |
| whose | Posesión | The man **whose** car was stolen |
| where | Lugar | The house **where** I live |
| when | Tiempo | The day **when** I arrived |

### Restrictivo vs no restrictivo

| Tipo | Pronombre | Comas | Ejemplo |
|---|---|---|---|
| Restrictivo | that / which / who | No | The book **that** is red |
| No restrictivo | which / who | Sí | My car**, which is red,** is fast |

**Nota**: *That* solo se usa en cláusulas restrictivas (sin comas).

### Preposición + relativo

| Formal | Informal |
|---|---|
| The man **to whom** I spoke | The man **(whom)** I spoke **to** |
| The book **in which** I read | The book **(which)** I read **in** |

**Nota**: El inglés formal coloca la preposición antes del relativo.

### Omisión del relativo

El pronombre relativo de objeto puede omitirse:

- The book (that) I read.
- The man (whom) I met.

## Chino

### 的 como enlazador relativo

El chino no tiene pronombres relativos. Usa *<ruby>的<rt>de</rt></ruby>* después de la cláusula:

| Tipo | Estructura | Ejemplo |
|---|---|---|
| Sujeto | V + 的 + N | <ruby>来<rt>lái</rt></ruby>**<ruby>的<rt>de</rt></ruby>**<ruby>人<rt>rén</rt></ruby> (el que vino) |
| Objeto | S + V + 的 + N | <ruby>我<rt>wǒ</rt></ruby><ruby>读<rt>dú</rt></ruby>**<ruby>的<rt>de</rt></ruby>**<ruby>书<rt>shū</rt></ruby> (el que leí) |
| Posesión | Poseedor + 的 + N | <ruby>我<rt>wǒ</rt></ruby><ruby>的<rt>de</rt></ruby><ruby>书<rt>shū</rt></ruby> (mi libro) |

### Omisión del sustantivo cabeza

Si el contexto es claro, el sustantivo puede omitirse:

| Completo | Omitido |
|---|---|
| <ruby>来<rt>lái</rt></ruby><ruby>的<rt>de</rt></ruby><ruby>人<rt>rén</rt></ruby> | <ruby>来<rt>lái</rt></ruby><ruby>的<rt>de</rt></ruby> |
| <ruby>红<rt>hóng</rt></ruby><ruby>的<rt>de</rt></ruby><ruby>车<rt>chē</rt></ruby> | <ruby>红<rt>hóng</rt></ruby><ruby>的<rt>de</rt></ruby> |

### Preposiciones en relativos chinos

La preposición se mantiene con el verbo:

- <ruby>我<rt>wǒ</rt></ruby><ruby>跟<rt>gēn</rt></ruby><ruby>他<rt>tā</rt></ruby><ruby>说<rt>shuō</rt></ruby><ruby>话<rt>huà</rt></ruby><ruby>的<rt>de</rt></ruby><ruby>人<rt>rén</rt></ruby> (La persona con quien hablé)
- <ruby>我<rt>wǒ</rt></ruby><ruby>从<rt>cóng</rt></ruby><ruby>北<rt>běi</rt></ruby><ruby>京<rt>jīng</rt></ruby><ruby>来<rt>lái</rt></ruby><ruby>的<rt>de</rt></ruby><ruby>时<rt>shí</rt></ruby><ruby>候<rt>hou</rt></ruby> (Cuando vine de Beijing)

### Ejemplos

- <ruby>我<rt>wǒ</rt></ruby><ruby>认<rt>rèn</rt></ruby><ruby>识<rt>shi</rt></ruby><ruby>的<rt>de</rt></ruby><ruby>人<rt>rén</rt></ruby> (La persona que conozco)
- <ruby>你<rt>nǐ</rt></ruby><ruby>买<rt>mǎi</rt></ruby><ruby>的<rt>de</rt></ruby><ruby>东<rt>dōng</rt></ruby><ruby>西<rt>xī</rt></ruby> (Las cosas que compraste)
- <ruby>我<rt>wǒ</rt></ruby><ruby>们<rt>men</rt></ruby><ruby>去<rt>qù</rt></ruby><ruby>的<rt>de</rt></ruby><ruby>地<rt>dì</rt></ruby><ruby>方<rt>fāng</rt></ruby> (El lugar donde fuimos)

## Comparación rápida

| Característica | Español | Inglés | Chino |
|---|---|---|---|
| Pronombres relativos | que, quien, el cual, cuyo | who, which, that, whose | Ninguno |
| Enlace | Pronombre + verbo | Pronombre + verbo | Cláusula + 的 + sustantivo |
| Sujeto relativizado | Conservado | Conservado | Omitido |
| Objeto relativizado | Omitido | Omitido | Omitido |
| Posesivo | cuyo (concuerda) | whose (invariable) | 的 después del poseedor |
| Preposición | Antes del relativo (formal) | Antes del relativo (formal) | Con el verbo |
| Restrictivo/no restrictivo | Comas para no restrictivo | that vs which | Sin distinción formal |

## Ejemplos en contexto

### Relativo de sujeto
- **ES**: El hombre **que** vino.
- **EN**: The man **who** came.
- **ZH**: <ruby>来<rt>lái</rt></ruby><ruby>的<rt>de</rt></ruby><ruby>人<rt>rén</rt></ruby>。

### Relativo de objeto
- **ES**: El libro **que** leí.
- **EN**: The book **(that)** I read.
- **ZH**: <ruby>我<rt>wǒ</rt></ruby><ruby>读<rt>dú</rt></ruby><ruby>的<rt>de</rt></ruby><ruby>书<rt>shū</rt></ruby>。

### Posesivo
- **ES**: El hombre **cuyo** hijo estudia.
- **EN**: The man **whose** son studies.
- **ZH**: <ruby>儿<rt>ér</rt></ruby><ruby>子<rt>zi</rt></ruby><ruby>学<rt>xué</rt></ruby><ruby>习<rt>xí</rt></ruby><ruby>的<rt>de</rt></ruby><ruby>男<rt>nán</rt></ruby><ruby>人<rt>rén</rt></ruby>。

## Errores comunes

1. **Hispanohablantes**: Olvidar quien después de preposición → La mujer con **quien** hablé

2. **Angloparlantes**: which en cláusula restrictiva → The book **that** I read (NO which)

3. **Angloparlantes**: Añadir pronombre relativo en chino → 汉语无关系代词，用 的

4. **Chinos**: Confundir that/which → that = restrictivo; which = no restrictivo
