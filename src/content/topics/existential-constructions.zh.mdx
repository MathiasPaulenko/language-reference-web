## 概述

存在句是断言某物存在的句子。

- **西班牙语**：*Haber* 无人称（*hay, había, hubo, habrá*）用于存在。*Hay que* 用于义务。*Hubo* 用于突发事件。
- **英语**：*There is/are* 用于存在。动词与真实名词一致。存在与拥有之间没有区别。
- **中文**：*<ruby>有<rt>yǒu</rt></ruby>* 用于存在和拥有。*<ruby>发<rt>fā</rt></ruby><ruby>生<rt>shēng</rt></ruby>* 用于事件。无动词一致或冠词。

## 西班牙语

### 无人称haber

**现在时：hay**
- Hay un problema.（有一个问题。）
- Hay muchos libros.（有很多书。）
- No hay nadie.（没有人。）
- ¿Hay alguna pregunta?（有问题吗？）

**过去未完成时：había**
- Había mucha gente.（有很多人。）
- No había suficiente comida.（没有足够的食物。）

**简单过去时：hubo**
- Hubo un accidente.（发生了一起事故。）
- Hubo problemas ayer.（昨天出了问题。）

**将来时：habrá**
- Habrá tiempo.（会有时间的。）
- Habrá muchos invitados.（会有很多客人。）

**虚拟式：haya**
- Espero que haya suficiente.（希望有足够的。）
- No creo que haya problema.（我不认为有问题。）

### Hay que + 动词原形

- Hay que estudiar.（必须学习。—— 一般义务）
- Hay que ser cuidadoso.（必须小心。）
- No hay que preocuparse.（不必担心。）

### 存在 + 位置

- En la mesa hay un libro.（桌子上有一本书。）
- Hay un libro en la mesa.（有一本书在桌子上。）
- En Madrid hay muchos museos.（马德里有很多博物馆。）

### 否定

- No hay dinero.（没有钱。）
- Ya no hay tiempo.（已经没有时间了。）
- No hubo problemas.（没有问题。）

### 存在拥有（tener）

- Tengo un libro.（我有一本书。）
- ¿Tienes hermanos?（你有兄弟姐妹吗？）
- No tiene sentido.（没有意义。）

## 英语

### There is/are

**单数**：
- There is a problem.（有一个问题。）
- There is no time.（没有时间。）
- Is there a doctor?（有医生吗？）

**复数**：
- There are many books.（有很多书。）
- There are no solutions.（没有解决方案。）
- Are there any questions?（有问题吗？）

**过去时**：
- There was a fire.（发生了一场火灾。）
- There were people waiting.（有人在等。）

**将来时**：
- There will be time.（会有时间的。）
- There will be problems.（会有问题的。）

**情态动词**：
- There must be a reason.（一定有原因。）
- There might be trouble.（可能有麻烦。）
- There should be more.（应该有更多。）

### 缩略形式

| 完整形式 | 缩略形式 |
|---|---|
| There is | There's |
| There is not | There isn't / There's not |
| There are not | There aren't |
| There was not | There wasn't |
| There were not | There weren't |

### There is/are + 形容词

- There is nothing wrong.（没有什么问题。）
- There are several options.（有几个选项。）
- There is little hope.（几乎没有希望。）

### 存在 + 位置

- There is a book on the table.（桌子上有一本书。）
- On the table there is a book.（桌子上有一本书。—— 更正式/文学化）
- There are many museums in Madrid.（马德里有很多博物馆。）

### 否定

- There isn't any money.（没有钱。）
- There is no justice.（没有正义。）
- There weren't any problems.（没有问题。）

### 存在 + 拥有（have）

- I have a book.（我有一本书。）
- Do you have siblings?（你有兄弟姐妹吗？）
- It doesn't make sense.（这没有意义。）

## 常见错误

1. **学英语的西班牙语使用者**：*Es un problema* → *Hay un problema*（使用ser表示存在）

2. **学英语的西班牙语使用者**：*Is there any students?* → *Are there any students?*（动词一致）

3. **学英语的中文使用者**：*There is many books* → *There are many books*（动词一致）

4. **学中文的英语使用者**：*一本书* → *有*一本书（省略有表示存在）

5. **学中文的西班牙语使用者**：混淆 没有（没有）与 不（不）
